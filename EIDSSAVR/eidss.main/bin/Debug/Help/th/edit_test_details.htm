ดูหรือแก้ไขรายละเอียดการทดสอบ

รายละเอียดการทดสอบให้ข้อมูลเกี่ยวกับการทดสอบตัวอย่างที่อยู่ในระบบ บางรายการแสดงในตารางบนแบบฟอร์ม หมวดห้องปฏิบัติการ สามารถแก้ไขรายละเอียดได้ เช่น การวินิจฉัยการทดสอบ เเละอื่นๆที่เรียงเพิ่มที่ด้านขวาถัดไป ข้อมูลบางอย่างอาจอยู่ในแท็บ การทดสอบ

สำหรับรายละเอียดเพิ่มเติมเกี่ยวกับผลลัพธ์ภายนอก อ่านได้ที่หมวด ตั้งค่าผลลัพธ์สำหรับการทดสอบในห้องปฏิบัติการภายนอก

1.จากเมนู สมุดบันทึก เลือก หมวดห้องปฏิบัติการ หน้าต่าง หมวดห้องปฏิบัติการ จะเปิดขึ้น

2.ค้นหาการทดสอบที่ต้องการ

3.คลิกปุ่ม แก้ไข () เพื่อเปิด แบบฟอร์มตัวอย่าง/รายละเอียดการทดสอบ หน้าต่าง รายละเอียดตัวอย่าง/การทดสอบ จะปรากฏพร้อมรายละเอียดตัวอย่าง ซึ่งมีข้อมูลอย่างละเอียดเกี่ยวกับตัวอย่างนั้นๆ

4.ขยายหมวด หมวดรายละเอียดการทดสอบ เพื่อแสดงข้อมูลอย่างละเอียดเกี่ยวกับการทดสอบ

5.แก้ไขรายละเอียดการทดสอบ

6.คลิกปุ่ม OK หน้าต่าง รายละเอียดตัวอย่าง/การทดสอบ จะปิดลง

7.คลิกปุ่ม บันทึก เพื่อยืนยันการบันทึก
