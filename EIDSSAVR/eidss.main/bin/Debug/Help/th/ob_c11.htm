ช่องข้อมูล

ID การระบาด

รหัสที่ระบบกำหนดให้เมื่อสร้างการระบาดและบันทึกเรียบร้อย

วันที่เริ่ม

วันที่ๆเริ่มการระบาด

วันสิ้นสุด

วันสิ้นสุดเหตุการณ์ระบาด

การวินิจฉัย/กลุ่มการวินิจฉัย

เลือกการวินิจฉัยหรือกลุ่มการวินิจฉัย จากรายการดร็อปดาวน์ *ช่องนี้จำเป็นต้องกรอก*

สถานะการระบาด

ใส่สถานะของการระบาด เลือกสถานะ (น่าจะเป็น สงสัย ยืนยัน หรือ ไม่ใช่การระบาด) จากรายการดร็อปดาวน์

สถานที่ที่เกิดการระบาดหลัก

กำหนดตำแหน่งทางภูมิศาสตร์ของการระบาด คลิกที่ปุ่ม เเละหน้าต่างดร็อปดาวน์ จากแบบฟอร์ม ตำแหน่งทางภูมิศาสตร์ จะถูกขยาย

ผู้ป่วยหลัก/เซสชั่นหลัก

เลือกเคส ผู้ป่วย หรือเซสชั่นหลัก ของการระบาด

คำอธิบาย

ใส่ลักษณะรายละเอียดการระบาด 

ตาราง รายงานของผู้ป่วย/เซสชั่นที่เกี่ยวข้อง มีข้อมูลต่อไปนี้:

ช่องข้อมูล

ID ผู้ป่วย/เซสชั่น

มีการแสดง ID กรณี/เซสชั่น

วันที่กรณี/เซสชั่น

มีการแสดงวันที่กรณี

กรณีผู้ป่วย:

ถ้าค่าของช่อง วันที่เริ่มเเสดงอาการ ไม่ว่าง วันนี้ก็จะแสดงขึ้น

ถ้าค่าของช่อง วันที่เริ่มเเสดงอาการ ว่าง เเละค่าของวันที่เเจ้งเตือน ช่องเวลาไม่ว่าง วันที่เเจ้งเตือนจะแสดงขึ้น

ถ้าค่าของช่อง วันที่เริ่มเเสดงอาการ เเละ วัน เวลาที่เเจ้งเตือน นั้นว่าง ค่าของช่อง วันที่ป้อน จะแสดงขึ้น

กรณีสัตวแพทย์:

ถ้าค่าของช่อง วันที่สืบสวน ไม่ว่าง วันนี้ก็จะแสดงขึ้น

ถ้าค่าของช่อง วันที่สืบสวน ไม่ว่าง วันที่ทำการป้อนข้อมูล จะแสดงขึ้น  

ประเภทผู้ป่วย

กรณีผู้ป่วย: ถ้าไม่มีการกรอกในช่อง การเเยกประเภทกรณีสุดท้าย การเเยกประเภทเบื้องต้น จะปรากฏ ไม่เช่นนั้น การเเยกประเภทกรณีสุดท้าย จะปรากฏ  

กรณีสัตวแพทย์: มีการแสดงการเเยกประเภทกรณี

การวินิจฉัย

มีการแสดงการวินิจฉัยกรณี

กรณีผู้ป่วย: มีการแสดง การวินิจฉัยสุดท้าย/การวินิจฉัยปัจจุบัน

กรณีสัตวแพทย์: ถ้าไม่มีการกรอกในช่อง การวินิจฉัยสุดท้าย      จะมีการแสดงการวินิจฉัยที่ยังไม่เเน่ชัด ไม่เช่นนั้น การวินิจฉัยสุดท้าย จะแสดงขึ้น

สถานที่สัมผัสเชื้อ

ตำแหน่งของกรณีนั้นจะเเสดงตามที่มันถูกกำหนดไว้ในฟอร์แม็ตของตำแหน่งทางภูมิศาสตร์

กรณีผู้ป่วย: สถานที่สัมผัสเชื้อ

กรณีสัตวแพทย์: ตำแหน่งฟาร์ม

ที่อยู่

ตำแหน่งของกรณีนั้นจะเเสดงตามที่มันถูกกำหนดไว้ในฟอร์แม็ตของที่อยู่

กรณีผู้ป่วย: ที่อยู่ปัจจุบัน

กรณีสัตวแพทย์: ที่อยู่ฟาร์ม

ผู้ป่วย/เจ้าของฟาร์ม

ชื่อของผู้ป่วย หรือเจ้าของฟาร์มจะแสดงในช่องนี้

ตาราง รายงานของผู้ป่วย/เซสชั่นที่เกี่ยวข้อง มีปุ่มฟังค์ชั่นต่อไปนี้:

ปุ่ม

เลือกผู้ป่วยหลัก

เลือกเคสผู้ป่วยหรือเซสชั่นหลักของการระบาดได้โดยคลิกที่ปุ่มนี้

เพิ่มสัตว์ป่วย

เมื่อผู้ใช้คลิกปุ่มนี้ หน้าต่างของรายการสัตว์ป่วยจะเปิดขึ้นมาเพื่อให้เลือกรายการให้เชื่อม โดยการคลิกไปที่ปุ่ม เลือก รายการที่เลือกจะถูกเพิ่มเข้าไปในตาราง

ถ้ารายการที่เลือกเคยถูกเลือกให้เชื่อมกับการระบาดอื่นอยู่แล้ว จะมีข้อความปรากฏว่า: ลบกรณีนี้ออกจากการระบาดอื่นหรือไม่?

ถ้าผู้ใช้คลิกที่ปุ่ม ใช่  รายการที่เลือกจะถูกลบไปจากการระบาดเก่า เเละเเนบไปกับการระบาดที่ผู้ใช้เปิดอยู่

คลิกไปที่ปุ่ม ไม่ จะไม่มีอะไรเกิดขึ้น เเละการเลือกถูกยกเลิก

ปุ่มนี้เปิดเลือกเพื่อเชื่อมรายการสัตว์ป่วยเข้ากับการระบาด ซึ่งขึ้นกับสิทธิ์ของผู้ใช้ที่ระบุในบัญชีผู้ใช้

เพิ่มผู้ป่วย

เมื่อผู้ใช้คลิกปุ่มนี้ หน้าต่างของรายการผู้ป่วยจะเปิดขึ้นมาเพื่อให้เลือกรายการให้เชื่อม โดยการคลิกไปที่ปุ่ม เลือก เคสที่เลือกจะถูกเพิ่มเข้าไปในตาราง

ถ้ารายการเคสผู้ป่วยที่เลือกเคยถูกเลือกให้เชื่อมกับการระบาดอื่นอยู่แล้ว จะมีข้อความปรากฏว่า: ลบกรณีนี้ออกจากการระบาดอื่นหรือไม่?

ถ้าผู้ใช้คลิกที่ปุ่ม ใช่  รายการที่เลือกจะถูกลบไปจากการระบาดเก่า เเละเเนบไปกับการระบาดที่ผู้ใช้เปิดอยู่

คลิกไปที่ปุ่ม ไม่ จะไม่มีอะไรเกิดขึ้น เเละการเลือกถูกยกเลิก

หากเคสผู้ป่วยที่เลือกเชื่อมกับการระบาด มีระบุไว้ในแท็บย่อย สรุปประเภทผู้ป่วยและผลการสอบสวน ใน ฟิลด์ กรณีนี้เกี่ยวข้องกับการระบาดไว้ว่า ไม่ ข้อความจะปรากฏว่า ไม่สามารถเพิ่มกรณีนี้ได้ เพราะมันถูกทำเครื่องหมายว่าไม่เกี่ยวข้องกับการระบาดใดๆ เคสผู้ป่วยนี้ไม่สามารถเชื่อมกับการระบาดได้

เพิ่มเซสชั่นพาหะนำโรค

เมื่อผู้ใช้คลิกปุ่มนี้ หน้าต่างของรายการเซสชั่นการเฝ้าระวังพาหะนำโรคจะเปิดขึ้นมาเพื่อให้เลือกรายการให้เชื่อม โดยการคลิกไปที่ปุ่ม เลือก รายการที่เลือกจะถูกเพิ่มเข้าไปในตาราง

ถ้ารายการที่เลือกเคยถูกเลือกให้เชื่อมกับการระบาดอื่นอยู่แล้ว จะมีข้อความปรากฏว่า: ลบกรณีนี้ออกจากการระบาดอื่นหรือไม่?

ถ้าผู้ใช้คลิกที่ปุ่ม ใช่  รายการที่เลือกจะถูกลบไปจากการระบาดเก่า เเละเเนบไปกับการระบาดที่ผู้ใช้เปิดอยู่

คลิกไปที่ปุ่ม ไม่ จะไม่มีอะไรเกิดขึ้น เเละการเลือกถูกยกเลิก

หมวด หมายเหตุ ใช้ในการบันทึกหมายเหตุใดๆก็ตามที่เกี่ยวกับการระบาด
