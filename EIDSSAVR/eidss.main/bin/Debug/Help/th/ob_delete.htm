การลบการระบาด

ข้อสำคัญ! ระบบไม่อนุญาตให้ลบการระบาด ในกรณีดังต่อไปนี้:

•มีการเชื่อมผู้ป่วยกับการระบาดนั้นๆ;

•มีการเชื่อมต่อสัตว์ป่วยกับการระบาดนั้นๆ;

•มีการเชื่อมต่อเซสชั่นการเฝ้าระวังพาหะนำโรคกับการระบาดนั้นๆ

 

1.คลิก สมุดบันทึก -> การระบาด รายการเหตุการณ์การระบาด จะเปิดขึ้น

รูปภาพ 10 รายการการระบาด

 

2.ค้นหา การระบาดที่ต้องการลบทิ้ง

ข้อแนะนำ หากไม่แน่ใจ ควรเปิดดูรายละเอียด โดยกดที่เลข ID การระบาด เพื่อเปิดดูรายละเอียดก่อนลบการระบาดนั้น

3.เลือกจากรายการ เเละคลิกปุ่ม หน้าต่างยืนยันจะเปิดขึ้น

รูปภาพ 11 ลบบันทึก

 

4.คลิก OK  เพื่อยืนยัน เเละลบการระบาด
