Heyvan xəstəlik hadisəsi bölməsində laborator testlərin və test nəticələrinin görüntüsü

Testlər məntəqədən kənar laboratoriyada keçirilibsə, istifadəçi test nəticələrinə Mal-qara və quş xəstəlik hadisələri formalarının Laborator testlər alt-pəncərəsində baxa bilər.

Şəkil 321 Laborator testlər alt-pəncərəsi

Testlər bölməsində keçirilmiş testlər barədə məlumatlar əks olunub və həmin məlumatlar yalnız oxumaq rejimində açılacaq. Amma Nəticələrin xülasəsi və intepretasiyası bölməsini redaktə etmək mümkündür.
