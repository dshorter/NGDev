Məlumatların sütunda filter olunması

Əgər bəzi filterlər tərtib cədvəldə istifadə olunacaq qruplaşdırıcı funksiyaların nəticələrinə tətbiq olunmalıdırsa onları Baxış rejimin alt-səhifədə yarat və tətbiq et. Filterin funksionallığı - əks istiqamətdə yerləşən seçim-xanalarını nişanlamaqla və daha çevik alət olan uygunlaşdırma filterlərini seçməklə cədvəldə görünməsini istədiyiniz məlumatları işarələmək üçün istifadə olunan standart filterlərdən ibarətdir.

Filterin açılan siyahılar xüsusi sütun üçün filter kriteriyalarını müəyyənləşdirmək üçün tərtib olunmuşdur.

Filterlə birlikdə Tərtib cədvəldə quraşdırılmış spesifik parametrlər Baxış rejimin alt-pəncərəsində dəyişdirilə bilməz.

Standart filter

Standart filterdən istifadə etmək üçün sütun başlığında olan filter işarəyə bas və açılmış siyahısından istədiyin bəndi seç.

Şəkil 562 Sütun filteri

Filteri uyğunlaşdır

Daha təkmil filter tələb olunduğu halda Istifadəçi tərəfindən köklənilən filteri istifadə et.

1.Sütun başlığında filter işarəyə bas və aşılmış siyahısından Istifadəçi tərəfindən köklənilən filter xanasını seç. Istifadəçi tərəfindən köklənilən filterin pəncərəsi açılacaq.

2.2 axtarış meyarına qədər istifadəçi tərəfindən köklənilən filteri qur.

3.Bəli düyməsini bas.
