Qrup/fərdi Keçirici təfərrüatları W03 formasının Laboratoriya Testləri alt-pəncərəsi

Laboratoriya Testləri keçirici/ qrupu üçün Laboratoriya modulunda həyata keçirilən laboratoriya testləri siyahısını nəzərdən keçirmək üçün istifadə olunur. Tab yalnız-oxunandır.

Şəkil 409 Laborator testlər alt-pəncərəsi

Məlumat xanaları

Nümunənin laborator qeydiyyat nömrəsi

Test üçün istifadə olunan nümunənin laboratoriya nömrəsi nümayiş olunur. Nümunənin laborator qeydiyyat nömrəsi nümunə Laboratoriya moduluna daxil edilən zaman Fərdi Nömrələmə Sxeminə uyğun olaraq avtomatik şəkidlə təyin edilir.

Nümunənin sahə qeydiyyat nömrəsi

Test üçün istifadə olunan nümunənin sahə nömrəsi.

Nümunə Növü

Test üçün istifadə olunan nümunənin tipi.

Toplanma tarixi

Test üçün istifadə olunan nümunənin toplandığı tarix.

Testin Adı

Testin adı.

Test Kateqoriyası

Əgər məlumdursa testin kateqoriyası nümayiş olunur.

Nəticə Tarixi

Testin başlama tarixi yaxud onun nəticəsinin daxil edildiyi tarix.

Testi icra etdi (tərəfindən)

Testin nətisəsini daxil edən işçinin adı.

Test keçirən müəssisə

Testi həyata keçirmiş təşkilat.

Test Nəticəsi

Testin nəticəsi.

Xəstəlik

Laboraotoriya modulunda test üçün Test Diaqnozu kimi təyin edilən xəstəlik. Əgər nəticə müsbətdirsə, xəstəliyin surəti sesiyanın Xəstəliklər sahəsinə köçürülür.
