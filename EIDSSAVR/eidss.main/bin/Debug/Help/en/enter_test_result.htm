Enter test result

Once a test is completed, it is time to register its result in the system. Results of the tests both performed in the current organization and in an external laboratory can be registered in the system. The results of the tests performed in the current laboratory need to be validated (confirmed). This means that if Individual Forms are used, the test results must first be entered, and then subsequently validated. For the period until the test's result is validated, the status of the test is Preliminary. However, in the Laboratory Section form a test can be entered and validated at the same time. If this occurs, the status of a test is immediately changed from Not started to Final.

Meanwhile the results received from an external laboratory are accepted as already confirmed. So their status becomes Final without an intermediate step of the Preliminary status. Only results of the tests assigned to samples with a Transferred out status can be entered as external results.

For more details, read the Sample Statuses and Test Statuses sections.
