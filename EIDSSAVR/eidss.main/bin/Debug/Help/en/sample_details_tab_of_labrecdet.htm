Sample Details tab of Laboratory Record Detailed form

The Sample Details tab is used to display sample detailed information, which cannot be displayed in the Laboratory Section form.

Fig. 506 Sample Details tab

 

Collection Date

The date when the sample has been collected.

Storage Location

Select a location in the storage for the sample:

1.Click on the Details button (). The Freezers window opens.

2.Double-click the desired location. The Freezers window closes. The assigned location is displayed in the field.

Functional Area

Select the department of the organization, which works with the sample.

Parent Sample ID

A parent sample for aliquots and derivatives. To view parental sample ID details, click the Details button (). The field is used for aliquots or derivatives only; for the remaining samples - the field is blank.

Notes

Enter some note if necessary.

Case/Session Type

The type of the case related to the sample:

•Human

•Avian

•Livestock

•Vector Surveillance

Nothing is displayed for an AS session.

Patient/Species/Vector Info

The following information can be displayed:

•Avian cases: a type of species.

•Livestock cases,:animal ID.

•Human cases: last, first, and middle names of the patient.

•AS sessions: the type of the animal and animal ID.

Tests Count

Total number of tests assigned to the sample.

Organization Sent To

The current organization is specified for those samples, which shall be transferred in. Once the samples are found, they can be transferred/accessioned in. For details, read the Accession in/ transfer in existing sample(s), linked to case/session (LS) section.

Collected by Institution

The organization that collected the sample.

Collected by Officer

Name of officer who collected the sample.

Transferred To

Specify an external organization to which the desired sample was transferred out. Once the samples are found, results for them can be registered. For details, read the Set result for test, performed in external laboratory (LS) section.

Destruction Method

Select the destruction method to be used. The field is used for the samples with a Marked for destruction status.
