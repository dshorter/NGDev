Veterinary Prophylactic Measure Matrix

Show/Hide Hidden Text

The Veterinary Prophylactic Measure Matrix is a tool designed to store and manage the list of prophylactic measures defined in the measure reference editor. The matrix is used in the vet aggregate actions entry.

There can be several versions of the matrix, one of which is currently active, the rest are the versions used in the past and/or matrix(es) not yet activated - a draft for a new matrix to be active in future.

View a version of the matrix (active or non-active):

1.In the System menu, click Configuration and then click Veterinary Prophylactic Measure Matrix. The Veterinary Prophylactic Measure Matrix window opens. If there are no matrix versions, the window is empty.

Fig. 71 Veterinary Prophylactic Measure Matrix window

2.The Version Name field contains the name of the currently active version. Activation Date displays the date when the version was activated.

3.In the Select Version field, select the desired matrix version. The matrix version is displayed. The Prophylactic Measures list of the activated version is read-only.
Create an active version of the prophylactic measures matrix:

1.In the System menu, click Configuration and then click Veterinary Prophylactic Measure Matrix. The Veterinary Prophylactic Measure Matrix window opens.

Fig. 72 Veterinary Prophylactic Measure Matrix window

2.If there already is an active version, click the New Matrix Version button. If not, skip this step.

3.In the Version Name field, enter the name of the version.

4.The Activation Date is an informative field. It is recommended to store the correct date here.

5.Click on the Click here to add a new row line of the table and add a new measure.

6.Define which measures for which animals for which diseases can be done.

7.Repeat steps 2-3 to create a list of all used prophylactic measures.

8.When the version is completed, click the Activate Matrix Version button. Note that no changes in the prophylactic measures can be done in an activated version.

9.Save the data and close the window.
Edit a version of the prophylactic measures matrix:

The non-active version can be edited. The active versions can be partially edited: the version name and its activation date can be changed, but the list of diseases cannot be modified except the order of items in it.

1.In the System menu, click Configuration and then click Veterinary Prophylactic Measure Matrix. The Veterinary Prophylactic Measure Matrix window opens.

Fig. 73 Veterinary Prophylactic Measure Matrix window

2.The Version Name field contains the name of the currently active version.

3.In the Select Version field, select the matrix version to edit. The matrix version is displayed.

4.Change version name and/or its activation date (for activated versions) and/or the list of disease (for the non-activated version).

5.Save the data and close the window.
