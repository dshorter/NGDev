Create a summary report

The Summary Report is the report compiled from selected aggregate cases.

Fig. 204 Human Aggregate Case Summary Form window

1.Click Search -> Human Aggregate Cases Summary. The Human Aggregate Case Summary Form opens.

2.Select Administrative Level and Time Interval Unit.

3.To add an existing case to the summary, click Select. The Human Aggregate Cases List opens.

Fig. 205 Human Aggregate Cases List

4.Search for the case to be included in the summary report and click Select. The case will be added to the summary, and will appear in the Selected Aggregate Cases section.

Note: Click New to create a new aggregate case and add it to the summary.

5.When all the needed cases are added to the Selected Aggregate Cases section, click Show Summary Info.

6.The summary will be displayed in the Summary Info section.
