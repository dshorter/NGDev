Create batch in Batches List form

1.From the Journals menu, select Batches. The Batches List window opens. It contains all batches created in the system.

2.Click the New button. The Batch Test Details form opens. It contains no tests.

Fig. 468 Batch Test Details form

3.Select a Test Name for a batch.

4.Add tests in one of the following ways:
First method:

a)Click the Add Tests button. The Laboratory Tests List window opens. It contains only those tests that conform to the batch:

•The test status is Not Started.

•The test does not belong to another batch.

•The test name is the same that is selected for the batch in the Test Name field.

b)Select the test to add to the batch.

c)Click the Select button (or add all tests by clicking the Select All button). The Laboratory Tests List window closes, and the selected tests are added to the list.

Second method:

a)Enter the laboratory ID of the sample related to the test in the Search test by lab sample ID field at the bottom of the form.

b)Click the Search button. All tests of the identified sample that meet the criteria will be added to the list of batch tests.

Third method:

a)Scan the barcode label of the sample related to the test in the Add test by scanning lab sample ID field or enter the barcode manually at the bottom of the form.

b)Click the Add button. All tests of the found sample that meet the criteria will be added to the list of batch tests.

5.If incorrect tests was added to the list, select the excess tests and click the Remove Test button.

6.Select the person to perform the test in the Tested by field.

7.Select the date of the test in the Date Test Started field and the result of the test in the Result field if it is known.

8.Click the Save button and confirm the data saving.

9.Click the OK button to close the Batch Test Details form.

If no test result is entered in EIDSS, their status and the status of the batch shall be In progress. For details about entering a result for the tests and closing a batch, read the Finalize a batch (IF) section.
