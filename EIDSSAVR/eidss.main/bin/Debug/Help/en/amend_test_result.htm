Amend test result

A Test result amendment is an operation of changing a validated and finalized test result. This procedure is used when a test result is found to be incorrect after it had been validated. A result of a test with another status (In Progress or Preliminary) does not need a special amendment procedure. To change such a result, use the same procedure which is used to initially enter a test result.

Each result amendment is saved, and can be found in the Amendment History tab of a test in the Laboratory Test Details L04 form or Laboratory Record Detailed Form

The Amended status is a dead-end for a test. To repeat an test with an Amended status, simply assign a new test instead.
