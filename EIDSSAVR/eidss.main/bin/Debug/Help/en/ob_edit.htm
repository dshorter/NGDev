Edit an outbreak

1.Click Journals -> Outbreaks. The Outbreaks List opens.

Fig. 386 Outbreaks List

2.Search for an Outbreak.

3.To edit an Outbreak, select it from list and click Edit. The Outbreak Details form opens.

Fig. 387 Outbreak Details

4.Edit the data in the fields of the form. Please refer to Outbreaks Details C11 for complete information about the form.

5.Click Ok to save the outbreak, and close the form.
