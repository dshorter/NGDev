Test-Test Result Matrix

Test -> Test Result Matrix is a tool designed to set reference results of tests.

Create a list of possible results for a test:

1.In the System menu, click Configuration and then click Test -> Test Result Matrix. The Test -> Test Result Matrix window opens.

Fig. 82 Test -> Test Result Matrix window

2.To create a list of possible results for laboratory tests, select the Laboratory Tests check-box.

3.In the Test Name field, select the test for which the results will be specified. The results added before are displayed in the Test Results for Selected Test area.

4.To add a result to the test, select it in the General Test Results List area and click the left arrow button (). The value is moved to the Test Results for Selected Test area.

5.To delete a result, select it in the Test Results for Selected Test area and click the right arrow button (). The value is moved to the General Test Results List area.

6.Check Indicative check-box for a test result, if this particular result indicates the detection of the diagnosis, for which this test is done. More than one test result can be checked as indicative.

7.To create a list of possible results for penside tests, select the Penside Tests radio-button.

8.Repeat steps 3-6 to create the list of possible results.

9.To create a new test result in the General Test Results List area, click the Add New Test Result button. The Base Reference Tables Editor window opens.

10.Add new test results and click Ok to close the window.

11.Save the data and close the window.
