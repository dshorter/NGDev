Persons List

Show/Hide Hidden Text

Any person identified during the case creation or other procedure is added to the Persons List. Use this list to search for a patient, farm owner, contact name etc.

View the Persons List:

1.In the Journals menu click Persons. Persons List is opened.

Fig. 125 Persons List

2.Use the search panel to enter the search criteria and click the Search button.

3.The list of persons meeting the search criteria will display in the right side of the Persons List window.

 
Add a person to the Persons List:

1.In the Persons List click the New button. The Person Details window opens.

Fig. 126 Person Details

2.Fill in the fields and click the Ok button. The person is added to the Persons List.
Edit a person's details in the Persons List:

1.In the Persons List select the desired person and click the Edit button to edit details.

Fig. 127 Person Details

2.Perform the necessary changes and click the Ok button. The person's details are saved.

 
Delete a person from the Persons List:

1.In the Persons List select the person to be deleted from the list and click the Delete button.

2.The confirmation window opens. Click Yes.

Fig. 128 Deleting a person from list

3.The person is deleted from the Persons List.

Note: a record cannot be deleted from the Persons List if the person is involved in a case or session.
