Statistics

Show/Hide Hidden Text

Data can be defined, which is used for statistics and analysis. Generally it is the population data, that is used in the Analysis Module.

Open the list of statistical data:

In the System menu, click Administration and then click Statistical Data. The Statistical Data List window opens.

Fig. 140 The list of statistical data (population)

Initially only 3 types of statistical data are available: Population, Population by Gender and Population by Age Group and Gender. If additional types are needed, add them in the Generic Statistic Types Editor.

Data can be filtered by type. To accomplish this, select the type in the Statistic Data Type parameter on the Search Panel and click the Search button.
Edit the existing data:

1.Select the item to edit and click the Edit button. The Statistical Data Details window opens.

Fig. 141 Statistial Data editing

2.Change the data in the fields.

3.Save the changes and close the window.
Add new data:

1.Click the New button. The Generic Statistical Data Details window opens. It is blank it initially opens.

Fig. 142 Generic Statistial Data Details window

2.Enter the data in the fields.

Fig. 143 Statistial Data addition

3.Save the changes and close the window.
Import new data:

1.Click the Load Data button. The Windows Open File window opens.

Fig. 144 Generic Statistial Data Details window

2.Select a file to import and click the Open button.

3.If the file contains no errors, the importing process will begin. After successful import a message screen will appear.

4.Click Ok to close the message screen.
Delete statistical data:

The data will be permanently deleted and not possible to use or restore them!

1.Select the item from the Generic Statistical Data List and click the Delete button. The Delete Record confirmation dialog opens.

2.Click Yes to confirm the deletion.
