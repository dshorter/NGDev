System Functions

Grant or restrict access rights to a user/user group for any system function:

1.In the Security menu, click System Functions. The System Functions window opens.

Fig. 154  System Functions List

2.Select the function for which change of access rights is required.

For detailed description of the system functions, see the System Functions List section.

3.Click the Permissions button. The Data Access Details window opens.

Fig. 155 Data access details

4.To add a user or group to the Actors pane, click the Add button. The Users and Groups List window opens.

Fig. 156 Users and Groups List

5.Select a group or a user to be added to the Actors list and click the Select button. The selected groups and persons appear in the Actors area of the Data Access Details window.

6.In the Actors pane select a group or a user for which access rights will be changed for the selected system function.

7.In the Permissions pane for each operation (Create/Read/Delete/Write/Access To Personal Data or Execute) define the access rights for the user/group checking either Deny or Allow check boxes.

8.Click Ok to save the changes and close the Data Access Details window.
