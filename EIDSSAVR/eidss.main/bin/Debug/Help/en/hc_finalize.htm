Finalize a Case

The case finalization procedure is a summary of the case. It should be performed only after the case investigation is finished.

 

1.Open the Case Investigation tab of the Human Case Report Form (H02) and switch to the Final Case Classification and Outcome sub-tab.

Fig. 180 Case Investigation tab - Final Case Classification and Outcome subtab

2.Select the Final Case Classification from the drop down menu.

3.Fill the other fields of the form if possible. Please refer to Human Case Report Form H02 for complete information on the form.

4.If the work with the case is finished, the Case Status should be set to Closed.

Note: cases with status Closed are not editable, and can be opened only as read-only.

5.Click Ok to save the data and close the form.
