Vet Aggregate Actions De-duplication Form V20

The Veterinary Aggregate Actions De-duplication form is used when actions de-duplication is required.

Fig. 374 Veterinary Aggregate Actions De-duplication form

Field/Variable

Description

De-duplication Group

Actions for de-duplication are displayed in groups. The group can be expanded or minimized by clicking the "+/-" button.

Case ID

Case ID is displayed.

Start Date

Start date is calculated based on the case’s time interval. The beginning of the interval is displayed.

Time Interval

Case’s time interval is displayed (Year, Quarter, Month, Week or Day)

Region

Case’s Region is displayed.

Rayon

Case’s Rayon is displayed.

Settlement

Case’s Settlement is displayed.

Organization

Entered by Institution attribute is displayed (organization’s abbreviation).

Date

Date entered is displayed.

Matrix Version

Matrix Version Name is displayed.

Template

Template Name is displayed.

Buttons

Search for Duplicates

Click the button to search for duplicates. The list of found duplicates will be updated. If duplicates are not found a message is displayed: “Duplicates are not found”.

Edit/View

Click the button to open the selected aggregate action in edit/read-only mode.

Delete

Clicks the button to delete the selected action.

The Summary Info section contains the grid which aggregates all selected cases data. Columns in this section are country specific and are defined in the Flexible forms designer and in Vet Aggregate Case Matrix Editor.
