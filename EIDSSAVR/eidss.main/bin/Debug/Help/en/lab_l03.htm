Laboratory Tests List L03

The Laboratory Tests List (L03) form is created to display all registered laboratory tests that match the search criteria. It consists of 2 parts.

•Search Panel

•Tests List

The Search Panel is created to perform the most sophisticated search by test details. The basic and most frequently used parameters are present in the panel as search fields.

The Tests List is a table where each line is a test record with laboratory test ID and a number of test parameters displayed in the table columns. If columns are not needed or irrelevant, hide them with the Column Chooser. Read about the following for more details; Item Lists, Column Chooser, Search Lists, and List and Search Forms Customization.

Search Criteria Fields

Test Name

Select the name of the desired test from a drop-down list.

Batch ID

Enter a unique ID of the batch to which the test belongs.

Status

Select the status of the desired test from a drop-down list. For details about test statuses, read the Test Statuses section.

Result Date

Enter the time interval when the result of the test was entered, validated, or amended for the last time.

Case ID/Session ID

Enter the case or session related to the tests's sample.

Test Diagnosis

Select the case or session diagnosis related to test's sample.

Patient

Select the patient for whom the case is entered.

Farm Owner

Select the farm owner from the veterinary case related to the sample.

Animal ID/Species ID/Vector ID

Test assigned to a sample linked to an avian case, enter a species ID.

Test assigned to a sample linked to a livestock case, enter an animal ID.

Test assigned to a vector surveillance session, enter a vector ID.

Test assigned to a sample linked to a human case, the field is not applicable.

Test assigned to a sample linked to an AS session, enter a species or animal ID.

Lab Sample ID

Enter a laboratory ID of the sample related to the desired test.

Local/Field Sample ID

Enter a local or field ID of the sample related to the desired test.

Accession Date From/To

Specify the time interval when the sample of the sample related to the desired test was accessioned.

Sample Type

Select the type of the sample of the sample related to the desired test.

Result List Columns

Batch ID

The number of the batch to which the test belongs.

Test Name

The name of the test.

Test Category

The category of the test.

Status

The current status of the test. For details about test statuses, read the Test Statuses section.

Test Result

The result of the test.

Result Date

The date of the test result entry, validation or amendment.

Lab Sample ID

The laboratory ID of the sample related to the test.

Sample Type

The type of the sample related to the test.

Local/Field Sample ID

The local or field ID of the sample related to the test.

Accession Date

The date when the sample related to the test was accessioned in.

Functional Area

The functional area (organization department) where the sample related to the test is stored and used.

Case ID/Session ID

The number of the case or session related to the test's sample.

Test Diagnosis

The diagnosis in the case or session related to the test's sample.

Patient/Farm Owner

The name of the human case or farm owner of the vet case related to the test's sample.

Animal ID/Species ID

A species ID of a sample linked to an avian case.

An animal ID of a sample linked to a livestock case.

Not possible to apply to human case, the field is blank.

Buttons

Make Batch

Creates a batch.

Select the tests with the same test type which are not in a batch and click the Make Batch button. The Batch Test Details L22 form with the selected tests opens.

This button is active if the Write permission to the Access to Laboratory Tests system function is granted.

Edit

Edits the selected test. This button is displayed if the Write permission to the Access to Laboratory Tests system function is granted.

Delete

Deletes selected test(s). Select the test(s) to be deleted in the grid and click the Delete button.

Completed tests cannot be deleted. If some of selected tests cannot be deleted, a warning message appears and these tests will not be deleted; all other selected tests will be deleted.

This button is displayed if the Delete permission to Access to Laboratory Tests system function is granted.
