Info Tab

The Info tab of a layout is used to edit the layout name and description, and share the layout. Moreover it stores information about the parent query, publishing status, and the latest data refresh.

Fig. 594 Info tab

Layout Name

Layout name is specified in English and in the local language (if applicable). The field is mandatory. The layout name is used to identify the layout in the Query/Folder/Layout Tree pane, it is printed in all reports, charts and maps.

Description

Layout description is a non-mandatory field, which can be used to specify details about the layout: some specific filters, sorting, etc.

Query Name

Read-only field which displays the name of the layout parent query.

Query Description

Read-only field which displays the description of the layout parent query.

Data Refresh Date and Time

Read-only field which displays the date and time of the latest data refresh.

Share Layout

Check-box designed to share the layout for other users of the layout site. For more details about layout sharing, read the Share layout section.

Publishing Status

Read-only field which displays the status of the layout - whether it is published or not. For more details about publishing a layout, read the Publish layout/query/folder section.
