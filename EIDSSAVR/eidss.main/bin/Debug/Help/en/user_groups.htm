User Groups

Show/Hide Hidden Text

Employees recorded in the system can be grouped. Use employee groups to easily grant access to a specific type of data.

Create a user group:

1.In the Security menu, click User Groups. The User Groups List window opens.

Fig. 146 User Groups List

2.Click the New button. The User Group Details window opens.

Fig. 147 User Groups Details

3.Enter the group name and brief description.

4.To add a user/user group to the created user group click the Add button. The User and Groups List window opens.

5.To select a user or a user group to be included into the created user group and click Select.

6.To define user group’s rights to the system functions open the System Functions tab.

Fig. 148 System Functions tab

7.Scroll through the system functions and choose the Deny either Allow checkbox to either deny or allow the appropriate function for the user group.

8.Click Ok to close the User Group Details window.

9.Click the Close button to close the User Groups List window.
Edit a user group:

1.In the Security menu, click User Groups. The User Groups List window opens.

Fig. 149 User Groups List

2.Click the Edit button. The User Group Details window opens.

Fig. 150 User Groups Details

3.Edit the group name or brief description.

4.To a user/user group to the created user group click the Add and in the User and Groups List window select a user/user group and click the Select button.

5.To define user group’s rights to the system functions open the System Functions tab.

Fig. 151 System Functions tab

6.In the Site drop-down field select the site on which the granted rights will be applied for the user group. Or click Apply to All Sites to apply the granted rights for the user group to all EIDSS sites.

7.Scroll through the system functions and choose the Deny either Allow checkbox to either deny or allow the appropriate function for the user group.

8.Click Ok to close the User Group Details window.

9.Click Close to close the User Groups List window.
Delete a user group:

1.In the Security menu, click User Groups. User Groups List opens.

Fig. 152 User Groups List

2.Select the group to delete.

3.Click the Delete button. The Delete Record window opens.

Fig. 153 Confirm Deletion

4.Click Ok to confirm the deletion.

5.Click the Close button to exit the Users Roles List window.
