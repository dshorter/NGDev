Laboratory Section L25

The Laboratory Section form is used to cover all Laboratory Module functionality within one window. Any laboratory operation starting with a sample creation and ending with a test result validation can be done in the same list form. Each line in the grid is a laboratory record: a sample either with or without. If more than one test is assigned to a sample, an additional line is created for the second test, and each test after it.

Fig. 501 Laboratory Section form

The Laboratory Section window consists of:

•Simple Search

•Search Panel

•List of laboratory records in the Laboratory Section tab

•List of laboratory records in the My Preferences tab

•Laboratory Record Detailed form

The Simple Search is a tool used to filter laboratory records (sample or test) by one search criterion.

The Search Panel  was created to perform the sophisticated search by sample test details. In order to open the search panel, click the Search panel shutter at the left edge of the window.

The Laboratory Section and My Preferences tabs look like a list form, but differ from any other EIDSS list form: the lists of the Laboratory Section form are editable. Depending on the cell that is clicked with the mouse, either sample or test details can be changed. The Laboratory Section tab contains the complete list of samples and tests entered in the system, while the My Preferences tab stores only those records that were manually added. For details, read the Add a record to preferences/ Remove a record from preferences list (IF) section.

In order to edit a value in a cell in the Laboratory Section and My Preferences tabs, click it and either type a new value to the cell or select it from a drop-down list or calendar.

The view of the the Laboratory Section and My Preferences tabs can be changed by hiding and unhiding their columns. For this, use the Column Chooser. Please note that columns that can be mandatory can never be hidden:

•Case/Session ID

•Local/Field Sample ID

•Sample Status

•Lab Sample ID

•Sample Type

•Accession Date

•Test Diagnosis

•Test Name

•Test Status

•Test Result

•Date Test Started

•Result Date

 

The Laboratory Record Details form is a hidden form that contains all information about the sample and the test. To open it, click the Plus button () prior to the laboratory record.
