The Farm Info Details window:

General Information tab

Contact info section contains the following parameters:

Field/Variable

Description

Farm ID

Farm ID is displayed. It is calculated automatically according to the numbering schema for new farm after the initial save.

Farm Name

Name of farm is entered.

Farm Owner Name

An already existing person in EIDSS is selected by clicking the

Fig. 381 Farm Info window

button or enter a new person in the text-box (Last Name, First Name, Middle Name are inputted limited by space). Use the empty field to clear, and unlink a person.

Phone

Farm phone is entered.

Fax

Farm fax is entered.

E-mail

Farm e-mail address is entered.

Longitude/Latitude (group of attributes)

Enter farm longitude and latitude.

The special button is located next to the attributes to provide the ability to point directly to the location on the map.

Longitude is a numerical field. The numbers are entered in the following format: x.xxxxx and are entered only in the range of (-180, +180).

Latitude is a numerical field. The numbers are entered in the following format: x.xxxxx and are entered only in the range of (-85, +85).

Point to the location directly on the map, and Longitude and Latitude are filled automatically.

Country

Name of country where farm is located are entered. Filled automatically if specified in Active Surveillance Session Address section.

Region

Only regions of the selected country are displayed in the list. Filled automatically if specified in Active Surveillance Session Address section.

Rayon

Only rayons of the selected region are displayed in the list. Filled automatically if specified in Active Surveillance Session Address section.

Town or Village

Only towns and villages of the selected rayon are displayed in the list. Filled automatically if specified in Active Surveillance Session Address section.

Street

Only streets of the selected town or village are displayed in the list. It is possible to input a new street that will be added to the reference list after the case is saved.

Postal code

Only post codes of the selected town or village are displayed in the list. It is possible to input a new postal code that will be added to the reference list after the case is saved.

Building/House/ Apt. (group of attributes)

Building, house and apartment details of farm’s location are entered. The sequence of these fields is customizable for different languages.

 

Ownership Structure section contains a group of radio-buttons. The farm's ownership structure is specified using these buttons.

 

Livestock Herds tab is designed to describe the treelike structure of livestock at a farm.

Field/Variable

Description

Farm/Herd/

Species

Treelike farm structure is entered:

-First level: Farm (filled automatically)

-Second level: Herds

-Third level: Species

The list of species is filtered by the Accessory value Livestock. Once selected, species cannot be added again to the same herd.

Herd ID is specified automatically after initial save.

Total

Total number of animals per species is entered. A total number of animals in every herd and farm is calculated automatically.

Buttons

New Herd

Click the New Herd button, and a new herd record will be added.

New Species

Click the New Species button, and a new species record will be added.

Delete

To delete, select the herd or species, and click Delete button.

Herd or species cannot be deleted while related animals exist. A warning message will appear: “Can’t delete herd/species. Delete all animals related with this herd/species before”.

Herd or species cannot be deleted while related vaccinations exist. A warning message will appear: “Can’t delete herd/species. Delete all vaccinations related with this herd/species before”.
