Laboratory Test Details L04

The Laboratory Test Details (L04) is a form used to display and edit a detailed information about a test.

 

Fig. 478 Laboratory Test Details form

It consists of the following tabs:

•General

•Test Details

•Sample Details

•Amendment History

 

Buttons

Paper Forms

Prints a result report for a test.

Amend Result

Initiates a test result amendment. The button is available for tests with a Final status only. For details, read the Amend a test result section.

Save

Saves the data in the form.

OK

Closes the Laboratory Test Details form.

Cancel

Cancels all changes made in the form.
