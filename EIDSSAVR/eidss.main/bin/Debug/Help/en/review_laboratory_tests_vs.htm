Review session laboratory tests and test results

Session laboratory tests are the list of all field tests performed for each individual vector/ pool of the session.

To review the session laboratory tests list, open the Lab Tests subtab of the Detailed Info tab, the Vector Surveillance Session form. The data in the tab is read-only. To edit samples, use the Laboratory module.

For a detailed description of each field of the Lab Tests subtab, read the Lab Tests subtab of Vector Surveillance Session W02 form section.
