Create an Urgent Notification

An urgent notification is a simplified Human Case creation procedure, where information is filled based on the Urgent Notification Paper Form.

1.In the Create menu, click Create -> Human Case. The Human Cases Report Form window opens.

Fig. 170 Human Case Report Form

2.Fill in the necessary data. Please refer to Human Case Report Form H02 for complete information on the form.

3.While filling in the data, EIDSS is checking if a similar case is already exists in the system. If such a case exists, then the Human Case List window opens automatically, allowing to select a case for de-duplication.

4.Click Ok. The urgent notification is now created.

The replication of the saved case will commence, and it will be transmitted to the EIDSS server.
