View or edit VS session

Vector Surveillance Session can be edited only if it is active, i.e. its status is In Progress.

 

1.From the Journals menu, select Vector Surveillance Sessions. The Vector Surveillance Sessions List window opens.

2.Find a desired VS session. For detailed description how to find a session, read the Search for a VS Session section.

3.Do one of the following:

•Double-click the session to edit.

•Select the session to edit and click the Edit button.

The session opens in the Vector Surveillance Session form and is ready for editing.

4.Edit general information of VS.

5.Edit information about the session with one of the following:

•Summary Information

•Detailed Information

•Register general information for vector/pool

•Register specific information for vector/pool

•Register samples for vector/pool

•Register fields test results for vector/pool

The list of VS session diseases is comprised of all diseases confirmed by either field or laboratory tests, which are performed for vector/ pools collected within the session.

6.Link VS session to an outbreak if necessary.

7.Save VS session.

8.Close VS session.

 

The replication of the saved vector surveillance session will commence and it will be forwarded to the EIDSS server and other subsystems if the Diagnosis field is changed, or if a field or laboratory test result is added or edited.
