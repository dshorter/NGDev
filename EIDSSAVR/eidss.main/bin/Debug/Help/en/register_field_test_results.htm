Register field test results for vector/pool

Once the list of samples collected for a pool/ individual vector is created, field tests can be added in the Field Tests tab of Pool/Individual Vector Details form if necessary. This step is optional. If no field tests are performed, leave the Field Tests tab of Pool/Individual Vector Details form empty. Save and close the VS session. Now the registered samples can be accessioned in, and laboratory tests can be assigned for them in the Laboratory module.

 

•Add a new field test, click the empty line with the message Click here to add a new row.

•Edit an existing field test, select the test and edit it in the grid.

•Delete field test(s), select the test(s) and click the Delete button.

 

For a detailed description of the fields, read the Field Tests tab of Pool/Individual Vector Details Form W03 section.

The tests performed for the session's vectors/pools are copied to the Field Tests subtab of Vector Surveillance Session form and are compiled in the list of session's field tests.
