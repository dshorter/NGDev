Search for a VS Session

The number of the vector surveillance sessions can be enormous. Under such circumstances it can be difficult to find a desired session without a special Search Panel. The Search panel is a filtration tool, used to extract only records that meet a list of entered search criteria.

 

1.From the Journals menu, select Vector Surveillance Sessions. The Vector Surveillance Sessions List window opens.

2.Select the filtration criteria in the fields of the Search Panel. For description of the search panel fields, read the Vector Surveillance Sessions List W01 section.

3.Click the Search button. The vector surveillance sessions that meet the filtration criteria are displayed in the list form.

 

For additional information about search panel, read the Search Lists section.
