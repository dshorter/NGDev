Farm Info V05

Farm Info (V05).

Fig. 378 Farm Info window

 

Field/Variable

Description

Farm Type

Farm Type is selected from the drop-down list. The farm type can be Avian or Livestock. The field is mandatory.

Farm ID

Farm ID is displayed. It is calculated automatically according to the numbering schema for a new farm after the first save.

Farm Name

Farm name is entered.

Farm Owner Name

Select an already existing person in EIDSS by clicking the

Fig. 379 Farm Info window

button or enter a new person in the text-box (Last Name, First Name, Middle Name are inputted limited by space).

Phone

Farm phone is entered.

Fax

Farm fax is entered.

E-mail

Farm e-mail address is entered.

Longitude/Latitude (group of attributes)

Enter the farm longitude and latitude.

A special button is located next to the attributes to provide the ability to point directly to the location on the map directly.

Longitude is a numerical field. The numbers are entered in the following format: x.xxxxx and are entered only in the range of (-180, +180).

Latitude is a numerical field. The numbers are entered in the following format: x.xxxxx and are entered only in the range of (-85, +85).

Pointing to the location on the map results in that Longitude and Latitude are filled automatically.

Date Last Updated

Date of the last data saving. Filled in automatically.

Country

Name of country where farm is located is entered.

Region

Only regions of the selected country are displayed in the list.

Rayon

Only rayons of the selected region are displayed in the list.

Town or Village

Only towns and villages of the selected rayon are displayed in the list.

Street

Only streets of the selected town or village are displayed in the list. It is possible to input a new street that will be added to the reference list after the case is saved.

Building/House/ Apt. (group of attributes)

Building, house and apartment details of farm’s location are entered. The sequence of these fields is customizable for different languages.

Postal code

Only post codes of the selected town or village are displayed in the list. It is possible to input a new postal-code that will be added to the reference list after the case is saved.
