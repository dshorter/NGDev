Transfers List C09

The Transfer List (L09) form is a journal of transfer transactions. Both in- and out-coming transactions are displayed in the list. To see a transaction details, double-click the transaction. The Sample Transfer window opens with detailed information and the list of transferred samples.

The Transfer List (L09) form consists of a search panel, and a list of transfers that meet the search criteria. Read special sections for details about Item Lists, Column Chooser, Search Lists, and List and Search Forms Customization.

Fig. 492 Transfer List

Search Parameters

Transfer ID

Enter the number of the desired transfer.

Transfer From

Enter the organization that has sent the samples.

Transfer To

Enter the organization to which the samples have been transferred.

Result List

Transfer ID

The transfer ID.

Transferred From

The organization that has sent the samples. If the sample has been transferred out, the current organization is displayed.

Transferred To

The organization to which the samples have been transferred. If the sample has been transferred in, the current organization is displayed.

Buttons

Edit

Opens the Sample Transaction window to transfer in samples or view the details of a transfer out operation.

Refresh

Updates the list of transfer transactions with the records added after the Transfers List window has been updated for the last time.

Close

Closes the Transfers List window.
