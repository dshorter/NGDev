Aliquots/Derivatives L06

The Aliquots/Derivatives (L06) form is used to register aliquots and derivatives created from a sample. It contains

•Upper left corner of the form displays a list of samples selected for creating aliquots and/or derivatives.

•A working pane at the left lower corner of the form hosts the parameters that define aliquots or derivatives shall be created, and the quantity of the new samples and the type of the newly created samples.

•The right pane of the Aliquots/Derivatives form is comprised of a list of created aliquots and derivatives.

Fig. 488 Aliquots/Derivatives form

Parental Samples

Sample ID

Sample ID of selected samples .

Count

Number of newly created aliquots/derivatives. The number is calculated automatically and cannot be changed manually.

Create Aliquot/Derivative

Select whether aliquots or derivatives shall be created.

Number of new samples

Specify the number of aliquots or derivatives.

Type of new samples or new derivatives

Select the type of the created derivatives from a drop-down list. The list of available types is defined by the type of the selected parental sample, and based on the Sample Type -> Derivative Type matrix. Available only for the case of derivatives created, when Create Derivatives is selected in the Create Aliquot/Derivative field.

Assigned Tests

Original Sample

Parental Lab Sample ID.

Lab Sample ID

Lab Sample ID of the newly created aliquot/derivative is displayed and can be changed.

Sample Type

Sample type of the newly created aliquot/derivative.

Functional Area

Select the organization department, which will work with the sample. Select from a drop-down list.

Storage Location

Select a storage location:

1.Click the Details button (). The Freezer form opens.

2.Select the desired location.

3.Click on the OK button. The selected location is displayed in the cell.

Comment

Comments.

Buttons

Create

Creates aliquots or derivatives. Click on the Create button when all required data is entered in the left pane. The newly created samples (aliquots and derivatives) are added to the list in the right pane.

Print Barcodes

Prints barcode labels for newly created aliquots and/or derivatives.

OK

Closes the window. If some aliquots/derivatives are created, the entered data will be saved and aliquots/derivatives creation confirmed.
