Vector Surveillance Sessions List W01

Vector Surveillance Sessions List form is used to find, view, edit, create and delete vector surveillance sessions.

Fig. 397 Vector Surveillance Sessions List window

Search Criteria Fields

Session ID

Enter the full ID of the session to be found if it is known.

Field Session ID

Enter the full field ID of the session to be found if it is known.

Status

Select the status of the session(s) to be found.

Vector type

Select the type of the vector(s) collected in the sessions to be found.

Species

Select the species of the vectors(s) collected in the sessions to be found. The field is available only if the Vector Type is selected.

Diagnoses

Select the diagnosis confirmed by the session(s) to be found.

Diagnoses Group

If sessions with the different diagnoses of the same group are to be found, select the group here.

Region

Select the region of the session(s) to be found.

Rayon

Select the rayon of the session(s) to be found. The field is available only if the Region is selected.

Settlement

Select the settlement where the session(s) took place. The field is available only if the Region and Rayon are selected.

Longitude

Enter the longitude coordinate of the area where the session(s) took place.

Latitude

Enter the latitude coordinate of the area where the session(s) took place.

Start Date From/ To

Select the time interval of the session(s)' start.

Close Date From/ To

Select the time interval of the session(s)' end.

Outbreak ID

Select the outbreak to which the desired session(s) is/are linked.

Data Entry Site

Select the site where the desired session(s) was registered.

Result List Columns

Session ID

The vector surveillance session ID is displayed.

Status

The status of the session is displayed.

Vectors types

The types of the vectors collected for the session are displayed and separated by a comma (,).

Diseases

The diseases confirmed by the session are displayed and separated by a comma (,).

Region

The region of the session holding is displayed.

Rayon

The rayon of the session holding is displayed.

Longitude

The longitude coordinate of the session holding is displayed.

Latitude

The latitude coordinate of the session holding is displayed.

Start Date

The date of the session start is displayed.

Close Date

The date of the session's status was changed to Closed is displayed.

Buttons

New

Creates a new vector surveillance session. For details, read the Create vector surveillance (VS) session section.

Edit

Opens a selected session to edit. For details, read the Edit VS session section.

Delete

Deletes a selected session. For details, read the Delete a VS Session section.

Refresh

Refreshes the list of displayed sessions.

Close

Closes the Vector Surveillance Sessions List window.
