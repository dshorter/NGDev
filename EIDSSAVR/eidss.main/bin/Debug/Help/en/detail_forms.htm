Detailed Forms

The detailed form contains fields of the selected list item. Mandatory fields are marked with the red frame, and the rest fields are optional or read-only.

For example, click on Create -> Human Case. The Human Case Report Form window opens.

Fig. 29 Detailed form example

General buttons description:

Button

Description

Save

Click the button to save the data entered in the form. All the mandatory fields should be filled in order to save the form. The form remains opened after the data is saved.  

Ok

Click the button to save the data and close the form. All the mandatory fields should be filled in order to save the form.

Cancel

Click the button to close the form without saving the data. A confirmation window with Yes/No button appears. Click Yes to close the form.

New

Click the button to close the current form without saving the data and to open the similar blank form. A confirmation window with Yes/No button appears. Click Yes to proceed.

Delete

Click the button to delete the related item (Human Case, Veterinary Case, Vector Surveillance Session etc.) from the database. A confirmation window with Yes/No button appears. Click Yes to proceed.

Clear

Click the button to clear the filling of the fields in the form. A confirmation window with Yes/No button appears. Click Yes to proceed.

Paper Forms

Click the button to open the paper form for printing. The list of all possible paper forms opens. Select the required paper form. Please refer to the Paper Forms section for more information on paper forms.

Select an item from the drop-down list:

1.Click the () button to expand the drop-down list.

Fig. 30 Drop-down list opened

2.Select an item from list. The drop-down list closes and the selected item is displayed in the field.

Select an item from the search list:

1.Click the () button to open the search list.

Fig. 31 Search list opened

2.Select an item from list and click Select. The search list closes and the selected item is displayed in the field.

Add an item to the drop-down list:

1.Click the () button to open the item's detailed window.

Fig. 32 Adding new person to the drop-down list

2.Fill in the mandatory fields and optional fields (if known) of the form.

3.Click Ok. The new item is displayed in the field and also the item is added to the drop-down list.

Edit an item of the drop-down list:

1.Click the () button to open the item's detailed window.

Fig. 33 Editing person's details

2.Edit the information if required. Fill in the mandatory fields and optional fields (if known).

3.Click Ok. The edited item is displayed in the field and also the item is updated in the drop-down list.
