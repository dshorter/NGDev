Link Active Surveillance Sessions to campaign

Show/Hide Hidden Text

The Active Surveillance Campaign can have a number of linked Active Surveillance Sessions. A new session can be created within the campaign or by selecting an existing session from the list. All operations are performed within the Sessions tab of the Active Surveillance Campaign form.

 

Create a new Active Surveillance Session

1.Click New Session. The Active Surveillance Session form opens.

Fig. 274 Active Surveillance Session

2.Please refer to Create an Active Surveillance Session section for more information.

3.Once the session is created and saved it appears in the list of linked sessions.

Fig. 275 Active Surveillance Campaign - Sessions tab

 

Select an existing session

1.Click Select Session. The Active Surveillance Sessions List opens.

Fig. 276 Active Surveillance Sessions List

2.Search for a session.

3.The list of sessions meeting the search criteria appears in search results list.

Fig. 277 Active Surveillance Sessions List

4.Select a session and click Select.

Note: the system checks if the following rules are complied in order to successfully link the session:

•The AS Session period must match and cannot be bigger than the AS Campaign period;

•The Disease and Species List of the AS Session must be either the same as Disease and Species List of the AS Campaign or represent a subset of campaign's Disease and Species List;

•The AS Session, which is already linked to another AS Campaign cannot be linked to the current AS Campaign.

5.The selected session will be added to the list of linked sessions.

Fig. 278 Active Surveillance Campaign - Sessions tab

 

View/edit a linked session

1.Select a session in the list of linked sessions, and click View Details. The Active Surveillance Session form opens.

Fig. 279 Active Surveillance Session

2.Observe the pertinent information and perform the necessary changes, then click Ok.

3.The linked session will be updated with new changes.

 

Remove a linked session

1.Select a session in the list of linked sessions and click Remove Session. The confirmation window appears.

Fig. 280 Removign a linked session

2.Click Yes to confirm the removal of the session.
