Person Details H04

Person Details (H04) form is used during the creating or editing a person in EIDSS database.

Fig. 244 Person Details form

Field/Variable

Description

Date Created

The date when the person was added to database is displayed. The field is filled in automatically.

Date Last Updated

The date when the data was last saved in database is displayed. The field is filled in automatically.

Name: Last; First; Middle

In the Last Name field the last name of the person is entered in a free-text form. Filling out the Last Name field is mandatory. The form could not be saved unless the Last Name is entered. The fields First and Middle are not mandatory fields, but they can be filled in with the first name and middle name of the person in a free-text form.

Date of birth

The date when the person was born is entered. The field is empty by default, and its value can be entered in two ways: directly typing in the field (date by default is the current date) or choosing the date from the calendar (date by default is the current date).

Sex

The Sex field identifies the sex of the person, and can be selected from a drop-down list: Male or Female. An empty field has to be selected to clear the contents of this field. Once the empty field is selected, the pop-up window asks if the user wants to clear or not clear the content. If the user confirms the deletion the field is cleared.

Personal ID Type

The person's personal ID type is selected from the drop-down list.

Personal ID

The person's personal ID is entered in this field. The field is active only after the personal ID type is selected.

Region/Rayon

These fields can be filled in from the drop-down lists of regions and corresponding rayons. The Rayon cannot be chosen unless the Region is specified. An empty field has to be selected to clear the content of the Region or Rayon field. Once the empty field is selected, the pop-up window asks if the user wants to clear or not clear the content. If the user confirms the deletion, the field is cleared.

Town or Village

The Town or Village information cannot be specified, unless the Rayon field is filled in. This field can be filled in from a drop-down list of towns in the corresponding rayon. In order to clear the content of Town or Village field the empty field has to be selected. As soon as the empty field is selected, the pop-up window asks if the user wants to clear or not clear the content. If the user confirms the deletion, the field is cleared.

Street

The Street field is a drop-down list. Initially this drop-down list is empty since, it is unnecessary for the system to store the full list of streets in all of the country’s settlements. Therefore, the streets of a settlement can be filled manually. The names of these streets will be available for this settlement in the drop-down list after manually being entered. The Street name can be given only if the Town or Village is also defined. The field is editable free-form text.

Postal Code

The Postal Сode field is filled in the same way as the Street field. The Postal Сode can be specified only if the Town or Village is also defined. The field is editable free-form text.

Building/House/Apt.

In the Building/House/Apt. free-form text field the building where the person lives is specified. Building/House/Apt. information can be given only if the Town or Village field is filled in.

Phone number

In the Phone Number free-form text field the telephone number of the person is specified.

Citizenship

The Citizenship field specifies the citizenship of the person. This can be done from a drop-down list. A new value to the drop-down list can be added in the form A01. A user can open this form by clicking the button in the Citizenship field. In order to clear the content of this field the empty field has to be selected. As soon as the empty field is selected, the pop-up window asks if the user wants to clear or not clear the content. If the user confirms the deletion, the field is cleared.

Longitude/Latitude

The longitude and latitude location can be entered using the up-down controls or by the use of Map button (GIS module will be opened).

Longitude is a numerical field. The numbers can be entered in the following format: x.xxxxx and can be entered only in the range of (-180, +180).

Latitude is a numerical field. The numbers are entered in the following format: x.xxxxx and can be entered only in the range of (-85, +85).

If a location is pointed on the map, Longitude and Latitude are filled automatically.

Name of Employer

The person’s Name of Employer can be specified. The employer is a place of work, a school, a kindergarten, an institution or other place which the person visits regularly, and where he/she interacts with a large number of people. Name of Employer is a free-form text field.

Address of Employer

Address of Employer field opens a separate drop-down window. In this window a user can specify:

-The Country which by default the value of the field equals to the country of the current site;

-The Region which depends on the chosen Country;

-The Rayon which depends on the chosen Region, and is not possible to define unless Region is selected;

-The Town or Village which depends on the chosen Rayon, and is not possible to define unless Rayon is selected;

-The Street and Postal Code and Building/House/Apt. which depend on the chosen Town or Village, and are not possible to be defined unless the Town or Village is selected.
