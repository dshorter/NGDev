Apply filters

Although there are filters in a query, there are cases when layout filters are required. Since a query is used for many layouts, it is reasonable to use those filters in common for all layouts in a query; and these, which are different for each layout, should be used in their appropriate layout.

For example, it is necessary to make several reports: a report per a disease. There are 2 ways to reach the target in the AVR:

First method

1.Create a query with all necessary data.

2.Leave only one diagnosis, remove the remained ones by a filter.

3.Create a layout for it.

4.Copy the query, change the filter.

5.Create a layout for it.

6.Repeat steps 4-5...

Second method

1.Create a query with all necessary data.

2.Create a layout for it with a filter that removes the irrelevant diagnoses.

3.Copy the layout and change the diagnosis in its filter.

4.Repeat step 3...

The 2nd method requires less efforts.

Create a filter in a layout

While in the layout, click the Edit Filters button. The Filter window opens.

Apply filters

 

After the filters are created, they are automatically activated.

If you need to activate filters after they were deactivated, select the Apply Pivot Grid Filters check-box in the Pivot Grid Settings pane of the Pivot Grid form.
