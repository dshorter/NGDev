Samples tab of Pool/Individual Vector Details W03 form

The Sample tab is used to add, edit, view and delete samples collected for the vector or pool. The samples from here are copied to the Samples subtab of Vector Surveillance Session (W02) form.

 

•Add a new sample, click the empty line with the message Click here to add a new row.

•Edit and an existing sample, select the sample and edit it in the grid.

•Delete sample(s), select the sample(s) and click the Delete button.

 

Fig. 409 Samples tab

Data fields

Lab Sample ID

The laboratory ID of the sample is assigned automatically in accordance with the Unique Numbering Schema.

Field Sample ID

Enter the ID of the sample used before its registration in EIDSS. The field is mandatory.

Sample Type

Select the type of the sample. The field is mandatory.

Collection Date

Select the date when the material was collected. The current date is used by default and can be changed manually.

Collected by Institution

Select the organization which collected the material for the sample.

Comment

The field is read-only and displays the comment which are entered in the Laboratory module.

Accession Date

The field  is read-only and displays the date when the sample was accessioned in the Laboratory module.

Sample Condition Received

The field is read-only and displays the condition of the sample when it was accessioned in the Laboratory module.

Buttons

Delete

Deletes selected sample(s).

1.Select sample(s) to delete.

2.Click the Delete button. The Confirmation window opens.

3.Click Yes to confirm the deletion and close the Confirmation window.

Only samples that have never been accessioned in and that have no field tests can be deleted.
