Batch Test Details L22

The Batch Test Details (L22) form is used to work with tests added to a batch. The form contains:

•a set of fields that display basic data of the batch

•a grid with a list of tests combined to the batch

•2 bottom field created to search for tests by the related samples

•the Sample Specific Test Parameters section which is the Test Details flexible form

•the Quality Controls section which is the Test Run flexible form

For details about flexible form creation, read the Flexible Forms Setup section.

The Sample specific Test Parameters section (Test Details flexible form) can be filled for every test in the grid. To display test's fields in the Sample specific Test Parameters section, select the test in the grid. The Quality Controls section (Test Run flexible form) can be filled out only for a batch rather than for a test of the batch. Templates for these flexible forms are defined by the test type used in the batch.

Data Fields

Batch Test ID

The ID of the batch. (new) is used for new batches until they are saved for the first time. Afterwards the batch ID is automatically assigned in accordance with the numbering scheme and can be changed manually.

Test Name

Select the test name to be performed for the batch. The field is mandatory and available if no tests are added to the batch. If a test is already in the list, only the test name available for the test is displayed in the Test Name field and cannot be changed.

The test name selected in this field filters the tests that can be added to the batch.

Location of Test Performed

by default, the current organization; cannot be changed.

Tested By

Select the employee who is to perform the test.

Validated By

Select the employee who is to validate the result. The availability of the field is defined by the Write permission to the Can Validate Test Result system function. For details about system functions, read the System Functions and System Functions List section.

Date Test Started

Enter the date when the batch test was started.

Result Date

Enter the date when the result of the batch test was received.

Columns

Sequence

Number order of records. Numeration starts from zero, no maximum number restriction is applied.

Lab Sample ID

The laboratory ID of the sample related to the test.

Diagnosis

A diagnosis for a test.

Sample Type

The type of the sample to which the test is assigned.

Category

The category of the test.

Result

Select a test result for the test.

Result Entered by

The current user is used by default as a person who entered the result and cannot be changed.

Buttons

Add Test

Adds test(s) to a batch.

Remove Test

Removes a selected test from the batch.

Add Group Result

Adds a result for a group of tests.

↑

Moves a selected record in the grid one position up.

↓

Moves a selected record in the grid one position down.

Close Batch

Closes a batch and changes the batch's status to Closed. Once a batch is closed, it becomes read-only. Only batches with a result defined for all tests can be closed.
