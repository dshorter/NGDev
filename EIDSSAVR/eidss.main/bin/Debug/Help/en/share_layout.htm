Share layout

Sharing a layout is designed to make it available for other users of the same site. Unlike the publishing, sharing a layout does not send it to all EIDSS sites and keeps it adjustable for the users of the creator's site.

 

1.From the AVR menu, click Launch AVR. The Analysis, Visualization, and Reporting Module window opens.

2.Do one of the following:

•In the Query and Layout Tree, double-click on the layout to share.

•Select the query to share and click the Edit Query/Layout button.

The Query and Layout Tree pane closes and a pane with the selected layout opens.

3.In the Info tab of the layout check the Share Layout check-box.

4.Save the layout by clicking the Save button.

 

The layout is shared and can be used and edited by other users of the current EIDSS site.
