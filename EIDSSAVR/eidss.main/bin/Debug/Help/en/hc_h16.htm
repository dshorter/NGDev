Human Aggregate Case De-duplication List H16

The Human Aggregate Cases De-duplication Form is used when case de-duplication is required.

Fig. 240 Human Aggregate Cases De-duplication form

Field/Variable

Description

Case ID

Case ID is displayed.

Start Date

Start date is based on the case time interval. The beginning of the interval is displayed.

Time Interval

Case time interval is displayed (Year, Quarter, Month, Week or Day).

Region

Case Region is displayed.

Rayon

Case Rayon is displayed.

Settlement

Case Settlement is displayed.

Organization

Organization which created the case in database is displayed (organization’s abbreviation).

Date

Date entered is displayed.

Matrix Version

Matrix Version Name is displayed.

Template

Template Name is displayed.

Buttons

Search for Duplicates

Clicking the button initiates the search for duplicates algorithm. The list of found duplicates is updated. If duplicates are not found a message is displayed: "Duplicates are not found”.

Edit

Click the button to open the Human Aggregate Case Details H06 form and edit the case.

Delete

Click the button to delete the selected case.
