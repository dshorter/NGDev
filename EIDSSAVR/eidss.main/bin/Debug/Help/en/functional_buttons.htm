Functional Buttons

The Functional buttons are buttons which are located at the bottom of the form and save or close the query, cancel changes, etc.

Fig. 604 Functional buttons

Cancel Changes

Cancel all changes made after the latest saving. For more details, read the Cancel changes (in a query) and Cancel changes (in a layout) sections.

Copy

Copy the opened query or layout. For more details, read the Copy layout and Copy query sections.

New

Create a new layout or query. For more details, read the Create layout and Create query sections.

Delete

Delete the opened query or layout. For more details, read the Delete layout and Delete query sections.

Save

Save changes. For more details, read the Save layout and Save query sections.

Close

Close the opened query or layout. For more details, read the Close layout and Close query sections.
