FFD - Sections and Fields for a Template

Show/Hide Hidden Text

If the list of sections and parameters does not contain the necessary parameter/section for the selected template, then add it.

       Adding a section:

1.In the Flexible Form Designer window, click the Parameters Editor tab to open it. The list of template types with sections and parameters is displayed in it.

Fig. 108 Flexible Forms Designer

2.Select the desired template type, and click the Add Section button (). A new section is added to the templates tree, its attributes are displayed in the upper-right area. Specify Default Name (in English) and National Name (in the local language) and Type (default or table).

Fig. 109 Flexible Forms Designer - Addition of a ne section

       Delete a section:

Select the section to be deleted and click the Delete Section button (). The section is removed from the tree.

       Add a parameter:

1.In the Flexible Form Designer window, click the Parameters Editor tab to open it. The list of template types with sections and parameters is displayed in the Parameter Editor.

Fig. 110 Flexible Forms Designer

2.Select the template type and section in which the parameter is to be (if applicable), and click the Add Parameter button (). A new parameter is added to the templates tree and is displayed in the working area, new parameter's attributes are displayed in the upper-right area.

3.Specify the following:

•Default Long Name – parameter’s long name in English

•Default Name – parameter’s long name in local language

•Type – type of the parameter (string, numeric or any other parameter type, defined in the Parameters Type Editor)

•Editor – the editor of the parameter, depends on the type of the parameter

•HA Code – select to which module parameter belongs

•Height – height of the parameter

•Label Size – label size of the parameter

•Scheme – position of the label

•Width – width of the parameter

•National Long Name – parameter’s long name in local language

•National Name – parameter’s name in local language

4.Use the mouse to change the size of the field. Moreover, select the value in the field can be selected.

       Delete a parameter:

Select the parameter to be deleted and click the Delete Parameter button (). The parameter is removed from the tree.
