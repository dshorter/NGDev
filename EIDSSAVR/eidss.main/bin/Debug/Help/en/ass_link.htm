Link an Active Surveillance Session to an existing campaign

Show/Hide Hidden Text

An Active Surveillance Session can be linked to an Active Surveillance Campaign. This action can be performed on both the Active Surveillance Session and the Active Surveillance Campaign forms.

Link the session to a campaign

1.In the Active Surveillance Session form in the Campaign ID field click the button. The Active Surveillance Campaigns List opens.

Fig. 287 Active Surveillance Campaigns List

2.Search for the required Active Surveillance Campaign, and click Select.

Note: the system checks if the following rules are complied in order to successfully link the session:

•AS Campaign must be in status Open;

•The AS Session period must match and cannot be bigger than the AS Campaign period;

•The Disease and Species List of the AS Session must be either the same as Disease and Species List of the AS Campaign or represent a subset of campaign's Disease and Species List;

•The AS Session, which is already linked to another AS Campaign cannot be linked to the current AS Campaign.

3.The session will be linked to the selected campaign. The Diseases And Species List section will be filled in automatically according to the campaign.

 

Edit the linked campaign

1.In the Active Surveillance Session form, if the campaign is already linked in the Campaign ID field, click the button. The Active Surveillance Campaign form opens.

Fig. 288 Active Surveillance Campaign

2.Edit the data in the fields of the form. Please refer to Active Surveillance Campaign V16 for complete information about the form.

3.Click Ok to save the campaign, and close the form.

 

Unlink the session from the campaign

Click the button to unlink the session from the campaign. All the fields related to the campaign will be cleared.
