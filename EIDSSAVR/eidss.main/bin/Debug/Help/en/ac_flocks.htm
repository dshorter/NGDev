Use the Flock Epi/Clinical Signs/ tab to register the information about flock/species in an Avian case.

Fig. 254 Flock Epi/Clinical Signs tab

Adding new flocks and species to the farm

1.Click the New Flock button to add a new flock. The new line will be added to the table.

Fig. 255 Flock Epi/Clinical Signs tab - adding new flock

2.Select a flock and click the New Species button to add a new species to the flock. The new line will be added to the table. Species selection is mandatory.

Fig. 256 Flock Epi/Clinical Signs tab - adding new species

3.Fill in other data for species: Total, Dead, Sick, Average Age and Start of Signs date.

Fig. 257 Herd Epi/Clinical Signs/Control Measures tab - adding new species

 

Farm - Epidemiological Information/Product Movement/Backward Tracing

If the Farm line is selected in the Flock/Species Info section, the Epidemiological Information/Product Movement/Backward Tracing is displayed in the right side of the form. It contains fields for additional information about the farm. It is divided into several sections.
 
Use the Clear button to clear all the data in the fields.

Fig. 258 Flock Epi/Clinical Signs tab - Farm selected
Species - Management/Clinical Signs/Necropsy Examination

If the Species line is selected in the Flock/Species Info section, the Management/Clinical Signs/Necropsy Examination is displayed in the right side of the form. It contains the fields for additional information about the selected species. It is divided into several sections.

 

Use the Clear button to clear all the data in the fields. Use the Copy and Paste buttons to copy the information to the other species.

Fig. 259 Flock Epi/Clinical Signs tab - species selected
