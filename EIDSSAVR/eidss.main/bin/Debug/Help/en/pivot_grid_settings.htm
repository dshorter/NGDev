Pivot Grid Settings

The Pivot Grid Settings tab contains the fields and parameters required to customize a pivot grid.

Fig. 598 Pivot Grid Settings pane

Apply Pivot Grid Filters

ON - Filters created by the Edit Filters button are applied to the pivot grid.

OFF - Filters created by the Edit Filters button are not applied to the pivot grid.

For details about the layout filters, read the Apply filters section.

Edit Filters

Click to create or edit filters for the layout. The created filters are applied only if the Apply Pivot Grid Filters check-box is selected.

Show Data in Pivot Grid

Display the data in the pivot grid. It is highly recommended to construct and configure a pivot grid with this check-box cleared. Otherwise each change in the pivot grid shall result in recalculating the displayed data, which can take a long time. For more details, read the Show data in pivot grid section.

ON - Data is displayed in the pivot grid.

OFF - Pivot grid is displayed without data.

Refresh Data

Extract the very latest data from the EIDSS database. It is strongly recommended to do it only in case it is really required, since it takes a long time. For more details, read the Refresh data section.

Compact Pivot Grid

Compress pivot grid row headers to clear a space for pivot grid data. For more details, read the Compact pivot grid section.

Freeze Row Headers

Prevent row headers from hiding caused by horizontal scrolling. For more details, read the Freeze row headers in pivot grid section.

Default Group Date

Select the default grouping for all dates in the pivot grid. For more details, read the Group date function section.

Show totals

Select the totals for the pivot grid: grand totals for rows or columns, subtotals, etc.

Show Missed Values in Pivot Grid

Add columns and rows with zero values for those dates for which no data is available. For more details, read the Show missed values section.

Aggregate Function

Choose an aggregate function to apply it to the selected field.  For more details, read the Apply aggregate functions section.

Precision

Specify the precision for the selected field with the applied aggregate function. For more details, read the Set functions default precision and Apply aggregate functions sections.

Parameters: Administrative Unit Field

Specify the administrative unit for the statistics to be used in the rate calculation. Available only if one of the Rate aggregate functions is used.

Date Field

Use this drop-down list to view all date fields grouped in a manner that differs from the default mode. For more details, read the Group date function section.
