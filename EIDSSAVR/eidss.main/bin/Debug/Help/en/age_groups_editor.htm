Age Groups Editor

The Age Groups Editor is a tool designed to define a list of age groups for a certain diagnosis if statistics for this diagnosis is collected by age groups.

Create a list of age groups:

1.In the System menu, click References and then click Age Groups Editor. The Age Groups Editor window opens.

Fig. 61 Age Groups Editor

2.Click on the Click here to add a new row line of the table and add a new type.

3.In the English Value column, type the English name of the age group.

4.Order column is used to define the order of items in the drop-down list.

5.Define the Lower and Upper Bound for the new age group.

6.Select the Interval Type (days, weeks, months or years) for the age group.

7.Save the data and close the window.
