Transfer in a sample (IF)

Transfer In is an accession in operation for a sample that has been transferred out from an external organization for tests or research and sent to the current organization. Samples received for transfer in can be found in the Transfers List L09 form.

Transferred out samples can be either transferred in from the Transfers List L09 form or accessioned in the Laboratory Section form.

 

1.From the Journals menu, select Samples Transfer. The Transfers List window opens. It contains a list of all packages sent out or in to the current organization.

2.Find a desired transfer package sent to the current organization that contain the samples to be transferred in and do one of the following:

•double-click the package

•select the package and click the Edit button. The Sample Transfer window opens.

3.Click the Transfer In button. Most fields are populated automatically:

•The current date is used as the Date Received, and can be manually changed if necessary.

•The Lab Sample ID is temporarily set to (New), and can be manually changed. If changed, the user-defined ID is used for the sample. If not changed, then the ID will get a permanent number in accordance with the Unique Numbering Schema after the form is saved.

•The Sample Received Condition field value is set to Accepted in Good Condition and can be manually changed. If manually changed, the Accession In Comment window opens for an explanation to be entered. Enter the explanation to the Comments field and click the OK button. The Accession In Comment window closes.

If the Sample Condition Received field value is changed to Rejected and saved, then the sample is hidden and cannot be found by a search.

 

Fig. 448 Sample Transfer form

4.Change details of transferred in samples if necessary. For details about columns, read the Sample Transfer L10 section.

5.Print barcode labels by clicking on the Print Barcodes button and stick them on the containers if necessary.

6.Click the Paper Forms button to print a transfer report if necessary.

7.Click the OK button. The Confirmation window opens.

8.Click the Yes button to confirm the transfer in operation.
