Active Surveillance Sessions List (V17)

Fig. 350 Active Surveillance Sessions List

Search pane section contains different fields used for session searching.

Fig. 351 Search Pane

Field/Variable

Description

Session ID

Session ID may be specified in this field.

Field Sample ID

Field Sample ID may be specified in this field.

Session Status

Session status (Open or Closed) is selected.

Session Disease

Session disease is selected.

Region

Region of the session is selected.

Rayon

Rayon is selected (list contains values from Rayon reference filtered by selected region).

Town or Village

Settlement is selected (list contains values from Settlements reference filtered by selected rayon).

Start Date

The bounds of session start date are entered.

End Date

The bounds of session end date are entered.

Date Entered

The bounds of data entry date are entered

Entered By

Session creator is selected.

Campaign ID

Campaign ID related to the session is entered.

Campaign Name

Campaign name related to the session is entered.

Campaign Type

Campaign type (Active Surveillance, Survey or Study) related to the session is selected.

Data Entry Site

Data Entry Site is selected.
Search results list section displays a list of sessions matching the entered search criteria.

Fig. 352 Search results list

Field/Variable

Description

Session ID

Session ID is displayed in this field.

Start Date

Session Start Date is displayed in this field.

End Date

Session End Date is displayed in this field.

Region

Region of the session is displayed in this field.

Rayon

Rayon of the session is displayed in this field.

Settlement

Settlement of the session is displayed in this field.

Campaign ID

Campaign ID related to the session is displayed.

Campaign Name

Campaign Name related to the session is displayed.

Buttons

New

Click the New button to create new session. Active Surveillance Session opens in a new window.

Edit

To edit, select a session in the search list, and click the Edit button. The selected session opens in a new window in edit-mode.

Delete

To delete, select a session in the search list, and click the Delete button. A warning message with Delete/Cancel buttons appears.
