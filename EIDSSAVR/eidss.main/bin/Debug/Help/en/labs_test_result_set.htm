Enter test result (LS)

1.From the Journals menu, select Laboratory Section. The Laboratory Section window opens.

2.Find the test(s) for a result to be entered. These are only tests with In progress or Preliminary statuses for which a result can be entered.

3.Enter its/their result in one of the following ways:

First method:

a)Select the desired test(s).

b)Right-click them and from the drop-down menu select Set Test Result, then the result itself. This procedure can be used for both a separate test and for several tests that have the same result.

Second method:

a)Click the Tests Status cell and select Preliminary from a drop-down list.

b)Click the Test Result cell and select the result from a drop-down list. This procedure can be used for a separate test only.

Third method:

a)Click the Test Result cell and select the result from a drop-down list. This procedure can be used for a separate test only. The Test Status is automatically changed to Preliminary.

In any of the cases the Date Test Started, Result Date, Tested by and Results Entered by shall be automatically populated by the current date and can be manually changed.

4.Click the Save button and confirm the saving.
