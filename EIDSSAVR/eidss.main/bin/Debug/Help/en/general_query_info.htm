General Query Info

The General Query Info pane contains the general information about the opened query (its name and description) and the Add all values of key references check-box which is used to create layouts for those administrative units and diagnoses with no entries in the EIDSS database.

Fig. 591 Query name and description

Query name

Query name needs is specified in English and in the local language (if applicable). The field is mandatory. The query name is used to identify the query in the Query/Folder/Layout Tree.

Description

Query description is a non-mandatory field which can be used to specify details about the query: some specific filters, search objects, etc.

Add all values of key references

Check-box created to add place holding records for missing diagnosis and administrative units. For more information about key values, read the Add key values section.
