Set result for test, performed in external laboratory (LS)

An external laboratory which performs sample testing, may or may not be equipped with EIDSS. If it is not, test results have to be registered by the sender of the samples.

An external test result can be entered for a sample with a Transferred Out status only. Although a transferred out sample contains no external tests, such a test will be automatically created while entering external results, and will immediately have a Final status. Internal tests if any will not be affected.

 

1.From the Journals menu, select Laboratory Section. The Laboratory Section window opens.

2.Find the test for an external result to be entered. This is only a test for a sample with a Transferred Out status which was transferred to an organization that has no EIDSS installed.

3.Enter the result in one of the following ways:

First method:

a)Click the plus button () prior to the sample related to the desired assigned test. A pane with sample details opens. It contains detailed information about the sample.

Fig. 458 Detailed form of laboratory record

b)Click the Test Details tab to open it. If no test is assigned yet, the tab is blank, and only the External Results check-box is available.

c)Click the External Results check-box to select it. This activates additional fields.

Second method:

a)Select the desired test.

b)Right-click them and from the drop-down menu select Set External Test Result.

4.In both cases, the following fields need to be filled out in the Test Details tab.

•The Tested by field is blank by default. Enter the person who has performed the external test. The field is mandatory.

•The Results Received from field stores the name of the organization to which the sample has been transferred out, and can be changed manually in case the sample is moved again.

•The Date Result Received field is populated automatically with the current date and can be changed manually.

•The Contact field is empty by default. Use it to keep contact data required to receive details about the performed test. The field is mandatory.

The Validated by field is populated with the current user and cannot be changed. For more details about the form, read the Laboratory Section L25 section.

The test status is changed to Final, the Date Test Started and Result Date shall be automatically populated by the current date and can be manually changed.

5.Enter the Test Name, Test Result in the Laboratory Section

6.Click the Save button and confirm the data was saved.
