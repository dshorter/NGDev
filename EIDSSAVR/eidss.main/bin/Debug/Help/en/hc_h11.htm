Human Case De-duplication Details H11

Show/Hide Hidden Text

The Human Case De-duplication Details form is used to compare two human cases.

Fig. 230 Selection of a survivior through cases comparison

Survivor is a case which will be left in the system; Superseded is a case which will be deleted from the system.

 

Notification tab is opened by default, and it is used to see the core differences between two cases. All fields which differ from one another are colored in red.

Field/Variable

Description

Case ID

The Case ID field shows the identification number of the human case.

Local Identifier

The Local Identifier field shows the local ID of the case.

Diagnosis

The Diagnosis field shows the case’s current diagnosis.

Diagnosis Date

The Diagnosis Date field shows the date of the initial diagnosis.

Last Name

The Last Name field shows the last name of the patient.

First Name

The First Name field shows the first name of the patient.

Middle Name

The Middle Name field shows the middle name of the patient.

Date of birth

The Date of birth field shows the date when the patient was born.

Age

The Age field shows the age of the patient in Age units.

Sex

The Sex field shows the gender of the patient.

Region

The Region field shows the region of patient’s current residence.

Rayon

The Rayon field shows the rayon of patient’s current residence.

Town or Village

The Town or Village field shows the settlement of patient’s current residence.

Street

The Street field shows the name of the street where the patient currently lives.

Postal Code

The Postal Code field shows the postal code of the patients’ current address.

House/Building/Aptm.

The House/Building/Aptm. field shows house number, building number and apartment of the patient’s current residence.

Phone Number

The Phone Number field shows the telephone number of the patient’s current residence.

Citizenship

The Citizenship field shows the citizenship of the patient.

Name of Employer

The Name of Employer field shows a place of work, a school, a kindergarten, an institution or any other place, which the patient visits regularly and where he/she interacts with a large number of people.

Address of Employer

The Address of Employer field shows the address of the organization defined in the field above.

Date of symptoms onset

The Date of symptoms onset field shows the date, when the first symptoms appeared.

Final State of the patient

The Final State of the Patient field shows if he/she is Dead or Alive.

The value of the field is taken from the field State of the patient at the time of notification.

Changed Diagnosis

The Changed Diagnosis field shows the diagnosis, which was selected instead of the initial diagnosis during the case’s investigation process.

Date of changed diagnosis

The Date of Changed Diagnosis field shows the date, when diagnosis was changed.

Type of current location

The Type of current location field shows the value of the field Current Location of Patient at the time of notification, which specifies the location of the patient.

Current location name

The Current location name field shows the name or address of the current location at the time of notification.

Additional Information

In the Additional Information field has additional important data is shown.

Samples tab is used to observe the list of samples collected for the cases. All samples have check-boxes. If the check-box is selected, then the samples will be added to/left in the samples of the survivor case. If the sample was accessioned in the Laboratory Module, then it is not possible to un-check the selected check-box.

Fig. 231 Case De-Duplication Samples tab

Field/Variable

Description

Sample ID

Sample ID field is filled in automatically after the test is assigned to the sample in the Laboratory Module.

Sample Type

Sample Type field defines the type of the collected sample.

Date of Collection

Collection Date field holds data about the date of the sample’s collection.

Number of Tests

Number of Tests field represents how many tests were assigned to the sample.
