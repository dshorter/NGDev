Employees/Users

Show/Hide Hidden Text

All the personnel involved in case investigation in human, veterinary and vector surveillance is recorded in EIDSS database.

View the list of employees/users:

1.In the System menu, click Administration and then click Employees. Employees List is opened.

Fig. 114 Employees List

 

Create a new employee/user:

1.Click the New button. The Employee Details window opens.

2.In the Personal Information tab, enter the names of the employee and his/her working position details.

Fig. 115 Employee Details - Personal Information

3.If it is planned for the employee to be a user of the system, click the Login tab. The Login tab opens.

Fig. 116 Employee Accounts

4.The Login tab contains the list of user accounts of the employee on different sites. Initially it is empty.

5.To add a new account, click the New button. The User Login window opens.

Fig. 117 Employe account details

6.Enter the employee's User Login (5 or more symbols), and Password. The password must meet the requirements, set in the Security Policy.

7.Repeat the password in the Confirm New Password field.

8.Click Ok to save the changes and to close the window.

9.Repeat steps 5 through 8 to add all the required accounts.

10.Save the data and close the Employee Details window.
Assign an employee to groups:

1.In the Employee Details window open the Groups tab.

Fig. 118 Employee Details - Groups tab

2.Click the Add button. The User Groups List window opens.

Fig. 119 User Groups List

3.Select a group from the list which should be assigned to the user and click Select.

4.The User Groups List window closes and the selected group is displayed on the Employee Details window. The access rights assigned to the group, to which the user was added, are now assigned to the user.

5.Click Save to save any performed changes.
Assign system functions to the user:

1.In the Employee Details window open the System functions tab.

Fig. 120 Employee Details - System functions tab

2.Scroll through the system functions and choose the Deny either Allow checkbox to either deny or allow the appropriate function.

3.Click Save to save any performed changes
Edit a user or an employee:

1.Select the user to edit.

2.Click the Edit button. The Employee Details window opens.

3.In the Personal Information tab, enter the names of the employee and his/her working position details.

Fig. 121 Employee Details - Personal Information

4.If it is planned for the employee to be a user of the system, click the Login tab. The Login tab opens.

Fig. 122 Employee Accounts

5.The Login tab contains the list of user accounts of the employee in different sites.

6.To add a new account, click the New button. The User Login window opens.

Fig. 123 Employe account details

7.Enter the employee's Site, User Login, which must be 5 or more symbols, and Password. The password must meet the requirements, set in the Security Policy.

8.Repeat the password in the Confirm New Password field. If the employee's login name is changed, password can be kept unchanged.

9.Click Ok to save the changes and to close the window.

10.Repeat steps 5 through 9 to add all the required accounts.

11.To delete, select an account and click the Delete button. Click Ok to confirm the deletion.

12.Save the data and close the Employee Details window.
Delete a user or an employee:

1.Select the employee to delete.

2.Click Delete button. The Delete Record dialog opens.

Fig. 124 Confirm Deletion

3.Click Ok to confirm the deletion.
