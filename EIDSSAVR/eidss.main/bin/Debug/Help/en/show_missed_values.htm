Show missed values

Some charts may need all cells to be populated even for those field values, for which data is not registered in the system. In this case, missed values can be added:

•Missing values in the source data for all date fields of the pivot grid (date, year, month, quarter, week);

For example, a report is required to show the number of cases for each month of a year. If no missed values are shown, only months with the entered cases registered in the system shall be displayed. However, if all months are required in the report, missed values have to be used for the date field. The added place holding data that is not real does not affect the database, and can be removed at any moment.

Missed values are added in 2 steps: first the place holding record needs to be created in the AVR database. After this, the field is marked as the one having the missed values, but the records are not shown in the pivot grid. For them to be shown in the pivot grid, the Show Missed Values in Pivot Grid check-box has to be selected. Any pivot grid change clears the check-box and thus hides the missed values.

Add missed values for a date field

1.Right-click the field that requires missed values.

2.From the drop-down list select Add Missed Values. The Date Diapason window opens.

3.Enter the dates range for the missed dates to be added.

4.Click OK. The Date Diapason window closes. The field is marked with a red border but no missed values appear in the grid yet.

Fig. 551 Marked field

 

5.Select the Show Missed Values in Pivot Grid check-box. The values will be added, and the grid will be recalculated.

Remove missed values

1.Right-click on the field missed values for those are going to be removed.

2.From the drop-down list select Remove Missed Values.
