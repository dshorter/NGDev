Репликация данных

ЭИСНЗ является распределённой системой, а её данные дублируются между сайтами ЭИСНЗ верхнего и нижнего уровней. Информация, введённая на сайте уровня санитарно-контрольного поста, дублируется на соответствующие сайты уровня станций эпидемиологического / эпизоотического мониторинга, а затем добавляется в Центральное хранилище данных. Данные с веб-сервера идут напрямую в ЦХД. Помимо этого, система поддерживает и обратное движение данных.

Архитектура связи системы обеспечивает высоконадёжную передачу данных посредством многоканального соединения (один основной канал связи дублируется несколькими отказоустойчивыми резервными каналами).

Рис. 39 Сайты ЭИСНЗ
(ЦХД) - Центральное хранилище данных; (ЦСЛ) - Центральная справочная лаборатория;
(СЭМ) - Станция эпидемиологического мониторинга; (Лаб) - Лаборатория;
(СКП) - Санитарно-контрольный пост; (АРМ) - Автоматическое рабочее место без установленной системы ЭИСНЗ.

В распределённой систем данные передаются между сайтами различных уровней. Репликация работает «вверх», а это значит, что для передачи данных между двумя связанными сайтами, процедуру репликации должен запустить сайт более нижнего уровня. Некоторая информация распределяется между сайтами, находящимися в рамках одного административного района.

Периодическая автоматическая репликация

Периодически, в соответствии с определённым графиком, происходит репликация всех данных.

Запускаемая пользователем репликация (вручную)

Для запуска процесса репликации вручную, пользователь должен войти в меню Система, и нажать пункт Репликация данных.

Срочная репликация

Ниже приведён список системных событий, вызывающих начало автоматической репликации данных:

•создана запись о новом случае заболевания,

•изменилась классификация случая заболевания,

•изменился диагноз случая заболевания,

•статус случая заболевания изменился на любой с «Закрыт»,

•создана запись о новой вспышке заболевания,

•к вспышке заболевания добавлен случай заболевания,

•изменился основной случай заболевания вспышки заболевания,

•сохранён тест с окончательным диагнозом,

•внесены изменения в тест.

Репликация начинается сразу же после нажатия кнопок Сохранить или ОК на соответствующих формах.
