Вложенная вкладка «Лабораторные тесты» формы W02 «Сессия надзора за переносчиками заболевания»

Вложенная вкладка лабораторные тесты формы сессия надзора за переносчиками заболевания используется для просмотра списка лабораторных тестов, проведённых в рамках сессии в лабораторном модуле.

Рис. 265 Вкладка «Лабораторные тесты»

Поля данных

Идентификатор группы переносчиков / отдельного переносчика заболевания

Указывается идентификатор группы переносчиков / отдельного переносчика заболевания, используемых(-ого) для проведения теста.

Тип переносчика заболевания

Указывается тип переносчика заболевания, использовавшегося для проведения теста.

Биологический вид

Указывается биологический вид переносчика заболевания, использовавшегося для проведения теста.

Лабораторный идентификатор пробы

Указывается лабораторный идентификатор пробы, использовавшейся для проведения теста. Лабораторный идентификатор пробы создаётся автоматически, в соответствии со Схемой уникальной нумерации, когда происходит приём пробы в лабораторном модуле.

Полевой идентификатор пробы

Указывается полевой идентификатор пробы, используемой для проведения теста.

Тип пробы

Указывается тип пробы, используемой для проведения теста.

Дата взятия

Указывается дата взятия пробы, используемой для проведения теста.

Название теста

Указывается название теста.

Категория теста

Отображается категория теста, если известна.

Дата результата

Вводится дата начала теста или ввода его результата.

Тестировавший работник

Указывается имя работника, вводившего результат теста.

Тестировавшее учреждение

Указывается проводившая тест организация.

Результат теста

Указывается результат теста.

Заболевание

Указывается заболевание, если его для такого теста указали в поле Проверяемый тестом диагноз в лабораторном модуле. При положительном результате, информация о заболевании копируется из поля Заболевания сессии.

Кнопки

Просмотр сведений о группе переносчиков / отдельном переносчике заболевания

Открывает форму Сведения о группе переносчиков / отдельном переносчике заболевания, в которой отображается информация о родительской группе переносчиков / отдельном переносчике заболевания, использовавшихся для полевого теста.
