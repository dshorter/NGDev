Связывание сессии НЗПЗ со вспышкой заболевания;

В некоторых случаях, сессии надзора за переносчиками заболеваний связаны со вспышками заболеваний. Если вспышка заболевания послужила поводом для проведении сессии, будет целесообразно отобразить это в ЭИСНЗ.

 

1.Находясь в меню Журналы, выберите пункт Сессия надзора за переносчиками заболеваний. Откроется окно Список сессий надзора за переносчиками заболеваний.

2.Найдите нужную сессию НЗПЗ. Для получения более подробного описания процесса нахождения сессии, прочтите раздел Поиск сессии НЗПЗ.

3.Проделайте одно из следующих действий:

•Дважды нажмите мышкой на предназначенной для редактирования сессии.

•Отметьте сессию для редактирования и нажмите кнопку Редактировать.

Такая сессия откроется в форме Сессия надзора за переносчиками заболевания и будет готова для редактирования.

4.В поле Идентификатор вспышки заболевания либо введите номер вспышки заболевания вручную, либо нажмите кнопку Список (), чтобы выбрать вспышку заболевания из поля Список вспышек заболеваний. Для получения более подробного описания процесса нахождения нужной вспышки заболевания, прочтите раздел Поиск вспышки заболевания.

5.Для сохранения идентификатора вспышки заболевания, нажмите кнопку Сохранить.

 

Для удаления связи между сессией и вспышкой заболевания, удалите запись о вспышке заболевания из поля Идентификатор вспышки заболевания сессии.
