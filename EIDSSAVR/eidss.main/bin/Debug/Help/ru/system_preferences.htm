Системные настройки

☝Настоятельно рекомендуется, чтобы конфигурацию «Системных настроек» проводил исключительно системный администратор.

Находясь в меню Система, нажмите пункт Системные настройки. Откроется окно Системные настройки.

Рис. 18  Окно «Опции»

Поле / переменное значение

Описание

Рабочий язык

Язык по умолчанию у приложения после его запуска.

Страна

Страна установки.

Принтер штрихкодов

Принтер по умолчанию для распечатки штрихкодовых наклеек.

Принтер документов

Принтер по умолчанию для распечатки документов.

Путь EPI Info

Путь к установленному программному обеспечению EPI Info.

Проект карты по умолчанию

Проект географической карты, по умолчанию открываемый в модуле АВО для указания местоположения случая заболевания. Наименования всех проектов географических карт, сохраняемых в ЭИСНЗ, отображаются в раскрывающемся списке.

Показ текстовых подсказок на панели инструментов

Выбирается для отображения названий кнопок в панели инструментов главного меню.

Показ предупреждения при загрузке большого макета

Появление на экране предупреждающего сообщения с кнопкой ОК при загрузке большого макета.

Показ подсказки на сохранение данных

Выбирается для включения показа диалогового окна с запросом на продолжение после закрытия любого окна.

Показ навигатора в форме H02

Выбирается для отображения панели навигации формы «Сообщение о случае заболевания человека».

Показ символа * для пустых значений в модуле АВО

Выбирается для отображения пустых значений данных АВО в виде символа «*».

Упрощённый режим лабораторного модуля

Выбирается для включения упрощённого режима лабораторного модуля (только лабораторный раздел); либо отключается, чтобы иметь иные способы доступа к функциям лабораторного модуля.

Использование кэшированных данные в АВО

Выбирается для автоматического обновления хранения данных АВО.

Область по умолчанию в панели поиска

Выбирается для автоматического заполнения поля Область в поисковых формах.

Печать географической карты в форме расследования случая заболевания животных

Выбирается для печати географической карты, когда печатается бумажная форма расследования случая заболевания животных.

Количество дней, за которые данные отображаются по умолчанию

Выбирается для отображения данных по умолчанию исключительно за определённый период.
