Вкладка «Общая информация» формы L04 «Сведения о лабораторном тесте»

Вкладка Общая информация формы Сведения о лабораторном тесте используется для просмотра и редактирования основных данных теста. Статус теста определяет доступность полей на вкладке Общая информация.

Рис. 341 Вкладка «Общая информация»

Название теста

Отображается наименование теста, которое невозможно изменить.

Категория

Отображается категория теста, если она известна, которую невозможно изменить.

Проверяемый тестом диагноз

Отображается диагноз теста, подлежащий подтверждению или отклонению, который невозможно изменить.

Статус

Выбирается для теста из раскрывающегося списка. Значение поля может меняться для тестов со статусами: «Не начат», «В процессе» и «Предварительный». Статус определяет доступность полей теста, а также то, что с ним можно делать в дальнейшем. Для получения более подробной информации, прочитайте раздел Статусы теста.

Результат

Отображается результат теста, если таковой был получен. Это поле является доступным тогда, когда статус теста меняется на «В процессе», «Предварительный» или «Окончательный», но только до первого сохранения. Для получения более подробной информации, прочитайте раздел Ввод результата теста.

Комментарии

Указываются любые необходимые комментарии. Это поле является доступным для редактирования тогда, когда статус теста меняется на «В процессе», «Предварительный» или «Окончательный», но только до первого сохранения.

Тестировавший работник

Используется для указания наименования тестировавшего лица, имя которого выбирается из раскрывающегося списка. Это поле является доступным тогда, когда статус теста меняется на «В процессе», «Предварительный» или «Окончательный», но только до первого сохранения.

Оператор ввода результатов

Автоматически отображается имя лица, вводившего результаты теста, которое нельзя изменить вручную.

Дата начала теста

Используется для указания даты начала проведения теста. Это поле является доступным тогда, когда статус теста меняется на «В процессе», «Предварительный» или «Окончательный», но только до первого сохранения.

Дата результата

Указывается дата, в которую был получен результат теста. Это поле является доступным тогда, когда статус теста меняется на «В процессе», «Предварительный» или «Окончательный», но только до первого сохранения.

Утвердившее лицо

Указывается имя лица, подтвердившего результат теста. Это поле является доступным тогда, когда статус теста меняется на «В процессе», «Предварительный» или «Окончательный», но только до первого сохранения.

Идентификатор группы

Отображается идентификатор группы, если тест является частью какой-либо из групп. Для открытия формы группы с целью просмотра сведений, нажмите кнопку Сведения ( ). Необходимая форма откроется в новом окне.

Лаборатория

Отображается название лаборатории, в которой проводился тест.

Идентификатор случая заболевания

Отображается идентификатор случая заболевания теста пробы, относящейся к медицинским или ветеринарным случаям заболеваний. Для открытия формы случая заболевания, нажмите кнопку Сведения (). Соответствующая ветеринарная или медицинская форма откроется в новом окне.

Идентификатор сессии

Отображается идентификатор сессии, относящейся к тесту пробы, связанной с сессией активного надзора (АН) или сессией надзора за переносчиками заболевания (НЗПЗ). Для открытия формы сессии, нажмите кнопку Сведения ( ). Форма сессии откроется в новом окне.
