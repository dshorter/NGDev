Операторы сравнения

Оператор сравнения является частью условия фильтрации и используется для выбора применяемого логического оператора.

Рис. 381 Оператор сравнения

Для выбора другого логического оператора раскрывающегося меню, нажмите на пункт оператора сравнения.

 

Рис. 382 Раскрывающееся меню операторов сравнения

В следующей таблице перечислены доступные для условий фильтрации операторы сравнения.

Оператор сравнения

Описание

Применение, исключения, примеры

РАВНО

Оператор, извлекающий значения, равные указанному значению.

Применяется для всех типов полей АВО.

Исключение:
Оператор Равно не может работать с полями «Дата» и «Время» (Дата регистрации, Дата последнего сохранения, и т.д.), в то время как поля данных обрабатываются правильно.

Для работы с полями «Дата» и «Время», замените оператор Равно набором из двух условий:

И

[Поле даты АВО] Больше или равно [Указанной даты]

[Поле даты АВО] Меньше [Даты, следующей за указанной]

НЕ РАВНО

Оператор, извлекающий значения, не равные указанному значению.

Применяется для всех типов полей АВО.

Исключение:
Оператор Не равно не может работать с полями «Дата» и «Время» (Дата регистрации, Дата последнего сохранения, и т.д.), в то время как поля данных обрабатываются правильно.

Для работы с полями «Дата» и «Время», замените оператор НЕ РАВНО набором из двух условий:

И

[Поле даты АВО] Больше или равно [Даты, следующей за указанной датой]

[Поле даты АВО] Меньше [Указанной даты]

БОЛЬШЕ

Оператор, извлекающий значения, превышающие указанное значение (в алфавитном порядке для текстовых полей).

Применяется для всех типов полей АВО, за исключением текстовых полей, связанных со справочными значениями ЭИСНЗ (напр., «Диагноз», «Тип пробы», «Название теста», и т.д.) или списками ( «Организации», «Сайты ЭИСНЗ»).

БОЛЬШЕ ИЛИ РАВНО

Оператор, извлекающий значения, превышающие или равные указанному значению (в алфавитном порядке для текстовых полей).

Применяется для всех типов полей АВО, за исключением текстовых полей, связанных со справочными значениями ЭИСНЗ.

СОДЕРЖИТ

Оператор, извлекающий значения, в которых содержится указанный текст.

Применяется исключительно для текстовых полей АВО, кроме текстовых полей, связанных со справочными значениями ЭИСНЗ или списками.

НЕ СОДЕРЖИТ

Оператор, извлекающий значения, в которых не содержится указанный текст.

Применяется исключительно для текстовых полей АВО, кроме текстовых полей, связанных со справочными значениями ЭИСНЗ или списками.

НАЧИНАЕТСЯ С

Оператор, извлекающий значения, начинающиеся с указанного текста.

Применяется исключительно для текстовых полей АВО, кроме текстовых полей, связанных со справочными значениями ЭИСНЗ или списками.

ЗАКАНЧИВАЕТСЯ НА

Оператор, извлекающий значения, заканчивающиеся указанным текстом.

Применяется исключительно для текстовых полей АВО, кроме текстовых полей, связанных со справочными значениями ЭИСНЗ или списками.

СООТВЕТСТВУЕТ МАСКЕ

Оператор, извлекающий значения, аналогичные указанному тексту. Текст может включать символ %, трактующийся как любое строковое значение.

Например, условие [поле АВО] является подобным «AB%D» извлекает такие значения, как «AB[Любое строковое значение]D» («ABXXXXD», «AB12345…D», и т.д.).

Применяется исключительно для текстовых полей АВО, кроме текстовых полей, связанных со справочными значениями ЭИСНЗ или списками.

НЕ СООТВЕТСТВУЕТ МАСКЕ

Оператор, извлекающий значения, не соответствующие указанному тексту. Текст может включать символ %, трактующийся как любое строковое значение.

Например, условие [поле АВО] не является подобным «AB%D» извлекает любые значения, за исключением таких значений, как «AB[Любое строковое значение]D» («ABXXXXD», «AB12345…D», и т.д.).

Применяется исключительно для текстовых полей АВО, кроме текстовых полей, связанных со справочными значениями ЭИСНЗ или списками.

ПУСТО

Оператор, извлекающий нулевое или пустое строковое значение.

Применяется для всех типов полей АВО.

НЕ ПУСТО

Оператор, извлекающий все строковые значения, за исключением нулевых или пустых.

Применяется для всех типов полей АВО.

СЕГОДНЯ

Оператор, извлекающий значения даты, равные сегодняшней.

Результат такого фильтра зависит от дня исполнения или открытия запроса АВО и/или макета, соответственно.

Применяется исключительно для полей АВО.

ВЧЕРА

Оператор, извлекающий значения даты, равные вчерашней.

Результат такого фильтра зависит от дня исполнения или открытия запроса АВО и/или макета, соответственно.

Применяется исключительно для полей АВО.

РАНЕЕ НА ЭТОЙ НЕДЕЛЕ

Оператор, извлекающий значения даты, соответствующие следующим критериям:

Первый день этой недели ≤ дата < вчера

Результат такого фильтра зависит от дня исполнения или открытия запроса АВО и/или макета, соответственно.

Применяется исключительно для полей АВО.

НА ПРЕДЫДУЩЕЙ НЕДЕЛЕ

Оператор, извлекающий значения даты, соответствующие следующим критериям:

Первый день последней прошедшей недели ≤ дата < первый день этой недели

Результат такого фильтра зависит от дня исполнения или открытия запроса АВО и/или макета, соответственно.

Применяется исключительно для полей АВО.

РАНЕЕ В ЭТОМ МЕСЯЦЕ

Оператор, извлекающий значения даты, соответствующие следующим критериям:

Первый день этого месяца ≤ дата < первый день последней прошедшей недели

Результат такого фильтра зависит от дня исполнения или открытия запроса АВО и/или макета, соответственно.

Применяется исключительно для полей АВО.

РАНЕЕ В ЭТОМ ГОДУ

Оператор, извлекающий значения даты, соответствующие следующим критериям:

Первый день этого года ≤ дата < первый день этого месяца

Результат такого фильтра зависит от дня исполнения или открытия запроса АВО и/или макета, соответственно.

Применяется исключительно для полей АВО.

В ПРЕДЫДУЩЕМ ГОДУ

Оператор, извлекающий значения даты, соответствующие следующим критериям:

Дата < первый день этого года

Результат такого фильтра зависит от дня исполнения или открытия запроса АВО и/или макета, соответственно.

Применяется исключительно для полей АВО.

 

Вследствие специального использования дат со значениями времени, следующий фильтр подходит для извлечения всех записей в течение определённого периода:

 

И

[Поле даты в АВО] больше или равно значения [Дата начала]

[Поле даты в АВО] меньше значения [Наступающий после даты окончания день]

 

Просьба иметь в виду, что подобных фильтрах необходимо использовать значение дня, наступающего после даты окончания периода.
