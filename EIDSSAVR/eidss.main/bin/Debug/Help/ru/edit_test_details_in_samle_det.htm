Просмотр и редактирование сведений о тесте в форме «Проба» (ОФ)

Настоящая операция доступна только в случае отсутствия включённого флажка на кнопке с фиксацией Упрощённый режим лабораторного модуля, расположенной в окне Системные настройки.

 

1.Находясь в меню Журналы, выберите пункт Пробы. Откроется окно Список проб.

2.Найдите и дважды нажмите мышкой по записям о пробе, относящейся к соответствующему тесту. Откроется окно Проба.

3.Нажмите на вкладке Тесты, чтобы открыть её. Вкладка «Тесты» содержит список всех тестов пробы: как назначенных, так уже проведённых.

Рис. 333 Вкладка «Тесты»

4.Добавьте тест в список тестов, назначенных для пробы, нажав кнопку Выбрать тест. Повторите это действие для всех необходимых тестов.

5.Тесты из списка удаляются нажатием кнопки Удалить тест.

6.Отредактируйте имеющиеся сведения о тестах. Для получения более подробной информации о форме «Проба» и её вкладках, прочитайте раздел Форма L02 «Проба».

7.Нажмите кнопку Сохранить и подтвердите сохранение данных.

8.Для закрытия окна Список проб, нажмите кнопку ОК.
