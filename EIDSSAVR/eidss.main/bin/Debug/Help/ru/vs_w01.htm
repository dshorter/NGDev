Форма W01 «Список сессий надзора за переносчиками заболеваний»

Форма Список сессий надзора за переносчиками заболеваний предназначена для поиска, просмотра, редактирования, создания и удаления сессии надзора за переносчиками заболеваний.

Рис. 259 Окно «Список сессий надзора за переносчиками заболеваний»

Поля критериев поиска

Идентификатор сессии

Вводится полный идентификатор разыскиваемой сессии, если он известен.

Полевой идентификатор сессии

Вводится полный полевой идентификатор разыскиваемой сессии, если он известен.

Статус

Выбирается статус разыскиваемой(-ых) сессии(-ий).

Тип переносчика заболевания

Выбирается тип разыскиваемого(-ых) переносчика(-ов) заболевания, отловленного(-ых) в рамках сессий.

Биологический вид

Выбирается биологический вид разыскиваемого(-ых) переносчика(-ов) заболевания, отловленного(-ых) в рамках сессий. Это поле является активным только в случае выбора поля Тип переносчика заболевания.

Диагнозы

Выбирается диагноз, подтверждённый разыскиваемой(-ыми) сессией(-иями).

Группа диагнозов

Здесь выбирается группа, если необходимо найти сессию одной группы, но с разными диагнозами.

Область

Выбирается область разыскиваемой(-ых) сессии(-ий).

Район

Выбирается район разыскиваемой(-ых) сессии(-ий). Это поле является активным только в случае выбора поля Область.

Населённый пункт

Выбирается населённый пункт, в котором проводилась(-ись) сессия(-ии). Это поле является активным только в случае выбора полей Область и Район.

Географическая долгота

Вводятся координаты долготы того места, где проводилась(-ись) сессия(-ии).

Географическая широта

Вводятся координаты широты того места, где проводилась(-ись) сессия(-ии).

Дата начала: с / по

Указывается временной интервал начала проведения сессии(-ий).

Дата закрытия: с / по

Указывается временной интервал окончания проведения сессии(-ий).

Идентификатор вспышки заболевания

Выбирается вспышка заболевания, с которой связана(-ы) соответствующая(-ие) сессия(-и).

Сайт ввода данных

Выбирается сайт, на котором была(-и) зарегистрирована(-ы) сессия(-и).

Столбцы таблицы с результатами поиска

Идентификатор сессии

Отображается идентификатор сессии надзора за переносчиками заболевания.

Статус

Отображается статус сессии.

Типы переносчиков заболеваний

Отображаются типы переносчиков заболеваний, отловленных в ходе сессии, названия которых разделены знаком запятой (,).

Заболевания

Отображаются заболевания, подтверждённые результатами сессии, названия которых разделены знаком запятой (,).

Область

Отображается название области места проведения сессии.

Район

Отображается название района места проведения сессии.

Географическая долгота

Отображаются координаты долготы места проведения сессии.

Географическая широта

Отображаются координаты широты места проведения сессии.

Дата начала

Отображается дата начала сессии.

Дата закрытия

Отображается дата, в которую статус сессии поменялся на Закрыта.

Кнопки

Новая

Нажатие создаёт новую сессию надзора за переносчиками заболевания. Для получения более подробной информации, прочитайте раздел Создание сессии надзора за переносчиками заболеваний (НЗПЗ).

Редактировать

Открывает выбранную сессию для редактирования. Для получения более подробной информации, прочитайте раздел Редактирование сессии НЗПЗ.

Удалить

Удаляет выбранную сессию. Для получения более подробной информации, прочитайте раздел Удаление сессии НЗПЗ.

Обновить

Нажатие обновляет список отображаемых сессий.

Закрыть

Закрывает окно Сессия надзора за переносчиками заболевания.
