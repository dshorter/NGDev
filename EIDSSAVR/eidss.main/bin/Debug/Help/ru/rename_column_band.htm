Переименование столбца / группы столбцов

Расположенные на вкладке Вид представления данных макета столбцы и группы столбцов можно переименовывать.

1.Нажмите правой кнопкой мыши на столбце или группе столбцов и в раскрывающимся меню выберите пункт либо Переименовать столбец, либо Переименовать группу столбцов.

Рис. 434 Переименование столбца

2.Проделайте одно из следующих действий:

•Если откроется окно Переименовать столбец. В поле Новое имя столбца введите новое наименование.

•Если откроется окно Переименовать группу столбцов. В поле Новое имя группы столбцов введите новое наименование.

Рис. 435 Окно переименования столбца

3.Нажмите ОК. Окно закроется. Столбец или группа столбцов получит новое наименование.
