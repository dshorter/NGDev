Опция расширенного поиска в лабораторном модуле

Расширенный поиск представляет собой инструмент фильтрации данных, используемый для извлечения из базы данных исключительно тех записей, которые соответствуют набору введённых критериев поиска. Для ввода критериев поиска используется инструмент «Панель поиска».

Расширенный поиск может использоваться как в отдельных формах, так и в лабораторном разделе.

Рис. 301 Панель поиска лабораторного раздела

1.Откройте списочную форму, используемую для отображения необходимых записей (форму Список проб для проб, форму Список лабораторных тестов для тестов, и т.д.).

2.В полях инструмента «Панель поиска» укажите критерии фильтрации. Каждая списочная форма использует свой уникальный набор критериев выбора.

3.Нажмите кнопку Найти (). На экране, в виде списочной формы, отобразятся только соответствующие критериям фильтрации записи.

 

В форме Лабораторного раздела список результатов найденных поиском лабораторных записей можно отфильтровать дополнительно: либо средствами простого поиска, либо вручную. Для получения более подробной информации о простом поиске, прочитайте раздел Опция простого поиска в лабораторном разделе. Для фильтрации вручную, выберите необходимые для удаления из списка записи, нажмите правой кнопкой мыши на этих записях, и из раскрывающегося меню выберите пункт Удалить. Из списка можно удалять только сохранённые записи.

Для получения дополнительной информации об инструменте «Панель поиска», прочитайте раздел Списки поиска.
