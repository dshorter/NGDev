Применение функций агрегирования

В сводной таблице можно выбирать способ отображения данных: в виде количества значений, в виде самого большого значения, и т.д. Функцией агрегирования называется функция, осуществляющая вычисления применительно к набору значений, а не к одному значению. Например, нахождение среднего или наименьшего значения в списке значений представляет собой функцию агрегирования.

По умолчанию каждое поле в ЭИСНЗ обладает применённой к нему функцией агрегирования. Если функция агрегирования по умолчанию не соответствуют предъявляемым требованиям, то либо измените функцию агрегирования по умолчанию (это следует сделать, если новая функция агрегирования будет применяться в большинстве случаев), либо измените функцию агрегирования так, чтобы она применялась здесь и сейчас.

Изменение функции агрегирования

1.Выберите в области данных те данные, для которых требуется поменять функцию агрегирования.

2.Убедитесь, что необходимое поле отображается в поле только для чтения Поля данных окна Настройки сводной таблицы.

3.В поле Функция агрегирования выберите функцию агрегирования, которая будет использоваться вместо старой.

4.В поле Точность укажите разрядность отображаемого значения.

5.В случае использования функции коэффициентного количества, в поле Параметры: административная единица выберите административную единицу, которая будет использоваться в расчётах для статистики.©

Среднее арифметическое значение, часто просто называемое как «среднее», представляет собой среднюю величину между двумя числами; т.е., например, среднее чисел 2 и 8 рассчитывается путём нахождения значения А, при этом 2 + 8 должны быть равны А + А. Значение А рассчитывается по формуле: А = (2 + 8)/2 = 5.

Если же увеличивается количество чисел в списке, среднее значение которого необходимо узнать, т.е., например, надо узнать среднее арифметическое чисел 2, 8, и 11, его рассчитывают с помощью уравнения, в котором значение А в уравнении A + A + A должно быть равным 2 + 8 + 11. Соответственно, значение А = (2 + 8)/2 = 7.
