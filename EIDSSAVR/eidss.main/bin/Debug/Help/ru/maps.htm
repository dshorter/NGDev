Географические расположения

Show/Hide Hidden Text

Основной целью модуля ГИС в ЭИСНЗ является динамическое визуальное картографическое отображение данных надзора, получаемых из информации, хранящейся в распределённой базе данных ЭИСНЗ.

Для указания местоположения созданного случая заболевания, фермы, больницы и т.д., можно использовать географическую карту. Как правило, для открытия географической карты используется нажатие кнопки Географическая карта ().

Рис. 34 Пример географической карты

Созданный случай заболевания можно отобразить на карте тремя способами:

•указав область, район и населённый пункт;

•указав точное местоположение посредством ввода данных географической долготы и широты;

•вручную проставив случай заболевания на карту.

Указание области, района и населённого пункта

1.Нажмите кнопку Географическая карта ().

2.Откроется окно Указать местоположение и на карте отобразится новый случай заболевания.

Рис. 35 Указание области, района и населённого пункта

3.Случай заболевания будет отображаться в виде красной точки, и располагаться на территории населённого пункта.

4.Нажмите кнопку OK для сохранения параметров нового случая заболевания.

Указание точного местоположения посредством ввода данных географической долготы и широты

1.Укажите географическую долготу и широту.

2.Нажмите кнопку Географическая карта ().

3.Откроется окно Указать местоположение и на карте отобразится новый случай заболевания.

Рис. 36 Точное место

4.Случай заболевания отображается в виде красной точки, а его местоположение указывается в окне географическая долгота / широта.

5.Нажмите кнопку OK для сохранения параметров нового случая заболевания.

Указание случая заболевания на географической карте вручную

1.Нажмите кнопку Географическая карта ().

2.Откроется окно «Указать местоположение»:

Рис. 37 Указание случая заболевания на географической карте вручную

3.Для размещения случая заболевания на карте нажмите курсором мыши на соответствующей точке. На карте появится красная точка, установленная пользователем вручную.

4.Нажмите кнопку OK для сохранения параметров нового случая заболевания.
