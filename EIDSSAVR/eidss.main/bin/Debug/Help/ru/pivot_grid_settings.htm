Настройки сводной таблицы

Вкладка Настройки сводной таблицы содержит поля и параметры, необходимые для настройки сводной таблицы.

Рис. 460 Окно настройки сводной таблицы

Применить фильтры сводной таблицы

ВКЛ. - к сводной таблице применяются фильтры, созданные кнопкой Редактировать фильтры.

ВЫКЛ. - к сводной таблице не применяются фильтры, созданные кнопкой Редактировать фильтры.

Для получения более подробной информации о фильтрах макета, прочитайте раздел Применение фильтров.

Редактировать фильтры

Нажатие даёт возможность создавать или редактировать фильтры макета. Чтобы применить созданный фильтр, активируйте флажок на кнопке с фиксацией Применить фильтры сводной таблицы.

Показать данные в сводной таблице

Нажатие отображает данные в сводной таблице. На время создания и настройки сводной таблицы настоятельно рекомендуется отключать эту кнопку с фиксацией. В противном случае, любое изменение в сводной таблице будет приводить к пересчитыванию отображаемых данных, что займёт продолжительное время. Для получения более подробных сведений, прочитайте раздел Отображение данных в сводной таблице.

ВКЛ. - в сводной таблице отображаются данные.

ВЫКЛ. - сводная таблица не содержит отображаемых данных.

Обновить данные

Служит для извлечения самых свежих данных из базы данных ЭИСНЗ. Настоятельно рекомендуется проводить эту операцию только в случае реальной необходимости в таких данных, поскольку её выполнение требует значительного времени. Для получения более подробных сведений, прочитайте раздел Обновление данных.

Включить компактный вид сводной таблицы

Эта кнопка уменьшает заголовки строк сводной таблицы, чтобы увеличить свободное пространство для данных такой таблицы. Для получения более подробных сведений, прочитайте раздел Компактный вид сводной таблицы.

Закрепить заголовки строк

Предотвращения скрытию заголовков строк, вызванному горизонтальным прокручиванием. Для получения более подробных сведений, прочитайте раздел Закрепление заголовков строк в сводной таблице.

Группировать по дате по умолчанию

Используется для выбора группирования по умолчанию всех дат в сводной таблице. Для получения более подробной информации, прочитайте раздел Функция группировки по дате.

Показать итоговые значения

Используется для выбора итоговых значений сводной таблицы: общие итоговые значения строк или столбцов, промежуточные итоговые значения и т.д.

Показать недостающие значения в сводной таблице

Служит для добавления столбцов и строк с нулевыми значениями для дат, по которым нет данных. Для получения более подробной информации, прочитайте раздел Показ недостающих значений.

Функция агрегирования

Выберите функцию агрегирования для её применения к выбранному полю.  Для получения более подробных сведений, прочитайте раздел Применение функций агрегирования.

Точность

Используется для указания разрядной точности выбранного поля применения функции агрегирования. Для получения более подробных сведений Настройка точности функций по умолчанию и Применение функций агрегирования.

Параметры: поле административной единицы

Применяется для указания административной единицы для статистики, которая будет использоваться при расчёте коэффициентного соотношения. Оно доступно только в случае использования функций коэффициентного агрегирования данных.

Поле даты

Настоящий раскрывающийся список используется для просмотра всех полей данных, сгруппированных способом, отличающимся от режима по умолчанию. Для получения более подробных сведений, прочитайте раздел Функция группировки по дате.
