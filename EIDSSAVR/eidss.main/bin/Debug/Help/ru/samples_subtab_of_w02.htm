Вложенная вкладка «Пробы» формы W02 «Сессия надзора за переносчиками заболевания»

Вложенная вкладка Пробы используется для просмотра проб, взятых в ходе сессии надзора за переносчиками заболеваний. Данные обо всех пробах, взятых у каждого отдельного переносчика / группы переносчиков заболевания в рамках сессии, комбинируются для построения списка проб сессии. Таблица, содержащая список проб, является нередактируемой. Для получения более подробной информации о пробах отдельного переносчика / группы переносчиков заболевания, прочитайте раздел Регистрация проб отдельного переносчика / группы переносчиков заболевания.

Рис. 263 Вкладка «Пробы»

Поля данных

Идентификатор группы переносчиков / отдельного переносчика заболевания

Отображается идентификатор, создаваемый автоматически в соответствии со Схемой уникальной нумерации.

Тип переносчика заболевания

Тип переносчика заболевания, использовавшийся для взятия пробы.

Биологический вид

Указывается биологический вид переносчик(-ов) заболевания.

Лабораторный идентификатор пробы

Отображается лабораторный идентификатор пробы, присваиваемый автоматически, в соответствии со Схемой уникальной нумерации.

Полевой идентификатор пробы

Указывается идентификатор пробы, использовавшийся до регистрации такой пробы в системе ЭИСНЗ.

Тип пробы

Указывается тип пробы.

Дата взятия

Указывается дата взятия материала.

Учреждение-сборщик

Указывается организация, взявшая материал для пробы.

Комментарий

Указывается комментарий для пробы, который вводится в лабораторном модуле.

Дата приёма

Указывается дата приёма пробы в лабораторный модуль.

Состояние пробы при получении

Указывается состояние пробы при её приёме в лабораторный модуль.

Кнопки

Просмотр сведений о группе переносчиков / отдельном переносчике заболевания

Открывает форму «Сведения о группе переносчиков / отдельном переносчике заболевания», в которой отображается информация о родительской группе переносчиков / отдельном переносчике заболевания пробы.
