Матрица «Группа диагнозов - диагноз»

Матрица «Группа диагнозов - диагноз» является инструментом, предназначенным для составления списка единичных диагнозов для группы диагнозов.

Для редактирования матрицы необходимо:

1.Находясь в меню Система, нажмите пункт Настройка, а затем нажмите Матрица «Группа диагнозов - диагноз». Откроется окно Матрица «Группа диагнозов - диагноз».

Рис. 89 Матрица «Группа диагнозов - диагноз» 

2.В поле Группа диагнозов выберите группу диагнозов, для которой будут указаны диагнозы, или добавьте её, нажав кнопку . Ранее добавленные группы диагнозов также будут отображаться.

3.Для добавления нового диагноза, нажмите строку Нажать для добавления новой строки таблицы, и выберите его из списка, или добавьте его, нажав кнопку .

4.Сохраните данные и закройте окно.
