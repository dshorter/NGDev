Форма L12 «Схема хранения»

Форма L12 Схема хранения используется для ввода подробной информации о холодильнике. Левая панель формы содержит схему внутренней структуры холодильника, со всеми полками, коробками и другими элементами хранилища; при необходимости, разрешается редактирование. В правой панели формы отображается либо основная информация о холодильнике (наименование холодильника, его тип, штрихкодовый номер и дополнительные примечания), либо основная информация о выбранном элементе хранилища холодильника.

Атрибуты холодильника

Наименование холодильника

Указывается уникальное наименование холодильника. Название должно содержать как минимум 1 символ, а его общая длина не может превышать 200 символов.

Тип хранения

Используется для выбора типа хранения из раскрывающегося списка.

Штрихкод холодильника

Указывается вручную или с помощью сканера, и состоит из букв и цифр. Длина штрихкода не может превышать 200 символов.

Для распечатки штрихкодовой наклейки холодильника, нажмите кнопку Печать штрихкодовых наклеек ().

Примечание

Вносятся примечания. Длина примечания не может превышать 200 символов.

Атрибуты полок и боксов

Наименование элемента хранилища

В этом поле указывается или редактируется название элемента хранилища. Название должно содержать как минимум 1 символ, а его общая длина не может превышать 200 символов.

Тип элемента хранилища

Используется для выбора типа элемента хранилища из раскрывающегося списка. Если необходимый тип отсутствует в списке, добавьте его нажатием кнопки Добавить ().

Количество мест хранения

Указывается количество мест хранения в элементе хранилища. Для существующего элемента хранилища, такое количество не может быть меньше количества уже хранящихся в нём проб.

Штрихкод элемента хранилища

Указывается вручную или с помощью сканера, и состоит из букв и цифр. Длина штрихкода не может превышать 200 символов.

Для распечатки штрихкодовой наклейки элемента хранилища, нажмите кнопку Печать штрихкодовых наклеек ( ).

Примечание

Указываются примечания, при необходимости. Длина примечания не может превышать 200 символов.

Кнопки

Новый

Служит для создания нового элемента хранилища в выбранном холодильнике или элементе хранилища.

1.Указывается холодильник или элемент хранилища, к которым добавляется новый элемент.

2.Нажмите кнопку Новый.

3.Введите основную информацию об элементе хранилища в правую панель.

Копировать

Служит для копирования выбранного элемента хранилища. Новая копия будет создана на том же месте, на котором располагался первоначальный элемент хранилища. Скопированы будут вся внутренняя структура и вся основная информация об элементе хранилища, кроме его штрихкодового номера.

Удалить

Служит для удаления выбранного элемента хранилища. Удалять можно только те элементы хранилища, равно как и его дочерние отделения, в которых нет хранящихся проб. Элемент хранилища удаляется вместе со всеми своими дочерними отделениями, если таковые имеются.
