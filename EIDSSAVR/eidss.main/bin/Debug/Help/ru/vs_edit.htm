Просмотр и редактирование сессии НЗПЗ

Редактирование сессии надзора за переносчиками заболевания возможно только если в её статусе указано значение В процессе.

 

1.Находясь в меню Журналы, выберите пункт Сессия надзора за переносчиками заболеваний. Откроется окно Список сессий надзора за переносчиками заболеваний.

2.Найдите нужную сессию НЗПЗ. Для получения более подробного описания процесса нахождения сессии, прочтите раздел Поиск сессии НЗПЗ .

3.Проделайте одно из следующих действий:

•Дважды нажмите мышкой на предназначенной для редактирования сессии.

•Отметьте сессию для редактирования и нажмите кнопку Редактировать.

Такая сессия откроется в форме Сессия надзора за переносчиками заболевания и будет готова для редактирования.

4.Отредактируйте общую информацию НЗПЗ.

5.Отредактируйте информацию о сессии, затронув следующие аспекты:

•Краткая информация;

•Подробная информация;

•Регистрация общей информации об отдельных переносчиках / группах переносчиков заболевания;

•Регистрация специальной информации по отдельному переносчику / группе переносчиков заболевания;

•Регистрация проб по отдельному переносчику / группе переносчиков заболевания;

•Регистрация результатов полевых тестов по отдельному переносчику / группе переносчиков заболевания;

Перечень заболеваний сессий НЗПЗ состоит из всех заболеваний, подтверждённых либо полевыми, либо лабораторными тестами, которые проводятся над отдельными переносчиками / группами переносчиков заболевания, отловленными в рамках такой сессии.

6.Свяжите сессию НЗПЗ со вспышкой заболевания, в случае необходимости.

7.Сохраните сессию НЗПЗ.

8.Закройте сессию НЗПЗ.

 

Начнётся репликация сохранённой сессии надзора за переносчиками заболевания, и её данные будут переданы на сервер ЭИСНЗ и другие подсистемы, если поле Диагноз было изменено либо было добавлено или редактировалось какое-либо поле или результат лабораторного теста.
