Вкладка «История изменений» формы «Сведения о лабораторной записи»

Вкладка История изменений используется для отображения информации о каждом изменении результатов тестов.

 

Рис. 370 Вкладка «История изменений»

Дата изменения

Отображается дата изменения.

Внёсшее изменения лицо

Отображается имя внёсшего изменения в результаты теста лица.

Внёсшая изменения организация

Отображается название внёсшей изменения в результаты теста организации.

Изменённый результат теста

Отображается новое значение результата теста.

Причина изменения

Отображается причина внесения изменения.
