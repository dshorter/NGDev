Приём в систему группы проб (ЛР)

Группу проб разных типов, относящихся к различным случаям заболеваний или сессиям, можно принять в систему одновременно, если у них одинаковый Локальный / полевой идентификатор пробы.

 

1.Находясь в меню Журналы, выберите Лабораторный раздел. Откроется окно Лабораторного раздела.

2.Нажмите кнопку Приём в систему группы проб. Откроется окно Приём в систему группы проб.

Рис. 308 Окно «Приём в систему группы проб»

3.Вручную или с помощью сканера, введите в поле Идентификатор полевой / локальный пробы полевой или локальный идентификатор пробы, используемый для всей группы, и нажмите кнопку Добавить.

4.Повторите шаг 3 для каждой принимаемой в систему группы.

5.Нажмите кнопку Закрыть. Окно Приём в систему группы проб закроется. Сведения о пробе меняются следующим образом:

•Значение текущей даты используется в качестве значения для поля Дата приёма в систему и, при необходимости, может быть изменено вручную.

•Значение поля Лабораторный идентификатор пробы временным устанавливается на Новый, и его можно менять вручную. Если его изменили, то для пробы будет использоваться определяемый пользователем идентификатор. Если изменения не было, то после сохранения формы идентификатор получит постоянный номер, в соответствии со Схемой уникальной нумерации.

•Значение поля Состояние пробы при получении устанавливается на значение Принято в хорошем состоянии, но может быть изменено вручную. Если его изменили вручную, откроется окно Комментарий о принимаемой в систему пробе, куда будет предложено ввести пояснение. Введите объяснение в поле Комментарии и нажмите кнопку ОК. Окно Комментарий о принимаемой в систему пробе закроется.

Если значение поля Состояние пробы при получении изменили на Отклонена и сохранили, проба в системе будет скрыта и её невозможно будет найти посредством поиска.

Рис. 309 Комментарий о принимаемой в систему пробе

6.Сохраните сеанс приёма в систему. Ей будет автоматически присвоен Лабораторный идентификатор пробы.
