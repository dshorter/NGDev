Форма Н04 «Сведения о лице»

Форма Н04 «Сведения о лице» используется во время создания или редактирования информации записи о человеке в базе данных ЭИСНЗ.

Рис. 244 Форма «Сведения о лице»

Поле / переменное значение

Описание

Дата создания

Отображается дата, в которую лицо было добавлено в базу данных. Это поле заполняется автоматически.

Дата последнего обновления

Отображается дата, в которую данные в последний раз сохранялись в базе данных. Это поле заполняется автоматически.

Ф.И.О.: фамилия, имя, отчество

В поле Фамилия, в произвольной текстовой форме указывается фамилия лица. Заполнение поля Фамилия является обязательным. Форму без указания информации в поле Фамилия сохранить невозможно. Поля Имя и Отчество не являются обязательными для заполнения, но в них в произвольной текстовой форме можно вписать имя и отчество человека.

Дата рождения

Вводится дата рождения лица. По умолчанию, это поле остаётся пустым, его значение можно ввести двумя способами: либо набрав значение с клавиатуры в само поле (при этом датой по умолчанию будет текущая дата), либо выбрав значение мышкой из календаря (датой по умолчанию будет текущая дата).

Пол

В поле Пол указывается пол лица, значение которого выбирается из раскрывающегося списка: Мужской или Женский. Для очистки содержимого, необходимо выбрать пустое значение этого поля. Как только будет выбрано пустое поле, во всплывающем окне появится запрос для пользователя на подтверждение удаления или оставления содержимого. При подтверждении пользователем удаления, поле будет очищено.

Тип личного идентификатора

Тип личного идентификатора человека выбирается из раскрывающегося списка.

Личный идентификатор

В это поле вводится личный идентификатор человека. Поле становится активным, только после выбора поля типа личного идентификатора.

Область / район

Значения для заполнения этих полей можно брать из раскрывающихся списков областей и соответствующих районов. Значение поля Район нельзя выбрать, пока не будет указано значение в поле Область. Для очистки содержимого полей Область или Район, необходимо выбрать пустое поле. Как только будет выбрано пустое поле, во всплывающем окне появится запрос для пользователя на подтверждение удаления или оставления содержимого. При подтверждении пользователем удаления, поле будет очищено.

Населённый пункт

Указание информации в поле Населённый пункт возможно только после заполнения поля Район. Значение для заполнения этого поля можно выбрать из раскрывающегося списка названий населённых пунктов соответствующего района. Для очистки содержимого поля Населённый пункт, необходимо выбрать пустое поле. Как только будет выбрано пустое поле, во всплывающем окне появится запрос для пользователя на подтверждение удаления или оставления содержимого. При подтверждении пользователем удаления, поле будет очищено.

Улица

Поле Улица представляет собой раскрывающийся список. Изначально этот раскрывающийся список пуст, поскольку системе нецелесообразно хранить полный перечень всех улиц всех населённых пунктов страны. Вследствие этого, улицы населённого пункта могут заполняться вручную. Названия таких улиц становятся доступными для такого населённого пункта в раскрывающемся списке после их занесения вручную. Заполнение поля Улица возможно только после заполнения поля Населённый пункт. Текстовая информация вносится в это поле в произвольной форме.

Почтовый индекс

Поле Почтовый индекс заполняется аналогично полю Улица. Заполнение поля Почтовый индекс возможно только после заполнения поля Населённый пункт. Текстовая информация вносится в это поле в произвольной форме.

Дом/корпус/квартира

В поле Дом/корпус/квартира, представляющее собой поле для текстовой информации в произвольной форме, указывается номер дома места проживания лица. Заполнение поля Дом/корпус/квартира возможно только после заполнения поля Населённый пункт.

Номер телефона

Поле для текстовой информации в произвольной форме Номер телефона служит для указания номера телефона лица.

Гражданство

В поле Гражданство указывается гражданская принадлежность лица. Значение можно выбрать из раскрывающегося списка. Новые значения в такой раскрывающийся список можно добавлять в форме А01. Пользователь может открыть эту форму нажав кнопку (), расположенную в поле Гражданство. Для очистки содержимого этого поля, необходимо выбрать пустое поле. Как только будет выбрано пустое поле, во всплывающем окне появится запрос для пользователя на подтверждение удаления или оставления содержимого. При подтверждении пользователем удаления, поле будет очищено.

Географическая долгота/широта

Значения географической долготы и широты могут вводиться с использованием кнопок управления «вверх»/«вниз», или посредством нажатия кнопки Карта (открывается модуль ГИС).

Поле Географическая долгота является числовым. Числа можно вводить в следующем формате: x.xxxxx, и исключительно в диапазоне от -180 до +180.

Поле Географическая широта является числовым. Числа вводятся в следующем формате: x.xxxxx, и исключительно в диапазоне от -85 до +85.

При указании определённой точки на карте, значения в полях Географическая долгота и Географическая широта заполняются автоматически.

Место работы

Можно указать Место работы человека. В этом поле указывается место работы, школа, детский сад, учреждение или другое место, которое пациент посещает на регулярной основе, вживую контактируя с большим количеством людей. Поле Место работы является полем для текстовой информации в произвольной форме.

Адрес места работы

Поле Адрес места работы открывает отдельное раскрывающееся окно. В этом окне пользователь может указать следующие данные:

-Страна, значением для которой, по умолчанию, будет значение поля той страны, в которой расположен текущий сайт;

-Область, значение которого зависит от выбранного значения поля Страна;

-Район, значение которого зависит от выбранного значения поля Область, и его невозможно указать, пока не будет выбрано значение в поле Область;

-Населённый пункт, значение которого зависит от выбранного значения в поле Район, и его невозможно указать до выбора значения в поле Район;

-Улица и Почтовый индекс, а также Дом/корпус/квартира, значения которых зависят от выбранного Населённого пункта, и которые указать невозможно, пока не будет выбрано значение в поле Населённый пункт.
