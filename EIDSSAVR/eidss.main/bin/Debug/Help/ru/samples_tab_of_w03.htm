Вкладка «Пробы» формы W03 «Сведения о группе переносчиков / отдельном переносчике заболевания»

Вкладка Пробы используется для добавления, редактирования, просмотра и удаления проб, взятых у отдельного переносчика или группы переносчиков заболевания. Из неё пробы копируются на вложенную вкладку «Пробы» формы W02 «Сессия надзора за переносчиками заболевания».

 

•Добавьте новую пробу; для этого нажмите на пустую строку с сообщением Нажать для добавления новой строки.

•Отредактируйте запись существующей пробы, для чего следует выбрать пробу и отредактировать её в таблице.

•Удалите пробу(-ы); для этого отметьте её/их и нажмите кнопку Удалить.

 

Рис. 271 Вкладка «Пробы»

Поля данных

Лабораторный идентификатор пробы

Лабораторный идентификатор пробы присваивается автоматически, в соответствии со Схемой уникальной нумерации.

Полевой идентификатор пробы

Указывается идентификатор пробы, использовавшийся до регистрации такой пробы в системе ЭИСНЗ. Это поле является обязательным для заполнения.

Тип пробы

Указывается тип пробы. Это поле является обязательным для заполнения.

Дата взятия

Указывается дата взятия материала. По умолчанию указывается значение текущей даты, которое можно изменить вручную.

Учреждение-сборщик

Указывается взявшая для пробы материал организация.

Комментарий

Поле является нередактируемым и отображает комментарий, который вводится в лабораторном модуле.

Дата приёма в систему

Поле является нередактируемым и отображает дату приёма пробы в систему в лабораторном модуле.

Состояние пробы при получении

Поле является нередактируемым и отображает состояние пробы в систему в лабораторном модуле.

Кнопки

Удалить

Удаляет выделенную(-ые) пробу(-бы).

1.Выберите предназначенную(-ые) для удаления пробу(-ы).

2.Нажмите кнопку Удалить. Откроется окно с запросом на подтверждение действия.

3.Нажмите Да для подтверждения удаления и закройте окно подтверждения.

Удалить получится только те пробы, которые не принимались и не имеют связанных с ними полевых тестов.
