Сайты ЭИСНЗ

Настоящая форма используется для просмотра списка всех сайтов с установленной ЭИСНЗ, а также для изменения настроек по умолчанию прав доступа пользователей одного сайта к другому сайту.

Для просмотра списка зарегистрированных сайтов необходимо:

В меню Безопасность, нажмите мышкой пункт Сайты ЭИСНЗ. Откроется окно Сайты ЭИСНЗ.

Рис. 160 Сайты с установленной ЭИСНЗ

Для редактирования настроек прав доступа какому-либо сайту необходимо:

1.Выберите сайт, права доступа к которому будут изменены, и нажмите кнопку Редактировать. Откроется окно Сведения о сайтах ЭИСНЗ.

Рис. 161 Сведения о сайтах с установленной ЭИСНЗ - вкладка «Общая информация»

2.Вкладка Общая информация является нередактируемой и содержит общую информацию о сайте. Нажмите на вкладке Полномочия для её открытия.

Рис. 162 Сведения о сайтах с установленной ЭИСНЗ - вкладка «Полномочия» 

3.Поле Сайт содержит краткое наименование сайта, а также пользователей и групп пользователей, которые отображаются в списке Действующие субъекты.

4.Для добавления пользователя или группы пользователей, выберите их сайт в поле Сайт и нажмите кнопку Добавить. Откроется окно Список пользователей и групп, в котором будет приведён список всех пользователей и групп пользователей.

Рис. 163 Окно «Список пользователей и групп»

5.Отметьте действующих субъектов и нажмите кнопку Выбрать. Окно Список пользователей и групп закроется, и действующие субъекты отобразятся в поле Действующие субъекты. Отметьте их один за одним и настройте их полномочия на доступ к сайту в поле Полномочия.

6.Повторите шаги 3-5 для пользователей и групп пользователей с разных сайтов.

7.Для удаления действующего субъекта, отметьте его или её в списке Действующие субъекты и нажмите кнопку Удалить.

8.По окончании изменения всех полномочий, сохраните изменения и закройте окно Сведения о сайтах ЭИСНЗ.
