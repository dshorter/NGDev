Системные функции

Для предоставления или ограничения прав доступа пользователя/группы пользователей в отношении любой системной функции необходимо:

1.В меню Безопасность, нажмите мышкой пункт Системные функции. Откроется окно Системные функции.

Рис. 154  Список системных функций

2.Выберите функцию, права доступа к которой необходимо изменить.

Для получения более подробных сведений о системных функциях, смотрите раздел Список системных функций.

3.Нажмите кнопку Полномочия. Откроется окно Сведения о доступе к данным.

Рис. 155 Сведения о доступе к данным

4.В поле Сайт, выберите сайт, для которого необходимо изменить права доступа к выбранной системной функции.

5.Для добавления пользователя или группы в окно Действующие субъекты, нажмите кнопку Добавить. Откроется окно Список пользователей и их групп.

Рис. 156 Список пользователей и их групп

6.Выберите группу или пользователя, которого необходимо добавить в список Действующие субъекты, и нажмите кнопку Выбрать. Выбранные группы и люди появятся в области Действующие субъекты окна Сведения о доступе к данным.

7.В панели Действующие субъекты выберите группу или пользователя, чьи права доступа в отношении выбранной системной функции будут изменены.

8.В панели Полномочия, в отношении каждой операции (Создание / чтение / удаление / запись / доступ к личным данным или исполнение), укажите права доступа для выбранного пользователя / группы, активируя флажки Запретить или Разрешить на кнопках с фиксацией.

9.Нажмите кнопку ОК для сохранения изменений, и закройте окно Сведения о доступе к данным.
