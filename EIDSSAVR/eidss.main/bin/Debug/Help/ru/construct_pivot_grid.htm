Построение сводной таблицы

Сводная таблица состоит из строк, столбцов и данных, лежащих на их пересечении. Например, чтобы узнать количество случаев заболеваний людей в какой-либо области с разбивкой по годам, необходимо переместить мышкой записи о медицинских случаях в область данных, переместить запись даты появления симптомов в область столбцов, а записи «Область» - в область строк.

Рис. 403 Пример сводной таблицы

1.Нажмите на вкладку Сводная таблица макета для его открытия.

2.Перетащите анализируемые данные из окна Настройка полей таблицы в область данных сводной таблицы.

Рис. 404 Перемещение полей сводной таблицы

3.Если предполагается отображение более одного типа данных (полей), добавьте его в область данных после добавления первого типа данных.

4.В область строк и область столбцов перетащите поля, с помощью которых данные должны анализироваться. Просьба учитывать, что данные, запланированные к использованию на оси Х диаграммы или на географической карте, следует перемещать в область строк.

5.Если предполагается отображение более одного типа данных (полей), добавьте его в ту же область данных после добавления первого типа данных. В последнем случае тип данных слева используется в качестве группы столбцов, а тип данных справа используется в качестве столбца. В одной и той же области можно использовать данные более, чем двух типов (полей).

Рис. 405 Использование группы столбцов

6.Для смены расположения поля внутри его области, переместите его на требуемое место.

7.Для использования какого-либо поля дважды (сгруппированного иным образом или с применёнными другими функциями агрегирования, скопируйте его ).

8.Переименуйте поля, которые необходимо использовать в диаграмме или на географической карте так, чтобы новые названия полей отображались на карте или в настройках диаграммы.
