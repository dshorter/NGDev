Удаление записи о вспышке заболевания

ВАЖНО! Запись о вспышке заболевания нельзя удалить, если такая вспышка:

•Связана с заболеваниями человека;

•Связана с заболеваниями животных;

•Связана с сессией надзора за переносчиками заболеваний.

 

1.Выберите Журналы -> Вспышки заболеваний. Откроется Список вспышек заболеваний.

Рис. 251 Список вспышек заболеваний

2.Выполните поиск вспышки заболевания.

3.Для удаления вспышки заболевания, выберите её из списка и нажмите кнопку Удалить. Откроется окно с запросом на подтверждение совершение действия.

Рис. 252  Удаление записи

4.Для подтверждения удаления записи о вспышке заболевания нажмите кнопку Да.
