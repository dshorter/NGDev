Поиск в режиме просмотра

Режим просмотра представляет собой режим расширенного поиска случая заболевания человека, дающий пользователю возможность использования любых полей формы «Случай заболевания человека» в качестве критериев поиска.

1.Перейдите в режиме просмотра Поиск -> Случай заболевания человека. Откроется окно формы Поиск случая заболевания человека. Для получения полной информации о форме, просьба изучить структуру формы Н09 Поиск случая заболевания человека.

Рис. 189 Окно формы «Поиск случая заболевания человека»

2.В поля формы введите данные, которые будут использоваться в качестве критериев поиска. В текстовых (строчных) полях могут использоваться подстановочные символы.

3.Нажмите кнопку Поиск.

4.Будут найдены только те случаи заболеваний, которые содержат все введённые данные. На экране одномоментно отображается только по одному найденному случаю. Если критериям поиска соответствует более одного случая заболевания, открывается диалоговое окно Результаты поиска случаев заболеваний людей.

Рис. 190 Диалоговое окно «Результаты поиска случаев заболеваний людей»

5.Нажмите Да, и диалоговое окно Результаты поиска случаев заболеваний людей закроется, и откроется окно Список случаев заболеваний людей. Оно содержит список записей о случаях заболеваний, соответствующих критериям поиска.

Рис. 191 Раздел «Результаты поиска»

6.Выберите тот случай заболевания человека, который необходимо открыть, и нажмите кнопку Выбрать. Случай заболевания откроется и будет готов к редактированию..
