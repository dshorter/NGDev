Обновление данных

Поскольку во время построения сводной таблицы приложение ЭИСНЗ занято, то для извлечения последних записей из базы данных становится целесообразным её периодическое обновление. Оно доступно лишь в момент отображения данных о случае заболевания в сводной таблице, т.е. когда включён флажок на кнопке с фиксацией Показать данные в сводной таблице). Следует помнить о том, что обновление может длиться очень продолжительное время!

 

1.Для обновления отображаемых в сводной таблице данных, нажмите кнопку Обновить данные, расположенную в на вкладке Сводная таблица макета. Откроется окно с предложением сохранения макета перед совершением действия.

2.Для сохранения макета нажмите кнопку Да, и обновите отображаемые в сводной таблице данные.
