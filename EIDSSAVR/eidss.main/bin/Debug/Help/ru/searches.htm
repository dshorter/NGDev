Списки поиска

Результаты стандартного поиска отображают записи, соответствующие указанным пользователем критериям поиска.

Для поиска записей в списке элементов необходимо:

1.Нажмите кнопку (), чтобы развернуть панель поиска.

2.Панель поиска откроется. Для осуществления поиска используйте поля панели. Некоторые поля заполняются автоматически (напр., Дата ввода данных); некоторые поля являются обязательными для заполнения, а некоторые - факультативными.

Например, находясь в меню Журналы, нажмите пункт Случаи заболевания животных. Откроется окно с панелью поиска Список случаев заболеваний животных.

В некоторых панелях поиска (напр., формы Список случаев заболеваний людей) есть опция, которая раскрывает дополнительные поля, при нажатии кнопки Дополнительно.
