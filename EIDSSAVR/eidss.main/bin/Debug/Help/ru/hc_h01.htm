Форма Н01 «Список случаев заболеваний людей»

Show/Hide Hidden Text

Список случаев заболеваний людей (Н01) представляет собой форму, в которой приведены все случаи заболеваний людей, совпадающие с критериями поиска.

Рис. 223 Окно «Список случаев заболеваний людей»

Панель поиска содержит различные поля, используемые для поиска случаев заболеваний. В панели поиска содержатся обязательные поля, заранее заполненные поля и факультативные поля, созданные с целью ограничения списка результатов поиска и улучшения работы системы.

Рис. 224 Раздел «Параметры поиска»

Поле / переменное значение

Описание

Зарегистрированные мной случаи заболевания

Отображает только те случаи заболеваний (при включении соответствующей кнопки с фиксацией), которые были внесены в систему текущим пользователем.

Идентификатор случая заболевания

В этом поле пользователь может указать определённый идентификатор случая заболевания.

Локальный идентификатор

В этом поле пользователь может указать локальный идентификатор случая заболевания.

Окончательный диагноз

В этом поле пользователь может указать окончательный диагноз случая заболевания.

Группа окончательного диагноза

В этом поле пользователь может указать группу окончательного диагноза случая заболевания.

Классификация случая заболевания

В этом поле пользователь может указать классификацию случая заболевания.

Статус случая заболевания

В этом поле пользователь может указать статус случая заболевания.

Учреждение-отправитель

В поле Учреждение-отправитель можно указать название организации, направившей извещение о случае заболевания.

Учреждение-получатель

В поле Учреждение-получатель можно указать название организации, получившей извещение о случае заболевания.

Диагноз

В поле Диагноз можно указать первоначальный диагноз случая заболевания.

Дата диагноза: с какого числа

В поле Дата диагноза: с какого числа указывается дата, после которой случаю заболевания было присвоено значение в поле Дата диагноза.  

Дата диагноза: по какое число

В поле Дата диагноза : по какое число может быть указана дата, до которой случаю заболевания было присвоено значение Дата диагноза.

Изменённый диагноз

В поле Изменённый диагноз можно указать значение последнего изменённого диагноза, поставленного случаю заболевания.

Дата изменения диагноза: с какого числа

В поле Дата изменения диагноза: с какого числа может быть указана дата, после которой случаю заболевания было присвоено значение Дата изменения диагноза.

Дата изменения диагноза: по какое число

В поле Дата изменения диагноза: по какое число может быть указана дата, до которой случаю заболевания было присвоено значение Дата изменения диагноза.

Локальный идентификатор пробы

В поле Локальный идентификатор пробы можно указать локальный идентификатор, присваиваемый пробе, взятой в связи со случаем заболевания.

Фамилия

В этом поле пользователь может указать фамилию пациента.

Имя

В этом поле пользователь может указать имя пациента.

Отчество

В этом поле пользователь может указать отчество пациента.

Дата рождения: с какого числа

Отображает исключительно те случаи заболеваний, которые затрагивают пациентов, у которых дата рождения больше или равна указанной.

Дата рождения: по какое число

Отображает исключительно те случаи заболеваний, которые затрагивают пациентов, у которых дата рождения меньше или равна указанной.

Дата ввода данных: с какого числа

Отображает исключительно те случаи заболеваний, у которых дата регистрации больше или равна указанной.

Дата ввода данных: по какое число

Отображает исключительно те случаи заболеваний, у которых дата регистрации меньше или равна указанной.

Область

В этом поле пользователь может указать административную область текущего места проживания пациента.

Район

В этом поле пользователь может указать административный район текущего места проживания пациента. Поле Район недоступно для редактирования до тех пор, пока не будет заполнено поле Область.

Населённый пункт

В этом поле пользователь может указать населённый пункт текущего места проживания пациента. Поле Населённый пункт недоступно для редактирования до тех пор, пока не будет заполнено поле Район.

Сайт ввода данных

В этом поле пользователь может указать сайт, на котором вводились данные.

Дополнительное поле поиска № 1

В этом поле пользователь может указать параметры поиска, которые будут применены для фильтрации случаев заболеваний в соответствии со значением, указанным в поле Дополнительное поле поиска № 3.

Дополнительное поле поиска № 2

В этом поле пользователь может указать оператор сравнения, который будет применён для фильтрации случаев заболеваний в соответствии со значением, указанным в поле Дополнительное поле поиска № 3.

Дополнительное поле поиска № 3

В этом поле пользователь может указать значение параметра из поля Дополнительное поле поиска № 1.
