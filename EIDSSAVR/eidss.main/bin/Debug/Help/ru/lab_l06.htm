Форма L06 «Аликвоты/дериваты»

Форма L06 Аликвоты/дериваты используется для регистрации аликвот и дериватов, созданных на основе пробы. В ней содержится следующая информация:

•В верхнем левом углу отображается список проб, выбранных для создания аликвот и/или дериватов.

•В рабочей панели в левом нижнем углу формы указываются параметры, которые определяют создаваемые аликвоты или дериваты, а также количество новых проб и тип новосозданных проб.

•Правая панель формы Аликвоты/дериваты состоит из списка созданных аликвот и дериватов.

Рис. 350 Форма «Аликвоты/дериваты»

Исходные пробы

Идентификатор пробы

Идентификаторы у выбранных проб.

Подсчёт

Количество новосозданных аликвот/дериватов. Оно рассчитывается автоматически и его нельзя изменить вручную.

Создание аликвоты/дериваты

Указываются создаваемые аликвоты или дериваты.

Количество новых проб

Указывается количество аликвот или дериватов.

Тип новых проб или новых дериватов

Используется для указания типа создаваемых дериватов, выбираемых в раскрывающемся списке. Список доступных типов определяется типом выбранной исходной пробы, и основывается на матрице Тип пробы -> тип деривата. Он доступен только при создании дериватов, когда в окне Создание аликвот/дериватов выбрано поле Создание дериватов.

Назначенные тесты

Первоначальная проба

Лабораторный идентификатор исходной пробы.

Лабораторный идентификатор пробы

Отображается лабораторный идентификатор пробы для новосозданной аликвоты/деривата, который можно изменять.

Тип пробы

Тип пробы новосозданной аликвоты/деривата.

Обрабатывающий отдел

Указывается отдел организации, который работает с пробой. Его выбирают из раскрывающегося списка.

Место хранения

Указывается место хранения:

1.Нажмите кнопку Сведения (). Откроется форма Холодильник.

2.Выберите необходимое место.

3.Нажмите кнопку ОК. В поле появится выбранное место хранения.

Комментарий

Комментарии.

Кнопки

Создать

Создание аликвот или дериватов. Нажимайте кнопку Создать, когда все необходимые данные будут введены в левой панели. После этого, новосозданные пробы (аликвоты и дериваты) добавятся в список в правой панели.

Печать штрихкодовых наклеек

Вызывает меню печати штрихкодовых наклеек для новосозданных аликвот и/или дериватов.

ОК

Закрывает окно. Если создаются какие-либо аликвоты/дериваты, введённые данные будут сохранены, а создание аликвот/дериватов будет подтверждено.
