Создание группы в форме «Список групп»

1.Находясь в меню Журналы, выберите пункт Группы. Откроется окно Список групп. В нём содержатся все группы, созданные в системе.

2.Нажмите кнопку Новая. Откроется форма Сведения о тесте группы. В нём не будет указано тестов.

Рис. 330 Форма «Сведения о тесте группы»

3.Укажите Название теста для группы.

4.Добавьте тесты одним из следующих способов:
Первый способ:

a)Нажмите кнопку Добавить тесты. Откроется окно Список лабораторных тестов. В нём содержатся только те тесты, которые соответствуют группе:

•статусом теста является Не начат.

•Тест не относится к другой группе.

•Названием теста является то же, которое выбрано для группы в поле Название теста.

b)Выберите тест для добавления к группе.

c)Нажмите кнопку Выбрать (или добавьте все тесты, нажав кнопку Выбрать все). Окно Список лабораторных тестов закроется, а выбранные тесты добавятся в список.

Второй способ:

a)Укажите лабораторный идентификатор пробы, относящийся к тесту в поле Найти тест по лабораторному идентификатору пробы, находящемся в нижней части формы.

b)Нажмите кнопку Найти. Все тесты указанной пробы, соответствующие критериям, добавятся в список тестов группы.

Третий метод:

a)Сканируйте штрихкодовую наклейку пробы, относящейся к тесту в поле Добавить тест сканированием лабораторного идентификатора пробы, или введите штрихкод вручную внизу формы.

b)Нажмите кнопку Добавить . Все тесты найденной пробы, соответствующие критериям, добавятся в список тестов группы.

5.Если в список добавились не те тесты, выберите их и нажмите кнопку Удалить тест.

6.Укажите выполняющего тест человека в поле Тестировавший работник.

7.Укажите дату проведения теста в поле Дата начала теста, а также результат теста в поле Результат, если таковой известен.

8.Нажмите кнопку Сохранить и подтвердите сохранение данных.

9.Для закрытия формы Сведения о тесте группы, нажмите кнопку ОК.

Если в ЭИСНЗ не введено никакого результата теста, их статусом, как и статусом группы, будет В процессе. Для получения более подробной информации о вводе результата тестов и закрытия группы, прочитайте раздел Закрытие группы (ОФ).
