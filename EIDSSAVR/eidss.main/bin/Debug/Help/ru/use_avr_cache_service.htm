Использование службы кэширования модуля АВО

Поскольку модуль АВО обрабатывает большие объёмы данных, вводимых в систему по всей стране, отображение извлечённых данных может занять продолжительное время, даже если и не использовать архивированную информацию. Для повышения скорости обработки данных и, тем самым, уменьшить время необходимое для отображения их на экране, была разработана Служба кэширования АВО. Её цель заключается в сохранении извлечённых макетом данных для их последующего использования во время открытия или редактирования макета.

Это означает, что используемые данные не будут абсолютно свежими, но очень близко к этому. Если необходимы самые последние данные, обновите их. Обновление освежит кэшированные данные до текущего состояния базы данных.

Довольно редко, но случается, что Служба кэширования АВО становится недоступной и на экране появляется следующее сообщение: Для продолжения работы без кэша АВО вы должны выключить опцию «Использовать кэшированные данные в АВО», находящуюся на форме С09 «Опции»; снимите флажок с кнопки с фиксацией Использовать кэшированные данные в АВО и работайте без кэширования до тех пор, пока служба кэширования не заработает вновь.

Пользователям настоятельно рекомендуется использовать Службу кэширования данных АВО, т.е. следить за тем, чтобы на кнопке с фиксацией Использовать кэширование данных АВО стояла галочка.

Включение Службы кэширования АВО

1.В главном меню ЭИСНЗ Система, выберите пунктСистемные настройки. Откроется окно формы С09 Системные настройки.

Рис. 447 Окно формы С09 «Системные настройки»

2.Активируйте кнопку с фиксацией Использовать кэширование данных АВО.

3.Нажмите ОК.
