Ввод краткой информации сессии НЗПЗ

После создания новой сессии надзора за переносчиками заболевания или открытия существующей для изменения её данных, , и указания её основной информации, необходимо ввести данные результатов сессии.

Краткая информация представляет собой краткие сведения об отловленных переносчиках заболеваний, месте их отлова и заболеваниях, подтверждённых результатами тестов, проведённых в отношении собранного материала. Информация о тестах: количество, тип и дата - не регистрируется; в систему попадают только подтверждённые заболевания.

 

1.Находясь в форме W02 Сессия надзора за переносчиками заболевания, перейдите на вкладку Краткая информация, чтобы открыть её.

2.Нажмите на пустой строке, в которой имеется надпись Нажмите здесь для добавления новой строки. Будет добавлена новая пустая строка.

3.Заполните поле Местоположение, указав информацию о месте отлова отдельного переносчика / группы переносчиков заболевания.

4.В столбце Тип переносчика заболевания выберите тип отдельного переносчика / группы переносчиков заболевания.

5.Выберите Биологический вид отдельного переносчика / группы переносчиков заболевания.

6.Введите значение Количество отловленных групп переносчиков / отдельных переносчиков заболевания, в рамках сессии.

7.Укажите Диагноз, подтверждённый тестами, если он уже известен, а также количество положительных результатов.

8.По возможности, заполните остающиеся столбцы.

9.Повторите шаги с 2-го по 7-й, чтобы добавить сведения обо всех отдельных переносчиках / группах переносчиков заболевания, отловленных в рамках сессии.

10.Для редактирования записи об отдельном(-ой) переносчике / группе переносчиков заболевания, выделите её.

11.Для удаления записи, выделите её и нажмите кнопку Удалить, находящуюся в середине вкладки Краткая информация.

12.Для сохранения краткой информации о сессии, нажмите кнопку Сохранить.

 

Заболевания, указанные в поле Заболевание (количество групп переносчиков /отдельных переносчиков заболевания с положительной реакцией) на вкладке Краткая информация, добавляются в поле Заболевания такой сессии. Помимо этого, типы переносчиков заболеваний из столбца Тип переносчика заболевания добавляются в поле Типы переносчиков заболеваний этой сессии.
