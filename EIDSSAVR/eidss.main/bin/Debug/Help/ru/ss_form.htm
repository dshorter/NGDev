Рис. 287 Форма «Основной синдромный надзор»

Форма «Основной синдромный надзор» состоит из:

Раздела «Общая информация», где отображается общая информация об отчёте.

Рис. 288 Раздел «Общая информация»

Поля данных

Идентификатор формы

Представляет собой уникальный идентификатор Формы «Основной синдромный надзор», который генерируется автоматически.

Дата регистрации

Отображается дата первого ввода информации отчёта в систему; генерируется автоматически.

Дата последнего сохранения

Отображается дата последнего сохранения информации отчёта в системе; генерируется автоматически.

Зарегистрировавший работник

Указывается имя пользователя, создавшего текущий отчёт «Основной синдромный надзор»; генерируется автоматически.

Сайт

Указывается наименование сайта создания текущего отчёта «Основной синдромный надзор»; генерируется автоматически.

Извещение

Представляет собой раскрывающийся список для выбора типа надзора. Список полей в форме зависит от варианта, выбранного в этом поле.

Название больницы / санитарно-контрольного поста

Представляет собой раскрывающийся список организаций, проводивших расследование.

Дата отчёта

Указывается дата отчёта. Её значение должно быть меньше или равно значению текущей даты.
Контактная информация содержит информацию о пациенте и его местоположении.

Рис. 289 Раздел «Контактная информация»

Поля данных

Фамилия

Фамилия указанного в отчёте лица.

Имя

Имя указанного в отчёте лица.

Отчество

Отчество указанного в отчёте лица.

Дата рождения

Дата рождения указанного в отчёте лица.

Возраст (годы)

Общее количество лет, округлённое в меньшую сторону, когда родилось указанное в отчёте лицо.

Возраст (месяцы)

Количество месяцев, превышающее округлённое в меньшую сторону количество лет, когда родилось указанное в отчёте лицо.

Пол

Пол пациента.

Область

Раскрывающийся список включает в себя все области страны, где проживает включённое в отчёт лицо.

Район

Раскрывающийся список включает в себя все районы выбранной области, где проживает включённое в отчёт лицо.

Населённый пункт

Раскрывающийся список включает в себя все поселения/населённые пункты в выбранном районе, где проживает включённое в отчёт лицо.

Улица

Поле для ввода текста в произвольном виде с указанием названия улицы, где проживает включённое в отчёт лицо.

Дом/корпус/квартира

Поле для ввода текста в произвольном виде с указанием номера дома или здания, где проживает включённое в отчёт лицо.

Почтовый индекс

Поле для ввода текста в произвольном виде с указанием почтового индекса адреса, где проживает включённое в отчёт лицо.

Номер телефона

Поле для ввода текста в произвольном виде с указанием номера основного контактного телефона включённого в отчёт лица.

Личный идентификатор

Уникальный идентификационный номер, присвоенный включенному в отчёт лицу.
Раздел «Клиническая информация» содержит дополнительную информацию о симптомах пациента. Содержание раздела зависит от значения поля «Извещение».

Рис. 290 Раздел «Клиническая информация»

Поля данных

Дата появления симптомов

День, месяц и год первого появления признаков или симптомов заболевания у указанного в отчёте лица.

Беременна

Раскрывающийся список со значениями «Да» / «Нет» / «Неизвестно», выбор одного из которых описывает наличие беременности у указанного в отчёте лица на момент составления отчёта.

Послеродовой период (6 недель)

Раскрывающийся список со значениями «Да» / «Нет» / «Неизвестно», выбор одного из которых описывает истекли ли 6 послеродовых недель у указанного в отчёте лица на момент составления отчёта. В случае наличия беременности у такого лица, поле не активно.

Температура больше 38°C

Раскрывающийся список со значениями «Да» / «Нет» / «Неизвестно», выбор одного из которых описывает наличие у указанного в отчёте лица температуры тела, равной или превышающей 38 градусов Цельсия.

Способ измерения

Раскрывающийся список, в котором отображается способ измерения температуры тела у указанного в отчёте лица. Пользователь выбирает одно из значений: «В ротовой полости», «В подмышечной впадине» или «Другой».

Если выбрано «Другой», укажите способ

Представляет собой поле для ввода произвольного текста с описанием способа, не совпадающего с предложенными вариантами, использовавшегося для измерения температуры тела указанного в отчёте лица.

Кашель

Раскрывающийся список со значениями «Да» / «Нет» / «Неизвестно», выбор одного из которых указывает на наличие кашля в клинических признаках.

Одышка

Раскрывающийся список со значениями «Да» / «Нет» / «Неизвестно», выбор одного из которых указывает на наличие одышки в клинических признаках.

Сезонная вакцинация против гриппа

Раскрывающийся список со значениями «Да» / «Нет» / «Неизвестно», выбор одного из которых описывает прохождение включённым в отчёт лицом сезонной вакцинации против гриппа.

Пациенту назначалось противовирусное лечение

Раскрывающийся список со значениями «Да» / «Нет» / «Неизвестно», выбор одного из которых указывает на приём включённым в отчёт лицом медицинского антивирусного препарата в рамках его лечения.

Наименование медицинского препарата

Поле для ввода текста в произвольном виде для указания наименования медицинского антивирусного препарата, предписанного включённому в отчёт лицу. Поле является активным только при наличии предписанного медицинского антивирусного препарата.

Дата принятия

Указывается дата принятия препарата. Поле является активным только при наличии предписанного медицинского антивирусного препарата.

Сопутствующие хронические заболевания или состояния

Представляют собой кнопки с фиксацией, с помощью которых можно указать другие заболевания и состояния, которые могут влиять на включённое в отчёт лицо. У пользователя есть возможность выбора следующих опций: заболевания дыхательной системы, астма, диабет, сердечно-сосудистые заболевания и ожирение: индекс массы тела больше на 30 пунктов; нет оценки, заболевания почек, печени, неврологические / физиологические, иммунодефицит, и / или неизвестной этиологии.
Раздел «Информация о пробах» содержит информацию о взятых пробах.

Рис. 291 Раздел «Контактная информация»

Поля данных

Дата взятия пробы

Указывает день, месяц и год, когда клиническая проба была получена от указанного в отчёте лица.

Идентификатор пробы

Представляет собой уникальный идентификационный номер, присваиваемый клинической пробе, взятой от указанного в отчёте лица.

Результат теста

Раскрывающиеся опции включают в себя следующие варианты: «Положительный», «Отрицательный» и «Неопределённый».

Дата результата

Указывает день, месяц и год, когда результат стал известен.
