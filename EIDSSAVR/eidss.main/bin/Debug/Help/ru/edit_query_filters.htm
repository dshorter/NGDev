Редактирование фильтров запроса

Для ограничения количества извлекаемых для макета записей, используйте фильтры. Использование фильтров настоятельно рекомендуется для повышения скорости обработки макета.

Критерий фильтра состоит из условий фильтрации, связанных объединяющими операторами.

Рис. 377 Фильтр

Части условий фильтрации

В следующем списке приведены части условий фильтрации, которые можно указать:

•Поля таблицы для сравнения;

•Операторы сравнения фильтров, такие как РАВНО, НЕ РАВНО, ПУСТО, НЕ ПУСТО, и т.д.;

•Опции фильтра, такие, как ЛЮБОЙ или ПОДОБНЫЙ, и значения критериев, такие как значения полей, выражения, наборы значений, и подзапросы.
