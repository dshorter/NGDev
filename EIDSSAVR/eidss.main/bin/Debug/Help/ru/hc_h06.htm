Форма Н06 «Сведения об агрегированном случае заболевания человека»

Форма Н06 «Сведения об агрегированном случае заболевания человека» используется в ходе создания агрегированного случая заболевания человека.

Рис. 235 Сведения об агрегированном случае заболевания человека

Поле / переменное значение

Описание

Отправитель извещения

Учреждение

В поле Учреждение указывается организация, направившая извещение с агрегированными данными.

Работник

В поле Работник указывается сотрудник такого учреждения, направивший извещение с агрегированными данными. Соответственно, поле Работник можно заполнить только если заполнено поле Учреждение.

Дата

В поле Дата указывается дата направления извещения с агрегированными данными.

 

Получатель извещения

Учреждение

В поле Учреждение указывается организация, получившая извещение с агрегированными данными.

Работник

В поле Работник указывается сотрудник такого учреждения, получивший извещение с агрегированными данными. Соответственно, поле Работник можно заполнить только если заполнено поле Учреждение.

Дата

В поле Дата указывается дата получения извещения с агрегированными данными. Значение в поле Дата должно быть больше или равно значению в поле Отправитель извещения: дата.

 

Регистратор

Учреждение

В поле Учреждение указывается организация, занёсшая в систему извещение с агрегированными данными. Это поле является нередактируемым и заполняется автоматически.

Работник

В поле Работник указывается сотрудник, занёсший в систему извещение с агрегированными данными. Это поле является нередактируемым и заполняется автоматически.

Дата

В поле Дата указываются дата и время введения извещения с агрегированными данными в систему. Это поле является нередактируемым и заполняется автоматически.

Общая информация

Идентификатор случая заболевания

Поле Идентификатор случая заболевания содержит уникальный номер случая заболевания, который присваивается такому случаю при первом сохранении. Идентификатор случая заболевания генерируется в соответствии со Схемой уникальной нумерации после первого сохранения формы.

Год

В поле Год указывается точный год сбора агрегированных данных.

Месяц

В поле Месяц указывается точный месяц сбора агрегированных данных. Его отображение зависит от настроек агрегирования.

Страна

В поле Страна указывается точная страна сбора агрегированных данных.Это поле является нередактируемым.

Область

В поле Область указывается точная область сбора агрегированных данных. Его отображение зависит от настроек агрегирования.

Район

В поле Район указывается точный район сбора агрегированных данных. Его отображение зависит от настроек агрегирования.

Населённый пункт

В поле Населённый пункт указывается точный населённый пункт сбора агрегированных данных. Его отображение зависит от настроек агрегирования.

 

Список диагнозов

Раздел Список диагнозов содержит таблицу. Наличие столбцов и строк в этой таблице зависит от конкретной страны, и определяется в Конструкторе настраиваемых форм (Шаблон - столбцы) и в Редакторе матрицы агрегированных данных по заболеваниям человека (Версия матрицы - строки).

Шаблон для Агрегированного случая заболевания человека выбирается из совокупности шаблонов Агрегированного случая заболевания человека, где Определителем является Страна.

Матрицей агрегированного случая заболевания человека по умолчанию является последняя активная матрица, но её можно заменить и другой матрицей из списка. В этом списке доступны только активные матрицы.

Версия матрицы

Это поле используется для выбора версии матрицы агрегированного случая заболевания человека. Версия выбирается из раскрывающегося списка.

Шаблон

Это поле используется для выбора шаблона агрегированного случая заболевания человека. Шаблон выбирается из раскрывающегося списка.

Примечание: отображение полей Страна / Область / Район / Населённый пункт и Год / Квартал / Месяц / Неделя зависит от настроек агрегирования.
