Если список разделов и параметров не содержит необходимых параметров / разделов для выбранного шаблона, добавьте их.

       Для добавления раздела необходимо::

1.В окне Конструктор настраиваемых форм нажмите мышью на вкладке Редактор параметров для его открытия. В нём появится список типов шаблонов с разделами и параметрами.

Рис. 108 Конструктор настраиваемых форм

2.Выберите необходимый тип шаблона и нажмите кнопку Добавить раздел (). В дерево шаблонов добавится новый раздел, а его атрибуты будут отображаться в верхнем правом углу. Укажите Название по умолчанию (на английском) и Национальное название (на местном языке) и Тип (по умолчанию или табличный).

Рис. 109 Конструктор настраиваемых форм - добавление нового раздела

       Для удаления раздела необходимо::

Отметьте раздел для удаления и нажмите кнопку Удалить раздел (). Раздел будет удалён из дерева.

       Для добавления параметра необходимо::

1.В окне Конструктор настраиваемых форм нажмите мышью на вкладке Редактор параметров для его открытия. В окне Редактор параметров отобразится список типов шаблонов с разделами и параметрами.

Рис. 110 Конструктор настраиваемых форм

2.Укажите тип шаблона и раздел, в которых должен быть параметр (в зависимости от обстоятельств), и нажмите кнопку Добавить параметр (). Новый параметр будет добавлен в дерево шаблонов и будет отображаться в рабочей области; при этом атрибуты нового параметра показываются в верхней правой части.

3.Укажите следующее:

•Полное название по умолчанию - полное наименование параметра на английском языке;

•Название по умолчанию - полное наименование параметра на местном языке;

•Тип - тип параметра (строка, числовой или любой другой тип параметра, определённый в Редакторе типов параметров);

•Редактор – редактор параметра, зависит от типа параметра;

•Код Ч/Ж – указывает принадлежность параметра к определённому модулю (человек/животное);

•Высота – высота параметра;

•Размер надписи – размер надписей параметра;

•Схема – расположение надписи;

•Ширина – ширина параметра;

•Полное национальное название по умолчанию - полное наименование параметра на местном языке;

•Национальное название – наименование параметра на местном языке.

4.Для изменения размера поля используйте мышку. Помимо этого, можно выбирать значение поля.

       Для удаления параметра необходимо::

Отметьте параметр для удаления и нажмите кнопку Удалить параметр ( ). Параметр будет удалён из дерева.
