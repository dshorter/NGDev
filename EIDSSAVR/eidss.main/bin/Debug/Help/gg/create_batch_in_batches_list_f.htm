შექმენით პარტია "პარტიების ჩამონათვალის" ფორმაში

1.ჟურნალების მენიუში შეარჩიეთპარტიები. გაიხსნება პარტიების ჩამონათვალის ფანჯარა. იგი შეიცავს სისტემაში შექმნილ ყველა პარტიას.

2. დააწკაპუნეთ ღილაკზე ახალი. გაიხსნება ფორმა ჯგუფური ტესტის დეტალები. იგი შეიცავს არც ერთ ტესტს.

ნახ. 272 "ჯგუფური ტესტის დეტალების" ფორმა

3.შეარჩიეთ ტესტის დასახელება პარტიისთვის.

4.ტესტები დაამატეთ ჩამოათვლილიდან ერთ–ერთი ხერხით:
პირველი მეთოდი:

a)დააწკაპუნეთ ღილაკზე ტესტების დამატება. გაიხსნება ფანჯარა ლაბორატორიული ტესტების ჩამონათვალი. იგი შეიცავს მხოლოდ იმ ტესტებს, რომლებიც შეესაბამება კონკრეტულ პარტიას:

•ტესტის სტატუსია არ არის დაწყებული.

•ტესტი არ მიეკუთვნება სხვა პარტიას.

•ტესტის დასახელება იგივეა, რაც პარტიისთვის შეირჩა ტესტის დასახელების ველში.

b)შეარჩიეთ პარტიაში დასამატებელი ტესტი.

c)დააწკაპუნეთ შერჩევის ღილაკზე (ან დაამატეთ ყველა ტესტი მონიშნეთ ყველაფერი ღილაკზე დაწკაპუნებით). დაიხურება ფანჯარა ლაბორატორიული ტესტების ჩამონათვალი და შერჩეული ტესტები დაემატება ჩამონათვალში.

მეორე მეთოდი:

a)ტესტთან დაკავშირებული ნიმუშის ლაბორატორიული ID შეიყვანეთ ფორმის ქვედა ნაწილში არსებულ ველში ტესტის ძებნა ნიმუშის ლაბორატორიული ID-ის მიხედვით.

b)დაკლიკეთ ძიების ღილაკზე. პარტიის ტესტების ჩამონათვალში დაემატება დაფიქსირებული ნიმუშის ყველა ის ტესტი, რომლებიც აკმაყოფილებს კრიტერიუმებს.

მესამე მეთოდი:

a)მოახდინეთ ტესტთან დაკავშირებული ნიმუშის შტრიხკოდის ეტიკეტის სკანირება ტესტის დამატება ნიმუშის ლაბორატორიული ID-ის სკანირებით ველში ან ხელით შეიყვანეთ შტრიხკოდი ფორმის ბოლოში.

b)დააწკაპუნეთ ღილაკზე დამატება. პარტიის ტესტების ჩამონათვალში დაემატება ნაპოვნი ნიმუშის ყველა ის ტესტი, რომლებიც აკმაყოფილებს კრიტერიუმებს.

5.თუ ჩამონათვალში დამატებული იყო არასწორი ტესტები, მონიშნეთ ზედმეტი ტესტები და დააწკაპუნეთ ღილაკზე ტესტის ამოღება.

6.ტესტის ჩამტარებელი პირი შეარჩიეთ ველში ვინ ჩაატარა ტესტირება.

7.ტესტის თარიღი შეარჩიეთ ველში ტესტის დაწყების თარიღი, ხოლო ტესტის შედეგი ველში შედეგი, თუ ცნობილია.

8.დააწკაპუნეთ დამახსოვრების ღილაკზე და დაადასტურეთ მონაცემთა დამახსოვრება.

9.ჯგუფური ტესტის დეტალების ფორმის დასახურად დააწკაპუნეთ ღილაკზე დიახ.

თუ დზეის-ში არ არის შეყვანილი არც ერთი ტესტის შედეგი, მათი სტატუსი და პარტიის სტატუსი იქნება პროცესშია. ტესტების შედეგის შეყვანის შესახებ დეტალური ინფორმაციის მისაღებად და პარტიის დასახურად, წაიკითხეთ სექცია პარტიის დასრულება (IF).
