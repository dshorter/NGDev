ვეტერინარული შემთხვევების ჩამონათვალი V01

ნახ. 67 ვეტერინარული შემთხვევების ჩამონათვალის ფანჯარა

ძებნის პანელი განყოფილება; მოიცავს სხვადასხვა ველებს, რომლებიც გამოიყენება შემთხვევის ძებნისთვის.

ნახ. 68 ძებნის მენიუ

ველი/ცვლადი

აღწერა

ჩემი შემთხვევები

თუკი ეს ოფცია მოქმედია, ეკრანზე გამოვა მხოლოდ მოცემული მომხმარებლის მიერ რეგისტრირებული შემთხვევები.

შემთხვევის ID

ამ ველში შეიძლება მიუთითოთ "შემთხვევის ID".

დიაგნოზი

დიაგნოზი შერჩეულია. ძებნა განხორციელდა სავარაუდო და საბოლოო დიაგნოზების მიხედვით.

დაბრუნება მხოლოდ საბოლოო დიაგნოზზე

შემოწმების შემთხვევაში, ეკრანზე გამოვა მხოლოდ ის შემთხვევები, რომლებიც საბოლოო დიაგნოზითაა განსაზღვრული.

დიაგნოზთა ჯგუფი

დიაგნოზთა ჯგუფი შერჩეულია.

შემთხვევის კლასიფიკაცია

შემთხვევის კლასიფიკაცია შერჩეულია.

შემთხვევის სტატუსი

შემთხვევის სტატუსი შერჩეულია.

ანგარიშის ტიპი

მოხსენების ტიპი შერჩეულია.

შემთხვევის ტიპი

შემთხვევის ტიპი შერჩეულია.

მონაცემთა შეყვანის თარიღი: -დან

ქვედა ზღვრული მონაცემების შეყვანის თარიღი შეყვანილია.

მონაცემთა შეყვანის თარიღი: -მდე

ზედა ზღვრული მონაცემების შეყვანის თარიღი შეყვანილია.

კვლევის თარიღი

შეყვანილია კვლევის ქვედა ზღვრული თარიღიდან.

კვლევის თარიღი

კვლევის ზედა ზღვრულ თარიღამდე შეყვანილია.

ნიმუშის ადგილობრივი ID

ადგილობრივი ნიმუშის ID შეყვანილია.

ცხოველთა საერთო რაოდენობა

ცხოველთა საერთო რაოდენობის ქვედა ზღვრული რაოდენობა შეყვანილია.

ცხოველთა საერთო რაოდენობა

ცხოველთა საერთო რაოდენობის ზედა ზღვრული რაოდენობამდე შეყვანილია.

მფლობელის სახელი

ფერმის მფლობელის სახელი შეყვანილია

მფლობელის გვარი

ფერმის მფლობელის გვარი შეყვანილია

რეგიონი

რეგიონი შერჩეულია.

რაიონი

შერჩეულია რაიონი და ჩამონათვალი შეიცავს მნიშვნელობებს რაიონის მახასიათებლის შესაბამისად, რომელიც გაფილტრულია შერჩეული რეგიონის მიხედვით.

ქალაქი ან სოფელი

დასახლება შერჩეულია, ჩამონათვალი, რომელიც შეიცავს მნიშვნელობებს დასახლებების მახასიათებლებიდან, გაფილტრულია შერჩეული რაიონის მიხედვით.

მონაცემთა შეყვანის სამუშაო ობიექტი

მონაცემთა შეყვანის ადგილი შერჩეულია.
ძებნის შედეგების ჩამონათვალის განყოფილება; ეკრანზე გამოაქვს შემთხვევათა ჩამონათვალი, რომელიც შეესაბამება ძებნის კრიტერიუმებს.

ნახ. 69 შედეგების ჩამონათვალის ძებნა

ველი/ცვლადი

აღწერა

შემთხვევის ID

ეკრანზე ჩანს შემთხვევის ID

მონაცემთა შეყვანის თარიღი

მონაცემთა შეყვანის თარიღი გამოდის ეკრანზე.

კვლევის თარიღი

კვლევის თარიღი გამოდის ეკრანზე.

დიაგნოზები

მოცემული შემთხვევის დიაგნოზ(ებ)ი გამოდის ეკრანზე.

ფერმის მისამართი

ფერმის მისამართი ეკრანზე გამოდის ასეთი თანმიმდევრობით: რეგიონი, რაიონი, დასახლება, ქუჩა, სახლი, ბინა (ინგლისური ენისთვის). სხვა ენების ფორმატი შეიძლება განსხვავებული იყოს.

ფერმის მფლობელი

ფერმის მფლობელის გვარი გამოტანილია ეკრანზე.

ცხოველთა საერთო რაოდენობა

ცხოველთა საერთო რაოდენობა გამოტანილია ეკრანზე.

შემთხვევის ტიპი

შემთხვევის ტიპი გამოტანილია ეკრანზე.

კლასიფიკაცია

კლასიფიკაცია გამოტანილია ეკრანზე.

სტატუსი

შემთხვევის სტატუსი გამოტანილია ეკრანზე.

ანგარიშის ტიპი

მოხსენების ტიპი გამოტანილია ეკრანზე.

ღილაკები

შინაურ ცხოველთა ახალი შემთხვევა

დააწკაპუნე ღილაკზე ახალი შემთხვევა. რათა შექმნა შინაური ცხოველის ახალი შემთხვევა. ცხოველის დაავადებათა კვლევის ფორმა იხსნება ახალ ფანჯარში.

ფრინველის ახალი შემთხვევა

დააწკაპუნე ღილაკზე ფრინველის ახალი შემთხვევა, რათა შექმნა ფრინველის ახალი შემთხვევა. ფრინველის დაავადებათა კვლევის ფორმა ხსნის ახალ ფანჯარას.

რედაქტირება

რედაქტირებისთვის შეარჩიეთ შემთხვევა ძებნის ჩამონათვალში, და დააწკაპუნეთ რედაქტირებისღილაკზე. შერჩეული შემთხვევა გაიხსნება ახალ ფანჯარაში რედაქტირების რეჟიმში.

წაშლა

წაშლისთვის შეარჩიეთ შემთხვევა ძებნის ჩამონათვალში, და დააწკაპუნეთ წაშლისღილაკზე. გამაფრთხილებელი მესიჯი წაშლა/გაუქმება გამოჩნდება ღილაკების სახით.
