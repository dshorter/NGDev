პულის/ინდივიდუალური ვექტორის W05 ფორმის კოპირება

ფორმა პულის/ინდივიდუალური ვექტორის კოპირება გამოიყენებაერთობლიობის/ინდივიდუალური ვექტორის ორიგინალის იმ დეტალების შესარჩევად, რომელთა კოპირება უნდა მოხდეს შთამომავალ ერთობლიობაში/ვექტორში.

ნახ. 216 "პულის/ცალკეული ვექტორის კოპირების" ფანჯარა

ვექტორის საერთო მონაცემები

საკონტროლო უჯრა აირჩევა დეფოლტის მეშვეობით და მისი წაშლა შეუძლებელია. ველების კოპირება ხდება შემდეგი წესების მეშვეობით:

•ახალი ერთობლიობის/ვექტორის ID იქმნება უნიკალური დანომვრის სქემის შესაბამისად და შეიძლება ხელით შესწორდეს

•ხოლო სესიის ID და ვექტორის ტიპი დაკოპირდება პულის/ ინდივიდუალური ვექტორის ორიგინალიდან

•ცარიელი რჩება პულის/ვექტორის ID ველი და შეიძლება ხელით შესწორდეს

მონაცემები ზოგადი ტაბულაციიდან კოპირდება ცვლილებების გარეშე.

ვექტორის მახასიათებელთა მონაცემები

შეარჩიეთ საკონტროლო უჯრა, რათა დააკოპიროთ ყველა მონაცემი ვექტორის მახასიათებელთა მონაცემების პულის/ ინდივიდუალური ვექტორის ორიგინალის ტაბულაციიდან.

შეგროვებული ნიმუშები

შეარჩიეთ საკონტროლო უჯრა, რათა დააკოპიროთ ნიმუშები შემდეგი მოანცემებით:

•სახეობა

•ნიმუშის ტიპი

•შეგროვების თარიღი

•აღებულია დაწესებულების მიერ

კოპირებული მონაცემები არის რედაქტირებადი. დარჩენილი ველები რჩება ცარიელი და შეიძლება შეივსოს ხელით.

საველე ტესტები

კოპირებისთვის შეარჩიე ტესტები შემდეგი მონაცემებით:

•ნიმუშის ტიპი

•მონაცემთა შეგროვება

•ტესტის სახელი

•ტესტის კატეგორია

•ტესტის ჩატარების თარიღი

•ტესტირებულია ინსტიტუტის მიერ

•დადასტურებულია

ტესტის შედეგებით და დაავადების ველები დატოვებულია ცარიელი.

როდესაც საველე ტესტების საკონტროლო უჯრები შეირჩევა, ნიმუშები ავტომატურად დაკოპირდება ნიმუშის კოპირების წესების შესაბამისად.
