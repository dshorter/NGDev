"ძირითადი სინდრომული ზედამხედველობის ფორმების ჩამონათვალი"

"ძირითადი სინდრომული ზედამხედველობის ფორმების ჩამონათვალი" არის ფორმა, სადაც ჩამოთვლილია ყველა ის "ძირითადი სინდრომული ზედამხედველობის" ანგარიში, რომლებიც აკმაყოფილებს ძებნის კრიტერიუმებს.

ნახ. 222 საბაზისო სინდრომული ზედამხედველობის ჩამონათვალი

ძებნის პანელი სექცია, რომელიც შეიცავს "ძირითადი სინდრომული ზედამხედველობის" ანგარიშების ძებნისას გამოყენებულ სხვადასხვა ველს.

ნახ. 223 ძებნის პარამეტრები

ცხოველის ID

ველში ფორმის ID შეიძლება მიეთითოს "ძირითადი სინდრომული ზედამხედველობის" ანგარიშის ID.

შეტყობინება

ჩამოსაშლელი სია, სადაც შეიძლება მოინიშნოს "გრიპის მსგავსი დაავადების" ან "სერიოზული უარყოფითი რეაქციების" შესახებ შეტყობინება.

საავადმყოფოს/საველე სადგურის სახელი

ჩამოსაშლელი სია, სადაც შეიძლება შეირჩეს საავადმყოფო ან საველე სადგური.

შეტყობინების თარიღი: -დან/-მდე

"ძირითადი სინდრომული ზედამხედველობის" ანგარიშის თარიღის ზედა და ქვედა ზღვრები.

სიმპტომების დაწყების თარიღი: -დან/-მდე

სიმპტომების დაწყების თარიღის ზედა და ქვედა ზღვრები.

გვარი

ანგარიშში შეტანილი პირის გვარი.

სახელი

ანგარიშში შეტანილი პირის სახელი.

სქესი

ჩამოსაშლელი სიის ოპციები იქნება "მამრობითი" და "მდედრობითი".

ასაკი (წელი)

ანგარიშში შეტანილი პირის ასაკის ზედა და ქვედა ზღვრები.

რეგიონი

ველში რეგიონი შეიძლება მიუთითოთ რეგიონის ზუსტი დასახელება.

რაიონი

ველში რაიონი შეიძლება მიუთითოთ რაიონის ზუსტი დასახელება.

ქალაქი ან სოფელი

ველში ქალაქი ან სოფელი შეიძლება მიუთითოთ დასახლების ზუსტი დასახელება.

ტესტის შედეგი

ჩამოსაშლელი სია, სადაც შეიძლება მოინიშნოს "დაუდგენელი", "უარყოფითი" ან "დადებითი"ტესტის შედეგი.

ადგილი

ჩამოსაშლელი სია, სადაც შეიძლება შეირჩეს დზეის-ის ადგილები.
ძებნის შედეგების ჩამონათვალის სექცია, რომელიც გვიჩვენებს იმ "ძირითადი სინდრომული ზედამხედველობის ფორმების" ჩამონათვალს, რომლებიც აკმაყოფილებს ძებნის მითითებულ კრიტერიუმებს.

ნახ. 224 ძებნის შედეგების ჩამონათვალი

ცხოველის ID
