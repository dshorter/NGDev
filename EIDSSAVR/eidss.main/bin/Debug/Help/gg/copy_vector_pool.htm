ვექტორი/პულის კოპირება

თუ საჭიროა მსგავსი ვექტორი/პულის გრძელი სიის შექმნა, გონივრული იქნება, შეიქმნას ერთი ვექტორი/პული და შემდეგ მოხდეს მისი კოპირება იმდენჯერ, რამდენიც აუცილებელია. შემდგომში შესაძლებელია ამ ასლების რედაქტირება.

პულის/ცალკეული ვექტორის კოპირებისას, შესაძლებელია იმ მონაცემთა ნაკრების მონიშვნა, რომლებიც უნდა გადავიდეს ამ პულთან/ვექტორთან ერთად.

 

1."ვექტორული ზედამხედველობის სესიის" ფორმის "პულები/ცალკეული ვექტორების" ქვეჩანართში მონიშნეთ ასლის სახით გადასატანი პული/ცალკეული ვექტორი.

2.დააწკაპუნეთ კოპირების ღილაკზე. გაიხსნება პულის/ცალკეული ვექტორის კოპირების ფანჯარა.

ნახ. 200 "პულის/ცალკეული ვექტორის კოპირების" ფანჯარა

3. მონიშნეთ პულთან/ცალკეულ ვექტორთან ერთად ასლის სახით გადასატანი მონაცემთა ნაკრები. მეტი ინფორმაციისთვის, წაიკითხეთ სექცია პულის/ცალკეული ვექტორის კოპირების W05 ფორმა.

4.დააწკაპუნეთ ღილაკზე შერჩევა . დაიხურება პულის/ცალკეული ვექტორის კოპირების ფანჯარა. პულის/ცალკეული ვექტორის დეტალების ფორმა გაიხსნება ახლადშექმნილი პულით/ცალკეული ვექტორით მისი რედაქტირებისთვის.

ველების კოპირება ხდება შემდეგი წესით:

•იქმნება ახალი პულის/ვექტორის ID დანომრვის უნიკალური სქემის შესაბამისად.

•ხდება სესიის ID და ვექტორის ტიპის კოპირება პირველადი პულის/ცალეკული ვექტორიდან.

•პულის/ვექტორის საველე ID არ ივსება და შესაძლებელია მისი ხელით შეცვლა.

•ასევე ხდება სესიის ნიმუშების კოპირებაც. თუ მოხდება ნიმუშების კოპირება, შეიყვანეთ მათი ID-ები.

სხვა მონაცემების კოპირება ხდება შერჩეული პარამეტრების შესაბამისად.

5.ჩაატარეთ ახლადშექმნილი პულის/ცალკეული ვექტორის რედაქტირება.

6.ცვლილებების დასამახსოვრებლად და ასლის შექმნის ოპერაციის დასადასტურებლად დააწკაპუნეთ დიახ ღილაკზე პულის/ცალკეული ვექტორის დეტალების ფორმაში. ახლა, თუ დააწკაპუნებთ გაუქმების ღილაკზე, არცერთი ახალი პული/ცალკეული ვექტორი არ შეიქმნება და კოპირების ოპერაცია აღარ შესრულდება.
