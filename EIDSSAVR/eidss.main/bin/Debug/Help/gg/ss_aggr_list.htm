ILI აგრეგირებული ფორმების ჩამონათვალი

Show/Hide Hidden Text

ILI აგრეგირებული ფორმების ჩამონათვალი არის ფორმა, სადაც ჩამოთვლილია ყველა ILI აგრეგირებული ანგარიში, რომლებიც აკმაყოფილებს ძებნის კრიტერიუმებს.

ნახ. 226 ILI აგრეგირებული ფორმების ჩამონათვალი

ძებნის პანელი განყოფილება, რომელიც მოიცავს სხვადასხვა ველებს, რომლებიც გამოიყენება ILI აგრეგირებული მოხსენებების ძებნისთვის.

ნახ. 227 ძებნის პარამეტრები

ცხოველის ID

ველში - ID-ის ფორმირება, შეიძლება განისაზღვროს ILI აგრეგირებული ანგარიშის ID.

დასაწყისის თარიღი

ILI აგრეგირებული სესიის პერიოდის დაწყების თარიღის შერჩევა აქ ხდება.

დასრულების თარიღი

ILI აგრეგირებული სესიის პერიოდის დასრულების თარიღი თარიღის შერჩევა აქ ხდება.

კვირეების -დან -მდე

ILI აგრეგირებული სესიის პერიოდის დაწყების კვირისა და დასრულების კვრის შერჩევა აქ ხდება. მოიძებნება მხოლოდ ის კვირეები, რომლებიც სრულად არის ჩართული მოცემულ პერიოდში.

საავადმყოფოს/საველე სადგურის სახელი

ჩამოსაშლელი სია, სადაც შეიძლება შეირჩეს საავადმყოფო ან საველე სადგური.

ადგილი

ჩამოსაშლელი სია, სადაც შეიძლება შეირჩეს დზეის-ის ადგილები.
ძებნის შედეგების ჩამონათვალის განყოფილება, რომელსაც ეკრანზე გამოაქვს საბაზისო სინდრომული ზედამხედველობის აგრეგირებული ფორმების ჩამონათვალი, რომელიც შეესაბამება ძებნის მითითებულ კრიტერიუმებს.

ნახ. 228 ძებნის შედეგების ჩამონათვალი

ცხოველის ID

ფორმის ID ეკრანზეა გამოტანილი.

კვირეები

ეკრანზე გამოდის კვირის ნომერი.

საავადმყოფოს/საველე სადგურის სახელი

ეკრანზე გამოდის საავადმყოფოს ან დაცვის სადგურის სახელი.
