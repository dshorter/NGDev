ნიმუშების ჯგუფის მიღება (LS)

სხვადასხვა ტიპის ისეთი ნიმუშების ჯგუფის, რომლებიც დაკავშირებულია სხვადასხვა შემთხვევებთან ან სესიებთან, ერთდროულად მიღება შესაძლებელია, თუ ამ ნიმუშებს ექნებათ ერთნაირი ნიმუშის ადგილობრივი/საველე ID.

 

1.ჟურნალების მენიუში შეარჩიეთ ლაბორატორიული სექცია. გაიხსნება ლაბორატორიული სექციის ფანჯარა.

2.დააწკაპუნეთ ღილაკზე ნიმუშების ჯგუფური მიღება . გაიხსნება ფანჯარა ნიმუშების ჯგუფური მიღება.

ნახ. 250 "ნიმუშების ჯგუფური მიღების" ფანჯარა

3.ნიმუშის საველე/ადგილობრივი ID ველში ხელით ან სკანერით შეიყვანეთ ნიმუშების მთელი ჯგუფისთვის გამოყენებული ნიმუშის საველე ან ადგილობრივი ID და დააწკაპუნეთ ღილაკზე დამატება.

4.მე-3 ნაბიჯი გაიმეორეთ ყველა მისაღები ჯგუფისთვის.

5.დაკლიკეთ დახურვის ღილაკზე. დაიხურება ფანჯარა ნიმუშების ჯგუფური მიღება. ნიმუშის დეტალები იცვლება შემდეგი სახით:

•მიმდინარე თარიღი გამოიყენება, როგორც ნიმუშის მიღების თარიღი და საჭიროების შემთხვევაში შესაძლებელია მათი ხელით შეცვლა.

•ნიმუშის ლაბორატორიული ID დროებით დაყენებულია სტატუსზე (ახალი) და მისი შეცვლა შესაძლებელია ხელით. ცვლილების შემთხვევაში ნიმუშისთვის გამოიყენება მომხმარებლის მიერ განსაზღვრული საიდენტიფიკაციო კოდი. თუ არ შეიცვლება, მაშინ ამ ID-ს დანომრვის უნიკალური სქემის შესაბამისად მიენიჭება მუდმივი ნომერი ფორმის დამახსოვრების შემდეგ.

•ნიმუშის მდგომარეობა მიღებისას დაყენებულია სიდიდეზე მიღებულია სათანადო მდგომარეობაში და მისი შეცვლა შესაძლებელია ხელით. ხელით შეცვლის შემთხვევაში გაიხსნება ფანჯარა კომენტარი ნიმუშის მიღებისას, სადაც შეძლებთ ახსნა-განმარტების შეყვანას. ველში კომენტარები შეიყვანეთ ახსნა-განმარტება და დააწკაპუნეთ დიახ ღილაკზე . დაიხურება ფანჯარა კომენტარი ნიმუშის მიღებისას.

თუ ნიმუშის მდგომარეობა მიღებისას ველის სიდიდე შეიცვალა სიდიდით დაწუნებულია და მოხდა მისი დამახსოვრება, ასეთი ნიმუში დაიმალება და შეუძლებელი იქნება მისი პოვნა ძებნის ფუნქციის გამოყენებით.

ნახ. 251 კომენტარი ნიმუშის მიღებისას

6.დაიმახსოვრეთ ნიმუშის მიღება. ლაბორატორიული ნიმუშის ID -ს მინიჭება ხდება ავტომატურად.

✓იმ არსებული ნიმუშის მიღება, რომელიც მიბმულია დაავადების შემთხვევაზე/სესიაზე (LS)

✓ნიმუშის რეგისტრაცია, რომელიც დაკავშირებულია ან არ არის დაკავშირებული შემთხვევასთან/სესიასთან (LS)
