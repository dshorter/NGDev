ვექტორის ტიპის-შეგროვების მეთოდის მატრიცა

ვექტორის ტიპის->შეგროვების მეთოდის მატრიცა არის ინსტრუმენტი, რომელიც განსაზღვრავს მეთოდების ჩამონათვალს, რომლებიც გამოიყენება სხვადასხვა ვექტორის ტიპების შესაგროვებლად.

მოახდინეთ მატრიცის რედაქტირება:

1.დააწკაპუნე სისტემისმენიუში გრაფაზე კონფიგურაცია და შემდეგ დააწკაპუნე გრაფაზე ვექტორის ტიპის -> შეგროვების მეთოდის მატრიცა. გაიხსნება ვექტორის ტიპის -> შეგროვების მეთოდის მატრიცის ფანჯარა.

ნახ. 515 ვექტორის ტიპის->შეგროვების მეთოდის მატრიცის ფანჯარა

2.ველში ვექტორის ტიპი, შეარჩიეთ ვექტორის ტიპი, რომელსაც უნდა დაემატოს შეგროვების მეთოდების ჩამონათვალი, ან დაამატეთ იგი ხელით დაწკაპუნებით ღილაკზე. ასევე განოისახება ადრე დამატებული მეთოდები.

3.ახალი მეთოდის დასამატებლად დააწკაპუნეთ გრაფაზე ახალი სტრიქონის დასამატებლად დააწკაპუნეთ აქ ცხრილის ხაზზე და შეარჩიეთ იგი ჩამონათვალიდან ან დაამატეთ იგი დაწკაპუნებით ღილაკზე

4.შეინახეთ მონაცემები და დახურეთ ფანჯარა.
