"ვექტორული ზედამხედველობის სესიის" W02 ფორმის "ლაბორატორიული ტესტების" ქვეჩანართი

ვექტორული ზედამხედველობის სესიის ფორმის ლაბორატორიული ტესტების ქვეჩანართი გამოიყენება "ლაბორატორიულ მოდულში" სესიისთვის ჩატარებული ლაბორატორიული ტესტების ჩამონათვალის სანახავად.

ნახ. 207 ლაბორატორიული ტესტების ჩანართი

მონაცემთა ველები

პულის/ვექტორის საიდენტიფიკაციო ნომერი

ტესტისთვის გამოყენებული პულის/ვექტორის ID.

ვექტორის ტიპი

ტესტისთვის გამოყენებული ვექტორის ტიპი.

სახეობა

ტესტისთვის გამოყენებული ვექტორის სახეობა.

ლაბორატორიული ნიმუშის საიდენტიფიკაციო კოდი

ეკრანზე ჩანს ტესტისთვის გამოყენებული ნიმუშის ლაბორატორიული ID. ნიმუშის ლაბორატორიული ID ავტომატურად იქმნება დანომრვის უნიკალური სქემის შესაბამისად, როდესაც ხდება ნიმუშის მიღება "ლაბორატორიულ მოდულში".

ველზე ხელმისაწვდომობის ID

ტესტისთვის გამოყენებული ნიმუშის სავალე ID.

ნიმუშის ტიპი

ტესტისთვის გამოყენებული ნიმუშის ტიპი.

შეგროვების თარიღი

თარიღი, როცა აღებული იყო ტესტისთვის გამოყენებული ნიმუში.

ტესტის სახელი

ტესტის დასახელება.

ტესტის კატეგორია

ეკრანზე ჩანს ტესტის კატეგორია, თუ ცნობილია.

შედეგის თარიღი

თარიღი, როცა დაიწყო ტესტი ან როცა მისი შედეგი შეიყვანეს.

ვინ ჩაატარა ტესტირება

იმ თანამშრომლის სახელი, ვინც ტესტის შედეგი შეიყვანა.

ტესტირებულია ინსტიტუტის მიერ

ორგანიზაცია, რომელმაც ტესტი ჩაატარა.

ტესტის შედეგი

ტესტის შედეგი.

დავადება

დაავადება, რომელიც "ლაბორატორიულ მოდულში" კონკრეტული ტესტისთვის დაყენებულია როგორც ტესტის დიაგნოზი. თუ შედეგი დადებითია, დაავადება ასლის სახით გადადის სესიის დაავადებების ველში.

ღილაკები

პულის/ვექტორის დეტალების ნახვა

ხსნის პულის/ცალკეული ვექტორის დეტალების ფორმას, სადაც ჩანს საველე ტესტის ძირითადი პული/ცალკეული ვექტორი.

იხ. აგრეთვე:

✓ტესტის დანიშვნა/მინიჭება
