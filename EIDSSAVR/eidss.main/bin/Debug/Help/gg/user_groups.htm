მომხმარებელთა ჯგუფები

Show/Hide Hidden Text

სისტემაში ჩაწერილი თანამშრომლები შეიძლება დაჯგუფდეს. გამოიყენეთ თანამშრომელთა ჯგუფები, რათა ადვილად მიანიჭოთ წვდომა მონაცემთა გარკვეულ ტიპზე.

შექმენით მომხმარებელთა ჯგუფი:

1.დააწკაპუნე უსაფრთხოების მენიუში გრაფაზე მომხმარებლის ჯგუფები. გაიხსნება მმხმარებელთა ჯგუფების ფანჯარა

ნახ. 506 მომხმარებელთა ჯგუფების სია

2. დააწკაპუნეთ ღილაკზე ახალი. გაიხსნება მომხმარებელთა ჯგუფების დეტალების ფანჯარა

ნახ. 507 მომხმარებლის ჯგუფების დეტალები

3.შეიყვანეთ ჯგუფის სახელი და მოკლე აღწერილობა.

4.შექმნილ მომხმარებელთა ჯგუფისთვის მომხმარებლის/მომხმარებლის ჯგუფის დამატების მიზნით დააწკაპუნეთ ღილაკზე დამატება . გაიხსნება მომხმარებლის და ჯგუფების სიების ფანჯარა.

5.მომხმარებლის ან მომხმარებლის ჯგუფის შესარჩევად, რომელიც უნდა ჩაერთოს შექმნილ მომხმარებლის ჯგუფში, დააწკაპუნეთ ღილაკს შერცევა.

6.სისტემის ფუნქციებთან მომხმარებლის ჯგუფის უფლებების განსაზღვრისთვის გახსენით სისტემის ფუნქციების ჩანართი.

ნახ. 508 სისტემის ფუნქციების ჩანართი

7.შეარჩიე საიტის ჩამოსაშლელ ველში ის საიტი, სადაც გამოყენებული იქნება მომხმარებლის ჯგუფისთვის მინიჭებული უფლებები. ან დააწკაპუნეთ გრაფაზე გამოიყენება ყველა საიტისთვის, რათა მინიჭებული უფლებები გამოყენებული იქნეს მომხმარებელთა ჯგუფის მიერ დზეის-ის ყველა საიტის მიმართ.

8.ჩამოყევით სისტემის ფუნქციებს და აირჩიეთ უარყოფის ან ნებართვისსაკონტროლო უჯრა, რათა უარყოთ ან დაუშვათ შესაბამისი ფუნქცია მომხმარებლის ჯგუფისთვის.

9.დააწკაპუნეთ Ok რათა დახუროთ მომხმარებლის ჯგუფის დეტალების ფანჯარა.

10.დააწკაპუნეთ ღილაკზე დახურვა, რათა დახუროთ ფანჯარა მომხმარებლის ჯგუფების სია.
მომხმარებლის ჯგუფის რედაქტირება:

1.დააწკაპუნე უსაფრთხოების მენიუში გრაფაზე მომხმარებლის ჯგუფები. გაიხსნება მმხმარებელთა ჯგუფების ფანჯარა

ნახ. 509 მომხმარებელთა ჯგუფების სია

2.დააწკაპუნეთ რედაქირების ღილაკზე. გაიხსნება მომხმარებელთა ჯგუფების დეტალების ფანჯარა

ნახ. 510 მომხმარებლის ჯგუფების დეტალები

3.მოაცდინეთ ჯგუფის სახელის ან მოკლე აღწერილობის რედაქტირება.

4.შექმნილი მომხმარებლი/ მომხმარებლის ჯგუფისთვის დასამატებლად დააწკაპუნეთ ღილაკზე დამატება და ფანჯარაში მომხმარებლის და ჯგუფების სია შეარჩიეთ მომხმარებელი/მომხმარებლის ჯგუფი და დააწკაპუნეთ შერჩევის ღილაკს.

5.სისტემის ფუნქციებთან მომხმარებლის ჯგუფის უფლებების განსაზღვრისთვის გახსენით სისტემის ფუნქციების ჩანართი.

ნახ. 511 სისტემის ფუნქციების ჩანართი

6.შეარჩიე საიტის ჩამოსაშლელ ველში ის საიტი, სადაც გამოყენებული იქნება მომხმარებლის ჯგუფისთვის მინიჭებული უფლებები. ან დააწკაპუნეთ გრაფაზე გამოიყენება ყველა საიტისთვის, რათა მინიჭებული უფლებები გამოყენებული იქნეს მომხმარებელთა ჯგუფის მიერ დზეის-ის ყველა საიტის მიმართ.

7.ჩამოყევით სისტემის ფუნქციებს და აირჩიეთ უარყოფის ან ნებართვისსაკონტროლო უჯრა, რათა უარყოთ ან დაუშვათ შესაბამისი ფუნქცია მომხმარებლის ჯგუფისთვის.

8.დააწკაპუნეთ Ok რათა დახუროთ მომხმარებლის ჯგუფის დეტალების ფანჯარა.

9.დააწკაპუნეთ ღილაკზე დახურვა, რათა დახუროთ ფანჯარა მომხმარებლის ჯგუფების სია .
მომხმარებლის ჯგუფის წაშლა:

1.დააწკაპუნე უსაფრთხოების მენიუში გრაფაზე მომხმარებლის ჯგუფები. მომხმარებლის ჯგუფების სია გაიხსნება.

ნახ. 512 მომხმარებელთა ჯგუფების სია

2.შეარჩიეთ წასაშლელი ჯგუფი.

3.დააწკაპუნეთ წაშლის ღილაკზე. გახისნება ჩანაწერის წაშლის ფანჯარა.

ნახ. 513 დაადასტურეთ წაშლა

4.დააწკაპუნეთ Ok რათა დაადასტუროთ წაშლა.

5.დააწკაპუნეთ ღილაკზე დახურვა, რათა გამოხვიდეთ მომხმარებელთა როლების სიის ფანჯრიდან.
