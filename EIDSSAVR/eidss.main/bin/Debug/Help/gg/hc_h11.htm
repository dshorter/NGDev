ადამიანის დაავადების შემთხვევის დედუპლიკაციის დეტალები H11

Show/Hide Hidden Text

ადამიანის დაავადების შემთხვევის დედუპლიკაციის დეტალები ფორმა გამოიყენება ადამიანის დაავადების ორი შემთხვევის შესადარებლად.

ნახ. 581 დასატოვებელი შემთხვევის შერჩევა დაავადების შემთხვევების შედარების გზით

დასატოვებელი არის დაავადების შემთხვევა, რომელიც დარჩება სისტემაში; წასაშლელი არის დაავადების შემთხვევა, რომელიც წაიშლება სისტემიდან.

 

შეტყობინების ჩანართი იხსნება დეფოლტით, რომელიც გამოიყენება ორ შემთხვევას შორის არსებითი განსხვავების სანახავად. ერთმანეთისგან განსხვავებული ყველა ველი აღნიშნულია წითელი ფერით.

ველი/ცვლადი

აღწერა

შემთხვევის ID

შემთხვევის ID ველში ნაჩვენებია ადამიანის დაავადების შემთხვევის საიდენტიფიკაციო ნომერი.

ადგილობრივი იდენტიფიკატორი

ადგილობრივი იდენტიფიკატორი ველში ნაჩვენებია დაავადების შემთხვევის ადგილობრივი ID.

დიაგნოზი

დიაგნოზი ველში ნაჩვენებია დაავადების შემთხვევის მიმდინარე დიაგნოზი.

დიაგნოზის თარიღი

დიაგნოზის თარიღი ველში ნაჩვენებია საწყისი დიაგნოზის თარიღი.

გვარი

გვარი ველში ნაჩვენებია პაციენტის გვარი.

სახელი

სახელი ველში ნაჩვენებია პაციენტის სახელი.

მამის სახელი

მამის სახელი ველში ნაჩვენებია პაციენტის მამის სახელი.

დაბადების თარიღი

დაბადების თარიღი ველში ნაჩვენებია პაციენტის დაბადების თარიღი.

ასაკი

ასაკი ველში ნაჩვენებია პაციენტის წლოვანება ასაკის ერთეულებში.

სქესი

სქესი ველში ნაჩვენებია პაციენტის სქესი.

რეგიონი

რეგიონი ველში ნაჩვენებია პაციენტის ამჟამინდელი საცხოვრებელი ადგილის რეგიონი.

რაიონი

რაიონი ველში ნაჩვენებია პაციენტის ამჟამინდელი საცხოვრებელი ადგილის რაიონი.

ქალაქი ან სოფელი

ქალაქი ან სოფელი ველში ნაჩვენებია პაციენტის ამჟამინდელი საცხოვრებელი ადგილის დასახლება.

ქუჩა

ქუჩა ველში ნაჩვენებია იმ ქუჩის დასახელება, სადაც ამჟამად ცხოვრობს პაციენტი.

საფოსტო კოდი

საფოსტო ინდექსი ველში ნაჩვენებია პაციენტის ამჟამინდელი მისამართის საფოსტო ინდექსი.

სახლი/კორპუსი/ბინა.

სახლი/კორპუსი/ბინა ველში ნაჩვენებია პაციენტის ამჟამინდელი საცხოვრებელი სახლის ნომერი, კორპუსის ნომერი და ბინა.

ტელეფონის ნომერი

ტელეფონის ნომერი ველში ნაჩვენებია პაციენტის ამჟამინდელი საცხოვრებელი ადგილის ტელეფონის ნომერი.

მოქალაქეობა

მოქალაქეობა ველში ნაჩვენებია პაციენტის მოქალაქეობა.

დამქირავებლის დასახელება

დამქირავებლის დასახელება ველში ნაჩვენებია სამუშაო ადგილი, სკოლა, საბავშვო ბაღი, რაიმე დაწესებულება ან ნებისმიერი სხვა ადგილი, სადაც პაციენტი რეგულარულად დადის და ურთიერთობს მრავალ ადამიანთან.

დამქირავებლის მისამართი

დამქირავებლის მისამართი ველში ნაჩვენებია ზედა ველში მოყვანილი ორგანიზაციის მისამართი.

სიმპტომების დაწყების თარიღი

სიმპტომების დაწყების თარიღი ველში ნაჩვენებია პირველი სიმპტომების გამოვლენის თარიღი.

პაციენტის საბოლოო მდგომარეობა

პაციენტის საბოლოო მდგომარეობა ველში ნაჩვენებია, პაციენტი გარდაიცვალა თუ ცოცხალია.

ამ ველის სიდიდე აღებულია ველიდან პაციენტის მდგომარეობა შეტყობინების მომენტში.

შეცვლილი დიაგნოზი

შეცვლილი დიაგნოზი ველში ნაჩვენებია ის დიაგნოზი, რომელიც საწყისი დიაგნოზის ნაცვლად იქნა შერჩეული შემთხვევის კვლევის პროცესის დროს.

დიაგნოზის შეცვლის თარიღი

დიაგნოზის შეცვლის თარიღი ველში ნაჩვენებია თარიღი, როცა შეიცვალა დიაგნოზი.

ამჟამინდელი ადგილსამყოფელის ტიპი

ამჟამინდელი ადგილსამყოფელის ტიპი ველში ნაჩვენებია პაციენტის ამჟამინდელი ადგილსამყოფელი ველის სიდიდე შეტყობინების მომენტში, რომელშიც მითითებულია პაციენტის ადგილსამყოფელი.

ამჟამინდელი ადგილსამყოფელის დასახელება

ამჟამინდელი ადგილსამყოფელის დასახელება ველში ნაჩვენებია ამჟამინდელი ადგილსამყოფელის მისამართი ან დასახელება შეტყობინების მომენტში.

დამატებითი ინფორმაცია

დამატებითი ინფორმაცია ველში ნაჩვენებია მნიშვნელოვანი დამატებითი ინფორმაცია.

"ნიმუშების" ჩანართი გამოიყენება დაავადების შემთხვევებიდან აღებული ნიმუშების ჩამონათვალზე დაკვირვებისთვის. ყველა ნიმუშს გააჩნია მოსანიშნი უჯრა. მოსანიშნი უჯრის მონიშვნის შემთხვევაში ნიმუშები დაემატება/დარჩება დასატოვებელი შემთხვევის ნიმუშებში. თუ ნიმუში მიღებული/რეგისტრირებული იყო "ლაბორატორიულ მოდულში", მაშინ შეუძლებელი იქნება მონიშნული უჯრის მონიშვნის მოხსნა.

ნახ. 582 "შემთხვევის დედუპლიკაციის ნიმუშების" ჩანართი

ველი/ცვლადი

აღწერა

ნიმუშის ID

ნიმუშის ID ველი ავტომატურად ივსება "ლაბორატორიულ მოდულში" ნიმუშისთვის ტესტის დანიშვნის შემდეგ.

ნიმუშის ტიპი

ნიმუშის ტიპი ველი განსაზღვრავს აღებული ნიმუშის ტიპს.

ნიმუშის აღების თარიღი

შეგროვების თარიღი ველში შეტანილია მონაცემები ნიმუშის აღების თარიღის შესახებ.

ტესტების რაოდენობა

ტესტების რაოდენობა ველში ნაჩვენებია, თუ რამდენი ტესტი დაინიშნა ნიმუშზე.
