AVR–ის მთავარი ფანჯარა

AVR–ის მთავარი მენიუ ძირითადი მოთხოვნის და სქემის ოპერაციების, მონაცემების იმპორტის, AVR პარამეტრების, AVR ოლაინ დახმარების და ა.შ. წვდომის საშუალებას იძლევა ყველაზე ხშირად გამოყენებადი ოპერაციები წარმოდგენილია ღილაკების სახით, ინსტრუმენტების პანელზე. მოთხოვნის და სქემის ხე უნდა შეიცავდეს მოთხოვნებს და მათ სქემებს, რომლებიც საქაღალდეების მიხედვითაა დაჯგუფებული. თუმცა, წინასწარი შეთანხმებით, დისპლეიზე გამოტანილია წინასწარ განსაზღვრული მოთხოვნები. გამოქვეყნებული მოთხოვნები და სქემები აღნიშნულია მუქი შრიფტით.
