ჯგუფის მიღების ფორმა გამოიყენება სხვადასხვა ტიპის იმ ნიმუშების ჯგუფის ერთდროულად მისაღებად, რომლებიც დაკავშირებულია სხვადასხვა შემთხვევებთან ან სესიებთან, მაგრამ გააჩნიათ ერთი და იგივე ლოკალური/საველე ნიმუშის ID.

ნახ. 301 "ნიმუშების ჯგუფური მიღების" ფანჯარა

მონაცემთა ველები

ნიმუშის ლოკალური/საველე საიდენტიფიკაციო კოდის სკანირება

შეიტანეთ ხელით ან შტრიხკოდის სკანერის მეშვეობით ნიმუშები ჯგუფის ლოკალური/საველე საიდენტიფიკაციო კოდი. მას შემდეგ, რაც დაკლიკავთ დამატების ღილაკზე, ნიმუშის შერჩევის ფანჯარა გაიხსნება ნიმუშების ჯგუფით, რომლისთვისაც შეტანილია ნიმუშის ლოკალური/საველე ID.

ნიმუშის ტიპი

ნიმუშის ტიპი.

შენახვის ადგილი

შეარჩიეთ ადგილი სანახში მიღებული ნიმუშისთვის:

1.დააწკაპუნეთ დეტალების ღილაკზე (). გაიხსნება საყინულეების ფანჯარა.

2.შეარჩიეთ სასურველი ადგილმდებარეობა.

3.დააწკაპუნეთ OK ღილაკზე. დაიხურება საყინულეების ფანჯარა. მითითებული ადგილი ჩანს გრაფაში

ხელმისაწვდომია მხოლოდ იმ ნიმუშებისთვის, რომლებიც მიღებულია კარგ ან ცუდ მდგომარეობაში (სტატუსით მიღებულია, მიღებული ნიმუშის მდგომარეობა არაა უარყოფილი).

ლოკალური/საველე ნიმუშის ID

ლოკალური/საველე ნიმუშის ID

შემთხვევის საიდენტიფიკაციო კოდი / სესიის საიდენტიფიკაიცო კოდი

შემთხვევის საიდენტიფიკაციო კოდი / სესიის საიდენტიფიკაიცო კოდი

შტრიხ-კოდი (ლაბორატორიული ნიმუშის საიდენტიფიკაციო კოდი)

ნიმუშის მიღების შემდეგ ავტომატურად მინიჭებული ნიმუშის ID შეიძლება შეიცვალოს ხელით. ხელმისაწვდომია მხოლოდ მიღების შემდეგ.

შეგროვების თარიღი

ნიმუშის აღების თარიღი

მიღების თარიღი

მიმდინარე თარიღი გამოიყენება, როგორცმიღების თარიღინიმუშის მიღების შემდეგ,და შეიძლება ხელით შეიცვალოს, საჭიროების შემთხვევაში. ხელმისაწვდომია მხოლოდ მიღების შემდეგ.

ნიმუშის მდგომარეობა მიღებისას

ნიმუშის მდგომარეობა მიღების შემდეგ მიღებულია კარგ მდგომარეობაში გამოიყენება წინასწარშეთანხმებით და შეიძლება ხელით შეიცვალოს, ჩამოსაშლელი სიიდან შერჩევის გზით. თუ შეარჩევთ უარყოფილია ან მიღებულია ცუდ მდგომარეობაში, უნდა დაუმატოთ განმარტება კომენტარების გრაფაში. ხელმისაწვდომია მხოლოდ მიღების შემდეგ.

კომენტარები

კომენტარები თუ შეარჩევთ უარყოფილია ან მიღებულია ცუდ მდგომარეობაში პუნქტს მიღებისას ნიმუშის მდგომარეობის გრაფაში, კომენტარები სავალდებულო გახდება, სიტუაციის განმარტებისთვის.

ხელმისაწვდომია მხოლოდ მიღების შემდეგ.

ფუნქცონალური არე

შეარჩიეთ ორგანიზაციის დეპარტამენტი, ნიმუშის მისაღებად, ჩამოსაშლელი სიიდან.

ხელმისაწვდომია მხოლოდ იმ ნიმუშებისთვის, რომლებიც მიღებულია კარგ ან ცუდ მდგომარეობაში (სტატუსით მიღებულია, მიღებული ნიმუშის მდგომარეობა არაა უარყოფილი).

ღილაკები

დამატება

გაიხსნება ნიმუშების სია, ნიმუშის ლოკალური/საველე საიდენტიფიკაციო კოდით ნიმუშის შერჩევის ფანჯარაში:

1.შეიტანეთ ხელით ან შტრიხკოდის სკანერის მეშვეობით ნიმუშების ჯგუფის ლოკალური/საველე საიდენტიფიკაციო კოდი ნიმუშის ლოკალური/საველე ID-ის გრაფაში

2.დაკლიკეთ დამატების ღილაკზე. გაიხსნება ნიმუშების შერჩევის ფანჯარა, ნიმუშების ჯგუფით, რომლებისთვისაც შეტანილია ნიმუშის ლოკალური/საველე საიდენტიფიკაციო კოდი.

შტრიხკოდების ამობეჭდვა

ბეჭდავს შტრიხ-კოდის ეტიკეტებს შერჩეული ნიმუშებისთვის.

1.შეარჩიეთ ნიმუშები, რომლენსაც შტრიხ-კოდის ეტიკეტები სჭირდება.

2.დაკლიკეთ შტრიხ-კოდების ბეჭდვის ღილაკზე. გაიხსნება შტრიხ-კოდების ბეჭდვის ფანჯარა, ნიმუშების შტრიხ-კოდების წინასწარ დათვალიერების რეჟიმში.

3.დაკლიკეთ ბეჭდვის ღილაკზე ().

ყველას მიღება

ბადეში ყველა ნიმუშის მიღებისთვის დაკლიკეთ ყველას მიღების ღილაკზე, მიღებული გახდება ყველა ნიმუში, რომელიც არ იყო მიღებული. ისეთი ატრიბუტები, როგორიცაა მიღების თარიღი, ნიმუშის მდგომარეობა მიღებისას, შენახვის ადგილმდებარეობა, კომენტარი და ფუნქციონალური ზონა გახდება რედაქტირებადი.

ამოღება

წაშლის ყველა შერჩეულ ნიმუშს ჯგუფის მიღების ფანჯრიდან. შესაძლებელია მხოლოდ იმ ნიმუშების მოცილება, რომლებიც ჯერ არაა მიღებული.

1.შეარჩიეთ სიიდან ამოსაღები ნიმუში.

2.დაკლიკეთ მოცილების ღილაკზე. გაიხსნება შეტყობინების ფანჯარა და მოითხოვს წაშლის დადასტურებას.

3.დააწკაპუნეთ OK ღილაკზე. დაიხურება შეტყობინების ფანჯარა, ნიმუში ამოიშლება ჯგუფის მიღების ფანჯრიდან

დამახსოვრება

ცვლილებების დამახსოვრება.

დიახ

დახურავს ჯგუფის მიღების ფანჯარას ცვლილებების შენახვის გარეშე, თუ არ მოხდება მათი შენახვა დახურვამდე.

გაუქმება

აუქმებს ყველა ცვლილებებს, იმ შემთხვევაშიც კი. თუ ისინი დამახსოვრებულია და ხურავს ფანჯარას.
