ნიმუშის მონაცემების ნახვა ან რედაქტირება

ტესტის მონაცემები წარმოადგენს სრულყოფილ ინფორმაციას სისტემაში შენახული ტესტის შესახებ. ზოგიერთი მონაცემის შეცვლა შესაძლებელია ლაბორატორიული სექციის ფორმის ცხრილში. ნაკლებად გამოყენებული ინფორმაცია დამალულია ნიმუშის დეტალების პანელში. მეტი ინფორმაციის მისაღებად, წაიკითხეთ სექციები ლაბორატორიული სექცია L25 და ნიმუშის მონაცემების რედაქტირება (LS).

იგივე ინფორმაციის პოვნა შესაძლებელია ნიმუშის ფორმაში. მეტი ინფორმაციის მისაღებად, წაიკითხეთ სექციები ნიმუშის მონაცემების რედაქტირება (IF) და ნიმუში L02.
