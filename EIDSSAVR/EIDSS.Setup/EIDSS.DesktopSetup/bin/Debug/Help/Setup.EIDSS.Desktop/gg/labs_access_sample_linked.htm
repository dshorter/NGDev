ის ნიმუშები, რომლებიც მიბმულია დაავადების შემთხვევაზე/სესიაზე, შეგიძლიათ ჯერ შეიყვანოთ ამ შემთხვევაში/სესიაში და მოგვიანებით მოახდინოთ მათი მიღება "ლაბორატორიულ მოდულში", ან ერთდროული მიღების ოპერაციით დაამატოთ ისინი "ლაბორატორიულ მოდულში". დაავადების შემთხვევაზე ან სესიაზე მიბმული ახალი ნიმუშების შექმნისა და მათი ერთდროული მიღების შესახებ მეტი ინფორმაციისთვის, წაიკითხეთ სექცია დაავადების შემთხვევაზე/სესიაზე მიბმული ნიმუშის შექმნა და მიღება (LS). შესაძლებელია მხოლოდ ისეთი ნიმუშების მიღება, რომელთა სტატუსია არ არის მიღებული.

 

1.ჟურნალების მენიუში შეარჩიეთ ლაბორატორიული სექცია. გაიხსნება ლაბორატორიული სექციის ფანჯარა.

2.იპოვეთ სასურველი ნიმუშ(ებ)ი: გამოიყენეთ ორგანიზაცია, სადაც გაიგზავნა ძებნის კრიტერიუმი, რომელიც დააყენეთ მიმდინარე ორგანიზაციაზე მხოლოდ იმ ნიმუშების გასაფილტრად, რომლებიც შეიძლება გადაიგზავნოს/მიიღოთ.

3.გააკეთეთ შემდეგიდან ერთ–ერთი:

•დააწკაპუნეთ ნაპოვნი ნიმუშის ნიმუშის მდგომარეობა მიღებისას უჯრაზე და ჩამოსაშლელ მენიუში, რომელიც ხელმისაწვდომია მხოლოდ ერთი ნიმუშისთვის, შეარჩიეთ ამ ნიმუშის მდგომარეობა. თუ მდგომარეობა არ არის სათანადო, გაიხსნება ფანჯარა კომენტარი ნიმუშის მიღებისას, სადაც შეძლებთ ახსნა-განმარტების შეყვანას. ველში კომენტარები შეიყვანეთ ახსნა-განმარტება და დააწკაპუნეთ დიახ ღილაკზე . დაიხურება ფანჯარა კომენტარი ნიმუშის მიღებისას.

•მაუსის მარჯვენა ღილაკით დააწკაპუნეთ მისაღებ ნიმუშ(ებ)ზე და ჩამოსაშლელ სიაში მონიშნეთ მიღება, ხოლო შემდეგ მისი (მათი) მდგომარეობა.
 

ნახ. 246 კომენტარი ნიმუშის მიღებისას

თუ მდგომარეობა არის სათანადო ან არასათანადო, მოხდება ნიმუშების განახლება შემდეგი სახით:

•ნიმუშების სტატუსი ავტომატურად იცვლება სტატუსით მიღებულია.

•მიმდინარე თარიღით შეივსო ნიმუშის მიღების თარიღის უჯრა და მისი შეცვლა შესაძლებელია.

•ნიმუშის ლაბორატორიული ID დროებით დაყენებულია სტატუსზე (ახალი) და მუდმივ ნომერს მიიღებს დამახსოვრების შემდეგ, დანომრვის უნიკალური სქემის შესაბამისად; თუ იგი ხელით არ იქნება შეცვლილი. ნიმუშის ID შეიძლება შეიცვალოს ხელით. თუ შეიცვლება, მაშინ გამოიყენება მითითებული სიდიდე.

თუ ნიმუშის მდგომარეობა მიღებისას ველის სიდიდე შეიცვალა სიდიდით დაწუნებულია და მოხდა მისი დამახსოვრება, ასეთი ნიმუში დაიმალება და შეუძლებელი იქნება მისი პოვნა ძებნის ფუნქციის გამოყენებით

4.დაიმახსოვრეთ ნიმუშის მიღება და ლაბორატორიული ნიმუშის ID მიენიჭება ავტომატურად.
