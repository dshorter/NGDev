"გაფართოებული ძებნა" წარმოადგენს ფილტრაციის ერთ-ერთ ინსტრუმენტს, რომელიც გამოიყენება მხოლოდ იმ ჩანაწერების მისაღებად, რომლებიც აკმაყოფილებს ძებნის შეყვანილი კრიტერიუმების ჩამონათვალს. შერჩევის კრიტერიუმების დასადგენად გამოიყენება "ძებნის პანელი".

გაფართოებული ძებნის გამოყენება შესაძლებელია როგორც "ინდივიდუალურ ფორმებში", ისე "ლაბორატორიულ სექციაში".

ნახ. 243 "ლაბორატორიული სექციის" ფორმის "ძებნის პანელი"

1.გახსენით ჩამონათვალის ფორმა, რომელიც გამოიყენება სასურველი ჩანაწერების ეკრანზე გამოსატანად (ნიმუშების ჩამონათვალის ფორმა ნიმუშებისთვის, ლაბორატორიული ტესტების ჩამონათვალის ფორმა ტესტებისთვის, და ა.შ.).

2."ძებნის პანელის" ველებში შეარჩიეთ ფილტრაციის კრიტერიუმები. თითოეული ჩამონათვალის ფორმა იყენებს შერჩევის კრიტერიუმების საკუთარ უნიკალურ ნაკრებს.

3.დააწკაპუნეთ ძებნის ღილაკზე (). ჩამონათვალის ფორმაში მოყვანილია ის ჩანაწერები, რომლებიც აკმაყოფილებენ ფილტრაციის კრიტერიუმებს.

 

ლაბორატორიული სექციის ფორმაში შესაძლებელია ძებნის ფუნქციით ნაპოვნი ლაბორატორიული ჩანაწერების შედეგების ჩამონათვალის დამატებით გაფილტვრა მარტივი ძებნის გამოყენებით ან ხელით. მარტივი ძებნის შესახებ მეტი ინფორმაციის მისაღებად წაიკითხეთ "მარტივი ძებნის" ფუნქცია "ლაბორატორიულ სექციაში" სექცია) ან ხელით. ხელით ფილტრაციის შემთხვევაში, მონიშნეთ ჩამონათვალიდან ამოსაღები ჩანაწერები, მაუსის მარჯვენა ღილაკით დააწკაპუნეთ ამ ჩანაწერებზე და გამოტანილ ჩამოსაშლელ მენიუში მონიშნეთ ფუნქცია ამოღება. ჩამონათვალიდან შესაძლებელია მხოლოდ დამახსოვრებული ჩანაწერების ამოღება.

დამატებითი ინფორმაცია ძებნის პანელის შესახებ იხ. ძებნის სიების სექცია.
