ტესტი -> ტესტის შედეგის მატრიცა წარმოადგენს ტესტების ეტალონური შედეგების განსაზღვრისთვის შექმნილ ინტრუმენტს.

შექმენით შესაძლო შედეგები სია ტესტისთვის:

1.დაკლიკეთ სისტემის მენიუში კონფიგურაცია და შემდეგ დაკლიკეთ ტესტი -> ტესტის შედეგის მატრიცაზე. გაიხსნება ტესტი -> ტესტის შედეგის მატრიცის ფანჯარა.

ნახ. 503 ტესტი -> ტესტის შედეგის მატრიცის ფანჯარა.

2.ლაბორატორიული ტესტებისთვის შესაძლო შედეგების სიის შესაქმნელად, შეარჩიეთ ლაბორატორიული ტესტების ალმიანი გრაფა.

3.შეარჩიეთ ტესტის სახელის ველში ტესტი, რომლისთვისაც იქნება მითითებული შედეგი. შედეგები ემატება, დისპლეიზე გამოტანამდე ტესტის შედეგებში, შერჩეული ტესტის ზონისთვის.

4.ტესტის შედეგის დამატებისთვის შეარჩიეთ ტესტის საერთო შედეგების სიის ზონა და დაკლიკეთ მარცხენა ისრის ღილაკზე (). სიდიდე გადადის ტესტის შედეგებში, შერჩეული ტესტის ზონაში.

5.ტესტის შედეის წასაშლელად შეარჩიეთ ის ტესტის საერთო შედეგების სიის არეში და დაკლიკეთ მარჯვენაისრის ღილაკზე (). სიდიდე გადადის საერთო ტესტის შედეგების სიის არეში.

6.მონიშნეთ ინდიკატიური ალმიანი გრაფა ტესტის შედეგისთვის, თუ ეს კონკრეტული შედეგი აღნიშნავს იმ დიაგნოზის დაფიქსირებას, რომლისთვისაც ჩატარდა ტსტი. ტესტის ერთზე მეტი შედეგი შეიძლება მოინიშნოს, როგორც ინდიკატიური.

7.ადრეული ტესტებისთვის შესაძლო შედეგების სიის შესაქმნელად, შეარჩიეთ ადრეული ტესტების რადიო-ღილაკი.

8.გაიმეორეთ 3-6 ნაბიჯები შესაძლო შედეგების სიის შესაქმნელად.

9.ახალი ტესტის შედეგის შესაქმნელად საერთო ტესტის შედეგების სიის არეში, დააწკაპუნეთ ახალი ტესტის შედეგის დამატების ღილაკზე. გაიხსნება საბაზო საინფორმაციო ცხრილების რედაქტორის ფანჯარა.

10.დაამატეთ ახალი ტესტის შედეგები და დაკლიკეთ Ok ფანჯრის დასახურად.

11.შეინახეთ მონაცემები და დახურეთ ფანჯარა.
