ანგარიშის დიაგნოზის ჯგუფი-დიაგნოზის მატრიცა წარმოადგენს ინსტრუმენტს დიაგნოზების ჯგუფების სიის მითითებისთვის, სპეციალური ანგარიშის ტიპისთვის.

მოახდინეთ მატრიცის რედაქტირება:

1.დაკლიკეთ სისტემის მენიუში კონფიგურაციაზე და შემდეგ დაკლიკეთ ანგარიშის დიაგნოზთა ჯგუფების მატრიცაზე. გაიხსნება ანგარიშის დიაგნოზის ჯგუფი -> დიაგნოზის მატრიცა ფანჯარა.

ნახ. 479 ანგარიშის დიაგნოზის ჯგუფი -> დიაგნოზის მატრიცა

2.სპეციალური ანგარიშის ტიპებიდან შეარჩიეთ სპეციალური ანგარიში, რომელიც წინასწარაა განკუთვნილი კონკრეტული ქვეყნისთვის.

3.დიაგნოზთა ჯგუფებიდან შეარჩიეთ დიაგნოზთა ჯგუფი ჩამოსაშლელიი სიიდან.

4.შასრულეთ საჭირო ცვლილებები ბადეში.

5.შეინახეთ მონაცემები და დახურეთ ფანჯარა.
