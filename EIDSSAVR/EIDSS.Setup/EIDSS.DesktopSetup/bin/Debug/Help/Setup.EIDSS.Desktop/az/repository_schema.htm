Nümunənin soyuducuya yerləşniridməsi zamanı nümunənin dəqiq yerini aXEMS-də qeydə adına bilər.

Nümunənin yerini qeyd edilməsindən əvvəl ilk öncə saxlanılma yerinin sxemini müəyyən etmək zəruridir. Bunu etməyin ən yaxşı yolu YXEMS-in məlumatlar bazasında müəssisə saxlanılma yerinin virtual surətini yaratmaqdır.

Saxlanılma yerinin sxeminin yaradılması:

1.Jurnallur menyusunda Nümunələsin saxlanılma yerinin sxemi bəndini klikləyin. Souuducuların Siyahısı pəncərəsi açıcacaq.

Şəkil 48 Saxlanılma Yerinin Sxemi

Pəncərə ilk dəfə açıldığı zaman saxlanılma yerinin sxemi hələ müəylən olmadığına görə heç bir bənd göstərilmir.

2.Yeni bir soyuducu yaratmaq üçün Yarat düyməsini klikləyin. Soyuducunun sxemi pəncərəai açılacaq.

3.Soyuducunun sxemi pəncərəsinin sağ bölməsindəli uahələri drldurun və Yaddaşa Sal düyməsini klikləyin. Daxil edilən element əlavə edilir və pəncərənin sol hissəsində nümayiş olunur.

Şəkil 49 Saxlanılma yerinin sxeminin elementi

4.Soldakı çərçivədə soyuducunu seçin və soyuducunun yeni bir alt-şöbəsini yaratmav üçün Yeni düyməsini klikləyin.

Şəkil 50 Saxlanılma yerinin sxeminin alt-elementi

5.Bölmənin adını doldurun və völmənin növünü müəyyən edin. Lazım olarsa, cari alt-şöbədə Yerləşmə yerinin sayı və Bölmənin ştrixkodunu təyin edin. Hər hansı əlavə Qeddi əlavə edin və Yaddaşa sal düyməsini kliklə. Daxil edilən element əlavə olunur və pəncərənin sol bölməsində nümayiş edilir.

Qeyd: Yerləşmə yerlərin sayı cari rölmə üçün nümunələrin mümkün sayını müəyyən edir.

6.Data çox bölməni(ləri) yarmtlaq üçün 4-5-ci addımları təkrarlayın. Daxil edilən bütün elementlər əlavə edilir və pəncərənin sol bölməsində nümayiş olunur.

Bir sxem elementinin surətini çıxarmaq:

1.Siyahıdad lazım olan bəndi seç və Surətini çıxar düyməsini kliklə.

2.Surəti çıxarılan sxem elementinin strukturu orijinalla eyni olacaq. Aktiv olan dialoq qutusunda yeni sxem elementinin atributlarını dəyişdirin.

3.Dəyişiklikləri yadda saxlamaq üçün Yaddaşa sal düyməsini klikləyin.

Bir sxem elementini silmək:

1.Silmək üçün elementi seçin.

2.Bir elementi yaxud bölməni silmək üçün Sil düyməsini kliklə.

UNREGISTERED EVALUATION VERSION:

2.Hesabatı çap etmək üçün Prineer işarəsini kliknəyin.
