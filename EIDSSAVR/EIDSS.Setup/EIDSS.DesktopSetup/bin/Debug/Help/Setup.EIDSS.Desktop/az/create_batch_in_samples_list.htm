Testlər Siyahısı formasında dəst yarat

1.Jurnallar menyununda Laboratoriya Testləri seç. Laboratoriya Testləei Styahısı pəncərəsi açılacaq.

2.Dəstdə birləşdiriləcək ttstləri tap. Yslnız Başlanmayıb statusunda olan və digər dəstə aid olmayan eyni adlı testlər eyni dəstdə birləşdirilə bilər.

3.Dəst Hazırla düyməsini bas. Dəst Test Təfərrüatları forması açılacaq.

Şəkil 465 Dəst Test Təfərrüatları forması

4.Əgər dəst üçün bühün testlər siyahıda derilsə, onları aşağırakı üsulların biri ilə daxil et:
Birinci metod:

a)Testləri Əlavə et düyməsini bas. Laboratariya Testlərr Siyahısı pəncərəsi açılacaq. O, yalnız dəstə uyğun gələn testləri ehtiva edir (onların test statusları Başlanmayıb otur və test adı Tesnin Adı sahəsində dəst üçün seçilən test adı ilə eyni olur).

b)Test(ləri) dəstə əlvvə etmək üçün onu seç.

c)Seç düyməsini, və ya bütün testləri Hamısını Seç düyməsini basmaqla əlavə et. Laborator Testlərin Siyohısı pəncərəsi bağlanır, seçilmiş testlər siyahıya əlavə olunacaq.

İkinci metod:

a)Testə aid olan nümunənin labomatoriya nömrəsini formanın aşağısındakı Nümunənin laboratoriya nömrəsi ilə testi axtar sahəsnnə haxil et.

b)Axtxr düyməsini bas. Müəyyən edilmiş nümunənin meyarına cavab verən bütün testlər dəst testlərinin siyahısına əlavə ediləcək.

Üçüncü metod:

c)Testin təyin olunduğu nümunənin ştrix kod nişanını Testi Nümunənin laboratoriya nömrəsini skan etməklə əlavə et sahəsinə skan edin və ya ştrix kodu formanın aşağısına əl ilə daxil et.

d)Əlavə et düyməsini bas. Müəyyən edilmiş nümunənin meyarına cavab verən bütün testlər dəst testlərinin siyahısına əlavə ediləcək.

5.Əgər siyahı dəstdə olmamalı testlərdən ibarətdirsə, artıq testləri seç və Testi Sİl düyməsini bas.

6.Testi iera edəcək şəxsin adını Testi icra etdi xanasına daxil et.

7.Testin tarixini Testin Başlama tarixi xanasına və, məlumdursa, testin nəticəsini Nəticə xanasına daxil et.

8.Yaddaşda sdxla düyməsini bas və məlumatların yaddaşda saxlanılmasını təsdiq et.

9.OK düyməsini basaraq Dəst Test Təfərrüatları formasını bağla.

Əgər testlər üçün heç bir nəticə müəyyən edilməyibsə, onların və dəstin statusu Davam edir olmalıdır. iestlər üçün nəticənin daxil edilməsi və dəstin bağlanması barədə ətraflı məlumatlar əldə etmək üçün, Dəsti yekunyaşdır(İF) bölməsinə bax.
