Fəal Müşahidə Sesiyasının redaktə et

1.Jurnallar -> Fəal Müşahidə Sesiyaları düymələrini bas. Fəal Müşahidə Sesiyalarıh Siyahısı açılacaq.

Şəkil 355 Fəal Müşahidə Sesiyalarının siyahısı

2.Fəal Müşaiidə Sesiyssını Axtar.

3.Fəal Müşahidə Sesiyasını redaktə etmək üçün sesiyalardan birini siyahıdan seç və Reaaktə et düyməsini bas. Fəal Müşahidə Sesiyası formasıaçılacaq.

Şəkil 356 Fəal Müşahidə Sesiyası

4.Formanın xanalarında məlumtaları redaktə et. Forma barəsində dolğun məlumatlar əldə etmək üçün Fəal Müşahidə Seaiyaiı V18 bölməsinə bax.

5.Sesiyanı yaddaşda saxlamaq və formanı bağlamaq üçün Ok düyməsini bas.

 

Yadda saxlanılmış sesiyanın replakasiyası başlayacaq və yenilənmiş məlumatlar YXEMS serverinə ötürüləcək.
