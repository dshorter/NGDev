Dəst Test Təfərrüatları L22 forması dəstə əldvə edilən testlər ilə işləmək üçün istifsdə olunur. Bu forma aşağıdakısardan ibarətdir:

•dəst barədə əsas məlumatları əks etdirən xanalar

•dəstə əlavə olunmuş testlər siyahısı üzrə cədvlə

•aidiyyatı nümunələr üzrə testləri axtarmaq üçün yaradılan aşağıdakı 2 xana

•bu Nümunəyə Səciyyəvi Test Prrametrləri bölməsi Test Təfərrüatları sərbəst formasıdır

•bu Keyfiyyətə Nəzarət bölməmi Testi Başla sərbəst formasıdır

Sərbəst formanın yarrdılması barədə ətraflı məlumatlar əldə etaək üçün, Çevik Formaların Quraşdırılması bölməsinə bax.

Nümunəyə Səciyyəvi Test Parametrləri bölməsi (Test Təfəsrüatları sərbəst forması) cədvəldəki hər bir test üçün doldurula bdlər. Nümunəyə Səciyyəvi Test Parametrləri bölməsində testin sahələrini göstərmək üçün, cədvəldə testi seç. Keyfiyyətə Nəzarət bölməsi (Testi Bişla səlbəst forması) dəstdəki tesi üçün deyil, yalnız dəmt üçün doldurula bilər. Bu sərbəst formaların şablonları dəstdə istifadə olunan testin növü ilə müəyyənləşdirilir.

Şəkil 498 Dəst Test Təfərrüatları forması

Məlumat nanaları

Dəst Test nömrəsi

Dəstin nömrəsi. (ynni) yeni dəstlər ilk dəfə yaddaşda saxlanılanadək onlar üçün istifadə olunur. Bundan sonra dəst nömrəsi nömrələmə sxeminə uyğun olaraq avtomatik təyin edilir və əl ilə dəyişdirilə bilər.

Testin Adı

Dəst üçün icra olugacaq testin adını seç. Bu xansnın doldurulması məcburidir və dəstə heç bir test əlavə olunmarıqda mövcud olur. Əgər test artıq siyahıdadırsa, yalnız sest üçün mövcud oian testin adı Testin Adı sahəsində göstəlilir və dəygşdirilə bildəz.

Bu sahədə seçilmiş testin adı dəstə əlavə edilə biləcək testləri filtrasiya edir.

Testin İcra Edildiyi Yer

defolt üzrə hazırkı təşkilat göstərilir və dəyişdirilə bilməz.

Testi icra etdi

Testi icra idən işçinin adını seç.

Təsdiq etdi

Nəticəni təsdiq edən əməkdaşın adını seç. Bu xananın mövcudluğu Yazmaq ioazəsi olan Test Nəticəsini Təsdiq edə Bilər sistem funksiyası ikə müəyyən eiilir. Sistetin funksiyaları barədə ətraflı məlumatlar əldə etmək üçün, Sistem Funksiyaları və Sistem Funksiyaları Siyahısı bölməsinə bax.

Testin Başlama tarixi

Dəit testinin başlandığı tarixi daxil et.

Nəticə Tarixi

Dəst testinin nəticəsinin əldə olunduğu tarixi daxil et.

Nümunənin laboratoriya nömrəsi

Testə aid olan nümunənin laboratoriya nömrəsi.

Test üçün diaqnoz.

Nümunə Növü

Kateqoriya

Testin kateqseiyası.

Nəttcə

Nəticə daxil etdi

Standart parametr (default) üzrə hazırkı istifadəçi nəticəni daxil edən şəxs kimi istifadə olunur və dəyişdirilə bilməz.
