Матрица «Тест - результат теста» является инструментом, предназначенным для составления справочных результатов тестов.

Для создания списка возможных результатов теста необходимо:

1.Находясь в меню Система, нажмите пункт Настройка, а затем нажмите Матрица «Тест - результат теста». Откроется окно Матрица «Тест - результат теста».

Рис. 82 Окно «Матрица «Тест - результат теста»

2.Для создания списка возможных результатов лабораторных тестов, активируйте флажок на кнопке с фиксацией Лабораторные тесты.

3.В поле Название теста выберите тест, результаты которого будут указаны. Ранее добавленные результаты отображаются в поле Результаты тестов для выбранного теста.

4.Для добавления результата к тесту, выберите его в поле Список общих результатов тестов, и нажмите мышкой на кнопке «Стрелка влево» (). Значение переносится в поле Результаты тестов для выбранного теста.

5.Для удаления результата, выберите его в поле Результаты тестов для выбранного теста и нажмите мышкой кнопку «Стрелка вправо» (). Значение переносится в поле Список общих результатов тестов.

6.Активируйте флажок на кнопке Индикативный  для какого-либо результата теста, если именно этот результат указывает на выявление диагноза, в отношении которого проводился настоящий тест. В качестве индикативных можно выделять и большее количество результатов тестов.

7.Для создания списка возможных результатов полевых тестов, активируйте радиокнопку Полевые тесты.

8.Для создания списка возможных результатов, повторите шаги 3-6.

9.Для создания нового результата теста в поле Список общих результатов тестов, нажмите кнопку Добавить новый результат теста. Откроется окно Редактор базы справочных таблиц.

10.Добавьте новый результат теста и нажмите кнопку OK, чтобы закрыть окно.

11.Сохраните данные и закройте окно.
