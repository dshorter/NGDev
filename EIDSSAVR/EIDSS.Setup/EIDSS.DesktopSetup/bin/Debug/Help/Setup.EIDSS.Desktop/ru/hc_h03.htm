Форма Н03 «Список лиц» используется для поиска пациента, владельца фермы или человека из списка связанных с заболеванием человека лиц.

Рис. 241 Форма «Список лиц»

Раздел «Параметры поиска»

Рис. 242 Раздел «Параметры поиска»

Поле / переменное значение

Описание

Имя

Вводится имя лица.

Отчество

Вводится отчество лица.

Фамилия

Вводится фамилия лица.

Дата рождения

Вводится дата рождения лица.

Область

Выбирается область текущего места жительства лица.

Район

Выбирается район текущего места жительства лица.

Тип личного идентификатора

Выбирается тип личного идентификатора лица.

Личный идентификатор

Вводится личный идентификатор лица.
Раздел «Результаты поиска»

Рис. 243 Раздел «Результаты поиска»

Поле / переменное значение

Описание

Фамилия

Отображается фамилия лица.

Имя

Отображается имя лица.

Отчество

Отображается отчество лица.

Дата рождения

Отображается дата рождения лица.

Адрес

Отображается текущее место жительства лица.

Тип личного идентификатора

Отображается тип личного идентификатора человека.

Личный идентификатор

Отображается личный идентификатор человека.

Дата создания

Отображается дата, в которую лицо было добавлено в базу данных.

Дата последнего обновления

Отображается дата последнего обновления данных.

Кнопки

Новая/новый

Нажмите эту кнопку для открытия пустой формы Сведения о лице, и создайте в базе данных запись о новом человеке.

Редактировать

Нажмите эту кнопку для открытия формы Сведения о лице, и отредактируйте данные по выбранному лицу.

Удалить

Нажмите эту кнопку для удаления записи о человеке из базы данных.
