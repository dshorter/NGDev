Приём в систему / получение передававшихся(-ейся) существующих(-ей) проб(-ы), связанных(-ой) со случаем заболевания / сессией (ЛР)

Связанные со случаем заболевания или сессией пробы могут быть либо изначально занесёнными в случай заболевания / сессию, а позже принятыми в систему с помощью лабораторного модуля, либо быть добавлены с одновременным приёмом в систему в лабораторном модуле. Для получения более подробной информации о создании и одновременном приёме в систему новой пробы, связанной со случаем заболевания / сессией, прочитайте раздел Создание и приём в систему пробы, связанной со случаем заболевания / сессией (ЛР). Приняты в систему могут быть только те пробы, статусом которых является Не принята в систему.

 

1.Находясь в меню Журналы, выберите пункт Лабораторный раздел. Откроется окно Лабораторный раздел.

2.Найдите необходимую(-ые) пробу(-ы), для чего воспользуйтесь критерием поиска Направлено в организацию, в котором укажите текущую организацию, чтобы отфильтровать к показу только те пробы, которые могли быть получены обратно / приняты в систему.

3.Проделайте одно из следующих действий:

•Нажмите по полю Состояние пробы при получении найденной пробы и в раскрывающемся меню выберите её состояние, которое будет доступно только для одной пробы.Если её/их состояние не нормальное, откроется окно Комментарий о принимаемой в систему пробе, куда будет предложено ввести пояснение. Введите пояснение в поле Комментарии и нажмите кнопку ОК. Окно Комментарий о принимаемой в систему пробе закроется.

•Правой кнопкой мыши нажмите на пробе(-ах), которую(-ые) необходимо принять в систему и в раскрывающемся меню выберите пункт Приём в систему, а затем укажите её(их) состояние.
 

Рис. 304 Комментарий о принимаемой в систему пробе

Если состояние проб(-ы) нормальное или посредственное, происходит следующее обновление их(её) данных:

•Значение Статус пробы автоматически меняется на Принята в систему.

•Значение текущей даты заполняет поле Дата приёма в систему и его можно изменить.

•Значение Лабораторный идентификатор пробы временно устанавливается на Новый, но после сохранения получает постоянный номер, в соответствии со Схемой уникальной нумерации, если до того его не поменяли вручную. Идентификатор пробы можно изменить вручную. После такого изменения будет использоваться указанное значение.

Если значение поля Состояние пробы при получении изменили на Отклонена и сохранили, проба в системе будет скрыта и её невозможно будет найти посредством поиска.

4.Сохраните сеанс приёма в систему и ей автоматически будет присвоен Лабораторный идентификатор пробы .
