Вложенная вкладка «Полевые тесты» формы W02 «Сессия надзора за переносчиками заболевания»

Вложенная вкладка Полевые тесты используется для просмотра полевых тестов, проведённых в ходе сессии надзора за переносчиками заболеваний. Данные обо всех полевых тестах, проведённых в отношении каждого отдельного переносчика / группы переносчиков заболевания в рамках сессии, комбинируются для построения списка полевых тестов. Таблица, содержащая список полевых тестов, является нередактируемой. Более подробно о вводе информации о полевых тестах проб отдельного переносчика / группы переносчиков заболевания, можно прочитать в разделе Регистрация результатов полевых тестов отдельного переносчика / группы переносчиков заболевания.

Рис. 264 Вкладка «Полевые тесты»

Поля данных

Идентификатор группы переносчиков / отдельного переносчика заболевания

Отображается идентификатор, создаваемый автоматически в соответствии со Схемой уникальной нумерации.

Тип переносчика заболевания

Указывается тип переносчика заболевания, использовавшийся для проведения полевого теста.

Биологический вид

Указывается биологический вид отдельного переносчика / группы переносчиков заболевания.

Полевой идентификатор пробы

Указывается полевой идентификатор пробы, которой было назначено проведение теста.

Тип пробы

Указывается тип пробы, которой было назначено проведение теста.

Дата взятия

Указывается дата взятия пробы, которой было назначено проведение теста.

Название теста

Указывается название теста.

Категория теста

Указывается категория теста.

Дата завершения теста

Указывается дата проведения теста.

Тестировавшее учреждение

Указывается организация, проводившая тест.

Тестировавший работник

Указывается лицо, проводившее тест.

Результат теста

Указываются результаты теста, если они известны. При положительном результате, информация о заболевании копируется из столбца Заболевание в поле Заболевания сессии надзора за переносчиками заболеваний.

Заболевание

Указывается заболевание, которое должно подтверждаться тестом. Если заболевание подтверждается, оно копируется в поле Заболевания сессии надзора за переносчиками заболеваний, в рамках которого проводился такой тест.

Кнопки

Просмотр сведений о группе переносчиков / отдельном переносчике заболевания

Открывается форма «Сведения о группе переносчиков / отдельном переносчике заболевания», в которой отображается информация о родительской группе переносчиков / отдельном переносчике заболевания, использовавшихся для полевого теста.
