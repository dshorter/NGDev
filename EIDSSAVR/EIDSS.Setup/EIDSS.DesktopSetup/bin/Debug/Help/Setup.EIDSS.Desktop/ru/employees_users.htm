Весь персонал, который так или иначе участвует в расследовании случаев заболеваний людей, животных и переносчиков заболеваний, заносится в базу данных ЭИСНЗ.

Для просмотра списка работников / пользователей необходимо:

1.Находясь в меню Система, нажмите пункт Администрирование, а затем нажмите Работники. Откроется Список работников.

Рис. 114 Список работников

 

Для создания записи о новом работнике /пользователе необходимо::

1.Нажмите кнопку Новый. Откроется окно Сведения о работнике.

2.На вкладке Персональные данные, введите имя работника и сведения о его/её рабочей должности.

Рис. 115 Сведения о работнике - персональные данные

3.Если планируется, что такой работник будет пользователем системы, нажмите вкладку Имя пользователя. Откроется вкладка Имя пользователя.

Рис. 116 Учётные записи работников

4.Вкладка Имя пользователя содержит список учётных записей пользователей работников на разных сайтах. Первоначально он является пустым.

5.Для добавления новой учётной записи нажмите кнопку Новая. Откроется окно Учётное имя пользователя.

Рис. 117 Сведения о учётной записи работника

6.Введите относящуюся к работнику информацию в поля Сайт, Учётное имя пользователя, (5 или более символов), и Пароль.Пароль должен соответствовать требованиям, установленным в политике безопасности.

7.Повторно введите пароль в поле Подтвердить новый пароль.

8.Для сохранения изменений и закрытия окна нажмите кнопку ОК.

9.Для добавления всех необходимых учётных записей, повторите шаги с 5-го по 8-й.

10.Сохраните данные и закройте окно Сведения о работнике.
Для закрепления работником за группой, необходимо::

1.Находясь в окне Сведения о работнике, откройте вкладку Группы.

Рис. 118 Вкладка «Сведения о работнике - группы»

2.Нажмите кнопку Добавить. Откроется окно Список групп пользователей.

Рис. 119 Список групп пользователей

3.Отметьте в списке группу, за которой должен быть закреплён пользователь, и нажмите мышкой Выбрать.

4.Окно Список групп пользователей закроется, а выбранная группа отобразится в окне Сведения о работнике. После этого, имеющиеся права доступа у группы, за которой был закреплён пользователь, будут доступны и такому пользователю.

5.Нажмите Сохранить для сохранения любых сделанных изменений.
Для закрепления системных функций за пользователем необходимо::

1.Находясь в окне Сведения о работнике, откройте вкладку Системные функции.

Рис. 120 Вкладка «Сведения о работнике - системные функции»

2.Из раскрывающегося списка выберите Сайт.

3.Пролистайте список системных функций и выберите либо кнопку-флажок Запретить, либо Разрешить, чтобы запретить или разрешить соответствующую функцию.

4.Нажмите Сохранить для сохранения любых сделанных изменений
Для редактирования записи пользователя или работника::

1.Выберите запись пользователя для редактирования.

2.Нажмите кнопку Редактировать. Откроется окно Сведения о работнике.

3.На вкладке Персональные данные, введите имя работника и сведения о его/её рабочей должности.

Рис. 121 Сведения о работнике - персональные данные

4.Если планируется, что такой работник будет пользователем системы, нажмите вкладку Имя пользователя. Откроется вкладка Имя пользователя.

Рис. 122 Учётные записи работников

5.Вкладка Имя пользователя содержит список учётных записей пользователей работников на разных сайтах.

6.Для добавления новой учётной записи нажмите кнопку Новая. Откроется окно Учётное имя пользователя.

Рис. 123 Сведения о учётной записи работника

7.Введите относящуюся к работнику информацию в поля Сайт, Учётное имя пользователя, (5 или более символов), и Пароль. Пароль должен соответствовать требованиям, установленным в политике безопасности.

8.Повторно введите пароль в поле Подтвердить новый пароль. Если учётное имя работника меняется, пароль можно оставить старым.

9.Для сохранения изменений и закрытия окна нажмите кнопку ОК.

10.Для добавления всех необходимых учётных записей, повторите шаги с 5-го по 9-й.

11.Для удаления учётной записи, отметьте её и нажмите кнопку Удалить. Нажмите ОК для подтверждения удаления.

12.Сохраните данные и закройте окно Сведения о работнике.
Для удаления записи пользователя или работника необходимо::

1.Выберите запись работника, предназначенную для удаления.

2.Нажмите кнопку Удалить. Откроется диалоговое окно Удалить запись.

Рис. 124 Подтверждение удаления

3.Нажмите ОК для подтверждения удаления.
