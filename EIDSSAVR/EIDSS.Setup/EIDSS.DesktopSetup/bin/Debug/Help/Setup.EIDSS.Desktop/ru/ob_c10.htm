Форма С10 «Список вспышек заболеваний» представляет собой форму, в которой приведены все вспышки заболеваний, совпадающие с критериями поиска.

Рис. 254 Окно «Список вспышек заболеваний»

Раздел панели поиска содержит различные поля, используемые для поиска вспышки заболевания.

Рис. &lt;%FIG++%&gt; Параметры поиска

Идентификатор вспышки заболевания

В этом поле указывается идентификатор вспышки заболевания.

Текущий диагноз

В поле Диагноз можно указать диагноз вспышки заболевания. Для очистки содержимого поля, в раскрывающемся списке необходимо выбрать пустое поле.

Группа диагнозов

В поле Группа диагнозов можно указать группу диагнозов вспышки заболевания. Для очистки содержимого поля, в раскрывающемся списке необходимо выбрать пустое поле.

Дата начала

В этом поле указываются границы начала вспышки заболевания.

Дата окончания

В этом поле указываются границы окончания вспышки заболевания.

Основное местоположение вспышки заболевания

В этом поле, в формате, используемом в столбце Основное местоположение вспышки заболевания списка Результаты поиска, указывается основное местоположение вспышки заболевания.

Для расширения результатов поиска можно использовать символ звёздочки (*).

По умолчанию механизм поиска проверяет два первых символа значения столбца «Основное местоположение вспышки заболевания» на соответствие тексту, указанному в параметре поиска поля Основное местоположение вспышки заболевания.

Статус вспышки заболевания

В этом поле может быть указан статус вспышки заболевания. Для очистки содержимого поля, в раскрывающемся списке необходимо выбрать пустое поле.

Пациент

В это поле вводится имя пациента.

Владелец фермы

В это поле вводится имя владельца фермы, чьи животные связаны со случаем заболевания.

 
Раздел «Список результатов поиска» отображает перечень вспышек заболеваний, чьё описание подходит введённым критериям поиска.

Рис. 255 Раздел «Список вспышек заболеваний»

Идентификатор вспышки заболевания

В этом поле указывается идентификатор вспышки заболевания.

Диагноз / группа диагнозов

В этом поле указывается специальный диагноз вспышки заболевания.

Дата начала

В этом поле указывается дата начала вспышки заболевания.

Дата окончания

В этом поле указывается дата окончания вспышки заболевания.

Основное местоположение вспышки заболевания

В этом поле отображается местоположение вспышки заболевания в том виде, как оно определено в поле Формат географического местоположения.

Статус вспышки заболевания

В этом поле указывается статус вспышки заболевания.

Пациент / владелец фермы

В этом поле отображается имя пациента или владельца фермы, на которой зафиксировано возникновение основного случая вспышки заболевания.
