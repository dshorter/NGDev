Вкладка «Общая информация» формы W02 «Сессия надзора за переносчиками заболевания»

Панель Общая информация используется для ввода, просмотра и редактирования основной информации о сессии НЗПЗ, включающей в себя: время проведения сессии НЗПЗ, место её проведения, типы отловленных переносчиков заболеваний, а также заболевания, которыми они могут заразить.

Рис. 261 Окно «Общая информация»

Поля данных

Идентификатор сессии

Идентификатор сессии надзора за переносчиками заболеваний (новый) используется для новой, не сохранённой сессии. После первого сохранения сессии, вместо (нового) идентификатора, подставляется идентификатор, который генерируется в соответствии со Схемой уникальной нумерации.

Дата начала

Вводится дата начала сессии.

Дата закрытия

Здесь отображается дата, в которую статус сессии поменялся на Закрыта, после чего это поле становится нередактируемым.

Продолжительность отлова

Вводится количество часов, потраченных на отлов переносчиков заболеваний в рамках сессии.

Полевой идентификатор сессии

Вводится идентификатор сессии, присвоенный ей до момента её регистрации в ЭИСНЗ.

Статус

Вводится текущий статус сессии НЗПЗ. Редактировать можно только те сессии, у которых указан статус В процессе. Сессии со статусом Закрыта сохраняются и могут быть открыты, но их нельзя редактировать.

Описание

Вводится описание сессии НЗПЗ.

Тип переносчика заболевания

В этом поле автоматически отображаются в режиме только для чтения типы переносчиков заболеваний, отловленных в ходе сессии. Перечень типов переносчиков заболеваний определяется переносчиками заболеваний, зарегистрированными в рамках сессии на вложенной вкладке Группы переносчиков / отдельные переносчики заболеваний вкладки Подробная информация и вкладки Краткая информация.

Местоположение

Точное местоположение сессии выбирается из списка или вводится вручную. Нажмите на поле и откроется специальное окно для ввода местоположения. После того, как местоположение будет указано, нажмите кнопку ОК для закрытия этого специального окна. Данные отображаются в поле Местоположение.

Идентификатор вспышки заболевания

Вводится вручную или выбирается из списка вспышек заболеваний, относящихся к сессии. Для открытия списка всех зарегистрированных вспышек заболеваний, нажмите кнопку Список ().

Диагнозы

В этом поле отображаются диагнозы, подтверждённые результатами полевых тестов или контрольных лабораторных тестов отдельных переносчиков / групп переносчиков заболевания, отловленных в ходе сессии.
