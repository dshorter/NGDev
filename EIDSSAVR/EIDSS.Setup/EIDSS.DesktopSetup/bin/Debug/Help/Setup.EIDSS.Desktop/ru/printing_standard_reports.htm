ЭИСНЗ содержит список стандартных отчётов, которые можно изменять и/или распечатывать.Пользователь может использовать только те отчёты с данными, право на доступ к которым у него имеется.

Для открытия стандартного отчёта необходимо:

Находясь в меню Отчёты, нажмите пункт Стандартные отчёты, выберите тип отчёта, а затем нажмите название нужного отчёта.
