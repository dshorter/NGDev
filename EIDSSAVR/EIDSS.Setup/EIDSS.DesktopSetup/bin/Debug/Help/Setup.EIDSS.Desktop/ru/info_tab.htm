Вкладка Информация макета используется для редактирования названия и описания макета, а также для предоставления к нему общего доступа. Помимо этого, на ней хранится информация об исходном запросе, статусе публикования и последнем обновлении данных.

Рис. 456 Вкладка «Информация»

Название макета

Название макета указывается на английском и местном языке (в зависимости от обстоятельств). Это поле является обязательным для заполнения. Название макета используется для идентификации макета в окне Дерево запросов / папок / макетов; оно распечатывается на всех отчётах, диаграммах и картах.

Описание

Описание макета не является обязательным для заполнения полем, и может использоваться для указания подробных сведений о макете: какие-либо специальные фильтры, сортировка, и т.д.

Имя запроса

Представляет собой нередактируемое поле, в котором отображается имя исходного запроса макета.

Описание запроса

Является нередактируемым полем, в котором отображается описание исходного запроса макета.

Дата и время обновления данных

Является нередактируемым полем, в котором отображается дата и время последнего обновления данных.

Общий доступ к макету

Представляет собой кнопку с фиксацией, предназначенную для предоставления общего доступа к макету пользователям сайта, на котором такой макет был создан. Для получения более подробных сведений о предоставлении общего доступа к макету, прочитайте раздел Общий доступ к макету.

Статус публикования

Является нередактируемым полем, в котором отображается статус макета - был ли он опубликован или нет. Для получения более подробных сведений о публикации макета, прочитайте раздел Публикация макета / запроса / папки.
