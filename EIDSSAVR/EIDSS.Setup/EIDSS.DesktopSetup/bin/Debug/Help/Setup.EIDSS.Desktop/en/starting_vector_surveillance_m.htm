Before the first vector session is entered, configure the Vector Surveillance (VS) module. The VS module configuration is comprised of a Unique Numbering Schema, Reference Tables configuration and Matrices configuration. All of them are already set up by default, but it is recommended to open them and ensure that their presets meet your process requirements.

Even if the reference tables are customized beforehand, they can be adjusted any time afterwards. However, matrices and the numbering schema have to be set up in advance, their improper configuration will prevent the module from functioning correctly.
