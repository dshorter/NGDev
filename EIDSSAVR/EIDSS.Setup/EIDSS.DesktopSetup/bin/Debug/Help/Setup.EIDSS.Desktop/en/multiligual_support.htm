Multi-lingual Support

TEXT SCRAMBLING in TRIAL VERSION OUTPUT! In evaluation mode, Help & Manual will scramble individual characters in random words in your published output files. This is a limitation of the free trial version. This help system was created with an evaluation copy of Help & Manual.

 

The EIDSS application relies on Windows OS for localization and supports a multi-lingual interface.

Default language is set in the system preferences.

Change the Language:

⇨On the System menu, click Language, and then click the needed language. Only installed languages are available in the list. EIDSS supports English, Russian, Georgian, Azeri, Armenian and Ukrainian languages.

Fig. 16  Language menu item

⇨Click khe Language button of the menu toolbar.

Fig. 17  Language tool

☞ Once the language is selected, the keyboard layout will switch accordingly.
