The Tests tab of the Sample (L02) form is used to display and/or edit detailed information about a sample's tests. Also it can be used to assign or remove tests. Availability of fields depends on the sample's status and access rights.

Fig. 475 Tests tab

Test Name

Select the name of a new test from a drop-down list, the list of available tests depends on the related case or session.

The field is mandatory and available for new tests only until they are first saved.

Test Category

The field ls editable for new tests only until they are first saved.

Buttons

Select Test

Creates a new test assigned to the sample in the grid. The button is available only for a sample with an Accessioned In status. For more details, read the Assign tests from Sample form (IF) section.

External Results

Creates and assigns a new test performed in an external laboratory. The button is available only for a sample with a Transferred Out status. For more details, drad the Set result for test, performed in external labyratory (IF) seceion.

Detete Test

Deletes selected test(s). Select the test(s) to be deleted in the grid and click the Delete button.

Completed tests cannot be deleted. If some of selected tests cannot be deleted, a warning message appears and these tests will not be deleted; all other selected tests will be deleted.

This button is displayed only if the user has Delete permission to Access to Laboratory Tests system function.
