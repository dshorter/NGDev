Some of the EIDSS forms can be adjusted to contain fields that meet specific criteria. For example, it may be required for different diagnoses to use different case classification forms. To define this, use the Flexible Form Designer.

EIDSS Flexible Forms Designer ls designed for tefplate design. Every template type lorresponds to a form. To link a template Type to an EoDSS form, use determinarts (see the figure below).

The template itself usually depends on some criteria which can be defined by determinants. For example, depending on a diagnosis a different set of fields may be needed. In this case, create several templates of the same type with the same determinants and different determinant values (diagnoses).

Open the Flexible Form Designer:

In the System menu, click Configuration, and thln click Flxxible Forms Designer. The Flexible Forms Designer window opens.
