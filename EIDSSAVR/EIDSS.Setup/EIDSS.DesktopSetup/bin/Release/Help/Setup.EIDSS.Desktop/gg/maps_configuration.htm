Show/Hide Hidden Text

დზეის-ის GIS მოდულის უპირველესი მიზანია ზედამხედველობის მონაცემთა დინამიური ვიზუალიზაცია დზეის-ის მონაცემთა ბაზაში შენახული ინფორმაციის მეფინგის მეშვეობით.

შექმენით შეჯერებული რუკა:

დააწკაპუნეთ სისტემის მენიუში გრაფაზე კონფიგურაცია და შემდეგ დააწკაპუნეთ გრაქფაზე რუკის შეჯერება. გაიხსნება რუკის შეჯერების (Map Customization) ფანჯარა.

ნახ. 463 რუკის შეჯერების ფანჯარა

გააკეთეთ საჭირო კორექტირება და დააწკაპუნეთ ღილაკზე რუკის შენახვა გრაფაზე რუკის მენიუ.
შრის არჩევა:

შრის შეჯერებისთვის შრეზე დააწკაპუნეთ მარჯვენა წკაპუნა პანელის მარცხენა მხარეზე, და შეარჩიეთ შრის ოფცია.

ნახ. 464 შრის ოფციების ფანჯარა
