მოთხოვნის ველების სიის რედაქტირება

სქემისთვის ამოსაღები ჩანაწერების რიცხვის შესაზღუდად გამოიყენეთ ფილტრები. რეკომენდირებულია ფილტრების გამოყენება სქემის დამუშავების სიჩქარის გასაზრდელად.

ფილტრის კრიტერიუმი შედგება ფილტრის პირობებისგან, რომლებიც კომბინირებულია კომბინირების ოპერატირებით.

ნახ. 319 ფილტრები.

ფილტრის პირობების ნაწილები

შემდეგი სია შეიცავს ფილტრის პირობების ნაწილებს, რომლების მითითებაც შესაძლებელია:

•ცხრილის ველები შედარებისთვის;

•ფილტრის შედარების ოპერატორები, როგორებიცაა ტოლია, არ არის ტოლი, ცარიელია, არაა ცარიელი, და ა.შ..;

•ფილტრის ოპციები, როგორიცაა ნებისმიერი ან მსგავსი, და კრიტერიუმების სიდიდეები, როგორებიცაა ველის სიდიდეები, გამოსახულებები, სიდიდეების სიმრავლეები, და ქვე–მოთხოვნები.
