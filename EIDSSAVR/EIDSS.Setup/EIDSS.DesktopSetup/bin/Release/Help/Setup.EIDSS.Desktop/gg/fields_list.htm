ველების სია

მოთხოვნის რედქტორი შეიცავს შემდეგ ველების სიებს:

 

•ხელმისაწვდომი ველების სია არის ყველა იმ ველების სია, რომლებიც მიეკუთვნება შერჩეულ ძიების ობიექტს და რომლების შერჩვაც არ ხდება მოთხოვნისთის.

•ხელმისაწვდომი ველების სია არის იმ ველების სია, რომლებიც შეარჩია მომხმარებელმა მოთხოვნისთის.

 

ორივე სია შეიცავს მხოლოდ იმ ველებს, რომლებიც მიეკუთვნება შერჩეულ ძიების ობიექტს. იმ ველების დასანახად რომლებიც ძიების სხვადასხვა ობიექტებს მიეკუთვნება, შეარჩიეთ სასურველი ძიების ობიექტი და მისი ველები ავტომატურად გამოჩნდება სიებში, ადრე შერჩევის შესაბამისად.
