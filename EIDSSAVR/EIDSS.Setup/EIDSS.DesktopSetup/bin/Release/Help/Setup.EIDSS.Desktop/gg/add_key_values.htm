ძირითადი სიდიდეების დამატება

ზოგჯერ შეიძლება საჭირო იყოს ანგარიშის შექმნა ამ საბოლოო დიაგნოზებისთვის და არსებული საცხოვრებელი რეგიონების/რაიონებისთვის, რომლებისთვისაც არ არის მონაცემები. ამ შემთხვევაში გამოიყენეთ ძირითადი სიდიდეები. ძირითადი სიდიდეები იკავებს ადგილს ცარიელი ჩანაწერებისთვის (დიაგნოზების და რეგიონების/რაიონებისთვის) რომლებიც შეიძლება დაემატოს მოთხოვნას, საჭიროებისამებრ.

მაგალითად, საჭიროა ანგარიშის შექმნა, ინფორმაციით, რიგი ადამიანის შემთხვევების შესახებ, გარკვეული დიაგნოზით, გურიის და აჭარის რეგიონების ყველა რაიონებისთვის.

 

ნახ. 336 გარდაქმნადი ბადე ძირითადი სიდიდეების გარეშე

ნახ. 337 გარდაქმნადი ბადე ძირითადი სიდიდეებით

მართალია ოზურგეთში, ქობულეთსა და შუახევში შემთხვევების რაოდენობა ნულია, რაიონები ნაჩვენებია რაიონების სიაში

ძირითადი სიდიდეების აქტივაცია

შეარჩიეთ ძირითადი მითითებების ყველა სიდიდის დამატება ალმიანი გრაფა საერთო მოთხოვნის ინფორმაციის ფანჯარაში.

ნახ. 338 საერთო მოთხოვნის საინფორმაციო ფანჯარა

ყველა განლაგებას, რომელიც შექმნილია მოთხოვნის საფუძელზე, აქტივიზირებული ძირითადი სიდიდეებით, თავის მხრივ ექნება აქტივიზირებული ძრითადი სიდიდეები. ძირითადი სიდიდეები დაემატება მოხსენებას, იმ შემთხვევაშიც კი, თუ ეს ოპცია უფრო გვიან შეიქმნება განლაგებაში.
