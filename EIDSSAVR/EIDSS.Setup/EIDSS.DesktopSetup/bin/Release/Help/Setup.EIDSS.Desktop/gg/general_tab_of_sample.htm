ნიმუშის L02 ფორმის ზოგადი ჩანართი

ნიმუშის (L02 ) ფორმის ზოგადი ინფორმაციის ჩანართი გამოიყენება ნიმუშის ინფორმაციის ეკრანზე გამოსატანად ან/და რედაქტირებისთვის. ველების ხელმისაწვდომობა დამოკიდებულია ნიმუშის სტატუსსა და წვდომის უფლებებზე.

 

პირველადი ნიმუშის საიდენტიფიკაციო კოდი

ეკრანზე ჩანს ალიქვოტების და დერივატების ძირითადი ნიმუში. ძირითადი ნიმუშის ID დეტალების სანახავად, დააწკაპუნეთ დეტალების ღილაკზე (). ეს ველი ივსება მხოლოდ ალიქვოტებისა და დერივატებისთვის.

ფუნქცონალური არე

მონიშნეთ ორგანიზაციის განყოფილება, რომელიც მუშაობს კონკრეტულ ნიმუშთან.

შენახვის ადგილი

საცავში შეარჩიეთ ნიმუშის ადგილმდებარეობა:

1.დააწკაპუნეთ დეტალების ღილაკზე (). გაიხსნება საყინულეების ფანჯარა.

2.შეარჩიეთ სასურველი ადგილმდებარეობა.

3.დააწკაპუნეთ OK ღილაკზე. დაიხურება საყინულეების ფანჯარა. ველში გამოჩნდება მინიჭებული ადგილმდებარეობა.

შემთხვევის ტიპი

შემთხვევის ტიპი, რომელიც დაკავშირებულიანიმუშთან:

•ადამიანის დაავადება

•ფრინველის დაავადება

•შინაური საქონლის დაავადება

•ვექტორული ზედამხედველობა

"აქტიურ ზედამხედველობასთან" დაკავშირებით არაფერი ჩანს ეკრანზე.

ინფორმაცია შემთხვევის შესახებ

შემთხვევის ID და დიაგნოზი

დაავადების შემთხვევის ფორმის გასახსნელად დააწკაპუნეთ დეტალების ღილაკზე (). დაავადების შემთხვევის ფორმაზე წვდომა განისაზღვრება ამ შემთხვევის ფორმაზე მომხმარებლის დაშვების უფლებით.

"აქტიურ ზედამხედველობასთან" დაკავშირებით არაფერი ჩანს ეკრანზე.

სესიის ID

ნიმუშთან დაკავშირებული სესიის ID.

სესიის ფორმის გასახსნელად დააწკაპუნეთ დეტალების ღილაკზე (). ამ ფორმაზე წვდომა განისაზღვრება ამ შემთხვევის ფორმაზე მომხმარებლის დაშვების უფლებებით.

დაავადების შემთხვევებთან დაკავშირებით არაფერი ჩანს ეკრანზე.

პაციენტის/ჯიშს/ვექტორის ინფორმაცია

შესაძლებელია შემდეგი ინფორმაციის ეკრანზე ჩვენება:

•ფრინველის დაავადების შემთხვევებისთვის: სახეობის ტიპი.

•შინაური საქონლის დაავადების შემთხვევებისთვის: ცხოველის ID.

•ადამიანის დაავადების შემთხვევებისთვის: პაციენტის გვარი, სახელი და მამის სახელი.

•აქტიური ზედამხედველობის სესიებისთვის: ცხოველის ტიპი და ცხოველის ID.

•ვექტორული ზედამხედველობის სესიებისთვის: ვექტორის ID.
