სიის და ძიების ფორმების რეგულირება

საჭიროების შემთხვევაში მომხმარებელს შეუძლია სიის და ძიების ფორმების რეგულირება. რეგულირება სასარგებლოა, თუ საჭიროა ფორმის ერთი ნაწილის აღნიშვნა.

ძიების პანელის რეგულირება:

1.დააწკაპუნეთ () ღილაკზე, ძებნის პანელის გასაფართოებლად.

2.გაიხსნება ძებნის პანელი გადაიტანეთ კურსორი ძებნის პანელის მარჯვენა კიდეზე.

3.გადაათრიეთ ძებნის პანელის კიდე და დაარეგულირეთ მისი სიდიდე საჭიროებისამებრ.

ნახ. 27 ძებნის პანელის რეგულირება

ერთეულების სიების რეგულირება:

მომხმარებელს შეუძლია სვეტების თანმიმდევრობის შეცვლა მათი გადათრევით. სვეტში ახალი ადგილის არჩევის შემდეგ გაუშვით თაგუნას ღილაკი.

ასევე შესაძლებელია სვეტში პუნქტების თანმიმდევრობის შეცვლა (ზრდადი ან კლებადი). უბრალოდ დააწკაპუნეთ სვეტის სათაურზე, და პატარა ისარი სვეტის სათაურში შეცვლის მიმართულებას. ასევე შესაძლებელია სვეტების დამალვა და გამოჩენა (იხ. პუნქტების ჩამონათვალის სექცია).

ნახ. 28 პუნქტების სიის პარამეტრების დაყენება
