ვექტორის ზედამხედველობის მოდულის დაწყება

პირველი ვექტორის სესიის შეყვანამდე, მოახდინეთ ვექტორის ზედამხედველობის (VS) მოდულის კონფიგურაცია. ვექტორის ზედამხედველობის მოდულის კონფიგურაცია შეიცავს უნიკალური დანომვრის სქემას, მისათითებელი ცხრილების კონფიგურაციას და კონფიგურაციის მატრიცებს. ყველა მათგანი უკვე შექმნილია დეფოლტის მეშვეობით, მაგრამ რეკომენდებულია მათი გახსნა და უზრუნველყოფა იმისა, რომ მათი წინასწარი ვერსია აკმაყოფილებდეს თქვენი პროცესის მთხოვნებს.

მაშინაც კი, როდესაც მისათითებელი ცხრილები წინასწარაა შეჯერებული, მათი შეჯერება ნებისმიერ დროს შეიძლება ამის სემდეგაც. თუმცა, მატრიცები და დანომვრის სქემა წინასწარ უნდა შეიქმნას, მათი არასწორი კონფიგურაცია ხელს შეუშლის მოდულის სწორ ფუნქციონირებას.
