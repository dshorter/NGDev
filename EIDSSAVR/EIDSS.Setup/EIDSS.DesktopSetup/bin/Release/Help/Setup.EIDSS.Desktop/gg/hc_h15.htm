ნახ. 586 "ადამიანის დაავადების აგრეგირებული შემთხვევების ჩამონათვალი"

ძებნის პარამეტრების სექცია შეიძლება დაიყოს 3 ჯგუფად: დროის შუალედის ძებნის პარამეტრები, ადმინისტრაციული ძებნის პარამეტრები და დამატებითი ძებნის პარამეტრები.

ნახ. 587 "ძებნის პარამეტრების" სექცია

ველი/ცვლადი

აღწერა

შემთხვევის ID

შემთხვევის ID ველში შეიძლება მიუთითოთ "აგრეგირებული შემთხვევის ID". ეკრანზე გამოჩნდება მხოლოდ ის შემთხვევები, რომელთა შემთხვევის ID მთლიანად ემთხვევა ერთმანეთს.

დროის შუალედის ერთეული

დროის შუალედის ერთეული ველში ნაჩვენებია დროის შუალედის რეგულატორები, რომელთათვისაც შეჯამდება კონკრეტული მონაცემები. ეს ველი სავალდებულო ველია, რომელიც ავტომატურად ივსება დაყენების პარამეტრების შესაბამისად.

დასაწყისის თარიღი

დაწყების თარიღის ველში მითითებულია ის თარიღი, რომლიდან დაწყებულიც შეგროვდა აგრეგირებული მონაცემები. შერჩეული დროის შუალედის ერთეული -ს მიხედვით გამოთვლება შერჩეული დაწყების თარიღი -ს პოზიცია, ხოლო ჩანაწერების გაფილტვრის დროს გამოყენებული იქნება შესაბამისი დროის შუალედის დასაწყისი.

დასრულების თარიღი

დასრულების თარიღის ველში მითითებულია ის თარიღი, როდემდეც შეგროვდა აგრეგირებული მონაცემები. შერჩეული დროის შუალედის ერთეული -ს მიხედვით გამოთვლება შერჩეული დასრულების თარიღი -ს პოზიცია, ხოლო ჩანაწერების გაფილტვრის დროს გამოყენებული იქნება შესაბამისი დროის შუალედის დასასრული.

ადმინისტრაციული დონე

ადმინისტრაციული დონე ველში ნაჩვენებია ადმინისტრაციული ფილტრის ის რეგულატორები, რომელთათვისაც შეჯამდება მონაცემები. ეს ველი სავალდებულო ველია, რომელიც ავტომატურად ივსება დაყენების პარამეტრების შესაბამისად.

რეგიონი

რეგიონი ველში მითითებულია ის ზუსტი "რეგიონი", სადაც შეგროვდა აგრეგირებული მონაცემები.

რაიონი

რაიონი ველში მითითებულია ის ზუსტი რაიონი, სადაც შეგროვდა კონკრეტული აგრეგირებული მონაცემები.

დასახლება

დასახლება ველში მითითებულია ის ზუსტი დასახლება, სადაც შეგროვდა კონკრეტული აგრეგირებული მონაცემები.
"ძებნის შედეგების" სექცია გამოიყენება აგრეგირებული შემთხვევის ანალიზისთვის შესარჩევად.
