"ნიმუშების დეტალების" ფორმაში (IF) ტესტის მონაცემების ნახვა ან რედაქტირება

ეს ოპერაცია შესაძლებელია მხოლოდ მაშინ, როცა ლაბორატორიული მოდულის გამარტივებული რეჟიმის მოსანიშნი უჯრა არ არის მონიშნული სისტემურ უპირატესობებში.

 

1.ჟურნალების მენიუში შეარჩიეთ ნიმუშები. გაიხსნება ნიმუშების ჩამონათვალის ფანჯარა.

2.იპოვეთ და ორჯერ დააწკაპუნეთ სასურველ ტესტთან დაკავშირებულ ნიმუშზე. გაიხსნება ნიმუშის ფანჯარა.

3.ტესტების ჩანართის გასახსნელად დააწკაპუნეთ ჩანართზე. ტესტების ჩანართში ჩამოთვლილია ნიმუშზე დანიშნული და ჩატარებული ყველა ტესტი.

ნახ. 275 "ტესტების" ჩანართი

4.ტესტების ჩამონათვალში ნიმუშზე დანიშნული ტესტის დასამატებლად დააწკაპუნეთ ღილაკზე ტესტის შერჩევა. ყველა სასურველი ტესტის დასამატებლად გაიმეორეთ მოცემული ეტაპი.

5.ჩამონათვალში ტესტის წასაშლელად დააწკაპუნეთ ღილაკზე ტესტის წაშლა.

6.მოახდინეთ ტესტების შესახებ არსებული მონაცემების რედაქტირება. ნიმუშის ფორმისა და მისი ჩანართების შესახებ ინფორმაციის მისაღებად წაიკითხეთ სექცია ნიმუში L02.

7.დააწკაპუნეთ ღილაკზე დამახსოვრება და დაადასტურეთ მონაცემთა დამახსოვრება.

8.ნიმუშების ჩამონათვალი ფანჯრის დასახურად დააწკაპუნეთ დიახ ღილაკზე.
