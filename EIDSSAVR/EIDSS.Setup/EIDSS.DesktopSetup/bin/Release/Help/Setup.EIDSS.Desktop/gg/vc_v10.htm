ვეტ აგრეგირებული შემთხვევის დეტალები V10

ფორმა ვეტერინარული აგრეგირებული შემთხვევის დეტალები V10 შეიცავს 3 ძირითად ნაწილს: შეტყობინების ინფორმაცია, ზოგადი ინფორმაცია და დეტალური ინფორმაცია.

ნახ. 90 ვეტერინარული აგრეგირებული შემთხვევის დეტალები

ეს შეტყობინების ინფორმაციის განყოფილება შეიცავს შემდეგ პარამეტრებს:

ვინ გააგზავნა შეტყობინება

ინსტიტუტი

ველში დაწესებულება მოცემულია ორგანიზაცია, რომელმაც გადააგზავნა შეტყობინება აგრეგირებული მონაცემებით.

თანამშრომელი

ველში თანამშრომელი მოცემულია თანამშრომლის სახელი, რომელმაც გადააგზავნა შეტყობინება აგრეგირებული მონაცემებით.

თარიღი

ამ ველში თარიღი მოცემულია თარიღი, როდესაც გაგზავნილი იქნა შეტყობინება აგრეგირებული მონაცემებით.

ვინ მიიღო შეტყობინება

ინსტიტუტი

დაწესებულება ველში მითითებულია ის ორგანიზაცია, რომელმაც მიიღო კონკრეტული შეტყობინება აგრეგირებული მონაცემებით.

თანამშრომელი

ველში თანამშრომელი მოცემულია თანამშრომლის სახელი, რომელმაც მიიღო შეტყობინება აგრეგირებული მონაცემებით.

თარიღი

ამ ველში თარიღი მოცემულია თარიღი, როდესაც მიღებული იქნა შეტყობინება აგრეგირებული მონაცემებით.

ვინ შეიყვანა

ინსტიტუტი

ველში დაწესებულება მოცემულია ორგანიზაცია, რომელმაც შეიყვანა შეტყობინება აგრეგირებული მონაცემებით.

თანამშრომელი

ველში თანამშრომელი მოცემულია თანამშრომლის სახელი, რომელმაც შეიყვანა შეტყობინება აგრეგირებული მონაცემებით.

თარიღი

ველში თარიღი მოცემულია თარიღი და დრო, როდესაც შეტყობინება აგრეგირებული მონაცემებით შეყვანილი იქნა სისტემაში.

ზოგადი ინფორმაციის განყოფილება შეიცავს შემდეგ პარამეტრებს:

 

ველი/ცვლადი

აღწერა

შემთხვევის ID

ველი შემთხვევის ID შეიცავს შემთხვევის უნიკალურს ნომერს, და შეივსება მაშინ, როდესაც საქმეს პირველად შეინახავენ. შემთხვევის ID გენერირდება დანომრვის უნიკალური სქემა -ს შესაბამისად ფორმის პირველად დამახსოვრების შემდეგ.

წელი

ველში წელი მოცემულია ზუსტი წელი, როდესაც აგრეგირებული მონაცემები შეგროვდა.

კვარტალი

ველში კვარტალი მოცემულია ზუსტი კვარტალი, როდესაც აგრეგირებული მონაცემები შეგროვდა.

თვალსაჩინოა აგრეგირებული პარამეტრების შესაბამისად.

ქვეყანა

ველში ქვეყანა დაფიქსირებულია დზეის-ის ქვეყანას, სადაც მოხდა აგრეგირებული მონაცემების შეგროვება.

რეგიონი

ველშირეგიონი დაფიქსირებულია კონკრეტული რეგიონი, სადაც მოხდა აგრეგირებული მონაცემების შეგროვება.

თვალსაჩინოა აგრეგირებული პარამეტრების შესაბამისად.

რაიონი

ველში რაიონი დაფიქსირებულია კონკრეტული რაიონი, სადაც მოხდა აგრეგირებული მონაცემების შეგროვება.

თვალსაჩინოა აგრეგირებული პარამეტრების შესაბამისად.

შენიშვნა: ქვეყანა/რეგიონი/რაიონი/დასახლება და წელი/კვარტალი/თვე/კვირა ველების ეკრანზე გამოჩენა დამოკიდებულია აგრეგირებულ პარამეტრებზე.

დეტალური ინფორმაციის განყოფილება შეიცავს ბადეს. ამ ბადეში სვეტები და რიგები ქვეყნის მონაცემებს ასახავს და განსაზღვრულია გრაფაში მოქნილი ფორმების დიზაინერი (შაბლონი-სვეტები) და გრაფაში ვეტ აგრეგირებული შემთხვევის მატრიცის რედაქტორი (მატრიცის ვერსია - რიგები).

მატრიცა ვეტ აგრეგირებული შემთხვევა არის დეფოლტით ბოლო აქტივირებული მატრიცა, მაგრამ იგი შეიძლება შეიცვალოს ჩამონათვალში მოცემული სხვა მატრიცით. ამ ჩამონათვალში ხელმისაწვდომია მხოლოდ აქტივირებული მატრიცები.

ველი/ცვლადი

აღწერა

მატრიცის ვერსია

ველი გამოიყენება მატრიცის ვერსიის შესარჩევად ვეტერინარული აგრეგირებული საქმისთვის. ვერსიის შერჩევა შესაძლებელია ჩამოსაშლელ სიაში.

შაბლონი

ველი გამოიყენება შაბლონის შესარჩევად ვეტერინარული აგრეგირებული საქმისთვის. შაბლონის შერჩევა შესაძლებელია ჩამოსაშლელ სიაში.
