აგრეგირებული რეჟიმი

აგრეგირება არის პროცესი, რომლის მეშვებითაც გროვდება სამედიცინო მოვლენები, რათა მოხდეს მათი მართვა როგორც ერთიანი აგრეგირებული მოვლენისა. აგრეგირება გამოიყენება ზოგიერთი უვნებელი დაავადებების შესაგროვებლად, რაც მოგვიანებით გამოიყენება სტატისტიკური მიზნებით.

აგრეგირების აწყობა:

1.დააწკაპუნეთ სისტემის მენიუში გრაფაზე კონფიგურაცია და შემდეგ დააწკაპუნეთ გრაფაზე აგრეგირებული რეჟიმი. გაიხსნება აგრეგირებული რეჟიმის ფანჯარა.

ნახ. 414 აგრეგირებული რეჟიმის ფანჯარა

2. გრაფისთვის აგრეგირებული შემთხვევის ტიპი = ადამიანის აგრეგირებული შემთხვევა განსაზღვრეთ ადმინისტრაციული ერთეული აგრეგირებისთვის ველში მინიმალური ადმინისტრაციული ერთეული (ქვეყანა, რეგიონი, რაიონი ან დასახლებული პუნქტი).

3.განსაზღვრეთ დროის ერთეული აგრეგირებისთვის ველში დროის მინიმალური ინტერვალის ერთეული (დღე, კვირა, თვე, კვარტალი ან წელიწადი).

4.მიმართვისთვის აგრეგირებული შემთხვევის ტიპი = ვეტ აგრეგირებული შემთხვევაგაიმეორეთ მე-2 და მე-3 ნაბიჯები.

5.გრაფისთვის აგრეგირებული შემთხვევის ტიპი = ვეტ აგრეგირებული მოქმედება გაიმეორეთ მე-2 და მე-3 ნაბიჯები.

6.შეინახეთ მონაცემები და დახურეთ ფანჯარა.
