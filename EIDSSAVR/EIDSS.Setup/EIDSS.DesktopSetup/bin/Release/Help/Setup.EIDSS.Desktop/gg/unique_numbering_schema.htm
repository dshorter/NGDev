უნიკალური ნუმერაციის დიაგრამა

Show/Hide Hidden Text

EIDSS შეიცავს სხვადასხვა ერთეულების ნაკრებებს, როგორებიცაა დაავადებების შემთხვევები, პაციენტები, ნიმუშები, ტესტები და ა.შ. ერთეულის უნიკალური იდენტიფიკაციისთვის EIDSS იყენებს უნიკალური ნუმერაციის დიაგრამას ზოგიერთ შემთხვევებში მომხმარებელს შეუძლია მიანიჭოს ერთეულს ID, ხოლო სხვა ID მიენიჭება ავტომატურად, უნიკალური ნუმერაციის დიაგრამის მეშვეობოთ, ავტომატური ნუმერაცია უზრუნველყოფს ერთეულებისთვის უნიკალური ID-ის მინიჭებას და ამგვარად, გამორიცხავს, რომ ორ ერთეულს ერთი და იგივე ID ჰქონდეს. მეტიც, ერთეულის საიდენტიფიკაციო ნომერი განსაზღვრავს ნომრის დაკავშირების წელს, ადგილს, ერთეულის ტიპს (ნიმუში, შემთხვევა თუ სხვა), და ა.შ. ნუმერაციის დიაგრამის განსაზღვრა ერთეულისთვის ნიშნავს სიმბოლოთა ჯგუფების თანმიმდევრობის განსაზღვრას, რაც ქმნის საიდენტიფიკაციო ნომერს.

მოახდინეთ ნუმერაცია EIDSS-ის გამოყენებამდე და ნუ შეცვლით მას! ნუმერაციის დიაგრამის ცვლილებამ შეიძლება სერიოზულად დააზიანოს მონაცემთა ბაზა.

უნიკალური ნუმერაციის დიაგრამათა სიის გახსნა:

დაკლიკეთ სისტემის მენიუში კონფიგურაცია, შემდეგ დაკლიკეთ უნიკალური ნუმერაციის დიაგრამა. გაიხსნება უნიკალური ნუმერაციის დიაგრამათა სიის ფანჯარა

ნახ. 504 უნიკალური ნუმერაციის დიაგრამათა სიის ფანჯარა

 

უნიკალური ნუმერაციის დიაგრამის რედაქტირება.:

1.შეარჩიეთ ID ტიპი, რომლისთვისაც უნდა მოხდეს ნუმერაციის დიაგრამის რედაქტირევა და დაკლიკეთ რედაქტირების ღილაკზე. გაიხსნება უნიკალური ნუმერაციის დიაგრამის დეტალების ფანჯარა

ნახ. 505 უნიკალური ნუმერაციის დიაგრამის ფანჯარა

ნუმერაციის ფორმატის სექცია განსაზღვრავს ნუმერაციას პრეფიქსის ველში. შეიტანეთ პრეფიქსი ID-ის არსებული ტიპისთვის. უზრუნველყავით, რომ პრეფიქსი განსხვავებული იყოს სხვა ტიპის ID-ების პრეფიქსებისგან. შეიტანეთ მომდევნო ნომრის სიდიდის ველში მრიცხველი. წინასწარ დათვალიერების ველში ნაჩვენებია ნუმერაციის შემოღები შედეგი.

2.შეინახეთ ცვლილებები და დახურეთ ფანჯარა
