ნიმუშის დეტალების ჩანართი ლაბორატორიული ჩანაწერების დეტალურ ფორმაში

ნიმუშის დეტალების ჩანართი გამოიყენება ნიმუშის შესახებ იმ დეტალური ინფორმაციის საჩვენებლად, რომელიც არ ჩანს ლაბორატორიული სექციის ფორმაში.

ნახ. 310 ნიმუშის დეტალების ჩანართი

 

შეგროვების თარიღი

თარიღი, როდესაც აღებულ იქნა ნიმუში.

შენახვის ადგილი

საცავში შეარჩიეთ ნიმუშის ადგილმდებარეობა:

1.დააწკაპუნეთ დეტალების ღილაკზე (). გაიხსნება საყინულეების ფანჯარა.

2.ორმაგად დაკლიკეთ სასურველ ადგილას. დაიხურება საყინულეების ფანჯარა. ველში გამოჩნდება მინიჭებული ადგილმდებარეობა.

ფუნქცონალური არე

მონიშნეთ ორგანიზაციის განყოფილება, რომელიც მუშაობს კონკრეტულ ნიმუშთან.

პირველადი ნიმუშის საიდენტიფიკაციო კოდი

მშობლის ნიმუში ალიქვოტების და დერივატებისთვის. ძირითადი ნიმუშის ID დეტალების სანახავად, დააწკაპუნეთ დეტალების ღილაკზე (). ეს ველი გამოიყენება მხოლოდ ალიქვოტების და დერივატებისთვის; დანარჩენი ნიმუშებითვის ველი ცარიელია.

შენიშვნები

საჭიროების შემთხვევაში შეიტანეთ შენიშვნა

შემთხვევის/სესიის ტიპი

შემთხვევის ტიპი, რომელიც დაკავშირებულია ნიმუშთან:

•ადამიანის დაავადება

•ფრინველის დაავადება

•შინაური საქონლის დაავადება

•ვექტორული ზედამხედველობა

"აქტიურ ზედამხედველობასთან" დაკავშირებით არაფერი ჩანს ეკრანზე.

პაციენტის/ჯიშს/ვექტორის ინფორმაცია

შესაძლებელია შემდეგი ინფორმაციის ეკრანზე ჩვენება:

•ფრინველის შემთხვევები: სახეობების ტიპი

•საქონლის შემთხვევები: ცხოველის საიდენტიფიკაციო კოდი

•ადაიანის დაავადების შემთხვევები: პაციენტის ბოლო, პირველი და შუა სახელები.

•AS სესიები: ცხოველის ტიპი და ცხოველის საიდენტიფიკაციო კოდი

ტესტების რაოდენობა

ნიმუშე დანიშნული ტესტების მთლიანი რაოდენობა

გაეგზავნა ორგანიზაციას

არსებული ორგანიზაციის მითითბა ხდება შემოსატანი ნიმუშებისთვის. ნიმუშების პოვნის შემდეგ შესაძლებელია მათი შემოტანა/მიღება. დეტალების შესახებ წაიკითხეთ არსებული ნიმუშების მიღების/შემოტანის პუნქტში, რომელიც დაკავშირებულია ლინკით შემთხვევის/სესიის (LS) სექციასთან

აღებულია დაწესებულების მიერ

ორგანიზაცია, რომელმაც შეაგროვა ნიმუშები

აღებულია თანამშრომლის მიერ

იმ ოფიცრის სახელი, რომელმაც შეაგროვა ნიმუშები

გადაეცა

მიუთითეთ გარეშე ორგანიზაცია, რომელშიც გატანილია სასურველი ნიუში. ნიმუშების პოვნის შემდეგ შესაძლებელია მათთვის შედეგების რეგისტრაცია. დეტალების შესახებ წაიკითხეთ გარეშე ლაბოატორიაში ჩატარებული ტსტის შედეგების განსაზღვრის (LS) სექციაში.

განადგურების მეთოდი

შეარჩიეთ გამოსაყენებელი განადგურების მეთოდი ველი გაოიყენება ნიმუშებისთვის, რომლებსაც აქვს მარკირებულია განადგურებისთვის სტატუსი.

✓ლაბორატორიული ჩანაწერების დეტალური ფორმა

✓შესწორების ისტორიის ჩანართი ლაბორატორიული ჩანაწერების დეტალურ ფორმაში
