შემთხვევის ნიმუშების რეგისტრაცია

ნიმუშების აღების პროცედურის ჩატარება შეიძლება საჭირო იყოს ავადმყოფისთვის დიაგნოზის დასასმელად. ყველა აღებული ნიმუში უნდა ჩამოითვალოს ადამიანის დაავადების შემთხვევის ანგარიშის ფორმა -ს შესაბამის სექციაში. მომხმარებელს ნიმუშების დამატება შეუძლია შემთხვევის რეგისტრაციის დროს და რეგისტრაციის შემდეგაც შემთხვევის რედაქტირების გზით.

1.გახსენით ადამიანის დაავადების შემთხვევის ანგარიშის ფორმა -ს (H02) ჩანართი შემთხვევის კვლევა და გადადით ქვე-ჩანართზე ნიმუშების აღება.

ნახ. 594 ნახ. 595 "შემთხვევის კვლევის" ჩანართი - "ნიმუშების აღების" ქვე-ჩანართი

2.აღებული ნიმუშები ველში მონიშნეთ დიახ.

3.დააწკაპუნეთ ახალი რიგის დასამატებლად დააწკაპუნეთ აქ სტრიქონზე. ახალი სტრიქონი დაემატება ნიმუშების ჩამონათვალში.

4.ჩამოსაშლელ სიაში მონიშნეთ ნიმუშის ტიპი.

შენიშვნა: შესაძლებელია ნიმუშების გაფილტვრა დიაგნოზის მიხედვით ოპციის გააქტიურება, რაც გაამარტივებს ნიმუშის ტიპი -ს შერჩევას.

ნახ. 596 "ნიმუშის ტიპის" შერჩევა

5.ცხრილში შეძლებისდაგვარად შეავსეთ სხვა ველებიც. ფორმის შესახებ სრული ინფორმაციის მისაღებად იხილეთ ადამიანის დაავადების შემთხვევის ანგარიშის H02 ფორმა.

6.შეიტანეთ აუცილებელი ინფორმაცია დამატებითი შენიშვნები მოთხოვნილი ტესტის და ნიმუშის შესახებ სექციაში.

7.მონაცემების დასამახსოვრებლად დააწკაპუნეთ დიახ ღილაკზე და დახურეთ ფორმა.
