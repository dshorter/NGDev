ჯამური ანგარიშის შექმნა

"ჯამური ანგარიში" არის შერჩეული აგრეგირებული შემთხვევების მონაცემების საფუძველზე შედგენილი ანგარიში.

ნახ. 540 "ადამიანის დაავადების აგრეგირებული შემთხვევის ჯამური ფორმის" ფანჯარა

1.დააწკაპუნეთ ძებნა -> ადამიანის დაავადების აგრეგირებული შემთხვევების ჯამური მონაცემები. გაიხსნება ადამიანის დაავადების აგრეგირებული შემთხვევის ჯამური მონაცემების ფორმა.

2.მონიშნეთ ადმინისტრაციული დონე და დროის შუალედის ერთეული.

3.ჯამურ ანგარიშში ნებისმიერი არსებული შემთხვევის დასამატებლად, დააწკაპუნეთ ღილაკზე შერჩევა. გაიხსნება ადამიანის დაავადების აგრეგირებული შემთხვევების ჩამონათვალი.

ნახ. 541 "ადამიანის დაავადების აგრეგირებული შემთხვევების ჩამონათვალი"

4.მოძებნეთ ჯამურ ანგარიშში შესატანი სასურველი შემთხვევა და დააწკაპუნეთ ღილაკზე შერჩევა. მოცემული შემთხვევა დაემატება ჯამურ ანგარიშში და გამოჩნდება შერჩეული აგრეგირებული შემთხვევები სექციაში.

შენიშვნა: ახალი აგრეგირებული შემთხვევის შესაქმნელად დააწკაპუნეთ ღილაკზე ახალი და დაამატეთ იგი ჯამურ ანგარიშში.

5.შერჩეული აგრეგირებული შემთხვევები სექციაში ყველა აუცილებელი შემთხვევის დამატების შემდეგ დააწკაპუნეთ ღილაკზე ჯამური ინფორმაციის ჩვენება.

6.ჯამური მონაცემები გამოჩნდება ჯამური ინფორმაციის სექციაში.
