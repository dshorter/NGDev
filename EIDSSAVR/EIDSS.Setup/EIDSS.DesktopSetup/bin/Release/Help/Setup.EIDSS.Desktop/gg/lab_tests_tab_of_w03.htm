პულის/ ინდივიდუალური ვექტორის დეტალების W03 ფორმის ლაბორატორიული ტესტების ჩანართი

ლაბორატორიული ტესტების ჩანართი გამოიყენება "ლაბორატორიულ მოდულში" ვექტორისთვის/პულისთვის ჩატარებული ლაბორატორიული ტესტების ჩამონათვალის სანახავად. ეს ჩანართი მხოლოდ წაკითხვადი ფორმატისაა.

ნახ. 215 ლაბორატორიული ტესტების ჩანართი

მონაცემთა ველები

ლაბორატორიული ნიმუშის საიდენტიფიკაციო კოდი

ეკრანზე ჩანს ტესტისთვის გამოყენებული ნიმუშის ლაბორატორიული ID. ნიმუშის ლაბორატორიული ID ავტომატურად ენიჭება დანომრვის უნიკალური სქემის შესაბამისად, როდესაც ხდება ნიმუშის მიღება "ლაბორატორიულ მოდულში".

ველზე ხელმისაწვდომობის ID

ტესტისთვის გამოყენებული ნიმუშის სავალე ID.

ნიმუშის ტიპი

ტესტისთვის გამოყენებული ნიმუშის ტიპი.

შეგროვების თარიღი

თარიღი, როცა აღებული იყო ტესტისთვის გამოყენებული ნიმუში.

ტესტის სახელი

ტესტის დასახელება.

ტესტის კატეგორია

ეკრანზე ჩანს ტესტის კატეგორია, თუ ცნობილია.

შედეგის თარიღი

თარიღი, როცა დაიწყო ტესტი ან როცა მისი შედეგი შეიყვანეს.

ვინ ჩაატარა ტესტირება

იმ თანამშრომლის სახელი, ვინც ტესტის შედეგი შეიყვანა.

ტესტირებულია ინსტიტუტის მიერ

ორგანიზაცია, რომელმაც ტესტი ჩაატარა.

ტესტის შედეგი

ტესტის შედეგი.

დავადება

დაავადება, რომელიც "ლაბორატორიულ მოდულში" კონკრეტული ტესტისთვის დაყენებულია როგორც ტესტის დიაგნოზი. თუ შედეგი დადებითია, დაავადება ასლის სახით გადადის სესიის დაავადებების ველში.
