გარდაქმნად ცხრილში მონაცემების ჩენება

დზეის-ში შენახულ მონაცემთა მოცულობა უზარმაზარია, და გარდაქმნადი/ძირითადი ბადის ყოველი ცვლილების დასრულებას შესაძლოა დიდი დრო დასჭირდეს. ამიტომ ძლიერ რეკომენდებულია, რომ გარდაქმნადი/ძირითადი ბადის კონსტრუირებისას გამოირთოს მონაცემთა გამოთხოვნა.

•მონაცემთა გამოთხოვნის თავიდან ასაცილებლად მოხსენით მონიშვნა კვადრატში მონაცემთა ჩვენება გადაწყობად ბადეში.

•ბადის კონსტრუირების დასრულების შემდეგ, უბრალოდ აღადგინეთ მონაცემთა გამოთხოვნა მონაცემთა ჩვენება გადაწყობად ბადეში კვადრატის მონიშვნით.  მონაცემები დაუყოვნებლივ ხელახლა გადაანგარიშდება მიმდინარე სტრუქტურისა და დაყენებული პარამეტრების შესაბამისად. ახლა გარდაქმნადი/ძირითადი ბადის (ველის, ფილტრების ან პარამეტრების) ყოველი ცვლილება გამოიწვევს მონაცემთა გადაანგარიშებას.
