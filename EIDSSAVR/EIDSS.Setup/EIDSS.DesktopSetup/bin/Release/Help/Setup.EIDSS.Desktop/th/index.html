This skin is particularly designed for HTML Help (.chm) files!

For Webhelp or eBook output, please select one of the "Webhelp" skins supplied with Help+Manual.

 

In CHM files, this skin creates a layout that makes the help file compatible with ultra-high resolution monitors, a new feature in Help+Manual. For more information on this topic, please refer to the in-depth article:

Zooming in on HTML Help

Making CHM files ready for ultra-high resolution monitors
