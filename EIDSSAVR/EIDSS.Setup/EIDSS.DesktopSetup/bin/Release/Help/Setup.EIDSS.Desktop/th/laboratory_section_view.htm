มุมมองหมวดห้องปฏิบัติการ

หมวดห้องปฏิบัติการ แสดงภาพรวมของโมดูลห้องปฏิบัติการ มีรายการตัวอย่าง เเละการทดสอบทั้งหมดใน EIDSS หากเปิดแล้วไม่มีรายการปรากฏในตารางผู้ใช้ควรดูในแผงการค้นหาเพื่อล้างบางค่าที่อาจใส่ในบางฟิลด์โดยอัตโนมัติ แล้วจึงเลือกใส่เกณฑ์สำหรับค้นหา รายการที่มีการเปลี่ยนแปลงแก้ไขจะปรากฏเป็นตัวหนาเพื่อให้ผู้ใช้ทราบว่าเป็นรายการที่ต้องยืนยัน โดย ยอมรับ (บันทึก) หรือ ถูกปฏิเสธ (ยกเลิก) ปุ่ม บันทึก ใช้ยอมรับการเปลี่ยนแปลงทั้งหมดในตาราง

 ในการยกเลิกการเปลี่ยนแปลง เลือกการเปลี่ยนแปลงที่ต้องการยกเลิก เเละคลิกขวาที่รายการนั้น เลือก ยกเลิกการเปลี่ยนแปลง จากเมนูดร็อปดาวน์

รูปภาพ 22 ตารางรายการหมวดห้องปฏิบัติการ

ผู้ใช้สามารถแก้ไขรายละเอียดของรายการที่แสดงในตารางได้ โดยคลิกที่ปุ่ม แก้ไข () หน้ารายละเอียดจะเปิดเพื่อให้เปลี่ยนแปลงแก้ไขข้อมูล อ่านเพิ่มเติมได้ที่หมวด แบบฟอร์มรายละเอียดตัวอย่าง/การทดสอบ

การแก้ไขบางข้อมูลทำได้ในเมนูคลิกขวา โดยผู้ใช้เลือกหนึ่งรายการหรือหลายรายการเเละคลิกขวา เลือกการทำงานที่ต้องการจากเมนูดร็อปดาวน์ การทำงานบางอย่างอาจถูกจำกัดตัวเลือกไว้สำหรับการเลือกแก้ไขบนรายการเดียวเท่านั้น

คอลัมน์รายการผลลัพธ์

ID ผู้ป่วย/เซสชั่น

หมายเลขเคสผู้ป่วยหรือเซสชั่นที่เกี่ยวข้องกับตัวอย่าง

รหัสตัวอย่างจากพื้นที่/ภาคสนาม

หมายเลขหรือรหัสของตัวอย่างตามที่ได้มาจากแหล่งเก็บตัวอย่าง เพื่ออ้างอิง

สถานะตัวอย่าง

สถานะของตัวอย่างเป็นหนึ่งในข้อมูลที่กำหนดค่าหรือการใส่ข้อมูลอื่นๆในการทำงานเพิ่มเติมของกับตัวอย่างนั้น อ่านเพิ่มได้ที่หมวด สถานะตัวอย่าง ค่าในช่องนี้จะเปลี่ยนโดยอัตโนมัติสอดคล้องกับการปฏิบัติงานกับตัวอย่างนั้น

ID รหัสตัวอย่าง

ID ของตัวอย่าง ซึ่งใช้ระบุตัวอย่างในห้องปฏิบัติการ

ระบบจะจัดการหมายเลขให้โดยอัติโนมัติหากผู้ใส่ข้อมูลไม่ระบุ

หมายเลขเอง

ประเภทตัวอย่าง

มีการเเสดงประเภทตัวอย่าง เเละสามารถเปลี่ยนได้เมื่อมีการสร้างตัวอย่าง อลิคว๊อท หรืออนุพันธ์ใหม่

วันที่รับ

วันที่มีการรับตัวอย่างเข้าโมดูลห้องปฏิบัติการ วันที่ปัจจุบันจะเป็นวันที่ใช้เป็นวันที่รับ แต่สามารถเปลี่ยนได้ด้วยตนเอง เมื่อมีการบันทึกข้อมูลเป็นครั้งเเรกเเล้วค่านี้จะไม่อนุญาติให้เปลี่ยนแปลง

สภาพตัวอย่าง

ที่ได้รับ

สภาพตัวอย่างที่ได้รับ หากตัวเลือกไม่ไช่ รับไว้ในสภาพดี ต้องใส่คำอธิบายในช่อง ความเห็น ช่องนี้จะว่างสำหรับตัวอย่างที่มีสถานะ ยังไม่ได้รับ สำหรับอลิคว๊อท เเละอนุพันธ์

การวินิจฉัย

การทดสอบ

ในเบื้องต้น การวินิฉัยการทดสอบจะเป็นการวินิจฉัยเดียวกับที่ระบุในเคสผู้ป่วย

หรือเซสชั่นที่เกี่ยวข้องกับตัวอย่างนั้น อาจเปลี่ยนแปลงได้ตามการทดสอบที่กำหนด

ชื่อการทดสอบ

ชื่อของการทดสอบที่กำหนด

สถานะการทดสอบ

สถานะปัจจุบันของการทดสอบ เเละสามารถเปลี่ยนได้ ยกเว้นการทดสอบที่มีสถานะ สุดท้าย หรือ ถูกแก้ไข

สำหรับรายละเอียดเพิ่มเติมเกี่ยวกับสถานะการทดสอบ อ่านได้ที่หมวด สถานะการทดสอบ

ผลการทดสอบ

ผลของการทดสอบ สามารถแก้ไขหรือเปลี่ยนผลการทดสอบที่มีสถานะ ขั้นต้น หรือ ถูกแก้ไข

วันที่เริ่มการทดสอบ

แสดงวันที่เริ่มการทำการทดสอบ สามารถเปลี่ยนแปลงได้

วันผลลัพธ์

วันที่ใส่ผลลัพธ์ พิสูจน์ หรือแก้ไขผลลัพธ์ วันที่ปัจจุบันจะใช้เป็นค่าตั้งต้นขณะผู้ใช้ใส่ข้อมูล เเละสามารถเปลี่ยนได้

กลุ่มลักษณะ

การทดสอบ

หมวดหมู่ตามลักษณะการทดสอบ หมวดหมู่เหล่านี้จัดการได้ในบรรณาธิการอ้างอิงพื้นฐานบนเดสท็อบ

รายละเอียด

การทดสอบเพิ่มเติม

แบบฟอร์มรายละเอียดการทดสอบยืดหยุ่น flexible form

ปุ่ม

เลือกทั้งหมด

เลือกรายการในโมดูลห้องปฏิบัติการทั้งหมด

บันทึก

บันทึกการเปลี่ยนแปลงทั้งหมด รายการที่มีการเปลี่ยนแปลงแต่ยังไม่ถูกบันทึกจะแสดงเป็น ตัวหนา

OK
