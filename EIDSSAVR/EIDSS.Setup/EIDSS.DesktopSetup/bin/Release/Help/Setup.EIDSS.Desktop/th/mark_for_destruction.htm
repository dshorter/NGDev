ทำเครื่องหมายระบุตัวอย่างสำหรับการทำลาย

การใส่ข้อมูลเพื่อทำลายตัวอย่างมีขั้นตอนต่อไปนี้:

1.ตัวอย่างจะถูกทำเครื่องหมายสำหรับการทำลาย (การทิ้ง) ตัวอย่างเหล่านี้มีสถานะ ทำเครื่องหมายรสำหรับการทำลายเป็นประจำ

2.การทำลายได้รับการอนุมัติ (ในระบบแสดงข้อมูล ยอมรับ) หลังจากนั้นตัวอย่างจึงจะถูกทำลาย สถานะของตัวอย่างจะเปลี่ยนเป็น ถูกทำลาย

ขณะที่รายการตัวอย่างถูกระบุให้ถูกทำลาย เเต่รายการยังไม่ถูกลบ จะสามารถยกเลิกการ (การทำลาย) รายการนั้นได้ รายละเอียด อ่านได้ที่หมวด ยกเลิกการทำลาย

เฉพาะตัวอย่างที่มีสถานะ ได้รับ ที่ผู้ใช้สามารถทำเครื่องหมายระบุสำหรับการทำลาย (การทิ้ง) ได้

1.จากเมนู สมุดบันทึก เลือก หมวดห้องปฏิบัติการ หน้าต่าง หมวดห้องปฏิบัติการ จะเปิดขึ้น

2.ค้นหา เเละเลือกการทดสอบที่ต้องการรุะบุทำเครื่องหมายเพื่อทำลาย

3.คลิกขวาที่รายการการทดสอบที่เลือกจากเมนูดร็อปดาวน์ เลือก การทำลายตัวอย่าง เเละเลือก ระบุรายการที่จะถูกทำลายเป็นประจำ เเละ เลือกวิธีการทำลายที่ต้องการ สถานะของตัวอย่างจะเปลี่ยนเป็น ทำเครื่องหมายสำหรับการทำลายเป็นประจำ

4.คลิกปุ่ม บันทึก เพื่อยืนยันการบันทึก

ดูรายการเหล่านี้ได้:

•หมวดห้องปฏิบัติการ L25

•ทำลายตัวอย่าง
