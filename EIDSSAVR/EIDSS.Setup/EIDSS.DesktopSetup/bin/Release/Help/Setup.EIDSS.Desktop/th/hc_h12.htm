การเปลี่ยนการวินิจฉัย H12

การวินิจฉัยในเคสผู้ป่วยอาจเปลี่ยนแปลงได้ โดยข้อมูลกการวินิจฉัยเดิมไม่ต้องแก้ไข

ผู้ใช้งานใส่ข้อมูลการวินิจฉัยใหม่ลงใน แบบฟอร์มรายงานผู้ป่วย ด้านล่างในส่วนข้อมูลทางคลีนิค โดยระบุในช่อง การวินิจฉัยล่าสุด (ถ้าทราบ)

หน้าต่าง การเปลี่ยนการวินิจฉัย (H12) เปิดขึ้นเพื่อให้ใส่การวินิจฉัยล่าสุดในเคสผู้ป่วยนั้น

รูปภาพ 46 ใส่ข้อมูลของการเปลี่ยนการวินิจฉัย 

ใส่เหตุผลการเปลี่ยนการวินิจฉัย ในช่อง กรุณาป้อนเหตุผลในการเปลี่ยนการวินิจฉัย โดยเลือกเหตุผลจากรายการดร็อปดาวน์ และคลิก OK เพื่อปิดหน้าต่าง

รูปภาพ 47 ใส่เหตุผลที่เปลี่ยนการวินิจฉัย
