ค้นหาการระบาด

1.คลิก สมุดบันทึก -> การระบาด

2.หน้ารายการเหตุการณ์ระบาดเปิดขึ้น

3.ในแผงการค้นหา ใส่เกณฑ์การค้นหาในช่องแผงการค้นหา ผู้ใช้งานสามารถกดปุ่มล้างสำหรับล้างข้อมูลในบางช่องที่ระบบใส่ไว้ เพื่อระบุการค้นหาที่ต้องการได้

รูปภาพ  12 รายการเหตุการณ์การระบาด

 

4.แก้ไขข้อมูล ดูรายละเอียดการใส่ข้อมูลได้ที่ รายละเอียดการระบาด C11  

5.คลิก ค้นหา ผลลัพธ์จะแสดงอยู่ด้านขวาของแบบฟอร์ม

6.เลือกจากรายการ เเละคลิกปุ่ม หรือกดที่เลข ID การระบาด เพื่อเปิด แก้ไข หรือ ลบ
