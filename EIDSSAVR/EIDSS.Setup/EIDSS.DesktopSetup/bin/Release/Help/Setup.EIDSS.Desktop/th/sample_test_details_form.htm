แบบฟอร์มรายละเอียดตัวอย่าง/การทดสอบ

แบบฟอร์มรายละเอียด บันทึกห้องปฏิบัติการ นั้นเป็นแบบฟอร์มที่เปิดขยายและย่อได้ มีข้อมูลเกี่ยวกับตัวอย่าง เเละการทดสอบ ผู้ใช้เปิดได้โดยคลิกรายการที่ต้องการในตารางรายการ คลิกไปที่ปุ่ม แก้ไข ()

รูปภาพ 24 แบบฟอร์มรายละเอียดตัวอย่าง/การทดสอบ

แบบฟอร์มมีหมวดต่างๆต่อไปนี้:

•แบบฟอร์มหมวดรายละเอียดตัวอย่างของตัวอย่าง/รายละเอียดการทดสอบ

•แบบฟอร์มหมวดรายละเอียดการทดสอบของตัวอย่าง/รายละเอียดการทดสอบ

•แบบฟอร์มหมวดรายละเอียดการทดสอบเพิ่มเติมของตัวอย่าง/รายละเอียดการทดสอบ

•หมวดประวัติการแก้ไขของตัวอย่าง/รายละเอียดการทดสอบ

มุมมอง รายละเอียดตัวอย่าง ใช้เพื่อเเสดงข้อมูลเกี่ยวกับตัวอย่าง

หมวด รายละเอียดการทดสอบ เเสดงถึงรายละเอียดเกี่ยวกับการทดสอบ

หมวด ประวัติการแก้ไข ใช้เพื่อเเสดงข้อมูลตามลำดับเวลาเกี่ยวกับการเปลี่ยนแปลงของสถานะ การทดสอบหลังจากทำการพิสูจน์ผลเเล้ว

ดูรายการเหล่านี้ได้:

•หมวดห้องปฏิบัติการ L25

•แท็บหมวดห้องปฏิบัติการ เเละการตั้งค่ากำหนดเองของฉัน

•แผงค้นหาของหมวดห้องปฏิบัติการ
