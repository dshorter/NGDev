กำหนดการทดสอบ L32

แบบฟอร์ม กำหนดการทดสอบ ใช้กำหนดการทดสอบที่วางแผนสำหรับตัวอย่าง เเละใส่ผลทดสอบ ตามงานที่เสร็จสิ้น หรือใส่ข้อมูลทั้งหมดในคราวเดียว

รูปภาพ 28 กำหนดการทดสอบ

ช่องข้อมูล

การวินิจฉัย

การทดสอบ

ในเบื้องต้น การวินิฉัยการทดสอบจะเป็นการวินิจฉัยเดียวกับที่ระบุในเคสผู้ป่วย หรือเซสชั่นที่เกี่ยวข้องกับตัวอย่างนั้น อาจเปลี่ยนแปลงได้ตามการทดสอบที่กำหนด

ชื่อการทดสอบ

เลือกชื่อการทดสอบจากรายการดร็อปดาวน์ ผู้ใช้อาจกาช่องเครื่องหมาย กรองชื่อการทดสอบโดยการวินิจฉัย เพื่อกรองให้แสดงค่าตัวเลือกของการทดสอบที่เหมาะสมกับ การวินิจฉัยที่ระบุ ในช่อง การวินิจฉัยการทดสอบ

ส่วนการทดสอบที่เหมาะกับแต่ละการวินิฉัยจัดการได้ใน บรรณาธิการอ้างอิง บนเดสท็อบ

ผลการทดสอบ

ใส่ผลของการทดสอบ หากช่องนี้ยังไม่มี การทดสอบนั้นจะอยู่ในสถานะ ยังไม่เริ่ม เมื่อใส่ผลการทดสอบ ข้อมูลนี้จะเปลี่ยนสถานะการทดสอบเป็น ขั้นต้น

กลุ่มลักษณะ

การทดสอบ

หมวดหมู่ตามลักษณะการทดสอบ หมวดหมู่เหล่านี้จัดการได้ในบรรณาธิการอ้างอิงพื้นฐานบนเดสท็อบ
