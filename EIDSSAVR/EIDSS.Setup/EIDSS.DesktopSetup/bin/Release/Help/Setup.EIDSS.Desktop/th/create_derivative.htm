สร้างอนุพันธ์

อนุพันธ์คือตัวอย่างใหม่ที่เกิดจากตัวอย่างเเม่ แต่มีการเปลี่ยนประเภท ประเภทตัวอย่างอนุพันธ์สำหรับตัวอย่างหลักมีระบุอยู่ใน ประเภทตัวอย่าง -> เมตริกซ์ประเภทตัวอย่าง

ตัวอย่างที่สร้างอนุพันธ์ได้ต้องมี สถานะตัวอย่าง เป็น ได้รับ ซึ่งใช้เป็นตัวอย่างหลักสำหรับการสร้างอนุพันธ์เพิ่ม ตัวอย่างของอนุพันธ์ที่เพิ่งสร้างใหม่จะได้รับสถานะ เป็น ได้รับ เช่นกัน

1.จากเมนู สมุดบันทึก เลือก หมวดห้องปฏิบัติการ หน้าต่าง หมวดห้องปฏิบัติการ จะเปิดขึ้น

2.ค้นหาเเละเลือกตัวอย่างที่เป็นตัวอย่างเเม่สำหรับการสร้างอนุพันธ์ เเละคลิกขวาที่รายการตัวอย่างนั้นๆ เเละจากเมนูดร็อปดาวน์ เลือก สร้างอนุพันธ์ หน้าต่าง สร้างอนุพันธ์ จะเปิดขึ้น

รูปภาพ 12 สร้างอนุพันธ์

3.ป้อนจำนวนอนุพันธ์ที่ต้องการสร้างในช่อง จำนวนตัวอย่าง

4.เลือกประเภทตัวอย่างของอนุพันธ์ใหม่ในช่อง ประเภทตัวอย่างใหม่ (อนุพันธ์)

5.คลิกปุ่ม OK ตัวอย่างใหม่จะถูกเพิ่มเข้าไปในแบบฟอร์ม หมวดห้องปฏิบัติการ อนุพันธ์ที่ถูกสร้างใหม่จะอยู่ในรายการในแบบฟอร์ม หมวดห้องปฏิบัติการ

6.คลิกปุ่ม แก้ไข () เพื่อเปิด แบบฟอร์มรายละเอียดตัวอย่าง/การทดสอบ แบบฟอร์ม รายละเอียดตัวอย่าง จะเปิดขึ้น

7.ใน รายละเอียดตัวอย่าง เพิ่มเติมข้อมูลต่อไปนี้:

•ID รหัสตัวอย่าง จะถูกตั้งค่าชั่วคราวตามที่ระบบจัดการให้ซึ่งจะเป็นเลขรหัสของตัวอย่างนี้ในระบบ แต่ผู้ใช้สามารถเปลี่ยนได้ด้วยตนเอง ถ้าเปลี่ยนเเล้วรหัสตัวอย่างจะถูกใช้สำหรับตัวอย่างนี้ถาวร

•สภาพตัวอย่างที่ได้รับ ช่องนี้มีค่าตั้งต้นที่ระบบใส่ไว้เป็น ยอมรับในสภาพที่ดี เเต่สามารถเปลี่ยนได้ด้วยตนเอง ถ้าผู้ใช้เลือกใส่ค่าอื่น หน้าต่าง เพิ่มในความเห็น จะปรากฏขึ้นเพื่อใส่คำอธิบาย ในช่อง ความเห็น เเละคลิกปุ่ม OK หน้าต่าง เพิ่มในความเห็น จะปรากฏ
               ถ้าผู้ใช้เลือกค่าในช่อง สภาพตัวอย่างที่ได้รับ เป็น ถูกปฏิเสธ เเละมีการบันทึกเรียบร้อย จะทำให้รายการตัวอย่างนี้จะไม่ถูกพบหรือเก็บในระบบ

8.คลิกปุ่ม OK เพื่อปิดหน้าต่าง รายละเอียดตัวอย่าง/การทดสอบ

9.คลิกปุ่ม บันทึก เเละยืนยันการลงรายการ

ดูรายการเหล่านี้ได้:

•สร้างอลิคว๊อท

•หมวดห้องปฏิบัติการ L25
