ดูหรือแก้ไขรายละเอียดตัวอย่าง

รายละเอียดตัวอย่างคือส่วนให้ข้อมูลเกี่ยวกับตัวอย่างที่อยู่ในระบบ บางรายการแสดงในตารางบนแบบฟอร์ม หมวดห้องปฏิบัติการ ในการแก้ไขรายละเอียดตัวอย่าง อ่านได้ที่ แบบฟอร์มตัวอย่าง/รายละเอียดการทดสอบ

สถานะของตัวอย่างจะกำหนดรายละเอียดที่แก้ไขได้หรือไม่ได้ ผู้ใช้ควรสังเกตว่ารายการที่ข้อมูลถูกแก้ไข จะขึ้นเป็นตัวหนาเพื่อเตือนให้ผู้ใช้ทำการบันทึกการเปลี่ยนแปลง ในขั้นตอนนี้ ผู้ใช้อาจเลือก ยอมรับ (บันทึก) หรือ ถูกปฏิเสธ (ยกเลิก) ได้

1.จากเมนู สมุดบันทึก เลือก หมวดห้องปฏิบัติการ หน้าต่าง หมวดห้องปฏิบัติการ จะเปิดขึ้น

2.ค้นหาตัวอย่างที่ต้องการ

3.คลิกปุ่ม แก้ไข () เพื่อเปิด แบบฟอร์มตัวอย่าง/รายละเอียดการทดสอบ หน้าต่าง รายละเอียดตัวอย่าง/การทดสอบ จะปรากฏพร้อมรายละเอียดตัวอย่าง ซึ่งมีข้อมูลอย่างละเอียดเกี่ยวกับตัวอย่างนั้นๆ

4.ขยาย หมวดรายละเอียดตัวอย่าง เพื่อแก้ไขข้อมูล

รูปภาพ 18 แท็บรายละเอียดตัวอย่าง/การทดสอบ

5.คลิกปุ่ม OK หน้าต่าง รายละเอียดตัวอย่าง จะปิดลง

6.คลิกปุ่ม บันทึก เพื่อยืนยันการบันทึก

ดูรายการเหล่านี้ได้:

•หมวดห้องปฏิบัติการ L25
