สถานะการทดสอบ

สถานะการทดสอบในโมดูลห้องปฏิบัติการ เป็นข้อมูลสำคัญแสดงขั้นตอนของการทดสอบเเละกำหนดงานในการทดสอบสำหรับตัวอย่างที่กำหนด ในการทดสอบนั้นๆ ผู้ใช้สามารถใช้สถานะการทดสอบเพื่อค้นหาการทดสอบ หรือตัวอย่างได้ สถานะการทดสอบเปลี่ยนทั้งโดยอัติโนมัติและเปลี่ยนได้โดยผู้ใช้ แล้วแต่กรณี
