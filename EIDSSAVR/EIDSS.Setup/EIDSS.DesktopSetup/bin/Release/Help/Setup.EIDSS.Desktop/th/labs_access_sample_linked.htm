รับตัวอย่างที่เชื่อมกับเคสผู้ป่วย

การลงรายการตัวอย่างรับเข้ามา ทำได้ทั้งแบบที่เชื่อมกับเคสผู้ป่วยหรือเซสชั่น หรืออาจเป็นการเพิ่มรายการรับเข้าในโมดูลห้องปฏิบัติการภายหลัง หรือเพิ่มในครั้งเดียวกันในโมดูลห้องปฏิบัติการ สำหรับรายละเอียดการสร้าง เเละการเพิ่มตัวอย่างใหม่ในครั้งเดียวกัน ซึ่งเชื่อมต่อกับกรณี หรือเซสชั่น อ่านได้ที่หมวด สร้าง เเละเพิ่มในตัวอย่าง เชื่อมต่อกับกรณี/เซสชั่น (LS)

1.จากเมนู สมุดบันทึก เลือก หมวดห้องปฏิบัติการ หน้าต่าง หมวดห้องปฏิบัติการ จะเปิดขึ้น

2.ค้นหาตัวอย่างที่ต้องการ: อาจใช้เกณฑ์การค้นหา สถาบันส่งไปที่ เเละใส่ค่าในช่อง ระบุเลือกเฉพาะตัวอย่างที่สามารถย้ายเข้า หรือเพิ่มเข้าได้

3.เลือกตัวอย่างที่ต้องการ

4.คลิกขวาที่ตัวอย่างที่ต้องการเพิ่มเข้า เเละจากเมนูดร็อปดาวน์ เลือก ลงรายการรับตัวอย่าง เเละเลือกสภาพของตัวอย่างที่ได้รับ

รูปภาพ 7 เพิ่มรายการรับตัวอย่างเข้ามา

ถ้าผู้ใช้เลือกสภาพตัวอย่างที่ไม่ดี หน้าต่าง เพิ่มความเห็น จะปรากฏเพื่อใส่คำอธิบายในช่อง ความเห็น เเละคลิกปุ่ม OK หน้าต่าง เพิ่มเข้าในความเห็น จะปิดลง

รูปภาพ 8 ช่องใส่ความเห็นเมื่อรับตัวอย่างที่สภาพไม่ดี

ระบบจัดการรายการตัวอย่างตามข้อมูลสภาพตัวอย่างที่รับเข้ามาทั้งที่และไม่ดี ดังนี้:

•สถานะตัวอย่าง จะถูกเปลี่ยนเป็น ได้รับ

•ช่อง วันที่เพิ่ม แสดงวันที่ปัจจุบัน ผู้ใช้สามารถเปลี่ยนได้ใน แบบฟอร์มตัวอย่างหมวดรายละเอียดตัวอย่าง/รายละเอียดการทดสอบ

•รหัสตัวอย่าง จะถูกตั้งค่าชั่วคราวตามที่ระบบจัดการให้ซึ่งจะเป็นเลขรหัสของตัวอย่างนั้นที่ใช้ในระบบ แต่ผู้ใช้สามารถเปลี่ยนได้ด้วยตนเอง ถ้าเปลี่ยนเเล้วรหัสตัวอย่างจะถูกใช้สำหรับตัวอย่างนี้ถาวร แบบฟอร์มตัวอย่างหมวดรายละเอียดตัวอย่าง/รายละเอียดการทดสอบ

ถ้าผู้ใช้เลือกค่าในช่อง สภาพตัวอย่างที่ได้รับ เป็น ถูกปฏิเสธ เเละมีการบันทึกเรียบร้อย จะทำให้รายการตัวอย่างนี้จะไม่ถูกพบหรือเก็บในระบบ

5.ระบบกำหนด ID รหัสตัวอย่าง ให้โดยอัตโนมัติ ตัวอย่างจะถูกเพิ่มข้อมูลการรับเข้า

ดูรายการเหล่านี้ได้:

•รับตัวอย่างใหม่ที่ยังไม่เชื่อมกับเคสผู้ป่วย

•การรับตัวอย่างเป็นกลุ่ม

•หมวดห้องปฏิบัติการ L25
