แก้ไขการระบาด

1.คลิก สมุดบันทึก -> การระบาด รายการเหตุการณ์การระบาด จะเปิดขึ้น

2.ค้นหา การระบาดที่ต้องการแก้ไขโดย เลือกจากรายการ เเละคลิกปุ่ม หรือกดที่เลข ID การระบาด เพื่อเปิดได้เช่นกัน

รูปภาพ 8 รายการการระบาด

 

3.ในการแก้ไขการระบาด ในแบบฟอร์ม รายละเอียดการระบาด

รูปภาพ 9 รายละเอียดการระบาด

 

4.แก้ไขข้อมูล ดูรายละเอียดการใส่ข้อมูลได้ที่ รายละเอียดการระบาด C11  

5.คลิก Ok เพื่อบันทึกการระบาด เเละปิดแบบฟอร์ม
