แบบฟอร์มหมวดรายละเอียดตัวอย่างของตัวอย่าง/รายละเอียดการทดสอบ

แท็บ รายละเอียดตัวอย่าง เเสดงข้อมูลและรายละเอียดอื่นๆ ที่ไม่อยู่บนแบบฟอร์ม หมวดห้องปฏิบัติการ

รูปภาพ 25 แท็บรายละเอียดตัวอย่าง

คอลัมน์รายการผลลัพธ์

ID ผู้ป่วย/เซสชั่น

หมายเลขเคสผู้ป่วยหรือเซสชั่นที่เกี่ยวข้องกับตัวอย่าง

รหัสตัวอย่างจาก

พื้นที่/ภาคสนาม

หมายเลขหรือรหัสของตัวอย่างตามที่ได้มาจากแหล่งเก็บตัวอย่าง เพื่อใช้อ้างอิง

สถานะตัวอย่าง

สถานะของตัวอย่างเป็นหนึ่งในข้อมูลที่กำหนดค่าหรือการใส่ข้อมูลอื่นๆ ในการทำงานเพิ่มเติมของกับตัวอย่างนั้น อ่านเพิ่มได้ที่หมวด สถานะตัวอย่าง ค่าในช่องนี้จะเปลี่ยนโดยอัตโนมัติสอดคล้องกับการปฏิบัติงาน

ID รหัสตัวอย่าง

ID ของตัวอย่าง ซึ่งใช้ระบุตัวอย่างในห้องปฏิบัติการ

ระบบจะจัดการหมายเลขให้โดยอัติโนมัติหากผู้ใส่ข้อมูลไม่ได้ระบุ

หมายเลขเอง

ประเภทตัวอย่าง

แสดงประเภทตัวอย่าง เเละสามารถเปลี่ยนประเภทได้สำหรับตัวอย่างอลิคว๊อท หรืออนุพันธ์ที่สร้างเพิ่มขึ้น เช่นตัวอย่างหลักเป็นเลือด แต่อนุพันธ์ที่สร้างจากเลือดอาจเป็นประเภทตัวอย่างอื่น

วันที่รับ

วันที่มีการรับตัวอย่างเข้าโมดูลห้องปฏิบัติการ วันที่ปัจจุบันจะเป็นวันที่ใช้เป็นวันที่รับ แต่สามารถเปลี่ยนได้ด้วยตนเอง เมื่อมีการบันทึกข้อมูลเป็นครั้งเเรกเเล้วค่านี้จะไม่อนุญาติให้เปลี่ยนแปลง

สภาพตัวอย่าง

ที่ได้รับ

ระบุสภาวะของตัวอย่างในเวลาที่เพิ่มเข้าไป ถ้าสภาวะนี้ต่างจากสภาวะที่ดี ต้องใส่คำอธิบายในคอลัมน์ ความเห็น หลังจากมีการบันทึกตัวอย่างเเล้ว ช่องนี้จะไม่สามารถใช้ได้ ช่องนี้จะว่างสำหรับตัวอย่างที่มีสถานะ ไม่ได้ถูกเพิ่มเข้า สำหรับอลิคว๊อท เเละอนุพันธ์

ID ของตัวอย่างหลัก

แสดงหมายเลขของตัวอย่างหลักที่ใช้สร้างอลิคว๊อท เเละอนุพันธ์ นั้น หากตัวอย่างนั้นไม่ใช่อลิคว๊อทหรืออนุพันธ์ - ช่องนี้จะว่าง

ประเภทเคส/เซสชั่น

ประเภทของเคสหรือเซสชั่นที่เชื่อมกับตัวอย่าง ซึ่งเชื่อมได้กับรายการดังนี้:

•ผู้ป่วย

•สัตว์ปีก

•ปศุสัตว์

•การเฝ้าระวังพาหะนำโรค

ไม่มีการแสดงรายการใดๆสำหรับเซสชั่น AS

ข้อมูลผู้ป่วย/

สายพันธุ์/

พาหะนำโรค

ข้อมูลเพิ่มเติมเกี่ยวกับตัวอย่างที่ควรป้อนลงไป:

•กรณีสัตว์ปีก: ประเภทสายพันธ์

•กรณีปศุสัตว์: ID สัตว์

•เคสผู้ป่วย: ชื่อและนามสกุลของผู้ป่วย

•เซสชั่น AS: ประเภทเเละ ID ของสัตว์

ระบุหน่วยงานภายนอกที่โอนย้ายตัวอย่างไป การลงข้อมูลผลทดสอบที่ได้รับ อ่านได้ที่ ตั้งค่าผลลัพธ์สำหรับการทดสอบในห้องปฏิบัติการภายนอก

วิธีการทำลาย

เลือกวิธีการทำลายที่จะใช้ ช่องนี้ใช้สำหรับตัวอย่างที่มีสถานะ ทำเครื่องหมายระบุรายการสำหรับการทำลายเป็นประจำ

ดูรายการเหล่านี้ได้:

•หมวดห้องปฏิบัติการ L25

•มุมมองหมวดห้องปฏิบัติการ

•แบบฟอร์มหมวดรายละเอียดการทดสอบของตัวอย่าง/รายละเอียดการทดสอบ

•แบบฟอร์มหมวดรายละเอียดการทดสอบเพิ่มเติม/รายละเอียดการทดสอบ

•แบบฟอร์มหมวดประวัติการแก้ไขของตัวอย่าง/รายละเอียดการทดสอบ

•ดู หรือแก้ไขรายละเอียดตัวอย่าง
