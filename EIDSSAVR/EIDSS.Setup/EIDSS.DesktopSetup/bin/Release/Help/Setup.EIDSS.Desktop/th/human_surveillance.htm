การเฝ้าระวังผู้ป่วย

โมดูลการเฝ้าระวังผู้ป่วยนั้นเป็นเครื่องมือใช้สำหรับการป้อน เเละการจัดการข้อมูลผู้ป่วย รวมไปถึงข้อมูลในการวินิจฉัย เเละการแยกประเภทผู้ป่วย ตัวบ่งชี้ทางประชากรศาสตร์ การสอบสวนการระบาดของโรค อาการของโรคทางคลีนิค ตัวอย่างทางห้องปฏิบัติการ เเละผลการทดสอบ การจัดการข้อมูลลงแบบฟอร์มผู้ป่วย เริ่มจาก ขั้นตอนเเรกคือการบันทึกผู้ป่วยใหม่ ผู้ใช้ต้องใส่ข้อมูลจำเป็นต้องกรอกตามที่ระบุไว้เเละป้อนข้อมูลที่ใช้ในการเเจ้งเตือน เมื่อข้อมูลเกี่ยวกับโรคติดต่อร้ายเเรงถูกบันทึกเเล้ว การเเจ้งเตือนจะเริ่มขึ้นโดยอัตโนมัติ ข้อมูลที่บันทึกจะถูกส่งไปเข้าระดับข้อมูลที่สูงขึ้น สามารถค้นหาในระบบเเละแก้ไขข้อมูลผู้ป่วยได้
