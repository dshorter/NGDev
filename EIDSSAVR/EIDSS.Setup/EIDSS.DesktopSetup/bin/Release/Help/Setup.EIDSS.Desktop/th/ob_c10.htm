รายการเหตุการณ์การระบาด C10

Show/Hide Hidden Text

รายการเหตุการณ์การระบาด(C10) คือแบบฟอร์มที่มีการลงรายการการระบาดทั้งหมด ที่ตรงกับเกณฑ์การค้นหา

รูปภาพ 13 หน้าต่างรายการการระบาด

 

หมวด แผงการค้นหา มีฟิลด์ให้เลือกใส่ค่าที่ใช้ค้นหาการระบาด

รูปภาพ 14 แผงการค้นหา

 

ID การระบาด

รหัสที่ระบบกำหนดให้เมื่อสร้างการระบาดและบันทึกเรียบร้อย

การวินิจฉัย

ระบบอาจใส่ค่าการวินิจฉัยของการระบาดในช่องนี้

กลุ่มการวินิจฉัย

ระบบอาจใส่ค่ากลุ่มการวินิจฉัยของการระบาดในช่องนี้

วันที่เริ่ม

วันที่ๆเริ่มการระบาด

วันสิ้นสุด

วันสิ้นสุดเหตุการณ์ระบาด

สถานที่ที่เกิดการระบาดหลัก

ตำแหน่งหลักของการระบาดนั้น อาจถูกระบุอยู่ในช่องนี้ ด้วยฟอร์แม็ต ที่ใช้ในคอลัมน์     สถานที่ที่เกิดการระบาดหลัก ใน รายการผลการค้นหา

ผู้ใช้อาจใช้เครื่องหมายดอกจัน (*) เพื่อขยายการค้นหาผลลัพธ์ที่ตรงกับค่าตำแหน่งหลักที่ระบุอยู่ใน แผงการค้นหาสถานที่ที่เกิดการระบาดหลัก

สถานะการระบาด

อาจมีการระบุสถานะการระบาดในช่องนี้ ในการล้างเนื้อหาในช่องออก เลือกช่องที่ว่างจากรายการดร็อปดาวน์

ผู้ป่วย

ชื่อของผู้ป่วยในการระบาด

เจ้าของฟาร์ม

ชื่อของเจ้าของฟาร์ม สำหรับกรณีที่มีสัตว์เกี่ยวข้อง

 
หมวดค้นหารายการผลลัพธ์ที่แสดงรายการการระบาดที่ตรงกับเกณฑ์การค้นหาที่ป้อน

รูปภาพ 15 หน้าต่างรายการการระบาด

 

ID การระบาด

การวินิจฉัย/กลุ่มการวินิจฉัย

การวินิจฉัยของการระบาด

วันที่เริ่ม

วันที่เริ่มต้นของการระบาดนั้น

วันสิ้นสุด

วันสิ้นสุดของการระบาดนั้น

สถานที่ที่เกิดการระบาดหลัก

ตำแหน่งพื้นที่ของการระบาดแสดงตามที่ได้ระบุ โดยฟอร์แม็ตตำแหน่งทางภูมิศาสตร์

สถานะการระบาด

ระบุสถานะการระบาด

ผู้ป่วย/เจ้าของฟาร์ม
