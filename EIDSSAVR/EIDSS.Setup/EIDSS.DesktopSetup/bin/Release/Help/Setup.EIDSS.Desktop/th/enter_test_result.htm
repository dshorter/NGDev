ป้อนผลการทดสอบ

หลังจากการทดสอบเสร็จสิ้นเเล้ว ผลของการทดสอบต้องบันทึกในระบบ ซึ่งสามารถลงผลการทดสอบที่ได้รับจากหน่วยอื่นที่ไม่ได้ใช้ EIDSS ได้ ผลลัพธ์ของการทดสอบต้องได้รับการพิสูจน์รับรอง (ยืนยันผลวินิจฉัย) ในแบบฟอร์ม หมวดห้องปฏิบัติการ อำนวยความสะดวกให้ใส่ข้อมูลโดยข้ามบางขั้นตอนได้ เช่น กำหนดการทดสอบ เเละรับรองพิสูจน์ได้ในเวลาเดียวกัน ถ้ารายการนี้ปรากฏ สถานะของการทดสอบจะถูกเปลี่ยนทันทีจาก ยังไม่เริ่ม เป็น สุดท้าย

•หากข้อมูลการทดสอบมาพร้อมกับผลการทดสอบที่มีข้อมูลการพิสูจน์รับรองผลวินิจฉัยเรียบร้อยแล้ว สถานะของการทดสอบจะเปลี่ยนเป็น สุดท้าย ทันที

•สำหรับผลทดสอบที่ได้จากห้องปฏิบัติการภายนอกมักจะเป็นการได้รับข้อมูลที่ครบถ้วนพร้อมผล จึงเป็นการลงข้อมูลรายการในคราวเดียว ดังนั้น สถานะของมันจะกลายเป็น สุดท้าย โดยไม่มีขั้นตอนอื่นๆ

•ตัวอย่างที่มีสถานะ ถูกย้ายออก ไปยังหน่วยงานที่ไม่ได้ใช้ EIDSS เท่านั้นที่จะใส่ข้อมูลการทดสอบที่กำหนดโดยระบุเป็นผลลัพธ์ภายนอกได้

•สำหรับรายละเอียด อ่านได้ที่หมวด สถานะตัวอย่าง เเละ สถานะการทดสอบ

ดูรายการเหล่านี้ได้:

•ป้อนผลลัพธ์การทดสอบ

•ตั้งค่าผลลัพธ์สำหรับการทดสอบในห้องปฏิบัติการภายนอก
