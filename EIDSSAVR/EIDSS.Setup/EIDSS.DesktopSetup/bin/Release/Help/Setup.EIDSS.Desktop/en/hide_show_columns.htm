Hide/show column/band

If the columns required in the pivot grid are not needed in the final report, hide them in the View tab. The data from the hidden column is not lost, and can be shown any moment it is required. However, hiding a column removes sorting. When a band is hidden with its child's bands and columns, the headers of the hidden children are not displayed in the Customization form.

There are 3 methods to hide a column. Use any of them, they lead to the same result.

First method to hide a column

Drag and drop the header of the column to be hidden to the view cell area until the cursor changes its image to a big 'X' sign.

Fig. 568 Column hiding.1st way

Second method to hide a column/band

1.Right-click a header of any table column/band.

2.From the drop-down menu select Column/Band Chooser. The Customization pop-up window opens.

3.Drag and drop the header of the column/band to be hidden to the Customization window. The column/band will be removed from the table.

Fig. 569 Column hiding. 2nd way

Third method to hide a column/band

1.Right-click the header of the column/band to hide.

2.From the drop-down menu, select Remove This Column or Remove Band.

Fig. 570 Column hiding. 3d way

Show a column/band

4.Right-click a header of any table column/band.

5.From the drop-down menu, select Column/Band Chooser. The Customization pop-up window opens.

6.Do one of the following:

•Drag and drop the name of the column/band to be unhidden from the Customization window to its place in the table.

Fig. 571 Column unhiding

•Double-click the name of the column to be unhidden in the Customization window. In the latter case the column will be restored to the right of the existing columns.
