Printing

The AVR module supports printing of the following AVR elements:

•Report

•Chart

•Map

Print a report/chart/map

1.Open the report/chart/map.

2.Click the Print button. The Preview window opens. It consists of a preview of the page to print and a toolbar.

Fig. 583 Printing preview

   The toolbar is designed to set up the page and start the printing.

Fig. 584 Preview toolbar

3.Set up the page.

4.Click the Print button to send the document to the printer.
