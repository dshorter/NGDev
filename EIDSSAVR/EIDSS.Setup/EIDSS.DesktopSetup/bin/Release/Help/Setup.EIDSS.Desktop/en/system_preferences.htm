System Preferences

☝It is strongly recommended that the System Preferences configuration were performed only by the system administrator.

In the System menu, click System Preferences. The System Preferences window opens.

Fig. 18  Options window

Field/Variable

Description

Startup Language

Default language of the application after start.

Country

Country of the installation.

Barcode Printer

Default printer for barcodes.

Document Printer

Default printer for documents.

EPI Info Path

A path to the installed EPI Info software.

Default Map Project

A map project that will be opened by default in AVR and point selection forms in cases. All map projects names that are saved in EIDSS are displayed in drop-down list.

Show text in toolbar

Select to display button names in the main menu toolbar.

Show save data prompt

Select for the confirmation dialog to appear after any window is closed.

Show navigator in H02 form

Select to display the navigation pane in the Human Case Report Form.

Lab Module Simplified mode

Select to enable Lab Module Simplified mode (Laboratory Section only) or unselect to enable different access to the Laboratory Module functionality.

Default Region in Search Panels

Select to auto fill the Region field in search forms.

Show warning for final case classification

Select to receive warning message if the entered data do not correspond the specified final case classification.

Show warning when big layout loading

A warning message with Ok button appears on the screen when big layout loading.

Print Map in Veterinary Investigation Forms

Select to print the map when the Veterinary Investigation paper form is printed.

Show * for blank values in AVR

Select to display the blank values of the AVR data as "*" symbol.

Filter samples by Diagnosis

Define the default state of the Filter samples by Diagnosis check-box in the following tabs:

•Human Case Research Form (H02) -> Case Investigation tab -> Samples Collection tab

•Livestock Case Investigation Form (V02) -> Samples tab

•Avian Case Investigation Form (V03) -> Samples tab

•Active Surveillance Session (V18) -> Detailed Info tab -> Farm/Animals/Samples subtab

Number of days for which data is displayed by default

Only the data over the defined period is displayed by default.
