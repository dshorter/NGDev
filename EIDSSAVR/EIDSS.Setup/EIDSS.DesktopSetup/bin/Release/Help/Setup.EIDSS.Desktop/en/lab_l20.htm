Select Tests L20

The Select Tests (L20) form is used to assign tests to samples.

Fig. 498 Select Tests form

Samples

Lab Sample ID

Lab Sample IDs of selected samples are displayed and cannot be changed.

Diagnosis

Select the diagnosis to confirm or refuse from a drop-down list. For human cases, only one diagnosis is available. Vet cases and AS sessions can have more than one available diagnosis, the final diagnosis is marked in bold. The current diagnosis of the case/session, related to the sample, is used by default. The field is mandatory.

Filter

Select Diagnosis to filter the test list

Select a diagnosis from a drop-down list to filter tests list. Only diagnoses of the case/session, related to the selected sample is linked, are available in the drop-down list. The field in not mandatory.

Tests

Test Name

Select the desired test.

Test Category

Presumptive is always displayed as a test category.

Assigned Tests

Lab Sample ID

Lab Sample ID of the sample related to the test.

Test Diagnosis

Diagnosis to be confirmed or refused by the test.

Test Name

Selected test type, cannot be changed. To change the test name, delete the wrong test by clicking the Remove button and add a new one.

Test Type

Select a category of the test from a drop-down list. The Presumptive category is used by default.  The categories are set up in the Reference Editor.

Buttons

Add

Adds selected tests to the list of assigned tests. A test cannot be assigned twice to the same sample.

Remove

Removes selected tests from the list of assigned tests.
