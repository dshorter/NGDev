Samples marked for disposition L07

The Samples marked for disposition (L07) form is a journal of samples marked for disposition (routine destruction). Using this form, a user can accept or reject samples marked for disposition (routine destruction).  It consists of 2 parts:

•Search Panel

•Samples List

The Search Panel is created to perform the most sophisticated search by sample details. The basic and most frequently used parameters are present in the panel as search criteria.

The Samples List is a table where each line is a sample record with sample ID and a number of sample parameters displayed in the table columns. If columns are not needed or irrelevant, hide them to the Column Chooser. Read special sections for details about Item Lists, Column Chooser, Search Lists, and List and Search Forms Customization.

Fig. 489 Samples marked for disposition form

Search Criteria Fields

Lab Sample ID

Enter laboratory ID of the desired sample.

Sample Type

Select a type of the desired sample.

Destruction Method

Select a destruction method specified while marking a sample for routine destruction.

Result List Columns

Lab Sample ID

Laboratory sample ID, the principal identification of a sample.

Sample Type

A type of a sample.

Destruction Method

A destruction method specified while marking a sample for routine destruction.

Buttons

Accept

Confirms a disposition (destruction) of a selected sample.

1.Select the sample(s) for destruction to be confirmed.

2.Click the Accept button. The Sample disposition window opens. For details about confirming a sample destruction, read the Destroy sample (IF) section.

Reject

Declines sample destruction request.

1.Select the sample(s) for destruction to be declined.

2.Click the Reject button. The samples are removed from the list. For details about rejecting a sample destruction, read the Reject destruction (IF) section.

Samples that shall not be destructed will return to the Accessioned In status which will be displayed in the Samples List form.
