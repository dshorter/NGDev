Batches List L21

The Batches List (L21) form is created to display all batches registered in EIDSS. It consists of 2 parts.

•Search Panel

•Batches List

The Search Panel is created to perform a by batch details. The basic and most frequently used parameters are present in the panel as search fields.

The Batches List is a table where each line is a batch with batch ID and a number of batch parameters displayed in the table columns. If some columns are not needed or irrelevant, hide them to the Column Chooser. Read special sections for detailed about Item Lists, Column Chooser, Search Lists, and List and Search Forms Customization.

Fig. 499 Batches List window

Search Criteria Fields

Batch ID

Enter the unique ID of the batch to be found.

Case ID/Session ID

Enter a case ID or Session ID related to the batch test.

Test Name

Select the name of the test used for the batch.

Lab Sample ID

Enter the sample related to the batch test.

Status

Select the status of the batch.

Date Test Started From/To

Enter time interval when the test was started.

Result Date

Enter time interval when the result of a test was entered, validated or amended for the last time.

Columns

Batch ID

The unique ID of the batch.

Test Name

The name of the test used for the batch.

Date Test Started

The day when the test started.

Result Date

The day when the test result was known.

Test Count

The number of tests in the batch.

Status

The status of the batch.

Buttons

New

Creates a new batch. For details, read the Create batch in Batches List form section.

Edit

Opens the batch to edit in the Batch Test Details L22 form.

Refresh

Updates the list of batches.

Close

Closes the Batches List window.
