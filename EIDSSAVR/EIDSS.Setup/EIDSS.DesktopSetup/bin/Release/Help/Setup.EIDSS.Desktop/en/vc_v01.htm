Field/Variable

Description

My Cases

Only the cases registered by current user will be displayed if this option is enabled.

Case ID

Case ID may be specified in this field.

Diagnosis

Diagnosis is selected. A search is performed by tentative and final diagnosis.

Return only Final Diagnosis

If checked, then only cases with final diagnosis specified are displayed.

Diagnoses Group

Diagnoses Group is selected.

Case Classification

Case classification is selected.

Case Status

Case status is selected.

Report Type

Report type is selected.

Case Type

Case type is selected.

Data Entry Date From

Lower bound of data entry date is entered.

Data Entry Date To

Upper bound of data entry date is entered.

Investigation Date From

Lower bound of investigation date is entered.

Investigation Date To

Upper bound of investigation date is entered.

Local Sample ID

Local sample ID is entered.

Total Animal Qty From

Lower bound of Total animal quantity is entered.

Total Animal Qty To

Upper bound of Total animal quantity is entered.

Owner First Name

Farm owner first name is entered.

Owner Last Name

Farm owner last name is entered.

Region

Region is selected.

Rayon

Rayon is selected and the list contains the values from the Rayon reference, filtered by selected region.

Town or Village

Settlement is selected, a list that contains values from the Settlements reference, filtered by selected rayon.

Data Entry Site

Data entry site is selected.
Search results list section; displays a list of cases matching the entered search criteria.

Buttons

New Livestock Case

Click the New Livestock Case button to create a new livestock case. The Livestock Disease Investigation Form opens in a new window.

New Avian Case

Click the New Avian Case button to create a new avian case. The Avian Disease Investigation Form opens in a new window.

Edit

To edit, select a case in the search list, and click the Edit button. A selected case opens in a new window in edit-mode.

Delete

To delete, select a case in the search list and click the Delete button. A warning message with Delete/Cancel buttons will appear.
