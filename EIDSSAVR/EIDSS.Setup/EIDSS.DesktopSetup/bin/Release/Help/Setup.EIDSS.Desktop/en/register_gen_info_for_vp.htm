Register general information for vector/pool

Once a new pool/ individual vector is created or an existing one is opened to edit, and the summary information is added, then enter general information for a vector/ pool  in the General tab of Pool/Individual Vector Details form.

The general information of a pool/ individual vector is the information which is nearly the same for each type of a vector: location and time of the vector collection, basic data of the collected vector.

For a detailed description of the fields, read the General tab of Pool/Individual Vector Details Form W03 section.
