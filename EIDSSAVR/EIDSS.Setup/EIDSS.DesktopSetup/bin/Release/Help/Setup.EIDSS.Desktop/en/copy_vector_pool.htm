Copy vector/pool

If a long list of similar vectors/ pools has to be created, it is reasonable to create one vector/ pool and then copy it as many times as required. The copies can be later edited.

When copying a pool/ individual vector, it is possible to select a set of details to be copied with it.

 

1.In the Pools/Individual Vectors subtab of Vector Surveillance Session form, select the pool/ individual vector to be copied.

2.Click the Copy button. The Copy Pool/Individual Vector window opens.

Fig. 396 Copy Pool/Individual Vector window

3.Select the sets of data to be copied with the pool/ individual vector. For details, read the Copy Poll/Individual Vector Form W05 section.

4.Click the Select button. The Copy Pool/Individual Vector window closes. The Pool/Individual Vector Details form opens with a newly created pool/ individual vector to edit.

The fields are copied in the following rules:

•A new Pool/ Vector ID is created in accordance with the Unique Numbering Schema.

•The Session ID and Vector Type are copied from the original pool/ individual vector.

•The Field Pool/ Vector ID is left blank and can be changed manually.

•Samples of the session are copied as well. If the samples are copied, enter IDs for them.

Other data are copied in accordance with the selected parameters.

5.Edit the newly created pool/ individual vector.

6.Click the OK button in the Pool/Individual Vector Details form to save the changes and confirm the creation of the copy. If the Cancel button is clicked now, no new pool/ individual vector is created, and copying is abandoned.
