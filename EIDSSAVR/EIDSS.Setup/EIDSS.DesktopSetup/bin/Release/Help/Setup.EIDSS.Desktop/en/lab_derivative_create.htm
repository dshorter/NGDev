Create derivative (IF)

This operation is available only if the Lab Module Simplified Mode check-box is clear in the System Preferences.

 

1.From the Journals menu, select Samples. The Samples List window opens.

2.Find and select the would-be parental sample(s) for derivative creation.

3.Click the Aliquots/Derivatives button. The Aliquots/Derivatives window opens.

4.Select a sample or several samples in the left pane, and select Create Derivative from a drop-down list.

5.In the Number of new samples field, enter the number of derivatives to be created.

6.Select the type of the derivatives to be created from a drop-down list. This list is defined by the Sample Type -> Derivative Type Matrix. For details, read the Sample Type-Derivative Type Matrix section.

7.Click the Create button. The new samples are listed in the right pane of the Aliquots/Derivatives window. The following columns are populated:

•The Original Sample cell is populated with the parental sample and cannot be changed.

•The Sample Type is the same  as parental sample.

•The Lab Sample ID is created in accordance with the Unique Numbering Schema and can be changed.

8.Repeat steps 4 and 5 for each parental sample.

9.Select the department of the organization in the Functional Area column, and Storage Location for each newly created sample. The fields are optional. For detailed description of the fields, read the Aliquots/Derivatives L06 section.

10.Print barcode labels by clicking on the Print Barcodes button and stick them on the containers if necessary

Fig. 451 Aliquots/Derivatives window

11.Once all required information is specified, click the OK button to save the created samples. The Confirmation window opens.

12.Click the Yes button to confirm the creation.
