Map Toolbar Buttons

Pan

Repositions a map within the window.

Refresh Map

Redraws the map with the most recent changes.

Country Extent

Zooms the view window to the full extent of the current country.

Info

Displays tabular data associated with the object.

Zoom In

Receives a closer area of the map.

Zoom Out

Receives a wider area of the map.

Zoom In/Out by clicking left/right button

Receives a closer area of the map by clicking the left button of the mouse, and a wider area of the map by clicking the right button of the mouse.

Measure Line

Measures distances point by point.

Data base Layer

Accesses the Database to add a layer to the map.

Shape File

Adds an ESRI shape file from different sources to the map.

Raster Layer

Adds a raster layer from different sources to the map.

Open Street Map

Automatically adds an OSM (Open Street Map) layer in the layer control panel. To make this tool active, enable the Internet connection.

Map Settings

Creates a layer for buffer zones.

Export Map to Image

Exports a map to a .jpeg file. Type the name of the file in the File Name window.

Administration Unit Mask

Displays a specified region or rayon.

Map Projects drop-down list

Opens previously saved map projects by selecting the map name in the names list of the Map Projects drop-down list.

Create Buffer Zones Layer

Creates a layer for buffer zones.

Circle Buffer Zone

Adds a circle buffer zone with the case or settlement as a center.

Independent Circle Buffer Zone

Adds a buffer zone with manually assigned point as a center.  

Polygon Buffer Zone

Creates a polygon buffer zone based on the district border.

Independent Polygon Buffer Zone

Manually creates of any polygon buffer zone.

Select Buffer Zone

Selects a buffer zone for updating or removing.
