Register and accession in sample, not linked to case/session (LS)

Samples not linked to a case or session can be registered with a simultaneous accession in the Laboratory module.

 

1.From the Journals menu, select the Laboratory Section. The Laboratory Section window opens.

2.Click the New Sample button. The Register a new sample window opens.

3.Enter the quantity of samples of the same type, which are linked to the same case/session to be registered.

4.Enter the case/session for the samples to be linked.

5.Select the samples' type. For more details about the Register a new sample window, read the Register a new sample L35 section.

6.Click the Add button. The entered samples are registered and ready for accession in.

7.Repeat steps 3 and 4 to add all desired samples.

8.Click the Close button. The Register a new sample window closes. The registered samples are in the list in the Laboratory Section window and have the following information:

•Case/Session ID and Sample Type entered in the Register a new sample window are used; the Sample Type can be manually changed.

•The current date is used as the samples' Accession Date and Collection Date which can be found in the detailed form. The dates can be manually changed if necessary until the form is saved for the first time.

•The Lab Sample ID is temporarily set to (New) and can be manually changed. If changed, the user-defined ID is used for the sample; and if not changed, then the ID will get a permanent number in accordance with the Unique Numbering Schema after the form is saved.

•The Sample Received Condition is set to Accepted in Good Condition but can be changed manually. If so, the Accession In Comment window opens for an explanation to be entered. Enter the explanation to the Comments field and click the OK button. The Accession In Comment window closes.
If the Sample Condition Received field value is changed to Rejected and saved, the sample is hidden and cannot be found by search.

9.Open the detailed form of the sample and specify Patient/Species/Vector Info.

10.Save the accession in. The Lab Sample ID is assigned automatically.

The samples are registered and accessioned in.
