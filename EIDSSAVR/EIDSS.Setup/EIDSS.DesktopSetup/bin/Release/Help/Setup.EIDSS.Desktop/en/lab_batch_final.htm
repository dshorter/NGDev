Finalize a batch (IF)

Batch finalization is an operation of entering a result of a batch, validation of this result and closure of the batch.

This operation is available only if the Lab Module Simplified Mode check-box is clear in the System Preferences.

 

1.From the Journals menu, select Batches. The Batches List window opens. It contains all batches created in the system.

2.Do one of the following:

a)Select the batch to finalize and click the Edit button.

b)Double-click the desired batch. The Batch Test Details form opens.

3.Enter the tests result by one of the following methods:

If the results match:

a)Click the Add Group Result button. The Select Test Result window opens.

b)Select the result in the Select test result field.

c)Click the OK button to save and close the window

If the results differ:

a)Select results individually in the Result column.

4.Enter the employee who has validated the test result in the Validated by field.

5.Enter the date when the result was received in the Result Date field. For more details, read the Batch Test Details (L22) section .

6.Click the Close batch button and confirm the closing.

7.Click the OK button to save and close the window.
