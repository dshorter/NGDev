Export map

When a final map is customized, it can be exported to a picture format file (JPG). The map is exported to file as is; this means that the map in the picture will look exactly as it looks in the Map Editor window.

 

1.Open the map to export.

2.Click the Export button. The Save As window opens.

3.Browse to the place for the file, enter its name in the File name field and click the Save button.
