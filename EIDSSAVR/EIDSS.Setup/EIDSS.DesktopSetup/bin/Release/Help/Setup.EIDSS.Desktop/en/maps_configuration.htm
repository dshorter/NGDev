Maps Configuration

Show/Hide Hidden Text

The primary purpose of EIDSS GIS Module is the dynamic visualization of surveillance data by mapping the information stored in EIDSS distributed database.

Create a customized map:

In the System menu, click Configuration and then click Map Customization. The Map Customization window opens.

Fig. 94 Map Customization window

Make the required corrections and click Save map on the Map menu.
Layer Options:

To customize a layer, right-click on the layer in the panel on the left, and select Layer Options.

Fig. 95 Layer Options window
