Freezers List L11

The Freezers List (L11) form is a list of laboratory freezers where samples are stored. It consists of a search panel and a list of freezers that meet the search criteria specified in the search panel.

From the Journals menu, select Freezers. The Freezers List window opens.

Fig. 494 Freezers List

Search Parameters

Functional Area

Enter a complete or partial name of the freezer to be found.

Note

Enter a complete or partial note of the freezer to be found.

Storage type

Select Freezer from a drop-down list.

Result List

Functional Area

Freezer name.

Note

Note related to the freezer.

Storage type

Type of the freezer.

Buttons

Copy

Copies the selected freezer with its subdivisions.

1.Select the source freezer.

2.Click the Copy button. The Copy Freezer window opens.

3.Enter the name of the desired freezer and click the OK button. The new freezer is added to the Freezers List window.

New

Creates a new freezer.

Click the New button. The Storage Schema L12 form opens.

Edit

Opens the selected freezer for edition.

1.Select the freezer to change.

2.Click the Edit button. The Storage Schema L12 form with freezer's details opens in an Edition mode.

Delete

Deletes the selected freezer.

Once a freezer is deleted, neither the freezer not its subdivisions can be further used or automatically restored.
Only freezers with no samples stored in it can be deleted.

1.Select the freezer to be deleted.

2.Click the Delete button. The Delete Record window opens asking for confirmation.

3.Click the Yes button. The Delete Record window closes. The deleted freezer is removed from EIDSS and the Freezers List window.
