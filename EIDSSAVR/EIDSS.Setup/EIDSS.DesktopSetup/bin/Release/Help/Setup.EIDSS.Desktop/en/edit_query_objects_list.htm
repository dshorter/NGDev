Edit query objects list

If a new query is created, or an existing one is opened for modifying, the search objects can be edited as follows:

 

1.To add an EIDSS object that contains the required information as a Search Object to the EIDSS query, click the Change Root Object button. When a drop-down list opens, select the desired EIDSS object. The fields of the object will be displayed in the Available fields list.

Fig. 513 Changing search object

2.To replace a search object, click the Change Root Object button. When a drop-down list opens, select the desired EIDSS object. The fields of the previous search object shall be removed from both the Available fields list and the Selected fields list; the fields of the new object will be displayed in the Available fields list instead of them.

3.If the desired information belongs to the search object child, add it as a child search object by clicking the Add Child Object button.

Fig. 514 Child search object

After the child object is added, its fields will be displayed in a new Available fields list. Root search object and child search objects have separate fields' lists.
The number of child objects is not limited; however it is strongly recommended to add only the required objects, since they affect the layout processing speed.

4.To delete a child object from the Query Search Object Tree, select it and click the Remove Child Object button. The fields of the child search object are removed from both the Available fields list and the Selected fields list.

 

For more information about search objects, read the Queries section.
