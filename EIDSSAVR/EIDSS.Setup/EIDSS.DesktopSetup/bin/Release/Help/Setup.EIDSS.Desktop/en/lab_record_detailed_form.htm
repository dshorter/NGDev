Laboratory Record Detailed form

Laboratory Record Details form is a hidden form that contains all information about the sample and the test if any. To open it, click the Plus button () prior to the laboratory record.

Fig. 505 Detailed form of laboratory record

The form contains the following tabs:

•Sample Details

•Test Details

•Amendment History

The Sample Details tab is used to display detailed information about a sample. The Test Details displays details about the test. The Amendment History tab is used to display chronological information about changes of test's status after its result validated.
