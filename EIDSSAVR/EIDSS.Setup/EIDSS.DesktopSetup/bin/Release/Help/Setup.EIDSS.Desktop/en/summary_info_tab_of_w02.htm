Summary Info tab of Vector Surveillance Session W02 form

The Summary Info tab is used to register information and materials gathered within a session briefly. Enter the data in the grid.

 

Fig. 404 Summary Info tab

Columns

Record ID

The ID of the pool/vector. (new) is used for a new, never saved vector/ pool. After the vector/ pool is saved for the first time, the ID generated according to the Unique Numbering Schema is used instead of the (new) one.

Location

Select or enter the exact location where the pool/ individual vector was collected. Click the field; a special window for location entry opens. Once the location is defined, click the OK button to close this special window. Data are displayed in the Location field.

The field is mandatory.

Collection Date

The date when the vector or pool was collected.

Vector Type

Select the type of the collected vector/ pool. The field is mandatory.

Species

Select the species of the vector(s). The field is mandatory.

Sex

Select the sex of the collected vector(s).

Number of Pools/ Vectors Collected

Enter the number of vectors/ pools with the selected parameters.

Disease (Number of Positive Pools/ Vectors)

Add diseases confirmed by tests.

1.Click the down-arrow button (). An additional window opens.

2.To add a new disease, click the empty line with the message Click here to add a new row. A new empty line is added.

3.Select a confirmed disease and a number of pools/ vectors proved positive. A disease can be entered only once.

If a disease is added by mistake, select it and click the Delete button in the additional window. The disease will be removed from the additional window, from the Disease (Number of Positive pools/ Vectors) column in the Summary Info tab and from the Diseases field of the session.

Buttons

Delete

Deletes selected record(s):

1.Select records(s) to delete.

2.Click the Delete button. The Confirmation window opens.

3.Click Yes to confirm the deletion and close the Confirmation window.
