Query Search Objects Tree

The Query Search Objects Tree pane contains the EIDSS object and its sub-objects (in the AVR these are the Child Objects) hosting the fields required for a query.

Fig. 592 Query Search Object Tree pane

The search objects tree consists of 2 levels: the first level contains one root search object; and all child objects are located at the same second level.
