Mark sample for disposition (destruction) (IF)

Only samples with a Accessioned In status can be marked for routine disposition (destruction). This operation is available only if the Lab Module Simplified Mode check-box is clear in the System Preferences.

 

1.From the Journals menu, select Samples. The Samples List window opens.

2.Find and select the sample(s) which shall be marked for routine destruction (disposition).

3.Click on the Mark for Disposition button. The Sample disposition window opens with the list of samples selected in the Samples List window. For more details about the Sample disposition window, read the Sample disposition L08 section.

Fig. 461 Sample disposition form

4.Click on the Save button to accept the marking of the samples. The Confirmation window opens.

5.Click on the Yes button to confirm the marking. The sample(s)' status changes to Marked for routine destruction.
