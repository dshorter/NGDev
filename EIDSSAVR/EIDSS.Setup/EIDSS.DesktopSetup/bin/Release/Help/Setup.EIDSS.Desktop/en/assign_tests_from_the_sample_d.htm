Assign tests from Sample form (IF)

This operation is available only if the Lab Module Simplified Mode check-box is clear in the System Preferences.

 

1.From the Journals menu, select Samples. The Samples List window opens.

2.Find the sample for which tests shall be assigned.

3.Do one of the following:

•double-click the sample

•select it and click the Edit button.

The Sample window opens.

4.Click the Tests tab to open it. The Tests tab contains the list of tests assigned to the sample. For more details about the Tests tab, read the Tests tab of Sample L02 form section.

5.Click the Select Test button. A new line for a new test is created in the grid.

6.Enter at least the Test Name. The fields are changed as follows:

•The Test Diagnosis is populated automatically with the diagnosis of the case or session related to the sample.

•The Test Status is available and defines the availability of the fields to the right of it. While the test has a Not Started status, no information about a test result can be specified. Once the status is changed to In Progress, the fields become editable. If the status of a test is changed to Final, most fields become mandatory. For more details, read the Test Statuses and Tests tab of Sample L02 form sections.

7.Repeat steps 5 through 7 to assign all necessary tests.

8.Click the Save button and confirm the data was saved.
