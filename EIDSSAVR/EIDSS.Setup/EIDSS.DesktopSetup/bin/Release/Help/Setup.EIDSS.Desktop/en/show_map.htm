Show map

Before running the map, ensure that the administrative units (regions and/or rayons) are added to the row area.

1.In the Map Default Data as Chart field, select the data to be displayed in the chart. For example, if it is necessary to show the number of human cases, select Human case - Case ID. More than one field can be used.

2.In the Map Default Administrative Unit field, select the geographical unit for the map set-up. Examples of different administrative units used for a map are shown below.

3.In the As Gradient field, select the data, which  shall be used to build the color gradient on the map. Only one field can be used.

Fig. 575 Map with Region administrative unit

 

Fig. 576 Map with Rayon administrative unit

4.Click the Open Map button.

The administration unit can be changed later in the Map Editor window.
