Repository Schema

When placing a sample in storage, the exact location of the sample within the storage can be recorded in EIDSS.

Before defining a sample location, it is necessary to define the storage schema first. The best way to do this is to create a virtual copy of the facility storage in the EIDSS data base.

Creating a repository schema:

1.In the Journals menu click Freezers. The Freezers List window opens.

Fig. 48 Repository Schema

When the window is opened for the first time, no items are listed, since the storage schema has yet to be defined.

2.Click the New button to create a new freezer. The Storage Schema window opens.

3.Fill in the fields in the right section of the Storage Schema window and click the Save button. The entered element is added and is displayed in the left part of the window.

Fig. 49 Storage scheme element

4.Select the freezer in the frame on the left and click the New button to create a subdivision of the freezer.

Fig. 50 Storage scheme sub-element

5.Fill in the Subdivision Name and define the Subdivision Type. Set the Number of locations in the current subdivision and Subdivision Barcode if required. Add any additional Notes and click the Save button. The entered element is added and is displayed in the left section of the window.

Note: the Number of locations defines the possible number of samples for current subdivision.

6.Repeat steps 4-5 to create more sub-element(s). All entered elements are displayed in the left section of the window.

Copy a schema element:

1.Select the target item from the list and click the Copy button.

2.The structure of the copied schema element will be the same as the original. Modify the attributes of the new schema element in the activated dialog box.

3.Click the Save button to save the changes.

Delete a schema element:

1.Select the element to delete.

2.Click the Delete button to remove an element or sub-element.

Print a container content report:

1.Click the Paper Forms button and select Container Content. The Report Viewer displays the report.

2.Click the Printer icon to print the report.
