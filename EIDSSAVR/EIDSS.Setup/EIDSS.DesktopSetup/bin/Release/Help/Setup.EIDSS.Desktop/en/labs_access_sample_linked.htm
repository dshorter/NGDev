Accession in/ transfer in existing sample(s), linked to case/session (LS)

Samples linked to a case or session can be either initially entered in the case/session and later accessioned in the Laboratory module, or added with a simultaneous accession in the Laboratory module. For details about creation and simultaneous accession in of a new sample, which is linked to a case or session, read the Create and accession in sample, linked to case/session (LS) section. Only samples with an Unaccessioned status can be accessioned.

 

1.From the Journals menu, select Laboratory Section. The Laboratory Section window opens.

2.Find the desired sample(s): use the Organization Sent To search criterion, and set it to the current organization to filter only samples which can be transferred in/accessioned in.

3.Do one of the following:

•Click in the Sample Condition Received cell of the found sample and select its condition from the drop-down menu which is available for one sample only. If the condition is not good, the Accession In Comment window opens for entering the explanation. Enter an explanation to the Comments field and click the OK button. The Accession In Comment window closes.

•Right-click the sample(s) to be accessioned in; and from the drop-down menu, select Accession In and then its (their) condition.
 

Fig. 442 Accession In Comment

If the condition is good or poor, the samples are updated as follows:

•The Sample Status is automatically changed to Accessioned In.

•The current date populated the Accession Date cell, and can be changed.

•The Lab Sample ID is temporarily set to (New) and shall get a permanent number, in accordance with the Unique Numbering Schema, after saving; if it is not changed manually. The sample ID can be changed manually. If so, the specified value is used.

If the Sample Condition Received field value is changed to Rejected and saved, then the sample is hidden and cannot be found by a search.

4.Save the accession in, and the Lab Sample ID is assigned automatically.

The samples are accessioned in.
