Register samples for vector/pool

Once a new pool is started or an existing one is opened to edit, the summary information, general information, and vector specific information are entered, then register samples for a vector/ pool in the Samples tab of Pool/Individual Vector Details form.

An individual vector collected within a vector surveillance session is separated to samples. A complete pool is considered as a single sample and then separated to aliquots.

The samples created within a session shall be registered in the Pool/ Individual Vector Details form of the VS session. After that field tests can be registered in the Field Tests tab of Pool/Individual Vector Details form. The samples can be accessioned in, and laboratory tests can be assigned to them in the Laboratory module.

 

•Add a new sample, click the empty line with the message Click here to add a new row.

•Edit an existing sample, select the sample and edit it in the grid.

•Delete sample(s), select the sample(s) and click the Delete button. Only samples that have never been accessioned in and that have no field tests can be deleted.

 

For a detailed description of the fields, read the Samples tab of Pool/Individual Vector Details Form W03 section.

The samples collected for the session's vectors/pools are copied to the Samples subtab of Vector Surveillance Session form and are compiled in the list of session's samples.
