Create an Active Surveillance Session

1.To create a new Active Surveillance Session press Create -> Active Surveillance Session. The Active Surveillance Session form opens in a new window.

Fig. 286 Active Surveillance Session

2.Fill in the fields of the form. Please refer to Active Surveillance Session V18 for complete information about the form.

3.Click Ok to save the session, and close the form.

The replication of the saved session will commence and it will be transmitted to the EIDSS server.
