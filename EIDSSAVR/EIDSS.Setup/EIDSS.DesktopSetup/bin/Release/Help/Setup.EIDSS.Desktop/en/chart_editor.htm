Chart Editor Form

The Chart Editor window includes of the Chart Diagram pane and the Grid Fields Customization pane. The Chart Diagram pane contains the chart in the View pane and the Chart Type field designed to change the type of the chart. The Grid Fields Customization pane is designed to change the view of the axes and chart.

Fig. 600 Chart Editor

Double-click on an axis to customize the view of it. The Grid Fields Customization pane changes. The 3D charts cannot be customized.

Fig. 601 Axis customization

 

Line Color

Color of the axis line.

Line Width

Width of the axis line.

Gridlines Color

Color of the auxiliary grid lines. Useful if the grid lines are visible (the Gridlines Visible check-box is selected).

Gridlines Styles

Style of the auxiliary grid lines (solid, dash, dot, dash-dot, or dash-dot-dot). Useful if the grid lines are visible (the Gridlines Visible check-box is selected).

Gridlines Visible

Make the grid lines visible:

ON - Gridlines of the chart are visible.

OFF - Gridlines of the chart are hidden.

Value Label Angle

Skew of the axis labels.

Value Label Font

Font for the axis labels.

Value Label Staggered Style

Change the way to arrange the axis labels.

ON - Values are disposed in staggered order.

OFF - Values are disposed in line order.

Text

Enter the title of the axis. Useful if the title is visible (the Visible check-box is selected).

Font

Change the font for the axis title. Useful if the title is visible (the Visible check-box is selected).

Alignment

Align the title of the axis. Useful if the title is visible (the Visible check-box is selected).

Visible

Make the axis title visible.

ON - Axis title is visible.

OFF - Axis title is hidden.

Tickmark Minor Count

Enter the number of not-labeled tickmarks between numbered ones.

Reverse

Mirror the chart vertically.

Auto Range

Set the minimum and maximum ends of the range. Remember that manually specified range is saved and can mismatch a changed chart.

ON - Range is set automatically.

OFF - Range can be set manually.

Range Max

Select the maximum end of the range. Available only if the Auto Range check-box is clear.

Range Min

Select the minimum end of the range. Available only if the Auto Range check-box is clear.

Position

Specify the position of the Y-axis (left or right). Not applicable for the X-axis.
