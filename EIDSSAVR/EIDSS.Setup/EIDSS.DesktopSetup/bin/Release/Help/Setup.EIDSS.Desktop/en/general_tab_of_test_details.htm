General tab of Laboratory Test Details L04 form

The General tab of the Laboratory Test Details form is used to view and edit basic data of a test. The test status defines the availability of the fields in the General tab.

Fig. 479 General tab

Test Name

The name of the test is displayed and cannot be changed.

Category

The category of the test is displayed if known and cannot be changed.

Test Diagnosis

The diagnosis for the test to confirm or reject is displayed and cannot be changed.

Status

Select the status of the test from a drop-down list. The field value can be changed for tests with Not Started, In Progress, and Preliminary statuses. The status defines what fields are available for the test and what can be done with it. For details, read the Test Statuses section.

Result

Enter the result of the test if obtained. The field is available when the test's status is changed to In progress, Preliminary, or Final until it is saved for the first time. For details, read the Enter test result section.

Comments

Enter any necessary comment. The field is editable when the test's status is changed to In progress, Preliminary, or Final until it is saved for the first time.

Tested By

Select the name of the tester from a drop-down list. The field is available when the test's status is changed to In progress, Preliminary, or Final until it is saved for the first time.

Result Entered By

The name of the person who entered the test results is automatically displayed and cannot be changed manually.

Date Test Started

Enter the date when the test was started. The field is available when the test's status is changed to In progress, Preliminary, or Final until it is saved for the first time.

Result Date

Enter the date when the result of the test is obtained. The field is available when the test's status is changed to In progress, Preliminary, or Final until it is saved for the first time.

Validated By

Select the name of the person who confirmed the test result. The field is available when the test's status is changed to In progress, Preliminary, or Final until it is saved for the first time.

Batch ID

Batch ID is displayed if the test belongs to any one. To open the batch form for details, click the Details button (). The desired form opens in a new window.

Laboratory

The laboratory where the test was performed.

Case ID

Case ID related to the test sample for vet and human cases. To open the case form, click the Details button (). The vet or human case form opens in a new window.

Session ID

Session ID related t the test sample is linked for AS or VS sessions. To open a session form, click the Details button (). The session form will be opened in a new window.
