Register a new sample L35

The Register a new sample form is used to add and accession in a new sample.

Fig. 510 Register a new sample

Data Fields

Number of Samples

Enter how many samples of the same type linked to the same case/session shall be registered.

Case/Session ID

Enter the case/session related to the new samples shall be linked. Once the case/session ID is specified, the Details button () can be used to view the case/session detailed information.

Sample Type

Select the type of the registered samples from a drop-down list.

Buttons

Add

Registers the entered samples without closing the Register a new sample window.

Close

Closes the window.
