FFD - Add a New Template

Basic template data:

1.In the Flexible Form Designer window, click the Templates Editor tab to open it.

2.Select the desired Template Type and click the Add Template button (). A New Template is added to the list, the template preview pane is empty.

Fig. 103 Creation of a new template

3.Enter template Default Name (in English) and National Name (in the local language), and Note.

4.Select the Is UNI Template? check-box to use the template as a UNI one - default for the current template type and for the country (to be used if no specific templates are defined).

Sections and parameters:

1.Select the template in the left pane needs to be edited. Click the Parameters button at the bottom of the left pane to open the list of parameters that can be used for the current template. Click to expand the list of existing templates of this type, click again near the template need to be edited to expand the list of available parameters. If the list does not contain the required parameter, add it.

Fig. 104 Click the Parameters button

2.To add a section/parameter, select one and click the location in the template preview pane to add. If the parameter belongs to a section, the parameter is added to the pane as well.

Fig. 105 Parameter addition

3.Use the mouse to move the parameter and/or change its size.

4.Change the Scheme of the parameter (define where its name should be located) and define whether the parameter is ordinary or a mandatory; this is selected in the Edit Mode list. Change Width, Height and Label Size if needed.

5.Repeat steps 2 through 4 to add all parameters needed.

6.To add labels, click the Label button and click the desired location for the label.

7.To add lines, click the Line button and click the desired location for the line in the preview pane.

8.To delete parameter/section, select it and click Delete Object button.

Rules for parameters:

1.To define the rules for the parameters on a template, select the needed template on the Template Editor. Click the Rules button at the bottom of the left pane. An empty Rules pane opens. Select the parameter on the right pane for which the rule shall be applied.

Fig. 106 Rules

2.Click the Add Rule button. A new rule is added to the list.

3.Right-click the field for which the rule is being created. From the right-click menu select the needed rule. If Value is selected, additional field Constants is displayed in the lower pane.

Fig. 107 Rule for one parameter

4.Specify the following:        

i.Click Actions. In the opened window from the list of parameters for the selected template choose the parameter, to which an action shall be applied after the parameter, for which the rule is created, is changed, and in the column Actions for the selected parameter check Clear and/or Disabled.

Example: parameter B should become non-editable if value for parameter A is specified. Open Actions window, select parameter B in the list and in column Actions check Disabled.

ii.In the field Check point specify when the rule shall be applied: On Load, On value changed, On Save (with notify) or On Save (with error).

iii.In the field Constants specify on which parameter’s value(s) the rule shall be applied. Click on the drop-down and in the opened window click Add Constant to add a value to the list, select it from the drop-down.

iv.In the Default Name field specify the name of the rule in English.

v.In the National Name field specify the name of the rule in the national language.

vi.In the Message Text field specify the text of the message in English (if needed), which is displayed if the rule is not verified.

vii.In the Message National Text field specify the text of the message in the national language (if needed), which is shown if the rule is not verified.

viii.In the field Not check the checkbox if the function of the rule (is Value or is Empty) shall be the opposite (is NOT Value or is NOT Empty).

Determinants for parameters:

1.To define the determinants for the parameters on a template, select the needed template on the Template Editor. At the bottom of the right pane click Add Determinant button. Determinants Editor window is opened. The left pane contains the list of available determinant types for the selected template type.

2.Select the determinant type in the left pane. Expand the list of determinants values by clicking , select the value of the determinant need to be added and click Ok. Or in Criteria field in the right pane enter the name of the determinant need to be added, click Find, the value will be found, click Ok. The value is added to the list of determinants.

3.To delete a determinant, select it in the list of determinants and click Delete Determinant.
