Laboratory Module

The Laboratory Management (LM) module is used to control samples recorded in the EIDSS database and laboratory tests made for them both in the organization and outside of it. All information entered in the Laboratory Management module is available in other EIDSS modules if required. Many fields in the Laboratory Management module are populated automatically to avoid input errors and repeating unnecessary work. And, vice versa, the data entered in the LM module are automatically populated in corresponding fields of the remaining EIDSS modules.

The Laboratory Management module gives the following methods to record samples and tests:

•Individual Forms (IF)
Operations are entered individually in specific menu sections. This approach is used to separate operations, and work with one sample at a time.

•Laboratory Section Form (LS)
All operations are entered in the same grid, which displays all samples and tests added to the database, their statuses and additional information. This approach is used to view the entire picture at a glance. The basic advantage of the Laboratory Section form is that no additional time is required to start operations, and that all samples and tests can be seen at once, in the same grid.  

Individual Forms (IF) and Laboratory Section Form (LS) can be used to perform the same operation: a user can access a sample and do a desired operation either in some individual forms or in the Laboratory Section form and have the same result. If only the Laboratory Section is used in the Laboratory Management module, Individual Forms can be deactivated (hidden from EIDSS menu). To do this, select the Lab Module Simplified Mode check-box in the System Preferences.
