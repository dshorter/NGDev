Diagnosis Change H12

Diagnosis Change (H12) form is opened when a user selects a diagnosis in the Changed diagnosis (if known) field of the Human Case Report Form.

Fig. 232 Diagnosis Change form

The form holds a single field: Please enter a reason for changing the diagnosis. The reason is selected from the drop-down list or is created in the Base Reference Tables Editor by clicking the button.
