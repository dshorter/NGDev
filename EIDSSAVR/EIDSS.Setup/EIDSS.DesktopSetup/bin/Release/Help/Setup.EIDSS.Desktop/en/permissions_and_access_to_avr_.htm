Permissions and Access to AVR Data

The access to the AVR function is controlled by user and group permissions.

Adjust access to the AVR for a user group

1.From the EIDSS menu select Security, then select User Groups. The User Groups List window opens.

2.Double-click on the desired user group. The User Group Details window opens.

3.Click the System Functions tab to open it.

4.Select the Site for the access adjustment.

5.Find the desired function and edit the permission.

 

Function

Description

Access to AVR Administration

Execute

ON - User of the group can publish/unpublish queries, layouts, or folders, and has access to the AVR Settings menu item ( Set functions default precision and Set search object fields default aggregate function).

OFF - User of the group cannot publish/unpublish queries, layouts, or folders, and has access to the AVR Settings menu item ( Set functions default precision and Set search object fields default aggregate function).

Access to Analysis, Visualization and Reporting module (AVR)

Create

ON - User of the group can create queries, folders and layouts in the AVR.

OFF - User of the group cannot create queries, folders and layouts in the AVR.

Delete

ON - User of the group can delete queries, folders and layouts in the AVR.

OFF - User of the group cannot delete queries, folders and layouts in the AVR.

Read

ON - User of the group can open and view queries and layouts in the AVR.

OFF - User of the group can open and view queries and layouts in the AVR.

Write

ON - User of the group can edit queries, folders and layouts in the AVR.

OFF - User of the group can edit queries, folders and layouts in the AVR.
