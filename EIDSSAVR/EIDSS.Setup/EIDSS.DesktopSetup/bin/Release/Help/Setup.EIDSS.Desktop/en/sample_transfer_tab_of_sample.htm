Sample Transfer tab of Sample L02 form

The Sample Transfer tab of the Sample (L02) form is used to display information about a sample's transfer in and out operations. None of the fields are available in the tab.

The Received From section is used for sample(s) which were transferred in. The Transferred To section is used for transferred out samples.

Fig. 476 Sample Transfer tab

'Received From' Section Fields

Institution

The source organization of the sample.

Transfer ID

The number of the transfer package in which the sample was received.

Lab Sample ID

The lab sample ID used in the external organization.

Date Received

The date when the sample was transferred in.

'Transferred To' Section Fields

Institution

The destination organization for the sample.

Transfer ID

The number of the transfer package in which the was transferred out.

Lab Sample ID

The lab sample ID used in the external organization is displayed is known.

Date Transferred

The date when the sample was transferred out.
