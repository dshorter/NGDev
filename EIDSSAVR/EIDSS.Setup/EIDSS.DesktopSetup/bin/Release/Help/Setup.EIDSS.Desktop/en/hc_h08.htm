Human Aggregate Case Details H06

Human Aggregate Case Summary (H08) form is used during the creation of the human aggregate case summary report.

Fig. 236 The Human Aggregate Case Summary

Field/Variable

Description

Summary Aggregate Settings

Administrative Level

The Administrative Level field specifies Administrative Level for which summary aggregate statistics must be shown.

Time Interval Unit

The Time Interval Unit field specifies time interval unit for which summary aggregate statistics must be shown.

Selected Aggregate Settings

Case ID

The Case ID is displayed.

Start Date

The date from which the aggregate data was collected is displayed.

Time Interval

The case’s time interval, which was applied during the creation of the case is displayed.

Region

The region where the aggregate data was collected is displayed.

Rayon

The rayon where the aggregate data was collected is displayed.

Settlement

The settlement where the aggregate data was collected is displayed.

Summary Info

The Summary Info section contains the grid which aggregates all selected cases data. Columns in this section are country specific and are defined in the Flexible Forms Designer and in Human Aggregate Case Matrix Editor.

The rows and the columns of the table are defined as conjunction of all rows and all columns in templates and matrixes, which were used for creation of the aggregate cases in the Summary Info section.

By default, when no cases are selected for the summary report, the last activated matrix and the unified template (current template) are shown.

Buttons

New

Click to open the Human Aggregate Case Details (H06) form and create a new Human Aggregate case. The new case will be added to the Selected Aggregate Cases section after it is saved.

View

Select an item from the Selected Aggregate Cases section and click View. The Human Aggregate Case Details (H06) form opens. The case can be observed, edited and saved.

Select

Click to open the Human Aggregate Cases List (H15) form. Search for Human Aggregate Case(s) to add to the Selected Aggregate Cases section.

Remove

Select an item from the Selected Aggregate Cases section and click Remove. The selected item will be removed from the Selected Aggregate Cases section.

Remove All

Click to remove all items from the Selected Aggregate Cases section.

Show Summary Info

Click the Show Summary button to recalculate the content of the Summary Info grid according to current aggregate cases, which comply with aggregate settings existing at the moment of form loading.
