General Information of Vector Surveillance Session W02 form

The General Information pane is used to enter, view, and edit the basic information about a VS session: time of the VS session, its location, types of the collected vectors, diseases with which they can infect.

Fig. 399 General Information pane

Data fields

Session ID

The ID of the Vector surveillance session. (new) is used for a new, not saved session. After the session is saved for the first time, the ID generated according to the Unique Numbering Schema is used instead of the (new) one.

Start Date

Enter the date when the session was started.

Close Date

The date when the status of the session was changed to Closed is displayed here and cannot be changed.

Collection effort

Enter the number of hours spent on the collection of the vectors within the session.

Field Session ID

Enter an ID of the session which has been assigned before the session is registered in EIDSS.

Status

Enter the current status of the VS session. Only sessions in the In progress status can be edited. Sessions in the Closed status are saved, and can be opened but cannot be changed.

Description

Enter a description of the VS session.

Vector Type

The types of vectors collected within the session are automatically displayed here and cannot be changed manually. The vectors registered for the session in the Pools/ Individual Vectors subtab of the Detailed Info tab and in the Summary Info tab define the list of vector types.

Location

Select or enter the exact location of the session. Click the field, a special window for location entry opens. Once the location is defined, click the OK button to close this special window. Data are displayed in the Location field.

Outbreak ID

Enter manually or select from the list the outbreak related to the session. To open the list of all registered outbreaks, click the List button ().

Diagnoses

The diagnoses confirmed by field tests or indicative laboratory tests of vectors/ pools collected within the session are displayed and cannot be changed manually.
