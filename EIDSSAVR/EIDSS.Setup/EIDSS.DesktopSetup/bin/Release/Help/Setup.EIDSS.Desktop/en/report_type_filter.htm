'Report Type' Filter

In addition to the filters described above, there are 2 special filters. Their role and construction differ them from ordinary filters. For example, veterinary queries (queries with veterinary cases, case samples, or tests as search objects) and combined queries (with zoonotic diseases) have a special filter designed to extract only cases with a specified Report Type. The filter refers to the Root Search Object only, and applies together with the other filters (as though combined with the And combining operator).

Although the 'Filter' term is used, this selection has nothing in common with the usual filters. It represents a drop-down list with the available values.

This filter is a necessity for veterinary queries; no query can be saved without this selection.

 

1.In the Vet Case - Filters pane, click the Report Type field.

Fig. 527 'Report Type' filter

From the drop-down list select Passive, Active or Both. Both is used to select all records without filtration by the Report Type parameter.
