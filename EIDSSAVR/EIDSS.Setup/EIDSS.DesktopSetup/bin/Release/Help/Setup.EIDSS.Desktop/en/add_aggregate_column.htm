Add aggregate column

When an aggregation is used in a pivot grid, the original data is replaced with the resulting one. Sometimes however both the original and resulting data are required in the same table. In this case, an aggregation column in the View tab table needs to be used.

Create an aggregation column

1.Select the column/band to which the aggregate function shall be applied.

2.Right-click the header of the column/band and from the drop-down menu select Add Aggregate Column. A new column named Aggregate Column will be added to the right of the column/band.

3.It is possible to rename the new column; right-click the header of the aggregate column and select Rename Column.

4.The added column shall contain no function yet. To add a function to an aggregate column, select the column and select the function in the Function field at the top of the tab.

5.Specify Parameters for the aggregate function, if required. No aggregate column can be used as a parameter for another aggregate column.

6.Specify the Precision for the aggregate column.

Precision

-1

Result is not rounded up.

0

Result is rounded up to the integer value.

>0

1, 2, 3, etc. Result is rounded up to the specified digit.

Aggregate functions available in a view table

Cumulative Percent

Sum of all cells above the cell in question divided by the column total and multiplied by 100%. For the function, a source column needs to be selected (in the Parameters: by column field). The function can be applied only to numeric/integer columns.

Max of Row

Maximum value of all cells of the same row within a band. The function can be applied only to numeric/integer columns.

Min of Row

Minimum value of all cells of the same row within a band. The function can be applied only to numeric/integer columns.

Percent of Column

The cell value divided by the column total and multiplied by 100%. The function can be applied only to numeric/integer columns. For the function, the cell source column needs to be selected (in the Parameters: by column field).

Percent of Row

The cell value divided by row total and multiplied by 100%. The function can be applied only to numeric/integer columns. For the function, the cell source column needs to be selected (in the Parameters: by column field).

Proportion

The cell value divided by a value of the same row of another selected column (Proportion = numerator/denominator). The function can be applied only to numeric/integer columns. For the function, 2 source columns need to be selected (in the Parameters: Numerator and Denominator fields).

Ratio

The cell value divided by a value of the same row of another selected column (Ratio = numerator/denominator). The function can be applied only to numeric/integer columns. For the function, 2 source columns need to be selected (in the Parameters: Numerator and Denominator fields).

It differs from the Proportion function: If both numerator and denominator are integers, the results are displayed in format X:Y and diminished to the single digit, e.g., 30:60  will be displayed as 1:2.
