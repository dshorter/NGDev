Veterinary Diagnostic Investigation Matrix

Show/Hide Hidden Text

Vet Diagnostic Investigation Matrix is a tool designed to store and manage the list of diagnostic investigations defined in the Basic reference editor used for diagnostics of diseases. The matrix is used in the vet aggregate actions entry.

There can be several versions of the matrix, one of which is currently active, the rest are the versions used in the past and/or matrix(es) not yet activated - a draft for a new matrix to be active in future.

View a version of the matrix (active or non-active):

1.In the System menu, click Configuration and then click Veterinary Diagnostic Investigation Matrix. The Veterinary Diagnostic Investigation Matrix window opens. If there are no matrix versions, the window is empty.

Fig. 68 Veterinary Diagnostic Measure Matrix window

2.The Version Name field contains the name of the currently active version. Activation Date displays the date when the version was activated.

3.In the Select Version field, select the desired matrix version. The matrix version is displayed. The investigation types of the activated version are read-only.
Create an active version of the Vet Diagnostic Investigation matrix:

1.In the System menu, click Configuration and then click Veterinary Diagnostic Investigation Matrix. The Vet Diagnostic Investigation Matrix window opens.

Fig. 69 Veterinary Diagnostic Measure Matrix window

2.If there already is an active version, click the New Matrix Version button. If not, skip this step.

3.In the Version Name field, enter the name of the version.

4.The Activation Date is an informative field. It is recommended to store the correct date here.

5.Click on the Click here to add a new row line of the table and add a new investigation type.

6.It is possible to define which investigations for which animals and probable diseases (the set: investigation type - species - diagnosis must be unique within a matrix).

7.Repeat steps 5 and 6 to create a list of all used investigation types.

8.When the version is completed, click the Activate Matrix Version button. Note that no changes in the diagnostic measures can be done in an activated version.

9.Save the data and close the window.
Edit a version of the Vet Diagnostic Investigation matrix:

The non-active version can be edited. The active versions can be partially edited: the version name and its activation date can be changed, but the list of diseases cannot be modified except the order of items in it.

1.In the System menu, click Configuration and then click Veterinary Diagnostic Investigation Matrix. The Vet Diagnostic Investigation Matrix window opens.

Fig. 70 Veterinary Diagnostic Measure Matrix window

2.The Version Name field contains the name of the currently active version.

3.In the Select Version field, select the matrix version to edit. The matrix version is displayed.

4.Change the version name and/or its activation date (for activated versions) and/or the list of diseases (for the non-activated version).

5.Save the data and close the window.
