Freeze row headers in pivot grid

It is very convenient to see headers even if a pivot grid is scrolled to the right. For this the Freeze Row Headers check-box is used. To freeze the row headers in position, select the Freeze Row Headers check-box and the row headers will remain in a fixed position while horizontal scrolling occurs. Freezing headers is applied to the whole table, i.e. all headers are frozen or released at the same time.
