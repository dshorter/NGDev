Group Accession In L13

The Group Accession In form is used to simultaneously accession a group of samples of different types which refer to different cases or sessions, but have the same local/field sample ID.

Fig. 497 Group Accession In window

Data Fields

Scan sample local/field ID

Enter manually or with a barcode scanner the local/field ID of the sample group. After the Add button is clicked,  the Select sample window opens with the group of samples that have the entered sample local/field ID.

Sample Type

Sample type.

Storage Location

Select a location in the storage for an accessioned sample:

1.Click on the Details button (). The Freezers window opens.

2.Select the desired location.

3.Click on the OK button. The Freezers window closes. The assigned location is displayed in the cell.

Available only for samples accessioned in with a good or bad condition (status Accessioned In, Sample Condition Received is not Rejected).

Local/Field Sample ID

Local/field sample ID.

Case ID/Session ID

Case ID/Session ID.

Barcode (Lab Sample ID)

An automatically assigned lab sample ID after sample’s accession, can be manually changed. Available after accession only.

Collection Date

Sample collection date.

Accession Date

The current date is used as an Accession Date once the sample is accessioned and can be manually changed using the calendar. Available after accession only.

Sample Condition Received

The condition of the sample once it is accessioned. Accepted in Good Condition is used by default and can be manually changed by selecting from a drop-down list. If Rejected or Accepted in Poor Condition is selected, an explanation has to be added to the Comments cell. Available after accession only.

Comments

Comments. If Rejected or Accepted in Poor Condition is selected in the Sample Condition Received cell, the Comments become mandatory for an explanation of the situation.

Available after accession only.

Functional Area

Select the department of the organization to receive the sample from a drop-down list.

Available only for samples accessioned in a good or bad condition (status Accessioned In, Sample Condition Received is not Rejected).

Buttons

Add

Opens the list of samples with the entered sample local/field ID in the Select sample window:

1.Enter manually or with a barcode scanner the local/field ID of the sample group to the Scan sample local/field ID field.

2.Click on the Add button. The Select sample window opens with the group of samples that have the entered sample local/field ID.

Print Barcodes

Prints barcode labels for the selected samples.

1.Select the samples which require barcode labels.

2.Click on the Print Barcodes button. The Print Barcode Labels window opens with the sample barcode preview.

3.Click on the Print button ().

Accession All

To accession all samples in the grid, click Accession All button, all non accessioned samples will be accessioned. Attributes such as Accession Date, Sample Condition Received, Storage Location, Comment and Functional Area become editable.

Remove

Deletes the selected sample from the Group Accession In window. Only samples that were not accessioned yet can be removed.

1.Select the sample to be removed form the list.

2.Click the Remove button. The Message window opens asking for deletion confirmation.

3.Click on the OK button. The Message window closes, the sample is removed from the Group Accession In window.

Save

Saves changes.

OK

Closes the Group Accession In window without saving any changes unless they were saved before.

Cancel

Cancels all changes, even if they were saved before, and closes the window.
