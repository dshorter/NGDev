Generic Statistical Types Editor

Initially only the following types are available in the EIDSS: Population, Population by Age Groups and Gender and Population by Gender. More statistical types can be added if required.

Create a list of statistical types:

1.In the System menu, click References, then click Generic Statistical Types Editor. The Generic Statistical Types Editor window opens.

Fig. 58 Generic Statistical Types Editor window

2.Click on the Click here to add a new row line of the table.

3.In the English Value column, type the English name of the statistical type.

4.In the Parameter Type column, select the appropriate type from the drop-down list.

5.Define if the new item is the Statistical Age Group or not.

6.Select the Period Type and Area Type from the drop-down lists.

7.Repeat steps 2-6 to create a list of all statistical types.

8.Save the data and close the window.
