Add key values

Sometimes it can be necessary to create a report for those final diagnoses and regions/rayons of the current residence for which no data is available. In this case use key values. The key values are place holding empty records (for diagnoses and regions/rayons) which can be added to a query when necessary.

For example, it is necessary to create a report with information about number of human cases for some diagnosis for all rayons of Guria and Adjara regions.

 

Fig. 532 Pivot grid without key values

Fig. 533 Pivot grid with key values

Although Ozurgeti, Qeda , Qobuleti and Shuajhevi have zero cases, these rayons are shown in the list of rayons.

Activate key values

Select the Add all values of key references check-box in the General Query Info pane.

Fig. 534 General Query Info pane

All layouts created on the basis of the query with activated key values will in turn have activated key values. Key values will be added to the report even if this option is cleared later in the layout.
