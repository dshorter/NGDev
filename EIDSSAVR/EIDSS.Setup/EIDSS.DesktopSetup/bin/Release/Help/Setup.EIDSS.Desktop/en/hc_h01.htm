Human Case List H01

Show/Hide Hidden Text

Human Case List (H01) is a form where all Human Cases that match the search criteria are listed.

Fig. 223 Human Cases List window

Search pane section, which contains different fields used for case searching. Search pane contains mandatory fields, pre-filled fields and optional fields in order to limit the search result list and to enhance system operation.

Fig. 224 Search Options section

Field/Variable

Description

My Cases

Only the cases registered by current user will be displayed with check-box enabled.

Case ID

User may specify Case ID in this field.

Local Identifier

User may specify a local case identifier in this field.

Final Diagnosis

User may specify case’s final diagnosis in this field.

Final Diagnosis Group

User may specify case’s final diagnosis group in this field.

Case Classification

User may specify case classification in this field.

Case Status

User may specify case status in this field.

Sent by facility

In the Sent by facility field the name of the organization which sent the notification may be specified.

Received by facility

In the Received by facility field the name of the organization which received the notification may be specified.

Diagnosis

In the field Diagnosis the initial diagnosis of a case may be specified.

Diagnosis date From

In the field Diagnosis date From, the date after which the Diagnosis Date of the case was assigned may be specified.

Diagnosis date To

In the field Diagnosis date To, the date before which the Diagnosis Date of the case was assigned may be specified.

Changed diagnosis

In the field Changed Diagnosis the last changed diagnosis of a case may be specified.

Changed diagnosis date From

In the field Changed diagnosis date From, the date after which the Changed Diagnosis Date of the case was assigned may be specified.

Changed diagnosis date To

In the field Changed diagnosis date To, the date before which the Changed Diagnosis Date of the case was assigned may be specified.

Local Sample ID

In the field Local Sample ID a local samples ID of a sample collected for a case may be specified.

Last Name

User may specify Last Name of a patient in this field.

First Name

User may specify First Name of a patient in this field.

Middle Name

User may specify Middle Name of a patient in this field.

Date of Birth From

Only cases in which patients have a date of birth greater than specified or equal will be displayed.

Date of Birth To

Only cases in which patients have a date of birth less than specified or equal will be displayed.

Data Entry Date From

Only cases in which the entry date is greater than specified or equal will be displayed.

Data Entry Date To

Only cases in which the entry date is less than specified or equal will be displayed.

Current Residence Region

User may specify a region of patient’s current residence in this field.

Current Residence Rayon

User may specify the rayon of patient’s current residence in this field. The Rayon field is non-editable until the Region field is filled in.

Current Residence Town or Village

User may specify the settlement of patient’s current residence in this field. The Town or Village field is non-editable until the Rayon field is filled in.

Data Entry Site

User may specify Data Entry Site in this field.

My Cases

Select the chech-box to view only those cases entered by the current user.

Date of Symptoms Onset

The date, when the first symptoms appeared may be specified in this field.

Notification Date

The date when the notification was sent may be specified in this field.

Date of Final Case Classification

 

The date when the final case classification was defined.

Location of Explosure: Region

The region of the case exact location.

Location of Explosure: Rayon

The rayon of the case exact location. Available only if the Location og Explosure: Region is selected.

Personal ID Type

The type of the personal ID used for the patient.

Personal ID

The personal ID used for the patient.
