Persons List H03

Show/Hide Hidden Text

Person List (H03) form is used to search for a patient, farm owner or a person from the contacts list of a human case.

Fig. 241 Person List form

Search parameters section

Fig. 242 Search Parameters section

Field/Variable

Description

First Name

The first name of the person is entered.

Middle Name

The middle name of the person is entered.

Last Name

The last name of the person is entered.

Date of birth

The date when the person was born is entered.

Region

The region of person’s current residence is selected.

Rayon

The rayon of person’s current residence is selected.

Personal ID Type

The personal ID type of the person is selected.

Personal ID

The personal ID of the person is entered.

Description

Buttons

New

Click this button to open the blank Person Details form, and create a new person in the database.

Edit

Click this button to open the Person Details form, and edit data for the selected person.

Delete

Click to remove the person from database.
