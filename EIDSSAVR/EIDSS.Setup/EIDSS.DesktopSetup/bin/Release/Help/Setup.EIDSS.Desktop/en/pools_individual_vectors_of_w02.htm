Pools/Individual Vectors subtab of Vector Surveillance Session W02 form

The Pools/Individual Vectors subtab is used to add, edit, view and delete vectors and pools within a vector surveillance session. The grid containing the list of pools/ individual vectors added to the vector surveillance session is read-only.

 

•Add a new pool/ individual vector, click the New button. The Pool/Individual Vector Details Form W03 opens for data entry. For a detailed description, read the Register vector/pool section.

•Edit and an existing pool/ individual record, select the sample and click the Edit button. The Pool/Individual Vector Details Form W03 opens for data change. For a detailed description, read the Edit VS session section.

•Delete pools/ individual vectors, select the pools/ individual vectors and click the Delete button in the middle of the Pools/Individual Vectors subtab.

 

Fig. 400 Pools/Individual vectors tab

Columns

Pool/vector ID

The ID of the pool/vector which was automatically created in accordance with the Unique Numbering Schema.

Field pool/vector ID

The ID of the pool or vector used before it is registered in EIDSS.

Region

The region where the vector/ pool was collected.

Rayon

The rayon where the vector/ pool was collected.

Settlement (nearest)

The settlement nearest to the location where the vector/ pool was collected.

Longitude

The longitude coordinates for the location where the vector/ pool was collected.

Latitude

The latitude coordinates for the place where the vector/ pool was collected.

Elevation (m)

The height above the sea level in meters where the vector/pool was collected.

Surrounding

The description of the location and area where the pool/vector was collected.

Collection Date

The date when the vector or pool was collected.

Collection Time

The time when the vector or pool was collected.

Collector

The person who collected the vector/pool.

Collected by Institution

The organization to which the person who collected the vector or pool belongs.

Species

The species of the vector(s).

Quantity

The number of collected individual vectors or vectors combined to the pool.

Sex

The sex of the collected vector(s).

Ectoparasites collected

Specify if parasites were or were not collected from the vector.

Host reference

The host animal of the collected vectors if the vectors were found on an animal.

Collection method

The method used to collect the vector or pool.

Basis of record

Information detailing how this vector or pool was collected - the vector or pool was intentionally found and collected, or brought by someone collected by accident.

GEO reference sources

The geo reference sources used to define the location where the pool/vector was collected.

Identified by

The person who identified the vector species.

Identified by Institution

The organization which identified the species of the vector.

Date Identified

The date when the species were identified.

Identification method

The Identification method field contains the identification method type, which was used for identifying a Pool/Individual vector.

Vector specific data

Data specific for a vector type, entered in the Vector Specific Data tab of Pool/Individual Vector Details Form W03. The data is divided by a comma (,) in the column.

Buttons

New

Creates a new pool or individual vector. For detailed description, read the Register vector/pool section.

Copy

Copies a selected individual vector/ pool with a selectable set of data. For details, read the Copy vector/pool section.

Edit

Opens the Pool/Individual Vector Details Form W03 window to edit a pool or individual vector:

1.Select a pool or individual vector to edit.

2.Click the Edit button. The Pool/Individual Vector Details Form W03 window opens.

3.Edit the pool/vector.

4.Click the OK button to save the changes.

Delete

Deletes selected pools/vectors.

1.Select pools/ individual vectors to delete.

2.Click the Delete button. The Confirmation window opens.

3.Click Yes to confirm the deletion and close the Confirmation window.

A vector cannot be deleted if at least one sample is accessioned in for it in the Laboratory module.
