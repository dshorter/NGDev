Sample Types Editor

The Sample Types Editor is a tool designed to manage the list of sample types that can be collected from animals, birds, vectors and human beings.

Create a list of sample types:

1.In the System menu, click References and then click Sample Types Editor. The Sample Types Editor window opens.

Fig. 54 Sample Types Editor window

2.Click on the Click here to add a new row line of the table.

3.In the English Value column, type the English name of the sample type.

4.In the Code column, type the code of a sample type to be printed in the barcodes.

5.In the Accessory Code column, select from whom the sample type can be collected.

6.Order column is used to define the order of items in the drop-down list.

7.Repeat steps 2-7 to create a list of all sample types.

8.Save the data and close the window.
