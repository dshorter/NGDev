Construct pivot grid

A pivot grid consists of the rows, columns and data that lie on their crossing. For example, to show the number of human cases per region for years, the Human Cases should be placed to the Data Area, the Data of Symptoms Onset should be placed to the Column Area, and the Region should be placed to the Row Area.

Fig. 541 Pivot Grid example

1.Click the Pivot tab of the layout to open it.

2.Drag and drop the data to be analyzed from the Grid Fields Customization pane to the Data Area of the Pivot Grid.

Fig. 542 Field moving to pivot grid

3.If more than one data type (fields) has to be displayed here, add it to the Data Area after the first data type.

4.Drag and drop the fields by which the data should be analyzed to the Row Area and Column Area. Please be aware that the data to be used in the X-axis of a chart or in a map needs to be placed to the Row Area.

5.If more than one data type (field) needs to be used here, add it to the same Area after the first data type. In the latter case, the data type on the left is used as a band, and the data type on the right is used as a column. More than two data type (fields) can be used in the same area.

Fig. 543 Usage of bands

6.To move a field within its area, drag-and-drop it to the desired place.

7.To use some field twice (grouped in a different way or with different aggregate functions applied, copy it).

8.Rename the fields which need to be used in a chart or map, so that the new field names could be displayed in a map and chart settings.
