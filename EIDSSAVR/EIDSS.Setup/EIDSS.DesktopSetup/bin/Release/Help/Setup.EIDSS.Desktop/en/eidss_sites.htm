EIDSS sites

This form is used to review the list of all EIDSS sites and to define default access settings for users from a site to another site.

View the list of registered sites:

In the Security menu click EIDSS Sites. The EIDSS Sites List window opens.

Fig. 160 EIDSS installation sites

Edit access rights to a site:

1.Select the site access rights to which will be changed and click the Edit button. The EIDSS Site Details window opens.

Fig. 161 EIDSS installation sitу details - General tab

2.The General tab is read-only and contains the general information on the site. Click the Permissions tab to open it.

Fig. 162 EIDSS installation sitу details - Permissions tab

3.To add a user or user group,click the Add button. The Users and Groups List window with the full list of users and user groups opens.

Fig. 163 Users and Groups List window

4.Select the actors to add and click the Select button. The Users Groups List window closes, and the actors are displayed in the Actors list. Select them one by one and edit their permissions for the site in the Permissions area.

5.Repeat steps 3-5 for the users and user groups from different sites.

6.To remove an actor, select him or her in the Actors list and click the Delete button.

7.After all permissions are edited, save changes and close the EIDSS Site Details window.
