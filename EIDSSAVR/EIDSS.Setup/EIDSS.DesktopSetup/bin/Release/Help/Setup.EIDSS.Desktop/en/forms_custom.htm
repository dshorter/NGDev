List and Search Forms Customization

The user can customize list and search forms if needed. The customization becomes useful if one part of the form should be highlighted.

Customize a Search Pane:

1.Click the () button to expand the search pane.

2.The Search pane opens. Move the cursor over the right edge of the search pane.

3.Drag the edge of the search pane and set its size as necessary.

Fig. 27 Search Pane customization

Customize an Item List:

The user may change the order of the columns by dragging them. When the new place for the column is selected, release the mouse button.

It is also possible to change the order of the items in a column (ascending or descending). Simply click the column header, and a small arrow in the column header changes its direction. It is also possible to hide and unhide columns (please refer to Item Lists section).
