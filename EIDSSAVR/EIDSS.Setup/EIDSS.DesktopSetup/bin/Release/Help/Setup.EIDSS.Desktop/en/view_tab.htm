View Tab

The View tab is created to customize reports, create maps and charts.

Fig. 599View tab

Function

Select function for additional aggregate column. For more details, read the Add aggregate column section.

for Aggregate Column

Read-only field displaying the aggregate column for which the function is being selected.

Precision

Specify the precision for the aggregate column after the function is applied. For details about default precision, read the Set functions default precision section, for details about aggregated column, read the Add aggregate column section.

Chart Default Series

Select the data to be presented as a chart. Only data from the data area of the pivot grid is available. More than one series can be displayed in a chart.  For details about chart construction, read the Charts in AVR section.

Chart Default X-axis

Select what should be presented in the X-axis of the chart. The fields from the row area of the pivot grid are available here.

Open Chart

Open the chart created on the basis of the view table. It can be customized in the Chart Editor window. For details of chart customization, read the Chart Editor section.

Map Default Data as Chart

Select the data to be presented as a chart in the map. Only data from the data area of the pivot grid is available. More than one series can be displayed in a chart.  For details of chart construction, read the Maps in AVR section.

As Gradient

Select the data to be used for coloring the map. Only one data series from the data area of the pivot grid is available. For details about map construction, read the Maps in AVR section.

Map Default Administrative Unit

Select the default administrative unit for the map. It can be changed in the Map Editor window later.

Open Map

Open the map created on the basis of the view table. It can be customized in the Map Editor window. For details of map customization, read the Maps in AVR section.
