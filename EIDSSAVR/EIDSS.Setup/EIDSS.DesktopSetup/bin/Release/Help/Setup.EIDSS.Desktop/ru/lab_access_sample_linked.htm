Приём в систему пробы, связанной или нет со случаем заболевания/сессией (ОФ)

Пробы могут быть либо изначально занесёнными в связанный(-ную) с ним случай заболевания / сессию, а позже принятыми в систему с помощью лабораторного модуля, либо быть добавлены с одновременным приёмом в систему в лабораторном модуле. Приняты в систему могут быть только те пробы, статусом которых является Не принята в систему. У ещё не принятых в систему проб столбец Состояние пробы при получении не заполнен.

Настоящая операция доступна только в случае отсутствия включённого флажка на кнопке с фиксацией Упрощённый режим лабораторного модуля, расположенной в окне Системные настройки.

 

1.Находясь в меню Журналы, выберите пункт Пробы. Откроется окно формы L01 Список проб.

2.Нажмите кнопку Приём в систему; из раскрывающегося меню выберите один из следующих пунктов, в зависимости от принимаемой в систему пробы: для регистрации связанной с медицинским случаем -

 

•Приём в систему пробы случая заболевания человека. Откроется окно Список случаев заболеваний людей.

•Приём в систему пробы случая заболевания животного для регистрации пробы, связанной с ветеринарным случаем. Откроется окно Список случаев заболеваний животных.

•Приём в систему пробы от сессии АН для регистрации пробы, связанной с сессией активного надзора. Откроется окно Список сессий активного надзора.

•Приём в систему пробы от сессии НЗПЗ для регистрации пробы, связанной с сессией надзора за переносчиками заболевания. Откроется окно Список сессий надзора за переносчиками заболеваний.

 

3.Найдите и дважды нажмите мышкой по случаю или сессии заболевания, относящихся к пробе. Откроется окно Приём в систему; его вид зависит от типа выбранного случая или сессии заболевания. Уже зарегистрированные пробы в отношении случая / сессия заболевания отображаются в панели Получение / регистрация проб. Она содержит подробный список проб, связанных с выбранным случаем / сессией заболевания.

4.Найдите нужную пробу (укажите полностью или частично идентификатор пробы в поле Поиск по локальному / полевому идентификатору пробы, и нажмите кнопку Найти).

5.Если необходимая проба ещё не находится в списке (этот шаг пропускается, если нужная проба уже отображается в списке):

a)Нажмите кнопку Новая. В панель Получение / регистрация проб добавится новая полупустая запись.

b)Укажите некоторые основные данные, необходимые для добавления пробы в систему. Для получения более подробных сведений о полях, прочитайте раздел формы L05 Приём в систему.

Рис. 302 Приём в систему

6.Если необходимая проба присутствует в списке, отметьте её и нажмите кнопку Приём в систему. Запись о пробе становится редактируемой, а поле Дата приёма в систему автоматически заполняется значением текущей даты, которое можно изменить.

7.Измените значение поля Состояние пробы при получении и, если это произошло, введите пояснение в поле Комментарий. Если значение поля Состояние пробы при получении изменили на значение Отклонена и сохранили, запись о пробе в системе будет удалена и её невозможно будет найти посредством поиска.

После этого, при необходимости, можно назначить проведение теста не покидая окно Приём в систему. Для получения более подробной информации, прочитайте раздел Назначение тестов во время приёма пробы в систему.

8.Распечатайте штрихкодовые наклейки для новопринятых в систему проб.

9.Сохраните изменения и закройте окно Приём в систему.

Проба будет принята в систему. Внесённые в записи случая / сессии заболевания изменения дублируются на другие сайты ЭИСНЗ в соответствии с правилами фильтрации.
