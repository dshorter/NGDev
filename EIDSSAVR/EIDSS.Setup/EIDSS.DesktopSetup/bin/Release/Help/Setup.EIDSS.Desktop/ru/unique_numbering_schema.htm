Схема уникальной нумерации

Show/Hide Hidden Text

ЭИСНЗ содержит различные наборы объектов, таких как случаи заболеваний, пациенты, пробы, тесты, и т.д. Для уникальной идентификации объектов ЭИСНЗ использует схему уникальной нумерации. В некоторых случаях, пользователь может назначать объекту идентификатор, при этом остальные идентификаторы назначаются автоматически, в соответствии со схемой нумерации. Автоматическая нумерация обеспечивает назначение объектам уникальных идентификаторов, и, тем самым, предотвращает присвоение одного и того же идентификатора двум разным объектам. Более того, номер объекта идентифицирует год присвоения номера, сайт, тип объекта (проба, случай заболевания или иное), и т.д. Определение схемы нумерации объектов означает определение строки групп символов, представляющих собой идентификатор.

Установите нумерацию до начала использования ЭИСНЗ, и в дальнейшем её не меняйте. Изменение схемы нумерации может привести к серьёзному повреждению базы данных.

Для открытия списка схем уникальной нумерации необходимо::

Находясь в меню Система, нажмите пункт Настройка, а затем нажмите Схема уникальной нумерации. Откроется окно Список схем уникальной нумерации.

Рис. 91 Окно «Список схем уникальной нумерации»

 

Для редактирования схемы уникальной нумерации необходимо::

1.Выберите значение в поле Тип идентификации, для которого редактируется схема нумерации, и нажмите кнопку Редактировать. Откроется окно Сведения о схеме уникальной нумерации.

Рис. 92 Окно «Схема уникальной нумерации»

В разделе Формат номера определяется нумерация: в поле Префикс введите префикс для текущего типа идентификации. Убедитесь, что такой префикс отличается от префиксов других типов идентификации. В поле Следующее числовое значение введите номер. В поле Предварительный просмотр отобразится результат нумерации.

2.Сохраните изменения и закройте окно.
