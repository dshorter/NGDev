Скрытие / отображение столбца / группы столбцов

Если необходимые в сводной таблице столбцы не требуются в окончательном отчёте, их отображение можно скрыть на вкладке Вид представления данных. Данные в скрытых столбцах не теряются и могут быть отображены в любой момент, когда они будут нужны. При этом скрытие столбца удаляет порядок его сортировки. При скрытии основных групп столбцов и их дочерних групп и столбцов, заголовки скрытых дочерних объектов в форме Настройка не отображаются.

Существует 3 способа скрытия столбца. Выбор любого из них приведёт к одинаковому результату.

Первый способ скрытия столбца

Перетащите заголовок скрываемого столбца на область вида ячеек и подождите, пока курсор мыши не превратится в знак большого «Х».

Рис. 430 Скрытие столбца - способ № 1

Второй способ скрытия столбца / группы столбцов

1.Правой кнопкой мыши нажмите на заголовке любого столбца /группы столбцов таблицы.

2.Из раскрывающегося меню выберите пункт Селектор столбца / группы столбцов. Откроется всплывающее окно Настройка.

3.Перетащите скрываемый заголовок столбца / группы столбцов на окно Настройка. Столбец / группа столбцов будут убраны из таблицы.

Рис. 431 Скрытие столбца - способ № 2

Второй способ скрытия столбца / группы столбцов

1.Правой кнопкой мыши нажмите на заголовке скрываемого / скрываемой столбца /группы столбцов.

2.Из раскрывающегося меню выберите пункт Убрать этот столбец или пункт Убрать группу столбцов.

Рис. 432 Скрытие столбца - способ № 3 

Отображение столбца / группы столбцов

4.Правой кнопкой мыши нажмите на заголовке любого столбца /группы столбцов таблицы.

5.Из раскрывающегося меню выберите пункт Селектор столбца / группы столбцов. Откроется всплывающее окно Настройка.

6.Проделайте одно из следующих действий:

•Перетащите открываемый для отображения заголовок столбца / группы столбцов из окна Настройка на его/их место в таблице.

Рис. 433 Раскрытие столбца

•Дважды нажмите на имени открываемого для отображения столбца в окне Настройка. В последнем случае, отображение столбца будет восстановлено справа от существующих столбцов.
