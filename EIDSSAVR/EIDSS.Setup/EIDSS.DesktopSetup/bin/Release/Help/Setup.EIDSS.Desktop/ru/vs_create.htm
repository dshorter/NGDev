Создание сессии надзора за переносчиками заболевания (НЗПЗ)

Информацию о сессии надзора за переносчиками заболевания можно внести в кратком или подробном виде. Подробная информация состоит из сведений о группах переносчиков / отдельных переносчиках заболевания, отловленных во время сессии, данные о месте и времени их отлова, а также содержит имя лица, производившего отлов материала. После того, как информация об отловленном материале будет зарегистрирована, укажите пробы, взятые у отловленного материала. Следующим шагом является регистрация полевых и лабораторных тестов, проведённых на собранном материале. Все вышеуказанные действия выполняются в модуле надзора за переносчиками заболеваний, за исключением лабораторных тестов, которые проводятся в лабораторном модуле и лишь отображаются в модуле надзора за переносчиками заболеваний.

Все пробы, включая пробы, зарегистрированные в модуле надзора за переносчиками заболеваний, доступны и в лабораторном модуле.

Краткая информация представляет собой очень краткие данные об отловленных переносчиках заболеваний, месте их отлова и заболеваниях, подтверждённых тестами, проведёнными на собранном материале. Информация о тестах: количество, тип и дата - не регистрируется; в систему попадают только подтверждённые заболевания.

 

1.Проделайте одно из следующих действий:

•Находясь в меню Создать, выберите пункт Сессия надзора за переносчиками заболевания.

•Находясь в меню Журналы, выберите пункт Сессия надзора за переносчиками заболеваний. Откроется окно Список сессий надзора за переносчиками заболеваний. Нажмите кнопку Новая.

•Выберите пункт Новая сессия надзора за переносчиками заболевания в меню панели инструментов.

Откроется окно Сессия надзора за переносчиками заболевания.

2.Введите общую информацию НЗПЗ.

3.Введите информацию о сессии. Это могут быть следующие данные:

•Краткая информация;

•Подробная информация;

•Регистрация отдельных переносчиков / групп переносчиков заболевания;

•Регистрация общей информации об отдельных / группах переносчиков заболевания;

•Регистрация специфической информации по отдельному переносчику / группе переносчиков;

•Регистрация проб по отдельному переносчику / группе переносчиков заболевания;

•Регистрация результатов полевых тестов по отдельному переносчику / группе переносчиков заболевания;

Перечень заболеваний сессии НЗПЗ состоит из всех заболеваний, подтверждённых тестами (полевыми или лабораторными), проведёнными в отношении отдельных переносчиков / групп переносчиков заболевания, отловленных в ходе сессии.

4.Свяжите сессию НЗПЗ со вспышкой заболевания, в случае необходимости.

5.Сохраните сессию НЗПЗ.

6.Закройте сессию НЗПЗ.

 

Начнётся репликация сохранённой сессии надзора за переносчиками заболевания, и её данные будут переданы на сервер ЭИСНЗ и другие подсистемы.
