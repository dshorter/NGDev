Населённые пункты

Show/Hide Hidden Text

Населённые пункты используются для лёгкого указания местоположения возникновения случая заболевания.

Откройте список населённых пунктов:

Находясь в меню Система, нажмите пункт Администрирование, а затем нажмите Населённые пункты. Откроется окно Список населённых пунктов.

Рис. 137 Список населённых пунктов

Для добавления нового населённого пункта необходимо::

1.Нажмите кнопку Новый. Откроется окно Сведения о населённом пункте. При первоначальном открытии оно будет пустым.

Рис. 138 Добавление нового населённого пункта

2.Введите данные о населённом пункте; заполнение всех полей является обязательным. Местоположение населённого пункта можно указать либо путём ввода координат в поля Географическая долгота и Географическая широта, либо с помощью географической карты.

3.Сохраните данные и закройте окно.
Для редактирования сведений о населённом пункте необходимо::

1.Отметьте населённый пункт для редактирования и нажмите кнопку Редактировать. Откроется окно Сведения о населённом пункте.

Рис. 139 Добавление нового населённого пункта

2.Внесите изменения в данные о населённом пункте. Местоположение населённого пункта можно указать либо путём ввода координат в поля Географическая долгота и Географическая широта, либо с помощью географической карты.

3.Сохраните данные и закройте окно.
Для удаления записи о населённом пункте необходимо::

1.Выберите населённый пункт для удаления и нажмите кнопку Удалить. Откроется окно Удалить запись.

2.Нажмите кнопку Дадля подтверждения операции удаления.
