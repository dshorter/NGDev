Вкладка «Сведения о пробе» формы L04 «Сведения о лабораторном тесте»

Вкладка «Проба» содержит информацию об относящейся к тесту пробе. Поля являются нередактируемыми.

Рис. 342 Вкладка «Сведения о пробе»

Идентификатор пробы

Отображается идентификатор пробы, связанной с тестом. Для открытия формы «Проба», нажмите кнопку Сведения (). Форма L02 Проба откроется в новом окне.

Тип пробы

Указывается тип пробы.

Дата взятия

Указывается дата взятия пробы.

Информация о пациенте / биологическом виде / переносчике заболевания

Отображается следующая информация:

•Случаи заболевания домашней птицы: биологический вид;

•Случаи заболевания домашнего скота: идентификатор животного;

•Случаи заболеваний людей: фамилия, имя и отчество пациента.

•Сессии АН: вид и идентификатор животного.

•Сессии НЗПЗ: идентификатор переносчика заболевания.
