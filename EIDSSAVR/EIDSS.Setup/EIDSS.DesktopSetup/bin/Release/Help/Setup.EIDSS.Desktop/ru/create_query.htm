Создание запроса

При создании/построении запроса, количество доступных для использования текущим пользователем полей ограничено правами доступа, которыми он обладает в системе.

1.Определите самые необходимые переменные значения.

2.Найдите, какой из объектов ЭИСНЗ содержит эту информацию. Для получения более подробной информации об объектах поиска - соответствии полей, см. раздел Список объектов поиска АВО и их полей.

3.Находясь в меню модуля АВО, нажмите пункт Запустить модуль АВО. Откроется окно Модуль анализа, визуализации и отчётности.

4.В меню Файл модуля АВО, нажмите на пункт Новый запрос.
Или нажмите кнопку панели инструментов Новый запрос. Вкладка Дерево запросов и макетов закроется, и откроется вкладка для нового запроса.

5.Добавьте объект поиска. Поля объекта отображаются в Списке доступных полей.

6.Найдите поля объекта, содержащие необходимые данные. Также эту информацию можно найти в разделе Список объектов поиска АВО и их полей.

7.Переместите их в Список выбранных полей.

8.Решите, какие другие поля данных могут потребоваться.

9.Тоже переместите их в Список выбранных полей.

10. Исключите ненужные данные при помощи фильтров.

11. Активируйте использование ключевых значений , при необходимости.

12. Укажите запросу название в поле Имя запроса на английском и местном языке.

13.Если требуется указать некоторые примечания, введите их в поле Описание.

14.Сохраните макет.

15.Закройте макет.

 

Созданный запрос доступен для всех пользователей этого же сайта. Они могут просматривать, редактировать или или удалить его.

Для получения более подробных сведений о форме «Конструктор запросов», прочитайте раздел Конструктор запросов.
