Указание результата теста, проводившегося во внешней лаборатории (ЛР)

В проводящей тест пробы лаборатории может быть (а может и не быть) установлена система ЭИСНЗ. Если она не установлена, результаты тестов должны регистрироваться лицом, отправившим туда пробы.

Результат теста внешнего учреждения можно вводить только в отношении пробы, имеющей статус Передана во внешнюю организацию. Даже если у пробы с таким статусом и не проводилось тестов внешней организацией, такой тест будет создан автоматически при вводе результатов теста внешнего учреждения, и сразу получит статус Окончательный. При этом существующие тесты, если таковые имеются, проведённые текущей организацией, никак не будут затронуты.

 

1.Находясь в меню Журналы, выберите Лабораторный раздел. Откроется окно Лабораторного раздела.

2.Найдите тест, в отношении которого планируется внести результат тестирования внешним учреждением. Это относится исключительно к тестам проб, имеющим статус Передана во внешнюю организацию, которые передавались во внешнее учреждение, не имеющее установленной системы ЭИСНЗ.

3.Введите результат одним из следующих способов:

Первый способ:

a)Нажмите кнопку плюс (), расположенную  напротив пробы, относящейся к необходимому назначенному тесту. Откроется панель со сведениями о пробе. Эти сведения изложены в подробной форме.

Рис. 320 Подробная форма лабораторной записи

b)Нажмите на вкладке Сведения о тесте для её открытия. Если теста ещё не назначалось, вкладка будет пуста и отображаться будет только кнопка с фиксацией Внешние результаты.

c)Нажмите кнопку с фиксацией Внешние результаты, чтобы её выбрать. Это активирует дополнительные поля.

Второй способ:

a)Выберите необходимый тест.

b)Нажмите на нём правой кнопкой мыши и из раскрывающегося меню выберите пункт Зарегистрировать результат внешнего теста.

4.В обоих случаях необходимо заполнить следующие поля на вкладке Сведения о тесте.

•По умолчанию поле Тестировавший работник не заполнено. Укажите лицо, проводившее тест во внешней организации. Это поле является обязательным для заполнения.

•В поле Направившая результаты организация хранится название организации, в которую передали пробу и, в случае последующей передачи пробы, эту информацию нельзя поменять вручную.

•Поле Дата получения результатов заполняется автоматически значением текущей даты, которое нельзя поменять вручную.

•Поле Контактное лицо не заполнено по умолчанию. Оно используется для хранения контактных данных, необходимых для получения сведений о проводившемся тесте. Это поле является обязательным для заполнения.

Поле Утвердившее лицо заполняется данными о текущем пользователе и их невозможно изменить. Для получения более подробных сведений о форме, прочитайте раздел формы L25 Лабораторный раздел.

Когда статус теста меняется на Окончательный, поля Дата начала теста и Дата результата автоматически заполняются значением текущей даты, которую можно изменить вручную.

5.В лабораторном разделе укажите значения в полях Название теста и Результат теста

6.Нажмите кнопку Сохранить и подтвердите сохранение данных.
