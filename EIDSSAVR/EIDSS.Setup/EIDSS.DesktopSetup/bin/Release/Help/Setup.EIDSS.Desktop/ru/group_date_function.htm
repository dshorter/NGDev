Функция группировки по дате

Даты, извлечённые с помощью модуля АВО из базы данных ЭИСНЗ, можно отображать различными способами: в качестве даты, сгруппированной помесячно, погодично, понедельно, и т.д. Группирование по умолчанию, применяемое ко всем датам в сводной таблице, можно выбрать в поле Группировка по дате по умолчанию окна Сводная таблица.

Рис. 410 Пример группирования по дате

Для изменения типа группирования по дате в поле «Дата», нажмите правой кнопкой мыши на соответствующем поле и из раскрывающегося меню выберите Группировка по дате, а затем выберите тип группирования, который необходимо применить для поля.
