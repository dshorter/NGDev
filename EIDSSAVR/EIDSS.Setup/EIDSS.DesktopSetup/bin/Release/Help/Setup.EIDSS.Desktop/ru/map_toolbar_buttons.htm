Кнопки панели инструментов карты

Панорамирование

Служит для выставления географической карты в пределах окна.

Обновление географической карты.

Запускает повторную отрисовку карты для отображения последних внесённых изменений.

Масштаб страны

Масштабирует окно просмотра для отображения карты всей загруженной страны.

Информация

Отображает табличные данные, связанные с объектом.

Приближение

Позволяет приблизить участок карты на экране.

Отдаление

Позволяет отдалить изображение для охвата большего участка карты.

Увеличение / уменьшение масштаба левой / правой кнопками мыши

Позволяет приблизить участок карты нажатием на левую кнопку мыши либо правой кнопкой отдалить вид, получив более широкий участок карты.

Измерительная линейка

Используется для измерения расстояний от точки до точки.

База данных слоёв

Позволяет получать доступ к базе данных для добавления слоя к географической карте.

Шейп-файл

Позволяет добавлять к карте из различных источников шейп-файлы (векторные изображения) в формате Института исследований окружающей среды, являющегося производителем геоинформационных систем.

Растровый слой

Позволяет добавлять к карте из различных источников растровые изображения.

Карты сервиса ОСМ

Автоматически добавляет в панель управления слоями слой карты, скачиваемой с бесплатного географического сервиса «Оупэн стрит мэп» или, сокращённо, «ОСМ». Для задействования этого инструмента необходимо наличие соединения с сетью Интернет.

Настройки карты

Создаёт слой для буферных зон.

Экспорт изображения карты в файл

Экспортирует географическую карту в файл формата *.jpeg. В окне Имя файла необходимо указать его название.

Маска административной единицы

Отображает указанную область или район.

Раскрывающийся список проектов карт

Позволяет открывать ранее сохранённые проекты карт путём выбора наименования соответствующей географической карты в раскрывающемся списке Проекты карт.

Создание слоя буферных зон

Создаёт слой для буферных зон.

Круговая буферная зона

Добавляет круговую буферную зону, центром которой служит место возникновения случая заболевания или населённый пункт.

Независимая круговая буферная зона

Позволяет добавить буферную зону с назначенной вручную точкой в качестве её центра.

Многоугольная буферная зона

Позволяет создавать многоугольную буферную зону, основываясь на границе района.

Независимая многоугольная буферная зона

Позволяет вручную создать любую многоугольную буферную зону.

Выбор буферной зоны

Позволяет выбрать буферную зону для обновления или удаления.
