Форма Н09 «Поиск случая заболевания человека»

Форма Н09 «Поиск случая заболевания человека» используется при выполнении поиска случая заболевания человека в режиме просмотра.

Рис. 226 Окно формы «Поиск случая заболевания человека»

Для получения информации о полях и разделах окна форма «Сообщение о случае заболевания человека», изучите форму «Сообщение о случае заболевания человека»..

 

Отличиями от формы Сообщение о случае заболевания человека являются следующие:

•В форме «Поиск случая заболевания человека» нет обязательных для заполнения полей;

•Все поля по умолчанию пустые;

•В форме «Поиск случая заболевания человека» нет вкладок Взятие проб и Тесты;

•Флажок любой кнопки с фиксацией можно снять;

•В справочники или журналы невозможно добавлять новые значения.
