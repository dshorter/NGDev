Можно определять, какие данные используются для статистики, а какие для анализа. Обычно, в модуле анализа используются данные о населении.

Для открытия списка статистических данных необходимо::

Находясь в меню Система, нажмите пункт Администрирование, а затем нажмите Статистические данные. Откроется окно Список статистических данных.

Рис. 140 Список статистических данных (население)

Первоначально доступно только 3 типа статистических данных: Население, Население по полу и Население по возрастным группам и полу. Если необходимы дополнительные типы, добавьте их используя Редактор типов общей статистики.

Данные можно фильтровать по типу. Для этого, укажите тип параметра в поле Тип статистических данных в панели поиска и нажмите кнопку Поиск.
Для редактирования существующих данных необходимо::

1.Отметьте объект для редактирования и нажмите кнопку Редактировать. Откроется окно Сведения о статистических данных.

Рис. 141 Редактирование статистических данных

2.Внесите изменения в данные полей.

3.Сохраните изменения и закройте окно.

Для добавления новых данных необходимо::

1.Нажмите кнопку Новые. Откроется окно Сведения о данных групповой статистики. При первоначальном открытии оно будет пустым.

Рис. 142 Окно «Сведения о данных групповой статистики»

2.Внесите изменения в данные полей.

Рис. 143 Добавление статистических данных

3.Сохраните изменения и закройте окно.
Для импорта новых данных необходимо: :

1.Нажмите кнопку Загрузить данные. Откроется окно Открыть файл Windows.

Рис. 144 Окно «Сведения о данных групповой статистики»

2.Выберите файл для импорта и нажмите кнопку Открыть.

3.Если файл не содержит ошибок, начнётся процесс импортирования. В случае успешного окончания операции импортирования, на экране появится сообщение.

4.Нажмите OK, чтобы закрыть окно сообщения.
Для удаления статистических данных необходимо::

Данные удаляются безвозвратно; их использование или восстановление будет невозможно!

1.Выделите объект в поле Список общестатистических данных и нажмите кнопку Удалить. Откроется диалоговое окно подтверждения на совершение действия Удалить запись.

2.Нажмите кнопку Дадля подтверждения операции удаления.
