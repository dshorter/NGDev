Форма L03 «Список лабораторных тестов»

Форма L03 Список лабораторных тестов создана для отображения всех зарегистрированных лабораторных тестов, удовлетворяющих критериям поиска. Она состоит из 2-х частей.

•Панель поиска

•Список тестов

Панель поиска была создана для осуществления сложного поиска, с применением сведений о тесте. В качестве полей поиска, в панели имеются основные и наиболее часто используемые параметры.

Список проб представляет собой таблицу, каждая из строк которой является записью о тесте, с лабораторным идентификатором теста и некоторым количеством параметров теста, отображаемых в столбцах таблицы. Если некоторые из столбцов не нужны или не актуальны, их можно скрыть, используя инструмент «Селектор столбцов». Читайте пункт дополнительной информации в разделах Список элементов, , , Селектор столбцов, Списки поиска, а также Настройка списочных и поисковых форм.

Рис. 339 Окно «Список лабораторных тестов»

Поля критериев поиска

Название теста

Используется для указания наименования соответствующего теста в раскрывающемся списке.

Идентификатор группы

Вводится уникальный идентификатор группы, с которой связан тест.

Статус

Используется для указания статуса соответствующего теста в раскрывающемся списке. Для получения более подробных сведений о статусах тестов, прочитайте раздел Статусы теста.

Дата результата

Указывается тот временной интервал, когда результат теста регистрировался, утверждался или изменялся в последний раз.

Идентификатор случая заболевания / идентификатор сессии

Используется для указания случая заболевания или сессии, связанной с пробой тестов.

Проверяемый тестом диагноз

Указывается диагноз случая заболевания или сессии, связанной с тестируемой пробой.

Пациент

Указывается пациент, для которого вводится случай заболевания.

Владелец фермы

Указывается владелец фермы из записи о случае заболевания животного, связанного с пробой.

Идентификатор животного / биологического вида / переносчика заболевания

В отношении теста, назначенного связанной со случаем заболевания домашней птицы пробе, введите идентификатор биологического вида.

В отношении теста, назначенного связанной со случаем заболевания домашнего скота пробе, введите идентификатор животного.

В отношении теста, назначенного сессии надзора за переносчиками заболевания, введите идентификатор переносчика заболевания.

В отношении теста, назначенного связанной со случаем заболевания человека пробе, полевой идентификатор недоступен.

В отношении теста, назначенного связанной с сессией АН пробе, введите идентификатор биологического вида или животного.

Лабораторный идентификатор пробы

Указывается лабораторный идентификатор пробы, связанной с необходимым тестом.

Локальный / полевой идентификатор пробы

Указывается лабораторный или полевой идентификатор пробы, связанной с необходимым тестом.

Дата приёма в систему: с/по

Указывается временной интервал, в течение которого в систему была принята проба от пробы, связанной с необходимым тестом.

Тип пробы

Указывается тип пробы от пробы, связанной с необходимым тестом.

Столбцы списка результатов

Идентификатор группы

Отображается номер группы, к которой принадлежит тест.

Название теста

Указывается название теста.

Категория теста

Отображается категория теста.

Статус

Отображается текущий статус теста. Для получения более подробных сведений о статусах тестов, прочитайте раздел Статусы теста.

Результат теста

Указывается результат теста.

Дата результата

Отображается дата ввода результата теста, утверждения или внесения изменений.

Лабораторный идентификатор пробы

Отображается лабораторный идентификатор пробы, связанной с тестом.

Тип пробы

Отображается тип пробы, связанной с тестом.

Локальный / полевой идентификатор пробы

Указывается локальный или полевой идентификатор пробы, связанной с тестом.

Дата приёма в систему

Указывается дата, когда в систему была принята проба, связанная с тестом.

Обрабатывающий отдел

Указывается обрабатывающий отдел (подразделение организации), в котором хранится или используется связанная с тестом проба.

Идентификатор случая заболевания / идентификатор сессии

Отображается номер случая заболевания или сессии, связанной с тестируемой пробой.

Проверяемый тестом диагноз

Отображается диагноз случая заболевания или сессии, связанной с тестируемой пробой.

Пациент / владелец фермы

Отображается Ф.И.О. пациента или владельца фермы, где произошёл случай заболевания животных, связанного с тестируемой пробой.

Идентификатор животного / биологического вида

Отображается идентификатор биологического вида, если проба связана со случаем заболевания домашней птицы.

Отображается идентификатор животного, если проба связана со случаем заболевания домашнего скота.

К случаям заболеваний людей не применяется; поле остаётся пустым.

Кнопки

Сформировать группу

Создаёт группу.

Укажите тесты с одинаковым типом теста, не входящие ни в какие группы, и нажмите кнопку Сформировать группу.   Откроется форма L22 Сведения о тесте группы с выбранными тестами.

Настоящая кнопка является активной исключительно при наличии права на запись в функции «Доступ к лабораторным тестам».

Редактирование

Позволяет редактировать выбранный тест. Настоящая кнопка отображается исключительно при наличии права на запись в функции «Доступ к лабораторным тестам».

Удаление

Позволяет редактировать выбранный(-ые) тест(-ы). Выберите в таблице тест(-ы) для удаления и нажмите кнопку Удалить.

Завершённые тесты удалить невозможно. Если некоторые из выбранных тестов удалить невозможно, появится предупреждающее сообщение, а тесты удалены не будут; все другие выбранные тесты будут удалены.

Настоящая кнопка отображается исключительно при наличии права на удаление в функции «Доступ к лабораторным тестам».
