Назначение теста

Назначение теста представляет собой операцию, возникающую в условиях планирования какого-либо теста над пробой. Для назначения тестов могут использоваться только пробы со статусом Принята в систему. Все назначенные пробе тесты можно найти в форме Проба (см. раздел Вкладка «Тесты» формы L02 «Проба») и форме Сведения о лабораторной записи (см. раздел Вкладка «Сведения о тесте» формы «Сведения о лабораторной записи»).

 

При использовании отдельных форм (ОФ), жизненный цикл теста является таким, как указано в:

 

1.Назначение теста. Первоначально тест создаётся для пробы и имеет статус Не начат . Бывает ситуация, когда результат теста вводится одновременно с назначением теста. В этом случае, шаги назначения теста и ввода результата теста совмещаются, а шаг проведения теста - опускается.

2.Проведение теста. Проведение теста начинается, но результаты либо ещё не получены, либо ещё не занесены в систему. Статус теста в такой ситуации будет В процессе. Этот шаг отсутствует, если результат теста вводится вместе с назначением теста.

3.Ввод результата теста. Такая ситуация возникает, когда результат теста известен и введён в ЭИСНЗ.. Статус теста при этом будет Предварительный.

4.Утверждение результата теста: тест прошёл проверку. Статус теста при этом будет Окончательный.

5.Отклонение результата теста (необязательный шаг). Ситуация, когда результат теста является изменённым, а статус теста меняется на Изменён.

 

При использовании лабораторного раздела (ЛР), жизненный цикл теста будет следующим: этапы

 

1.назначения теста и ввода его результатов могут проводиться одновременно или по очереди. Когда тест создаётся для пробы первоначально, его статусом является Не начат. После ввода его результата, статус теста меняется на Предварительный.

2.Проведение теста. Проведение теста начинается, но результаты либо ещё не получены, либо ещё не занесены в систему. В этом случае, статус для тестов со статусом Не начат можно указать вручную. Статус теста в такой ситуации станет В процессе. Этот шаг отсутствует, если результат теста вводится вместе с назначением теста.

3.Ввод результата теста. Такая ситуация возникает, когда результат теста известен и введён в ЭИСНЗ. . Статус теста при этом будет Предварительный.

4.Утверждение результата теста: тест прошёл проверку. Статус теста при этом будет Окончательный.

5.Отклонение результата теста (необязательный шаг). Ситуация, когда результат теста является изменённым, а статус теста меняется на Изменён.
