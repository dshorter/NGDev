Операции в лабораторном модуле

В настоящем разделе приведено описание операций, которые можно выполнять в лабораторном модуле.

Один раздел, состоящий из подразделов, предназначен для выполнения одной операции. Подразделы содержат страницы, описывающие различные методы выполнения операции. Разделы отмечаются аббревиатурами: либо ОФ (для отдельных форм), либо ЛР (для лабораторного раздела). Все методы являются правильными и взаимозаменяемыми.
