Форма L07 «Отмеченные для уничтожения пробы»

Форма L07 Отмеченные для уничтожения пробы представляет собой журнал проб, отмеченных для уничтожения (стандартной ликвидации). Используя эту форму, пользователь может принимать или отклонять отмеченные для уничтожения пробы (стандартной ликвидации).Она состоит из 2-х частей:

•Панель поиска;

•Список проб.

Инструмент «Панель поиска» создан для осуществления самых сложных типов поиска по информации о пробе. В качестве критериев поиска, в панели указываются основные и наиболее часто используемые параметры.

Форма «Список проб» представляет собой таблицу, каждая из строк которой является записью о пробе, с идентификатором пробы и некоторым количеством параметров пробы, отображаемых в столбцах таблицы. Если некоторые из столбцов не нужны или не актуальны, их можно скрыть, используя инструмент «Селектор столбцов». Для получения более подробной информации, прочитайте специальные разделы Список элементов, , Селектор столбцов, , Списки поиска, а также Настройка списочных и поисковых форм.

Рис. 351 Форма «Отмеченные для уничтожения пробы»

Поля критериев поиска

Лабораторный идентификатор пробы

Используется для указания лабораторного идентификатора соответствующей пробы.

Тип пробы

Используется для указания типа необходимой пробы.

Способ уничтожения

Указывается способ уничтожения при отметке пробы для стандартного уничтожения.

Столбцы списка результатов

Лабораторный идентификатор пробы

Отображается лабораторный идентификатор пробы, являющийся основным средством опознания пробы.

Тип пробы

Тип пробы.

Способ уничтожения

Указывается способ уничтожения при отметке пробы для стандартного уничтожения.

Кнопки

Принять

Подтверждает уничтожение (ликвидацию) выбранной пробы.

1.Укажите пробу(-ы), уничтожение которой(-ых) следует подтвердить.

2.Нажмите кнопку Принять.Откроется окно Уничтожение пробы. Для получения более подробной информации о подтверждении удаления пробы, прочитайте раздел Удаление пробы (ОФ).

Отклонить

Используется для отклонения уничтожения пробы.

1.Укажите пробу(-ы), уничтожение которой(-ых) следует отклонить.

2.Нажмите кнопку Отклонить. Пробы будут удалены из списка. Для получения более подробной информации об отклонении уничтожения пробы, прочитайте раздел Отклонение уничтожения (ОФ).

Не удалённым пробам возвращается статус Принята в систему, отображающийся в форме Список проб.
