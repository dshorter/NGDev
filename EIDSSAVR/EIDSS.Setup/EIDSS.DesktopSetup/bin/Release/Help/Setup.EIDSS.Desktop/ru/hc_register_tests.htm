Регистрация результатов тестов в отношении случая заболевания

Процедура регистрации теста необходима для выполнения различных анализов взятых проб. Все результаты тестов должны быть занесены в соответствующий раздел формы Сообщение о случае заболевания человека. В большинстве случаев, информация о проведённых тестах поступает из Лабораторного модуля. Иногда может возникнуть необходимость ручного ввода результатов тестов. Проведение такой операции доступно только для пользователей, обладающих соответствующими правами доступа. Пользователь может добавлять информацию о результатах тестов как во время регистрации случая заболевания, так и после неё, во время его редактирования.

1.Откройте вкладку Тесты, расположенную в форме Н02 Сообщение о случае заболевания человека.

Рис. 176 Вкладка «Тесты»

2.Установите статус у поля Тесты проведены в положение Да.

3.Нажмите кнопку Добавить результат. В список тестов добавится новая строка и откроется раскрывающийся список, в котором можно будет выбрать локальную пробу.

Рис. 177 Окно формы «Сообщение о случае заболевания человека»

Примечание: типу пробы присваивается статус Неизвестен, если врач-эпидемиолог уверен в отношении результата теста, но не имеет в своём распоряжении достоверной информации о типе пробы. Если поле Тип пробы имеет статус Неизвестен, появляется новый раздел Внешняя организация, позволяющий ввести данные другой лаборатории, не принадлежащей текущему сайту.

4.Заполните таблицу. Для получения полной информации о форме, просьба обратиться к форме Н02 Сообщение о случае заболевания человека.

Примечание: Полевой идентификатор пробы, Проверяемый тестом диагноз , Название теста и Результат / наблюдение являются обязательными для заполнения полями. Перед их заполнением, необходимо заполнить вкладки Пробы и Диагнозы.

5.Нажмите Сохранить для сохранения добавленного(-ых) результата(-ов) лабораторного(-ых) теста(-ов).

Начнётся репликация сохранённого случая заболевания, и он будет передан на сервер ЭИСНЗ.
