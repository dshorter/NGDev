Форма L32 «Назначение теста»

Форма Назначение теста используется для ввода данных теста(-ов), проведение которого(-ых) запланировано или выполнено в отношении проб(-ы), а также для ввода его/их результата(-ов), если тест(-ы) уже завершён/завершены. Эти шаги могут выполняться по отдельности либо одновременно.

Рис. 371 Форма «Назначение теста»

Поля данных

Проверяемый тестом диагноз

Используется для выбора из раскрывающегося списка диагноза с целью его подтверждения или отклонения. В раскрывающемся списке содержатся все текущие диагнозы, указанные для случаев заболеваний / сессий, с которыми связаны выбранные тесты. Это поле является обязательным для заполнения.

Фильтрация названий тестов по диагнозу

Положение ВКЛ. - в раскрывающемся списке Название теста отображаются только те тесты, которые подходят диагнозу, указанному в поле Проверяемый тестом диагноз. Подходящие тому или иному диагнозу тесты можно указывать в меню Диагноз - Матрица лабораторного теста.

Положение ВЫКЛ. - в раскрывающемся списке Название теста отображаются все тесты.

Название теста

Используется для выбора теста в раскрывающемся списке. При включении кнопки с фиксацией Фильтровать названия тестов по диагнозу, в раскрывающемся списке будут отображаться только те тесты, которые подходят диагнозу, указанному в поле Проверяемый тестом диагноз. Это поле является обязательным для заполнения.

Результат теста

Указываются результаты теста, если он завершён. Поле не является обязательным для заполнения. Если результат указан, статус теста меняется на Предварительный. В его отсутствие, тест получает статус Не начат.

Категория теста

Используется для выбора категории тестов из раскрывающегося списка. В нём доступны категории, указанные в Редакторе справочников. Поле не является обязательным для заполнения.

Кнопки

Добавить

Регистрирует введённые тесты не закрывая окно Назначить тест .

Закрыть

Закрывает окно.
