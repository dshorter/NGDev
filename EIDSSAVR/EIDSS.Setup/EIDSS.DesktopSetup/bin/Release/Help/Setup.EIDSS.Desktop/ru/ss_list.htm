Список форм «Основной синдромный надзор»

Show/Hide Hidden Text

Список форм «Основной синдромный надзор» представляет собой форму, в которой приведён перечень всех отчётов «Основной синдромный надзор», совпадающих с критериями поиска.

Рис. 292 Список форм «Основной синдромный надзор»

Радел панели поиска, содержащий разные поля, используется для поиска отчётов «Основной синдромный надзор».

Рис. 293 Параметры поиска

Идентификатор формы

В поле Идентификатор формы можно указать идентификатор отчёта «Основной синдромный надзор».

Извещение

Представляет собой раскрывающийся список, в котором присутствуют на выбор два значения: уведомление относится к ГПЗ (гриппоподобному заболеванию) или к ТОРЗ (тяжёлому острому респираторному заболеванию).

Название больницы / санитарно-контрольного поста

Представляет собой раскрывающийся список, в котором присутствуют на выбор два значения: «Больница» или «Санитарно-контрольный пост».

Дата отчёта: с/по

Верхняя и нижняя границы диапазона дат отчёта «Основной синдромный надзор».

Дата появления симптомов: с/по

Верхняя и нижняя границы диапазона дат появления симптомов.

Фамилия

Фамилия указанного в отчёте лица.

Имя

Имя указанного в отчёте лица.

Пол

Варианты раскрывающегося списка включают в себя опции «Мужской» и «Женский».

Возраст (годы)

Верхние и нижние границы диапазона возраста указанного в отчёте лица.

Область

В поле Область может быть указано точное наименование области.

Район

В поле Район может быть указано точное наименование района.

Населённый пункт

В поле Населённый пункт может быть указано точное наименование населённого пункта.

Результат теста

Представляет собой раскрывающийся список, где пользователь может выбрать один из следующих вариантов: «Неопределённый», «Отрицательный» или «Положительный» результат теста.

Сайт

Представляет собой раскрывающийся список, в котором можно выбрать соответствующей сайт ЭИСНЗ.
В разделе «Список результатов поиска» отображается перечень форм «Основной синдромный надзор», чьё описание подходит введённым критериям поиска.
