Вложенная вкладка «Группы переносчиков / отдельные переносчики заболеваний» формы W02 «Сессия надзора за переносчиками заболевания»

Вложенная вкладка Группы переносчиков / отдельные переносчики заболеваний используется для добавления, редактирования, просмотра и удаления отдельных переносчиков и групп переносчиков заболевания в рамках сессии надзора за переносчиками заболеваний. Таблица, содержащая список групп переносчиков / отдельных переносчиков заболевания и добавляемая к сессии надзора за переносчиками заболеваний, является нередактируемой.

 

•Для добавления новой группы переносчиков / отдельного переносчика заболевания, нажмите кнопку Новая. Для ввода данных откроется форма W03 Сведения о группе переносчиков / отдельном переносчике заболевания. Для получения более подробного описания, прочитайте раздел Регистрация отдельного переносчика / группы переносчиков заболевания.

•Отредактируйте существующую запись о группе переносчиков / отдельном переносчике заболевания; для этого отметьте пробу и нажмите кнопку Редактировать. Для ввода данных откроется форма W03 Сведения о группе переносчиков / отдельном переносчике заболевания. Для получения более подробного описания, прочитайте раздел Редактирование сессии НЗПЗ.

•Удалите группу переносчиков / отдельных переносчиков заболевания; для этого отметьте группу переносчиков / отдельных переносчиков заболевания и нажмите кнопку Удалить, расположенную посередине вложенной вкладки Группы переносчиков / отдельные переносчики заболеваний.

 

Рис. 262 Вложенная вкладка «Группы переносчиков / отдельные переносчики заболеваний»

Столбцы

Идентификатор группы переносчиков / отдельного переносчика заболевания

Отображается идентификатор группы переносчиков / отдельного переносчика заболевания, который был создан автоматически, в соответствии со Схемой уникальной нумерации.

Полевой идентификатор группы переносчиков / отдельного переносчика заболевания

Отображается идентификатор группы переносчиков или отдельного переносчика заболевания, использовавшийся исключительно до момента их регистрации в ЭИСНЗ.

Область

Отображается область, в которой происходил отлов отдельного переносчика / группы переносчиков заболевания.

Район

Отображается район, в котором происходил отлов отдельного переносчика / группы переносчиков заболевания.

Ближайший населённый пункт

Отображается ближайший от места отлова отдельного переносчика / группы переносчиков заболевания населённый пункт.

Географическая долгота

Отображаются координаты долготы места отлова отдельного переносчика / группы переносчиков заболевания.

Географическая широта

Отображаются координаты широты места отлова отдельного переносчика / группы переносчиков заболевания.

Высота над уровнем моря, м

Отображается в метрах высота над уровнем моря места отлова отдельного переносчика / группы переносчиков заболевания.

Среда обитания

Приводится описание условий и места обитания отловленной группы переносчиков / отдельного переносчика заболевания.

Дата отлова

Отображается дата отлова отдельного переносчика или группы переносчиков заболевания.

Время отлова

Отображается время отлова отдельного переносчика или группы переносчиков заболевания.

Сборщик переносчиков заболеваний

Указывается лицо, которое осуществило отлов отдельных переносчиков / групп переносчиков заболевания.

Учреждение-сборщик

Указывается организация, к которой принадлежит лицо, осуществившее отлов отдельных переносчиков или групп переносчиков заболевания.

Биологический вид

Указывается биологический вид переносчик(-ов) заболевания.

Количество

Указывается количество отловленных отдельных переносчиков или переносчиков заболеваний, объединённых в группу.

Пол

Указывается пол отловленного(-ых) переносчика(-ов) заболевания.

Собраны эктопаразиты

Указывается факт обнаружения и сбора наружных паразитов с переносчика заболевания.

Ссылка на переносчика-хозяина

Указывается животное-хозяин собранных переносчиков заболевания, если на нём были обнаружены такие переносчики.

Метод отлова

Указывается метод, использовавшийся для отлова отдельного переносчика или группы переносчиков заболевания.

Исходные данные для исследования

Указывается подробная информация об обстоятельствах отлова отдельного переносчика или группы переносчиков заболевания: отловлены в плановом порядке или принесены кем-либо при случайном обнаружении.

Точки геопозиционирования

Указываются точки геопозиционирования, использовавшиеся для определения места отлова группы переносчиков / отдельного переносчика заболевания.

Обнаруживший работник

Указывается лицо, определившее биологический вид переносчика заболевания.

Обнаружившее учреждение

Указывается организация, обнаружившая биологический вид переносчика заболевания.

Дата обнаружения

Указывается дата обнаружения биологического вида.

Метод обнаружения

Поле «Метод обнаружения» содержит данные о типе метода обнаружения, использовавшегося для обнаружения группы переносчиков / отдельного переносчика заболевания.

Специфические данные о переносчике заболевания

Указываются данные, относящиеся к типу переносчика заболевания и заносимые на вкладку «Специфические данные о переносчике заболевания» формы W03 «Сведения о группе переносчиков / отдельном переносчике заболевания». Данные в столбце отделяются знаком запятой (,).

Кнопки

Новая/новый

Используется для создания новой записи о группе переносчиков или отдельном переносчике заболевания. Для получения более подробного описания, прочитайте раздел Регистрация отдельного переносчика / группы переносчиков заболевания.

Копировать

Используется для копирования выбранного отдельного переносчика / группы переносчиков заболевания с выбранным набором данных. Для получения более подробной информации, прочитайте раздел Копирование отдельного переносчика / группы переносчиков заболевания.

Редактировать

Открывает окно формы W03 Сведения о группе переносчиков / отдельном переносчике заболевания для редактирования данных группы переносчиков или отдельного переносчика заболевания:

1.Отметьте группу переносчиков или отдельного переносчика заболевания для редактирования.

2.Нажмите кнопку Редактировать. Откроется окно формы W03 Сведения о группе переносчиков / отдельном переносчике заболевания.Отредактируйте данные о группе переносчиков / отдельном переносчике заболевания.Для сохранения изменений, нажмите кнопку ОК.УдалитьУдаляет выделенные записи групп переносчиков / отдельных переносчиков заболевания.Отметьте группу переносчиков / отдельных переносчиков заболевания, предназначенную для удаления.Нажмите кнопку Удалить. Откроется окно с запросом на подтверждение действия.Нажмите Да для подтверждения удаления и закройте окно подтверждения.Запись о переносчике заболевания нельзя удалить, если в её рамках была принята хотя бы одна проба в лабораторном модуле.См. также:Создание сессии надзора за переносчиками заболевания (НЗПЗ);
