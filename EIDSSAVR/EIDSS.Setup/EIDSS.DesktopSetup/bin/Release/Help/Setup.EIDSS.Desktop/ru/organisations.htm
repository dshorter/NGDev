Организации

Show/Hide Hidden Text

Все организации (объекты), которые так или иначе участвуют в расследовании случаев заболеваний людей, животных и переносчиков заболеваний, заносятся в базу данных ЭИСНЗ. При работе со списком организаций, старайтесь избегать создания повторных записей об организациях, уже имеющихся в системе: дублирующиеся записи могут негативно влиять на результаты анализа и поиска данных.

       Для создания записи об организации в системе необходимо::

1.Находясь в меню Система, нажмите пункт Администрирование, а затем нажмите Организации. Откроется Список организаций.

Рис. 129 Список организаций

2.Нажмите кнопку Новая. Откроется окно Сведения об организации.

Рис. 130 Информация об организации

3.Укажите краткое и полное наименование, а также дополнительную информацию об организации. Краткое наименование используется во время входа в систему.

4.Нажмите на вкладку Отделы. Вкладка Отделы откроется.

Рис. 131 Отделы организации

5.Укажите отделы, подразделения, офисы, и т.д. организации. Для этого, нажмите мышкой на верхней строке окна и укажите название отдела. Нажмите кнопку Добавить и продолжите ввод отделов.

6.По окончании ввода отделов, нажмите кнопку ОК. Откроется окно Подтверждения на совершение действия.

7.Нажмите Да, чтобы подтвердить ввод организации.

8.Для закрытия окна Список организаций, нажмите кнопку Закрыть.

       Для редактирования записи об организации в системе необходимо::

1.Находясь в меню Система, нажмите пункт Администрирование, а затем нажмите Организации. Откроется Список организаций.

Рис. 132 Список организаций

2.Отметьте организацию для редактирования. Нажмите кнопку Редактировать. Откроется окно Сведения об организации.

Рис. 133 Информация об организации

3.Укажите краткое и полное наименование, и/или дополнительную информацию об организации. Краткое наименование используется во время входа в систему.

4.Нажмите на вкладке Отделы. Откроется вкладка Отделы.

Рис. 134 Отделы организации

5.Укажите новый отдел, подразделение, офис и т.д. организации. Для этого, нажмите мышкой на верхней строке окна и укажите название отдела. Нажмите кнопку Добавить.

6.Для удаления записи об отделе, отметьте его и нажмите кнопку Удалить.

Запись удаляемого отдела окончательно удаляется из системы и больше уже не будет доступна.

7.По окончании изменений, нажмите кнопку ОК. Откроется окно Подтверждения на совершение действия.

8.Для подтверждения нажмите Да.

9.Для закрытия окна Список организаций, нажмите кнопку Закрыть.

       Для удаления организации из системы необходимо::

1.Находясь в меню Система, нажмите пункт Администрирование, а затем нажмите Организации. Откроется Список организаций.

Рис. 135 Список организаций

2.Отметьте организацию для удаления. Нажмите кнопку Удалить. Откроется окно Удалить запись.

Рис. 136 Подтверждение удаления

Запись удаляемой организации будет безвозвратно удалена из системы. Работники этой организации лишатся прав доступа в систему.

3.Для подтверждения нажмите «Да».

4.Для закрытия окна Список организаций, нажмите кнопку Закрыть.
