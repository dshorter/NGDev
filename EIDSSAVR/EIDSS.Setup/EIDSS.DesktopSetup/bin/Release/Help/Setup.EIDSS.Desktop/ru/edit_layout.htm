Редактирование макета

Редактирование опубликованных макетов невозможно. При создании макета, количество доступных для использования текущим пользователем полей ограничено правами доступа, которыми он обладает в системе.

До тех пор, пока к макету не будет открыт общий доступ или пока он не будет опубликован, редактировать его может только его автор. После открытия автором макета общего доступа к нему, любой пользователь этого сайта, обладающий правами доступа к модулю АВО, может изменять такой макет. После публикации макета, он становится видимым в режиме только для чтения пользователям всех сайтов данной страны, обладающим правами доступа к модулю АВО.

 

1.Находясь в меню модуля АВО, нажмите пункт Запустить модуль АВО. Откроется окно Модуль анализа, визуализации и отчётности.

2.Проделайте одно из следующих действий:

•Находясь в разделе Дерево запросов и макетов, дважды нажмите на макете, который предполагается редактировать.

•Выберите запрос для редактирования и нажмите кнопку Редактировать запрос / макет.

Окно Дерево запросов и макетов закроется, и откроется окно с выбранным макетом.

3.Измените имя и описание макета.

4.Измените сводную таблицу.

5.Измените вид макета. Вид представления данных является окончательным отчётом. Данные в нём могут отображаться либо в виде диаграммы, либо в виде географической карты.

6.Настройте свойства диаграммы или карты, если это необходимо.

7.Сохраните макет.

8.Закройте макет.

 

Для получения более подробных сведений о форме Редактор макетов, прочитайте раздел Редактор макетов.
