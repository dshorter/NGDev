Вкладки «Лабораторный раздел» и «Мои личные настройки»

Вкладки Лабораторный раздел и Мои личные настройки созданы для хранения списка и работы с зарегистрированными пробами и тестами, отвечающими критериям поиска. На вкладке Лабораторный раздел содержатся все записи. На вкладке Мои личные настройки отображаются только добавленные вручную записи. Для получения более подробной информации о том, как добавить запись на вкладку Мои личные настройки, прочитайте раздел Добавление записи в личные настройки / удаление записи из списка личных настроек.

На обоих вкладках: Лабораторный раздел и Мои личные настройки отображается таблица, в которой каждая строка является записью о пробе, имеющей (или не имеющей) связанный с ней тест. В каждой строке отображается номер пробы, а в столбцах пробы отображаются параметры тестов. Если некоторые из столбцов не нужны или не актуальны, их можно скрыть, используя инструмент «Селектор столбцов». Для получения более подробной информации, прочитайте специальные разделы Список элементов, , Селектор столбцов, , Списки поиска, а также Настройка списочных и поисковых форм.

Рис. 366 Список лабораторных записей

Лабораторную запись можно изменять в таблице. Для этого, выделите изменяемое поле и введите нужное значение либо выберите его. После изменения, запись выделяется жирным шрифтом и красной рамкой. Сделанные изменения можно либо принять (сохранить), либо отклонить (отменить). Для принятия всех изменений в таблице, нажмите кнопку Сохранить. Для отмены некоторых из сделанных изменений, выберите их, а затем нажмите на них правой кнопкой мыши и из раскрывающегося меню выберите пункт Отменить изменения .

При отсутствии необходимого параметра в таблице, нажмите кнопку Плюс (), расположенную напротив лабораторной записи. Отобразится подробная информация о лабораторной записи, которую можно будет изменить. Для получения более подробных сведений, прочитайте раздел формы Сведения о лабораторной записи.

Такие отсутствующие в таблице операции можно запускать из меню, выводимому при нажатии правой кнопки мыши. Для этого выберите одну или несколько лабораторных записей, нажмите на них правой кнопкой мыши и выберите необходимую операцию из раскрывающегося меню. В ряде случаев операция с несколькими записями одновременно не возможна, и это зависит как от самой операции, так и от параметров выбранных записей.

 

Столбцы списка результатов

Идентификатор случая заболевания / идентификатор сессии

Отображается идентификатор либо случая заболевания, либо сессии, связанных с пробой. Для просмотра подробной формы с информацией о случае заболевания или сессии, нажмите кнопку Сведения ().

Локальный / полевой идентификатор пробы

Отображается локальный или полевой идентификатор пробы, который копируется из данных случая заболевания / сессии, связанной с пробой.

Статус пробы

Отображается статус пробы, являющийся одним из наиболее важных параметров, определяющим доступность полей и операций пробы. Для получения более подробных сведений о статусах проб, прочитайте раздел Статусы проб. Поле всегда изменяется автоматически, в соответствии с выполняемой операцией.

Лабораторный идентификатор пробы

Отображается лабораторный идентификатор пробы, являющийся основным средством опознания пробы. Если пробы не приняты в систему, это поле остаётся пустым. При приёме пробы в систему, значение поля Лабораторный идентификатор пробы автоматически устанавливается на Новая , и его можно изменять вручную. Если изменение отсутствует, идентификатор рассчитывается автоматически, в соответствии со Схемой уникальной нумерации.

Тип пробы

Указывается значение типа пробы, которое можно изменять при создании новой пробы, аликвоты или деривата.

Дата приёма в систему

Отображается дата приёма пробы в систему. При приёме пробы в систему, значение текущей даты используется автоматически в качестве даты её приёма в систему и её значение можно изменять вручную. После первого сохранения записи по окончании её приёма в систему, поле становится нередактируемым.

Состояние пробы при получении

В поле указывается состояние пробы во время её приёма в систему. Если состояние пробы является любым кроме статуса «Хорошее», в столбец Комментарий потребуется вписать пояснение. После сохранения пробы, поле становится нередактируемым. Поле остаётся пустым для проб со статусом Не принята в систему, для аликвот и дериватов.

Проверяемый тестом диагноз

Отображается диагноз, требующий либо подтверждения, либо отказа на основании результатов теста. Первоначально используется диагноз случая заболевания или сессии, связанных с пробой, который можно изменять при назначении пробе теста.

Название теста

Указывается название теста. Наименование теста указывается при назначении пробе теста.

Статус теста

Отображается текущий статус теста, который, как правило, можно изменять вручную, за исключением тестов, имеющих статус Окончательный или Изменённый. Для получения более подробных сведений о статусах тестов, прочитайте раздел Статусы теста.

Результат теста

Отображается результат теста, если таковой имеется. Результат можно вводить или изменять только по тестам со статусом Предварительный или В процессе, или во время присвоения таких статусов.

Дата начала теста

Отображается значение даты начала теста, которое можно изменять.

Дата результата

Отображается дата ввода результата теста, утверждения или внесения изменений, при их наличии. При вводе результата, утверждения или изменения, в качестве даты по умолчанию используется текущая дата, значение которой можно изменять вручную до тех пор, пока результат не будет сохранён.

Категория теста

Отображается категория теста. Первоначально категории изменяются в Редакторе справочников.

Дополнительные сведения о тесте

Отображается информация из Настраиваемой формы «Сведения о тесте». Для получения более подробной информации, прочитайте раздел Конфигурация настраиваемых форм .

Кнопки

Выбрать все

Выбирает все лабораторные записи, отображаемые в списке.

Сохранить

Сохраняет все изменения, внесённые в лабораторные записи, но ещё не сохранённые. Изменения выделяется жирным шрифтом и красной рамкой.

ОК

Закрывает окно Лабораторный раздел.

Отменить

Отменяет все несохранённые изменения.
