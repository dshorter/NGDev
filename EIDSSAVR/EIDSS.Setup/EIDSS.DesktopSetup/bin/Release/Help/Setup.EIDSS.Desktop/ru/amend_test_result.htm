Изменение результата теста

Изменение результата теста является операцией изменения утверждённого и окончательного результата теста. Эта процедура используется тогда, когда результат теста признаётся неправильным уже после того, как он был утверждён. Результат теста, имеющего отличающийся статус от (В процессе или Предварительный), не требует специальной процедуры внесения изменений. Для изменения такого результата, используйте ту же процедуру, которая применяется для первоначального ввода результата теста..

Каждое изменение результата сохраняется и отображается на вкладке История изменений теста формы L04 Сведения о лабораторном тесте, или формы Сведения о лабораторной записи

Статус Изменённый представляет собой окончательный этап жизненного цикла теста. Для повторения теста со статусом Изменённый, следует просто назначить новый тест.
