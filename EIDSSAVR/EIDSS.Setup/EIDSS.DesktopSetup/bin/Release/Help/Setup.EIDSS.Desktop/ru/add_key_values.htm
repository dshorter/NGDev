Добавление ключевых значений

В некоторых случаях требуется создание отчёта по окончательным диагнозам и областям/районам текущего места жительства, по которым данных нет. В этом случае используйте ключевые значения. Ключевые значения представляют собой пустые записи (для диагнозов и областей/районов), которые можно добавлять в запрос по мере необходимости.

Например, необходимо создать отчёт с информацией о количестве случаев заболеваний людей по какому-либо диагнозу для всех районов областей Гурия и Аджария.

Рис. 395 Сводная таблица с ключевыми значениями

Хотя в районах Озургети, Кеда, Кобулети и Шуахеви нулевой процент случаев заболеваний, эти районы тоже отображаются в списке районов.

Активация ключевых значений

Активируйте кнопку с фиксацией Добавить все ключевые справочные значения, находящуюся в окне Общая информация запроса.

Рис. 396 Окно общей информации запроса

У всех макетов, созданных на основе запроса с активированными ключевыми значениями, ключевые значения тоже будут активированы. Ключевые значения будут добавлены в отчёт даже если позже в макете эта опция будет отключена.
