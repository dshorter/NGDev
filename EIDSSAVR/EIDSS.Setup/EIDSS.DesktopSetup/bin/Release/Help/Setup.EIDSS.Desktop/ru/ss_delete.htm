Удаление отчёта «Основной синдромный надзор»

1.Выберите Журналы -> Формы «Основной синдромный надзор. Откроется Список форм «Основной синдромный надзор».

Рис. 279 Список форм «Основной синдромный надзор»

2.Произведите поиск отчёта «Основной синдромный надзор».

3.Для удаления отчёта «Основной синдромный надзор», выберите нужный отчёт из списка и нажмите кнопку Удалить. Откроется окно с запросом на подтверждение совершение действия.

Рис. 280 Удаление записи

4.Для подтверждения удаления отчёта нажмите Да.

Примечание: помимо этого, отчёт «Основной синдромный надзор» можно удалить и в окне Формы «Основной синдромный надзор» просто нажав кнопку Удалить, а затем - Да в окне подтверждения на совершение действия.
