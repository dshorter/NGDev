Ввод результата теста (ЛР)

1.Находясь в меню Журналы, выберите Лабораторный раздел. Откроется окно Лабораторный раздел.

2.Найдите тест(-ы), чей результат планируется внести. Результат можно ввести только для тестов со статусом В процессе или Предварительный.

3.Введите его/их результат одним из следующих способов:

Первый способ:

a)Выберите необходимый(-ые) тест(-ы).

b)Нажмите на них правой кнопкой мыши и из раскрывающегося меню выберите пункт Указать результат теста, а затем сам результат. Эту процедуру можно использовать как для отдельного теста, так и для нескольких тестов с одинаковым результатом.

Второй способ:

a)Нажмите на поле Статус теста и из раскрывающегося списка выберите пункт Предварительный.

b)Нажмите на поле Результат теста и из раскрывающегося списка выберите результат. Такую процедуру можно использовать только для отдельного теста.

Третий метод:

a)Нажмите на поле Результат теста и из раскрывающегося списка выберите результат. Такую процедуру можно использовать только для отдельного теста. Значение Статус теста автоматически меняется на Предварительный.

Во всех указанных случаях, поля Дата начала теста, Дата результата, Тестировавший работник и Оператор ввода результатов автоматически заполняются значением текущей даты, которое можно поменять вручную.

4.Нажмите кнопку Сохранить и подтвердите сохранение.
