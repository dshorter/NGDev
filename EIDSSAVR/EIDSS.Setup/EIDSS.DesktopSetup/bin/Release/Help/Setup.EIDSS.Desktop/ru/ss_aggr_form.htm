Форма «Агрегированные данные ГПЗ»

Рис. 295 Форма «Агрегированные данные ГПЗ»

Поля данных

Идентификатор формы

Представляет собой уникальный идентификатор Формы «Агрегированные данные ГПЗ», который генерируется автоматически.

Дата регистрации

Отображается дата первого ввода информации отчёта в систему; генерируется автоматически.

Дата последнего сохранения

Отображается дата последнего сохранения информации отчёта в системе; генерируется автоматически.

Зарегистрировавший работник

Отображается имя пользователя, создавшего текущий отчёт «Агрегированные данные ГПЗ»; запись генерируется автоматически.

Сайт

Отображается наименование сайта создания текущего отчёта «Агрегированные данные ГПЗ»; генерируется автоматически.

Год

Представляет собой раскрывающийся список, используемый для выбора соответствующего года.

Неделя

Представляет собой раскрывающийся список, используемый для выбора соответствующей недели года.

Таблица

Название больницы / санитарно-контрольного поста

Представляет собой раскрывающийся список, в котором присутствуют на выбор два значения: «Больница» или «Санитарно-контрольный пост».

Возрастные группы

Возрастные группы делятся в следующем порядке: 0-4, 5-14, 15-29, 30-64 и старше 65. Для каждой группы выявленных гриппоподобных заболеваний вводится соответствующее число.

Всего ГПЗ

Указывается общее количество гриппоподобных заболеваний.

Всего госпитализированных

Указывается общее количество пациентов, поступивших на лечение в больницу.

Пробы ГПЗ

Указывается количество проб, взятых для расследования случаев гриппоподобных заболеваний.
