Группы пользователей

Show/Hide Hidden Text

Зарегистрированных в системе работников можно группировать. Используйте группирование работников для быстрого предоставления им прав доступа к определённому типу данных.

Для создания группы пользователей необходимо::

1.Находясь в меню Безопасность, нажмите мышкой Группы пользователей. Откроется окно Список групп пользователей.

Рис. 146 Список групп пользователей

2.Нажмите кнопку Новая. Откроется окно Сведения о группе пользователей.

Рис. 147 Сведения о группе пользователей

3.Укажите название группы и её краткое описание.

4.Для добавления пользователя / группы пользователей к созданной группе пользователей, нажмите кнопку Добавить. Откроется окно Список пользователей и групп пользователей.

5.Для выбора пользователя или группы пользователей, которые будут включены в созданную группу пользователей, нажмите Выбрать.

6.Для определения прав группы пользователей в отношении системных функций, откройте вкладку Системные функции.

Рис. 148 Вкладка «Системные функции»

7.В раскрывающемся поле Сайт, выберите сайт, на котором такой группе пользователей будут присвоены назначаемые права. Или нажмите пункт Применить ко всем сайтам, чтобы применить предоставляемые права группе пользователей на всех сайтах ЭИСНЗ.

8.Прокрутите весь список системных функций, делая выбор на кнопках с фиксацией Запретить или Разрешить, тем самым запрещая или разрешая доступ к соответствующей функции группе пользователей.

9.Нажмите OK, чтобы закрыть окно Сведения о группе пользователей.

10.Для закрытия окна Список групп пользователей, нажмите кнопку Закрыть.
Для редактирования группы пользователей необходимо::

1.Находясь в меню Безопасность, нажмите мышкой Группы пользователей. Откроется окно Список групп пользователей.

Рис. 149 Список групп пользователей

2.Нажмите кнопку Редактировать. Откроется окно Сведения о группе пользователей.

Рис. 150 Сведения о группе пользователей

3.Отредактируйте наименование группы или краткое описание.

4.Для добавления пользователя / группы пользователей к созданной группе пользователей нажмите Добавить, а в окне Список пользователей и групп пользователей выберите пользователя / группу пользователей и нажмите кнопку Выбрать.

5.Для определения прав группы пользователей в отношении системных функций, откройте вкладку Системные функции.

Рис. 151 Вкладка «Системные функции»

6.В раскрывающемся поле Сайт, выберите сайт, на котором такой группе пользователей будут присвоены назначаемые права. Или нажмите пункт Применить ко всем сайтам, чтобы применить предоставляемые права группе пользователей на всех сайтах ЭИСНЗ.

7.Прокрутите весь список системных функций, делая выбор на кнопках с фиксацией Запретить или Разрешить, тем самым запрещая или разрешая доступ к соответствующей функции группе пользователей.

8.Нажмите OK, чтобы закрыть окно Сведения о группе пользователей.

9.Для закрытия окна Список групп пользователей, нажмите кнопку Закрыть.

Для удаления группы пользователей необходимо::

1.Находясь в меню Безопасность, нажмите мышкой Группы пользователей. Откроется окно Список групп пользователей.

Рис. 152 Список групп пользователей

2.Выберите группу для удаления.

3.Нажмите кнопку Удалить. Откроется окно Удалить запись.

Рис. 153 Подтверждение удаления

4.Нажмите ОК для подтверждения удаления.

5.Для закрытия окна Список ролей пользователей, нажмите кнопку Закрыть.
