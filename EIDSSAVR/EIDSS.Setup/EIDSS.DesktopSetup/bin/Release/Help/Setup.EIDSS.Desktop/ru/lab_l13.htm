Форма L13 «Приём в систему группы»

Форма Приём в систему группы проб используется для одновременного приёма в систему группу проб разных типов, относящихся к различным случаям заболеваний или сессиям, но имеющих одинаковый локальный / полевой идентификатор пробы.

Рис. 359 Окно «Приём в систему группы проб»

Поля данных

Сканировать локальный/полевой идентификатор пробы

Указывается вручную или при помощи штрихкодового сканера локальный/полевой идентификатор группы проб. После нажатия кнопки Добавить, открывается окно Выбор пробы с группой проб, имеющих введённый локальный/полевой идентификатор пробы.

Тип пробы

Тип пробы.

Место хранения

Указывается положение места хранения принятой в системы пробы:

1.Нажмите кнопку Сведения (). Откроется окно Холодильники.

2.Выберите необходимое место.

3.Нажмите кнопку ОК. Окно Холодильники закроется. В поле будет отображаться назначенное местоположение.

Это поле доступно только для проб, принятых в систему в хорошем или плохом состоянии (статус у поля Принята в систему, Состояние пробы при получении не является Отклонена).

Локальный / полевой идентификатор пробы

Указывается локальный / полевой идентификатор пробы.

Идентификатор случая заболевания / идентификатор сессии

Указывается идентификатор случая заболевания / идентификатор сессии.

Штрихкод (Лабораторный идентификатор пробы)

Автоматически присваиваемый после приёма пробы в систему лабораторный идентификатор пробы, который можно менять вручную. Поле становится доступным только после приёма в систему.

Дата взятия

Указывается дата взятия пробы.

Дата приёма в систему

Указывается значение текущей даты, которое используется в качестве значения для поля Дата приёма в систему после того, как проба будет принята в систему; значение можно поменять вручную при помощи календаря. Поле становится доступным только после приёма в систему.

Состояние пробы при получении

Указывается состояние пробы после её приёма в систему. Значение Принята в хорошем состоянии используется по умолчанию, но его можно поменять вручную, выбрав значение из раскрывающегося списка. Если выбрано одно из двух значений: Отклонена или Принята в плохом состоянии, в поле Комментарии необходимо будет ввести пояснение. Поле становится доступным только после приёма в систему.

Комментарии

Указываются комментарии. Если выбрано одно из значений Отклонена или Принята в плохом состоянии в поле Состояние пробы при получении, заполнение поле комментариев становится обязательным для пояснения обстоятельств.

Поле становится доступным только после приёма в систему.

Обрабатывающий отдел

В раскрывающемся списке выбирается получающий пробу отдел организации.

Это поле доступно только для проб, принятых в систему в хорошем или плохом состоянии (статус у поля Принята в систему, Состояние пробы при получении не является Отклонена).

Кнопки

Добавить

Открывает список проб с введённым локальным/полевым идентификатором пробы в окне Выбор пробы:

1.Укажите вручную или при помощи штрихкодового сканера локальный/полевой идентификатор группы проб в поле Сканировать локальный/полевой идентификатор пробы.

2.Нажмите кнопку Добавить. Откроется окно Выбор пробы с группой проб, имеющих введённый локальный/полевой идентификатор пробы.

Печать штрихкодовых наклеек

Вызывает меню печати штрихкодовых наклеек выбранных проб.

1.Отметьте пробы, которым нужны штрихкодовые наклейки.

2.Нажмите кнопку Печать штрихкодовых наклеек. Откроется окно Печать штрихкодовых наклеек с возможностью предварительного просмотра штрихкодовой наклейки пробы.

3.Нажмите кнопку Печать ().

Принять все пробы

Служит для приёма в систему всех проб в таблице; для этого нажмите кнопку «Принять все пробы» и все, не принятые в систему, пробы будут приняты. Такие атрибуты, как «Дата приёма в систему», «Состояние пробы при получении», «Место хранения», «Комментарий» и «Обрабатывающий отдел» становятся доступны для редактирования.

Удалить

Позволяет удалять отмеченные пробы в окне Приём в систему группы проб. Удалять можно только ещё не принятые в систему пробы.

1.Отметьте удаляемую из списка пробу.

2.Нажмите кнопку Удалить. Откроется окно Сообщение, с просьбой подтвердить удаление.

3.Нажмите кнопку ОК. Окно Сообщение закроется, а проба из окна Приём в систему группы проб будет удалена.

Сохранить

Служит для сохранения изменений.

ОК

Закрывает окно Приём в систему группы проб без сохранения каких-либо изменений, если только их не сохраняли до этого.

Отменить

Отменяет все изменения, даже если их ранее сохраняли, и закрывает окно.
