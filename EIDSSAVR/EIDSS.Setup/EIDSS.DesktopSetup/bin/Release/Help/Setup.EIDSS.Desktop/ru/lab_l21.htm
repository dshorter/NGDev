Форма L21 «Список групп»

Форма L21 Список групп создаётся для отображения всех групп, зарегистрированных в ЭИСНЗ. Она состоит из 2-х частей.

•Панель поиска

•Список групп

Панель поиска имеет возможность осуществления поиска по сведениям о группе. В качестве полей поиска, в панели имеются основные и наиболее часто используемые параметры.

Форма «Список групп» представляет собой таблицу, каждая из строк которой является записью о группе, с идентификатором группы и некоторым количеством параметров группы, отображаемых в столбцах таблицы. Если некоторые из столбцов не нужны или не актуальны, их можно скрыть, используя инструмент «Селектор столбцов». Для получения более подробной информации, прочитайте специальные разделы Список элементов, , Селектор столбцов, , Списки поиска, а также Настройка списочных и поисковых форм.

Рис. 361 Окно «Список группы»

Поля критериев поиска

Идентификатор группы

Указывается уникальный идентификатор группы, который необходимо найти.

Идентификатор случая заболевания / сессии

Используется для указания идентификатора случая заболевания или сессии, связанных с тестируемой группой.

Название теста

Указывается название теста, применяемого в отношении группы.

Лабораторный идентификатор пробы

Указывается проба, относящаяся к тесту группы.

Статус

Указывается статус группы.

Дата начала теста: с/по

Указывается временной интервал, когда было начато проведение теста.

Дата результата

Указывается временной интервал, когда результат теста регистрировался, утверждался или изменялся в последний раз.

Столбцы

Идентификатор группы

Представляет собой уникальный идентификатор группы.

Название теста

Указывается название теста, применяемого в отношении группы.

Дата начала теста

Указывается день начала проведения теста.

Дата результата

Указывается день, в который стал известен результат теста.

Количество тестов

Отображается количество тестов в группе.

Статус

Отображается статус группы.

Кнопки

Новая

Нажатие создаёт запись о новой группе. Для получения более подробной информации, прочитайте раздел Создание группы в форме «Список групп».

Редактировать

Открывает группу для редактирования в Форме L22 Сведения о тесте группы.

Обновить

Обновляет список групп.

Закрыть

Закрывает окно Список групп.
