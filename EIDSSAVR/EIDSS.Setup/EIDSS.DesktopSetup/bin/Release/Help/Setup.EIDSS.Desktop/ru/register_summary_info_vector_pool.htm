Регистрация краткой основной информации об отдельных переносчиках / группах переносчиков заболевания

После создания новой записи о группе переносчиков / отдельном переносчике заболевания или открытия существующей записи с целью её редактирования, открывается форма W03 «Сведения о группе переносчиков / отдельном переносчике заболевания».

Краткая информация о группе переносчиков / отдельном переносчике заболевания включает в себя уникальный идентификатор, полевой идентификатор и тип группы переносчиков / отдельного переносчика заболевания. Поле Идентификатор сессии предназначено только для чтения и отображает сессию, в рамках которой был отловлен отдельный переносчик / группа переносчиков заболевания. Типы переносчиков заболеваний, добавляемые в окно краткой основной информации, копируются в поле Тип переносчика заболевания сессии надзора за переносчиками заболеваний. Для ознакомления с подробным описанием полей, прочитайте раздел «Краткая информация» формы W03 «Сведения о группе переносчиков / отдельном переносчике заболевания».
