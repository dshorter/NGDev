Вкладка «Полевые тесты» формы W03 «Сведения о группе переносчиков / отдельном переносчике заболевания»;

Вкладка Полевые тесты используется для добавления, редактирования, просмотра и удаления полевых тестов, проведённых в отношении отдельного переносчика / группы переносчиков заболевания.

 

•Добавьте новый полевой тест; для этого нажмите на пустую строку с сообщением Нажать для добавления новой строки.

•Отредактируйте запись существующего полевого теста; для этого отметьте тест и отредактируйте его в таблице.

•Удалите полевой(-ые) тест(-ы); для этого отметьте тест(-ы) и нажмите кнопку Удалить.

Рис. 272 Вкладка «Полевые тесты»

Поля данных

Полевой идентификатор пробы

Укажите полевой идентификатор пробы, которой было назначено проведение теста.

Тип пробы

По умолчанию отображается тип выбранной пробы, который нельзя изменить.

Дата взятия

По умолчанию отображается дата взятия выбранной пробы, которую нельзя изменить.

Название теста

Указывается название теста. Это поле является обязательным для заполнения.

Категория теста

Указывается категория теста.

Дата завершения теста

Указывается дата проведения теста.

Тестировавшее учреждение

Указывается проводившая тест организация.

Тестировавший работник

Указывается проводившее тест лицо.

Результат теста

Указываются результаты теста, если они известны. При положительном результате, информация о заболевании копируется из столбца Заболевание в поле Заболевания сессии надзора за переносчиками заболеваний, к которой относится тест.

Заболевание

Указывается заболевание, которое должно подтверждаться тестом. Если заболевание подтверждается, оно копируется в поле Заболевания сессии надзора за переносчиками заболеваний, в рамках которого проводился такой тест.

Кнопки

Удалить

Удаляет выделенный(-ые) тест(-ы).

1.Выберите тест(-ы) для удаления.

2.Нажмите кнопку Удалить. Откроется окно с запросом на подтверждение действия.

3.Нажмите Да для подтверждения удаления и закройте окно подтверждения.
