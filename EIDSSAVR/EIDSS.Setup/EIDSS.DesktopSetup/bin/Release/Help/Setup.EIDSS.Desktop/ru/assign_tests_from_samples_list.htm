Назначение тестов в форме «Список проб» (ОФ)

Настоящая операция доступна только в случае отсутствия включённого флажка на кнопке с фиксацией Упрощённый режим лабораторного модуля, расположенной в окне Системные настройки .

 

1.Находясь в меню Журналы, выберите пункт Пробы. Откроется окно Список проб.

2.Найдите и выберите пробу, для которой планируется назначить проведение тестов.

3.Нажмите кнопку Выбрать тест. Откроется форма Выбор тестов . Выбранные пробы находятся в левой панели. Для получения более подробных сведений о форме Выбор тестов , прочитайте раздел формы L20 Выбор тестов.

Рис. 317 Форма «Выбор тестов»

4.Выберите пробу(-ы) для назначения тестов.

5.Для фильтрации тестов по диагнозу, сделайте свой выбор в поле Выбор диагноза, что отфильтрует поле нужного теста.

6.В правой верхней панели укажите назначаемые тесты. Проделайте следующие действия:

•выберите назначаемые тесты в правой панели и нажмите кнопку Добавить. Выбранные тесты добавятся в нижнюю правую панель.

•Дважды нажмите мышкой на назначаемом тесте.

7.Нажмите кнопку OK, чтобы сохранить изменения и закрыть окно.
