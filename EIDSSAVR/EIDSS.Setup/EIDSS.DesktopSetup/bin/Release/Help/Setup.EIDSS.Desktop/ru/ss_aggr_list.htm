Список форм «Агрегированные данные ГПЗ»

Show/Hide Hidden Text

Список форм «Агрегированные данные ГПЗ» представляет собой форму, в которой приведён перечень всех отчётов «Агрегированные данные ГПЗ», совпадающих с критериями поиска.

Рис. 296 Список форм «Агрегированные данные ГПЗ»

Радел панели поиска, содержащий разные поля, используется для поиска отчётов «Агрегированные данные ГПЗ».

Рис. 297 Параметры поиска

Идентификатор формы

В поле Идентификатор формы можно указать идентификатор отчёта «Агрегированные данные ГПЗ».

Дата начала

Указывается дата начала периода «Агрегированные данные сессии ГПЗ».

Дата окончания

Указывается дата окончания периода «Агрегированные данные сессии ГПЗ».

Недели: с - по

Указывается состоящий из недель начала и окончания период «Агрегированные данные сессии ГПЗ». Будут найдены только полные недели, входящие в такой период.

Название больницы / санитарно-контрольного поста

Представляет собой раскрывающийся список, в котором присутствуют на выбор два значения: «Больница» или «Санитарно-контрольный пост».

Сайт

Представляет собой раскрывающийся список, в котором можно выбрать соответствующей сайт ЭИСНЗ.
В разделе «Список результатов поиска» отображается перечень форм «Агрегированные данные основного синдромного надзора», чьё описание подходит введённым критериям поиска.

Рис. 298 Список результатов поиска

Идентификатор формы

Отображается идентификатор формы.

Недели

Отображается номер недели года.

Название больницы / санитарно-контрольного поста

Отображается наименование больницы или санитарно-контрольного поста.
