Вкладка «Общая информация» формы W03 «Сведения о группе переносчиков / отдельном переносчике заболевания»;

Вкладка Общие данные используется для добавления, просмотра и редактирования подробной информации об отдельном переносчике или группе переносчиков заболевания: местоположении, времени и методе сбора материала, количестве отловленных переносчиков или групп переносчиков заболевания, биологических видах и поле, обнаружившем и осуществившем его/её отлов лице.

Поле «Комментарии», расположенное внизу вкладки, может использоваться для добавления текстовой информации в произвольной форме, при необходимости.

Рис. 269 Вкладка «Общая информация»

Поля данных об отлове

Местоположение

Выберите или введите точное местоположение отлова группы переносчиков / отдельного переносчика заболевания. Нажмите мышкой на поле и откроется специальное окно для ввода местоположения. После того, как местоположение будет указано, нажмите кнопку ОК для закрытия этого специального окна. Данные отображаются в поле Местоположение.

Это поле является обязательным для заполнения.

Высота над уровнем моря, м

Вводится высота в метрах над уровнем моря места отлова отдельного переносчика / группы переносчиков заболевания.

Среда обитания

Приводится описание условий и места обитания отловленной группы переносчиков / отдельного переносчика заболевания.

Точки геопозиционирования

Вводятся точки геопозиционирования, использовавшиеся для определения места отлова группы переносчиков / отдельного переносчика заболевания.

Исходные данные для исследования

Вводится информация об обстоятельствах отлова отдельного переносчика или группы переносчиков заболевания: отловлены в плановом порядке или принесены кем-либо при случайном обнаружении.

Ссылка на переносчика-хозяина

Указывается животное-хозяин собранных переносчиков заболевания, если на нём были обнаружены такие переносчики. В раскрывающемся списке содержится перечень отдельных переносчиков заболеваний, отловленных в ходе сессии.

Поле отображается и доступно только для групп переносчиков заболевания.

Учреждение-сборщик

Указывается организация, к которой принадлежит лицо, осуществившее отлов отдельных переносчиков или групп переносчиков заболевания. Это поле является обязательным для заполнения.

Сборщик переносчиков заболеваний

Указывается лицо, осуществлявшее отлов отдельных переносчиков / групп переносчиков заболевания.

Дата отлова

Указывается дата отлова отдельного переносчика или группы переносчиков заболевания. Это поле является обязательным для заполнения.

Время отлова

Указывается время отлова отдельного переносчика или группы переносчиков заболевания: в дневное или ночное время.

Метод отлова

Указывается метод, использовавшийся для отлова отдельного переносчика или группы переносчиков заболевания.

Собраны эктопаразиты

Указывается наличие или отсутствие паразитов на переносчике заболевания.

Данные о переносчике заболевания

Количество

Указывается количество отловленных отдельных переносчиков или переносчиков заболеваний, объединённых в группу. Это поле является обязательным для заполнения.

Биологический вид

Указывается биологический вид переносчик(-ов) заболевания. Это поле является обязательным для заполнения.

Пол

Указывается пол отловленного(-ых) переносчик(-ов) заболевания.

Обнаружившее учреждение

Указывается организация, обнаружившая биологический вид переносчика(-ов) заболевания.

Обнаруживший работник

Указывается лицо, обнаружившее биологический вид переносчика заболевания.

Метод обнаружения

Указывается способ, использовавшийся для обнаружения биологического вида переносчика(-ов) заболевания.

Дата обнаружения

Указывается дата обнаружения биологического вида.

Комментарий

Указывается дополнительная информация, если это необходимо.
