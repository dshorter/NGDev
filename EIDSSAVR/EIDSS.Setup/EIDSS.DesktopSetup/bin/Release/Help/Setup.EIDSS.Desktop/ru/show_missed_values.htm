Показ недостающих значений

Некоторым диаграммам может потребоваться заполнение всех ячеек, и даже тех значений полей, данные которых не зарегистрированы в системе. В этом случае, можно добавлять следующие недостающие значения:

•Недостающие значения в исходных данных для всех полей даты сводной таблицы (дата, год, месяц, квартал, неделя);

Например, требуется отчёт, в котором бы было показано количество случаев заболеваний по каждому месяцу года. При отсутствии отображения недостающих значений, будут показаны только те месяцы, в течение которых проводилась регистрация случаев заболеваний. Поэтому, если для показа в отчёте требуется отображать все месяцы, для поля даты необходимо использование недостающих значений. Добавленная ячейка, содержащая условно-придуманную информацию (служащую исключительно для отображения поля), не затрагивает базу данных и в любой момент может быть удалена.

Недостающие значения добавляются в два шага: вначале, необходимо создать запись-заполнитель в базе данных АВО. После этого поле отмечается в качестве содержащего недостающие значения, но записи в сводной таблице не отображаются. Для их показа в сводной таблице, установите флажок Показать недостающие значения в сводной таблице. Любое изменение сводной таблицы снимает флажок кнопки с фиксацией и, тем самым, скрывает недостающие значения.

Добавление недостающих значений для поля даты

1.Правой кнопкой мыши нажмите на поле, в которое надо внести недостающие значения.

2.Из раскрывающегося списка выберите пункт Добавить недостающие значения. Откроется окно Диапазон дат.

3.Укажите диапазон недостающих дат, которые надо добавить.

4.Нажмите ОК. Окно Диапазон дат закроется. Поле будет выделено красной рамкой, но в таблице ещё не будет видно недостающих значений.

Рис. 413 Выделенное поле

 

5.Включите кнопку с фиксацией Показать недостающие значения в сводной таблице. Значения будут добавлены, а таблица пересчитана.

Удаление недостающих значений

1.Правой кнопкой мыши нажмите на поле, недостающие значения которого необходимо удалить.

2.Из раскрывающегося списка выберите пункт Удалить недостающие значения.
