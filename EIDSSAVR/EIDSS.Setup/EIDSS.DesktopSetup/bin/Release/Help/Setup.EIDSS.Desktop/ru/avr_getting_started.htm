Запуск модуля АВО

Главное меню модуля АВО разработано для получения доступа к основным операциям с запросом и макетами, работы с импортом данных, настройки самого модуля АВО, получения онлайн справки по нему, и т.д. Запуск наиболее часто используемых операций дублируется при помощи кнопок быстрого вызова, расположенных на панели инструментов быстрого доступа. Окно Дерево запросов и макетов содержит запросы и их макеты, сгруппированные по папкам. Тем не менее, по умолчанию там отображаются только заранее заданные сохранённые запросы. Опубликованные запросы и макеты выделяется жирным шрифтом.

Рис. 373 Главное окно модуля АВО

Для открытия модуля АВО, нажмите на пункт Запустить модуль АВО, расположенный в меню АВО. Откроется окно Модуль анализа, визуализации и отчётности. По умолчанию окно Модуль анализа, визуализации и отчётности содержит Главное меню АВО, Панель инструментов быстрого доступа АВО и Дерево запросов и макетов.

Верхний уровень дерева содержит запросы. Самый нижний уровень состоит из макетов. Для удобства макетные формы можно группировать по папкам.

•Чтобы открыть список макетов и папок запроса, нажмите на значок «плюс», расположенный рядом с его названием ( ).

•Для открытия папки, дважды нажмите по нему мышкой.

•Для создания папки размещения выбранного в дереве запроса, нажмите кнопку Новая папка, расположенную на панели инструментов.

•Для удаления, выберите необходимую папку и нажмите кнопку Удалить запрос / папку / макет.

•Для создания макета на основе запроса, выберите запрос и папку, которую хотите добавить, и нажмите кнопку Новый макет, расположенную на панели инструментов. Для получения более подробной информации о создании макета, прочитайте раздел Создание макета.
