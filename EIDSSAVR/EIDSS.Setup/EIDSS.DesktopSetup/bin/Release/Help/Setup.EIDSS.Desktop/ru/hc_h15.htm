Рис. 237 Список агрегированных случаев заболеваний людей 

Раздел «Параметры поиска» можно разделить на 3 группы: по параметрам временного интервала, по административным параметрам и по дополнительным параметрам поиска.

Рис. 238 Раздел «Параметры поиска»

Поле / переменное значение

Описание

Идентификатор случая заболевания

В поле Идентификатор случая заболевания можно указать идентификатор агрегированного случая заболевания. Отображаются лишь те случаи, у которых имеется абсолютное совпадение Идентификаторов случая заболевания.

Единица временного интервала

В поле Единица временного интервала отображаются регуляторы временного интервала, по которым суммируются данные. Поле является обязательным для заполнения, что происходит автоматически, в соответствии с настройками.

Дата начала

В поле Дата начала отображается дата, с которой происходил сбор агрегированных данных. В соответствии с выбранным значением поля Единица временного интервала, вычисляется положение выбранной Даты начала, и к фильтрации записей применяется значение начала соответствующего временного интервала.

Дата окончания

В поле Дата окончания отображается дата, до которой происходил сбор агрегированных данных. В соответствии с выбранным значением поля Единица временного интервала, вычисляется положение выбранного значения в поле Дата окончания, и к фильтрации записей применяется значение окончания соответствующего временного интервала.

Административный уровень

В поле Административный уровень отображаются регуляторы административного фильтра, по которому обобщаются данные. Поле является обязательным для заполнения, что происходит автоматически, в соответствии с настройками.

Область

В поле Область указывается точная область сбора агрегированных данных.

Район

В поле Район указывается точный район сбора агрегированных данных.

Населённый пункт

В поле Населённый пункт указывается точный населённый пункт сбора агрегированных данных.

Раздел «Результаты поиска» используется для выбора агрегированных случаев заболеваний, по которым проводится анализ.

Рис. 239 Раздел «Результаты поиска»

Поле / переменное значение

Описание

Идентификатор случая заболевания

Отображается Идентификатор случая заболевания .

Дата начала

Отображается дата, с которой происходил сбор агрегированных данных.

Временной интервал

Отображается временной интервал случая заболевания, который применялся для создания такого случая.

Область

Отображается область, в которой происходил сбор агрегированных данных.

Район

Отображается район, в котором происходил сбор агрегированных данных.

Населённый пункт

Отображается населённый пункт, в котором происходил сбор агрегированных данных.
