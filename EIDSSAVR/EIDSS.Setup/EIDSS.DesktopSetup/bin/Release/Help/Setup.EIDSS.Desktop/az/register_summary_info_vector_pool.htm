Keçirici/qrup üçün yekun əsas məlumatları qeydiyyata almaq

Yeni bir qrup/fərdi keçirici yaradılsa yaxud mövcud biri redaktə edilmək üçün açılsa Qrup/fərdi Keçirici təfərrüatları W03 Forması açılacaq.

Qrup/fərdi keçirici barədə yekun məlumatları fərqləndirici identifikator, sahə Nömrəsi və qrup/keçiricinin növündən ibarətdir. Sesiya Nömrəsi sahəsi yalnız-oxunandır və keçirici/qrupun toplandığı sessiyanı nümayiş etdirir. Yekun əsas məlumatlar lövhəsinə əlavə edilən keçirici tiplərinin surəti keçirici müşahidə sessiyasının Keçirici Tipi sahəsinə köçürülür. Xanalar barədə ətraflı məlumatlar əldə etmək üçün Qrup/fərdi Keçirici təfərrüatları W03 formasının yekun bölməsini oxuyun.

Əlavə olaraq, bax:

✓Qrup/fərdi Keçirici təfərrüatları W03 formasının yekun bölməsi

✓Keçirici/qrup üçün ümumi məlumatları qeydə almaq

✓Keçiricilər/qrup üçün spesifik məlumatları qeydə almaq

✓Keçirici/qrup üçün nümunələri qeydiyyata almaq

✓Keçirici/qrup üçün sahə testi nəticələrini qeydiyyata almaq

✓Bir keçirici/qrup üçün laboratoriya testlərini və test nəticələrini nəzərdən keçirmək
