Sorğunun filterlərinin redaktə edilməsi

Cədvəl üçün ekstraksiya olunacaq qeydiyyatların sayını məhdudlaşdırmaq üçün filterlərdən istifadə et. Cədvəlin emal olunma sürətini artırmaq məqsədilə filterlərdən istifadə olunması tövsiyyə olunur.

Filter meyarı Birləşdirici operatorlar tərəfindən birləşdirilmiş filter şərtlərindən ibarətdir.

Şəkil 513 Filter

Filter Şərtlərinin Hissələri

Aşağıdakı siyahı təyin oluna bilən filter şərtlərinin hissələrindən ibarətdir:

•Müqayisə üçün cədvəl xanaları;

•Filter müqayisə operatorlar olan Bərabərdir, Bərabər deyildir, Boşdur, Boş deyildir, və s.

•İSTƏNİLƏN və ya KİMİ filter versiyaları və xana dəyərləri, ifadələr, dəyər qrupları və alt sorğular kimi meyar dəyərləri.
