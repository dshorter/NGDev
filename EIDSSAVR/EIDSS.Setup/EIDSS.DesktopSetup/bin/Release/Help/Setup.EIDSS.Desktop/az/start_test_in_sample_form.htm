Nümunə formasında (İF) testi başla

Bu əməliyyat yalnız Laborator Modulunun Sadələşdirilmiş Rejimi seçim-xanası Sistem Seçimləri bölməsində boş olduqda mümkündür.

1.Jurnallar menyusunda, Nümunələr düyməsini bas. Nümunələrin Siyahısı pəncərəsi açılacaq.

2.Testlərin yaradılması üçün nümunəni tap.

3.Aşağıdakılardan birini yerinə yetirin:

•nümunənin üzərində iki dəfə düyməni bas

•onu seç, və sonra Redaktə et düyməsini bas

Nümunə pəncərəsi açılacaq.

4.Testlər alt-pəncərəsini basaraq onu aç. Testlər alt-pəncərəsi nümunəyə təyin olunan testlərin siyahısını göstərir. Testlər alt-pəncərəsi barəsində əlavə məlumatlar əldə etmək üçün, Nümunə L02 formasının Testlər alt-pəncərəsi bölməsinə bax.

5.Status xanasında, Davam edir seç. İndi test başlandı.

6.Yaddaşda saxla düyməsini bas və məlumatların yaddaşda saxlanılmasını təsdiq edin.

Əlavə olaraq, bax:

✓Laboratoriya Testinin Təfərrüatları formasında (İF) testə başla

✓Nümunə L02

✓Nümunələrin Siyahısı L01
