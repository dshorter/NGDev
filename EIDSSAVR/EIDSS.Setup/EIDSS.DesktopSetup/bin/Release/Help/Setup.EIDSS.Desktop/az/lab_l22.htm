Dəst Test Təfərrüatları L22

Dəst Test Təfərrüatları L22 forması dəstə əlavə edilən testlər ilə işləmək üçün istifadə olunur. Bu forma aşağıdakılardan ibarətdir:

•dəst barədə əsas məlumatları əks etdirən xanalar

•dəstə əlavə olunmuş testlər siyahısı üzrə cədvlə

•aidiyyatı nümunələr üzrə testləri axtarmaq üçün yaradılan aşağıdakı 2 xana

•bu Nümunəyə Səciyyəvi Test Parametrləri bölməsi Test Təfərrüatları sərbəst formasıdır

•bu Keyfiyyətə Nəzarət bölməsi Testi Başla sərbəst formasıdır

Sərbəst formanın yaradılması barədə ətraflı məlumatlar əldə etmək üçün, Çevik Formaların Quraşdırılması bölməsinə bax.

Nümunəyə Səciyyəvi Test Parametrləri bölməsi (Test Təfərrüatları sərbəst forması) cədvəldəki hər bir test üçün doldurula bilər. Nümunəyə Səciyyəvi Test Parametrləri bölməsində testin sahələrini göstərmək üçün, cədvəldə testi seç. Keyfiyyətə Nəzarət bölməsi (Testi Başla sərbəst forması) dəstdəki test üçün deyil, yalnız dəst üçün doldurula bilər. Bu sərbəst formaların şablonları dəstdə istifadə olunan testin növü ilə müəyyənləşdirilir.

Şəkil 498 Dəst Test Təfərrüatları forması

Məlumat xanaları

Dəst Test nömrəsi

Dəstin nömrəsi. (yeni) yeni dəstlər ilk dəfə yaddaşda saxlanılanadək onlar üçün istifadə olunur. Bundan sonra dəst nömrəsi nömrələmə sxeminə uyğun olaraq avtomatik təyin edilir və əl ilə dəyişdirilə bilər.

Testin Adı

Dəst üçün icra olunacaq testin adını seç. Bu xananın doldurulması məcburidir və dəstə heç bir test əlavə olunmadıqda mövcud olur. Əgər test artıq siyahıdadırsa, yalnız test üçün mövcud olan testin adı Testin Adı sahəsində göstərilir və dəyişdirilə bilməz.

Bu sahədə seçilmiş testin adı dəstə əlavə edilə biləcək testləri filtrasiya edir.

Testin İcra Edildiyi Yer

defolt üzrə hazırkı təşkilat göstərilir və dəyişdirilə bilməz.

Testi icra etdi

Testi icra edən işçinin adını seç.

Təsdiq etdi

Nəticəni təsdiq edən əməkdaşın adını seç. Bu xananın mövcudluğu Yazmaq icazəsi olan Test Nəticəsini Təsdiq edə Bilər sistem funksiyası ilə müəyyən edilir. Sistemin funksiyaları barədə ətraflı məlumatlar əldə etmək üçün, Sistem Funksiyaları və Sistem Funksiyaları Siyahısı bölməsinə bax.

Testin Başlama tarixi

Dəst testinin başlandığı tarixi daxil et.

Nəticə Tarixi

Dəst testinin nəticəsinin əldə olunduğu tarixi daxil et.

Sütunlar

Ardıcıllıq

Qeydiyyatların nömrələmə qaydası. Nömrələmə sıfırdan başlanır, maksimal nömrə məhdudiyyəti yoxdur.

Nümunənin laboratoriya nömrəsi

Testə aid olan nümunənin laboratoriya nömrəsi.

Diaqnoz

Test üçün diaqnoz.

Nümunə Növü

Testin təyin olunduğu nümunənin növü.

Kateqoriya

Testin kateqoriyası.

Nəticə

Test üçün test nəticəsini seç.

Nəticə daxil etdi

Standart parametr (default) üzrə hazırkı istifadəçi nəticəni daxil edən şəxs kimi istifadə olunur və dəyişdirilə bilməz.

Düymələr

Testi Əlavə et

Dəstə test(ləri) əlavə edir.

Testi Sİl

Dəstdən seçilmiş testləri silir.

Qrup Nəticəsini Əlavə et

Qrup testləri üçün nəticəni əlavə edir.

Cədvəldə seçilmiş qeydiyyatı bir mövqe yuxarı keçirir.

↓

Cədvəldə seçilmiş qeydiyyatı bir mövqe aşağı keçirir.

Dəsti Bağlayın

Dəsti bağlayır və dəstin statusunu dəyişərək Bağlanıb edir. Dəst bağlandıqda, onu yalnız oxumaq olur. Yalnız bütün testlər üçün nəticəsi müəyyən edilmiş dəstlər bağlana bilər.
