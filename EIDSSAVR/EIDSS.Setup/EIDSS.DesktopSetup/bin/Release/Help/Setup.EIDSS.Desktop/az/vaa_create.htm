Aqreqat baytarlıq tədbirini yarat

1.Yarat -> Aqreqat Baytarlıq Tədbiri düymələrinə bas və Yeni Aqreqat Baytarlıq Tədbiri yarat. Aqreqat Baytarlıq Tədbirlərinin Təfərrüatları forması yeni pəncərədə açılacaq.

Şəkil 370 Aqreqat Baytarlıq Tədbirlərin Təfərrüatları

2.Formanın xanalarını doldur. Forma barəsində dolğun məlumatlar əldə etmək üçün Aqreqat Baytarlıq Tədbirlərinin Təfərrüatları V12 bölməsinə bax.

3.Seçim-siyahısından Matrisanın versiyasını seç və ya yenisini yarat.

4.Seçim-siyahısından şablonu seç və ya yenisini yarat.

5.Tədbiri yaddaşda saxlamaq və formanı bağlamaq üçün Ok düyməsini bas.
