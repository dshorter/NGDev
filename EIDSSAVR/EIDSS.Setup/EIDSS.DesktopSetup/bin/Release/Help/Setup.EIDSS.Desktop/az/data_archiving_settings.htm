Məlumatların arxivləşdirilməsini köklə

Məlumatların arxivləşdirilməsini köklə istifadəçini məlumat arxivləşdirməsi cədvəli barədə məlumatlandıran və redaktə edilə bilməyən pəncərədir. Normal olaraq istifadəçi yalnız arxivləşdirilməmiş məlumatlarla işləyir, lakin arxivləşdirilmiş məlumatlar lazım olan halda, istifadəçi Fayl menyusunda Arxiv məlumatlarına bağlanmaq bəndini seçir. Bu seçimdə YXEMS proqramı yalnız-oxumaq rejimində mövcuddur.

Məlumatın arxivləşmə prosesinin kökləmələri pəncərəsinə baxış

Sistem menyusunda Məlumatların arxivləşdirilməsini köklə bəndini klikləyin. Məlumatların arxivləşmə prosesinin kökləmələri pəncərəsi açılacaq.
