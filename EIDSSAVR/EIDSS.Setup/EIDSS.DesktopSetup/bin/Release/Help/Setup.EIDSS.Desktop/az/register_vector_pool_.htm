Keçirici/qrupu qeydiyyata almaq

Əgər bir sessiya haqqında məlumatlar təfsilatı ilə qeydiyyata alınarsa, o zaman fərdi keçirici yaxud qrup ilk olaraq qediyyatdan keçirilməlidir. Fərdi keçirici fərdi şəkildə toplana bilən keçiricidir (məsələn, siçovul, siçan və s.). Bir qruplaşdırma fərdi olaraq toplanmayan keçiricilər dəstidir (məsələn, ağcaqanadlar, milçəklər və s.). Fərdi keçirici kimi yaxud qruplarla toplanacaq keçiricilərin siyahısı əvvəlcədən təyin olunur və nəzərdən keçirilə bilər və/ya Keçirici Tipləri Redaktorunda dəyişdirilə bilər.

 

1.Keçiricilərin Müşahidəsi Sesiyası formasında Yeni düyməsini kliklə. Qrup/fərdi Keçirici təfərrüatları pəncərəsi açılacaq.

2.Qrup/fərdi Keçirici təfərrüatları formasının Yekun bölməsində qrup/fərdi keçirici haqda əsas məlumatlar daxil edin.

3.Qrup/fərdi Keçirici təfərrüatları formasının Ümumi alt-pəncərəsində keçirici/qrup üçün ümumi məlumatları qeydiyyatdan keçirin.

4.Qrup/fərdi Keçirici təfərrüatları formasının Keçirici barədə Spesifik Məlumatlar alt-pəncərəsində keçirici/qrup üçün spesifik məlumatları qeydiyyatdan keçirin.

5.Qrup/fərdi Keçirici təfərrüatları formasının Nümunələr alt-pəncərəsində keçirici/qrup üçün nümunələri qeydiyyatdan keçirin.

6.Qrup/fərdi Keçirici təfərrüatları formasının Sahə Testləri alt-pəncərəsində keçirici/qrup üçün sahə test nəticələrini qeydiyyatdan keçirin.

7.Laboratoriya modulunda sessiya nümunələrinin daxil edilməsi.

8.Nümunələr üçün laboratoriya testləri təyin etmək.

9.Laboratoriya modulunda laboratoriya testlərini icra edin və test nəticələrini qeydiyyatdan keçirin. Testlər Keçiricilərin Müşahidəsi Sesiyasını Laboratoriya Testləri formasında nəzərdən keçirilə bilər..

10.Dəyişiklikləri yadda saxlamaq və Qrup/fərdi Keçirici təfərrüatları pəncərəsin bağlamaq üçün OK düyməsini kliklə.

 

Yenicə yaradılmış keçirici/ qrup Keçiricilərin Müşahidəsi Sesiyası formasında nümayiş olunur. Keçirici / qrup üçün icra edilən testlər ilə təsdiqlənən xəstəliklər sesiyanın Xəstəliklər sahəsinə əlavə edilir (Keçiricilərin Müşahidəsi Sesiyası forması).

Əlavə olaraq, bax:

✓Keçirici müşahidəsi (KM) sessiyası yaratmaq

✓KM sessiyasını redaktə etmək

✓KM yekun məlumatlarını daxil edin

✓ Keçiricilərin Müşahidəsi Sesiyasını Yekunlaşdırmaq (bağlamaq)

✓KM sessiyasını yadda saxlamaq

✓KM sessiyası pəncərəsini bağlamaq
