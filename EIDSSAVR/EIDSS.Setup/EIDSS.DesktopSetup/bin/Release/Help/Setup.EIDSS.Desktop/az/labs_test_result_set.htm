Test nəticəsini daxil et (LB)

1.Jurnallar menyusunda Laboratoriya bölməsini seç. Laboratoriya Bölməsi pəncərəsi açılacaq.

2.Nəticəsi daxil edilməli olan test(ləri) tap. Yalnız Davam edir və ya İlkin statuslarında olan testlərin nəticəsi daxil edilə bilər.

3.Onun/onların nəticəsini aşağıdakı yollardan biri ilə daxil et:

Birinci metod:

a)İstədiyin test(ləri) seç.

b)Onlara sağ düyməni bas və açılan menyuda Test Nəticəsini təyin et düyməsini seç, sonra isə nəticənin özünü seç. Bu prosedura həm fərdi test üçün həm də eyni nəticəyə malik olan bir neçə test üçün istifadə oluna bilər.

İkinci metod:

a) Testlərin Statusu xanasında düyməni bas və açılan siyahıdan İlkin seç.

b) Test Nəticəsi xanasında düyməni bas və açılan siyahıdan nəticəni seç. Bu prosedura yalnız fərdi test üçün istifadə oluna bilər.

Üçüncü metod:

a) Test Nəticəsi xanasında düyməni bas və açılan siyahıdan nəticəni seç. Bu prosedura yalnız fərdi test üçün istifadə oluna bilər. Test Statusu avtomatik İlkin olaraq dəyişdirilir.

Hər hansı bir halda Testin Başlama Tarixi, Nəticə Tarixi, Test Kimin Tərəfindən keçirilib və Nəticə Kimin Tərəfindən Daxil edilib cari tarix ilə avtomatik doldurulur və əl ilə dəyişdirilə bilər.

4.Yaddaşda saxla düyməsini bas və məlumatların yaddaşda saxlanılmasını təsdiq et.
