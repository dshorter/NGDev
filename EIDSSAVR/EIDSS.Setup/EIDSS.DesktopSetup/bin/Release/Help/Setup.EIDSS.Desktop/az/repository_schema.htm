Saxlanılma yerinin Sxemi

Show/Hide Hidden Text

Nümunənin soyuducuya yerləşdirilməsi zamanı nümunənin dəqiq yerini YXEMS-də qeydə alına bilər.

Nümunənin yerini qeyd edilməsindən əvvəl ilk öncə saxlanılma yerinin sxemini müəyyən etmək zəruridir. Bunu etməyin ən yaxşı yolu YXEMS-in məlumatlar bazasında müəssisə saxlanılma yerinin virtual surətini yaratmaqdır.

Saxlanılma yerinin sxeminin yaradılması:

1.Jurnallar menyusunda Nümunələrin saxlanılma yerinin sxemi bəndini klikləyin. Soyuducuların Siyahısı pəncərəsi açılacaq.

Şəkil 48 Saxlanılma Yerinin Sxemi

Pəncərə ilk dəfə açıldığı zaman saxlanılma yerinin sxemi hələ müəyyən olmadığına görə heç bir bənd göstərilmir.

2.Yeni bir soyuducu yaratmaq üçün Yarat düyməsini klikləyin. Soyuducunun sxemi pəncərəsi açılacaq.

3.Soyuducunun sxemi pəncərəsinin sağ bölməsindəki sahələri doldurun və Yaddaşa Sal düyməsini klikləyin. Daxil edilən element əlavə edilir və pəncərənin sol hissəsində nümayiş olunur.

Şəkil 49 Saxlanılma yerinin sxeminin elementi

4.Soldakı çərçivədə soyuducunu seçin və soyuducunun yeni bir alt-şöbəsini yaratmaq üçün Yeni düyməsini klikləyin.

Şəkil 50 Saxlanılma yerinin sxeminin alt-elementi

5.Bölmənin adını doldurun və Bölmənin növünü müəyyən edin. Lazım olarsa, cari alt-şöbədə Yerləşmə yerinin sayı və Bölmənin ştrixkodunu təyin edin. Hər hansı əlavə Qeydi əlavə edin və Yaddaşa sal düyməsini kliklə. Daxil edilən element əlavə olunur və pəncərənin sol bölməsində nümayiş edilir.

Qeyd: Yerləşmə yerlərin sayı cari bölmə üçün nümunələrin mümkün sayını müəyyən edir.

6.Daha çox bölməni(ləri) yaratmaq üçün 4-5-ci addımları təkrarlayın. Daxil edilən bütün elementlər əlavə edilir və pəncərənin sol bölməsində nümayiş olunur.

Bir sxem elementinin surətini çıxarmaq:

1.Siyahıdan lazım olan bəndi seç və Surətini çıxar düyməsini kliklə.

2.Surəti çıxarılan sxem elementinin strukturu orijinalla eyni olacaq. Aktiv olan dialoq qutusunda yeni sxem elementinin atributlarını dəyişdirin.

3.Dəyişiklikləri yadda saxlamaq üçün Yaddaşa sal düyməsini klikləyin.

Bir sxem elementini silmək:

1.Silmək üçün elementi seçin.

2.Bir elementi yaxud bölməni silmək üçün Sil düyməsini kliklə.

Konteynerin tərkibi hesabatını çap etmək:

1.Kağız Formalar düyməsini kliklə və Konteynerin tərkibi seçin. Hesabata baxış hesabatı nümayiş etdirir.

2.Hesabatı çap etmək üçün Printer işarəsini klikləyin.
