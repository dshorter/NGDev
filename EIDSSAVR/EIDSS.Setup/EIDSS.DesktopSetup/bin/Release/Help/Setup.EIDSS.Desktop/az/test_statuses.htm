Test Statusları

Testin statusları Laboratoriya modulu üçün çox əhəmiyyətlidir. O, testin həyat dairəsində testin hazırkı yerini müəyyən edir, bu səbəbədən də o, test ilə və hatta testin təyin olunduğu nümunə ilə aparılması mümkün olan əməliyyatları müəyyən edir. Bundan əlavə, testin statusu testlərin və ya nümunələrin axtarılması üçün istifadə oluna bilər.

Testin statusu avtomatik və yaxud əl ilə vəziyyətdən asılı olaraq dəyişdirilə bilər.

 

Test Statusunun Adı

Təsvir

Başlanmayıb

Test təyin olunub lakin başlanmayıb.

Davam edir

Test təyin olunub və başlanıb, lakin nəticə hələ əldə olunmayıb.

İlkin

Testin nəticəsi əldə olunub, lakin hələ təsdiqlənməyib (yekunlaşmayıb).

Yekun

Testin nəticəsi təsdiqlənib (yekunlaşıb).

Düzəliş olunub

Nəticəyə düzəliş olunub.

Silinib

Bu testin qeydiyyatı silinib. Silinib statusu müvəqqətidir və silinməsi hələ yaddaşda saxlanılmayan testlərə tətbiq olunur. Bundan sonra, test sistemdən daimi pozulur və heç vaxt bərpa, tapıla və ya istifadə oluna bilməz.
