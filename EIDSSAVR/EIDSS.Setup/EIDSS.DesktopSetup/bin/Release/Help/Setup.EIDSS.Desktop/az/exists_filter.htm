'Mövcuddur' filteri

Digər xüsusi sorğu filteri 'Mövcuddur' filteridir. Bu, sistemdə xüsusi uşaq obyektlərinin mövcud olması və ya olmaması faktı əsasında valideyn axtarış obyektlərini və ya eyni növdən olan digər başqa xüsusi uşaq obyektlərinin mövcud olması və ya olmaması faktı əsasında uşaq axtarış obyektlərini filtrasiya etməyə imkan verir. Filterdə istifadə olunan uşaq obyektləri sorğuda axtarış uşaq obyektləri kimi istifadə oluna və ya olunmaya bilər. Bundan əlavə, 'Mövcuddur' filteri Seçilmiş xanalar siyahısında xanaların olmasını tələb etmir.

Məsələn, bəzi hesabatlara yalnız nümunələri olan insan xəstəlik hadisələrinin daxil edilməsi tələb olunur. İnsan xəstəlik hadisəsi Axtarış Obyekti kimi seçilməlidir və <İnsanlar arasında xəstəlik hadisəsi -  Numunələr> onun filtrasiya meyarı olacaqdır.

'Mövcuddur' Filterini yarat

1. Xananın adı düyməsini bas. Seçim-menyusu açılacaq.

Şəkil 526 Tabelikdəki obyektinin seçilməsi

2.Seçim-menyusundan mötərizə ilə işarələnmiş istədiyin tabelikdəki obyektini seç.

3.Tabelikdəki obyekti seçildikdə Mövcuddur müqayisə operatoru avtomatik şəkildə filter şərtlərinə əlavə olunur.

Şəkil 527 'Mövcuddur' filterinin nümunəsi

Bu filter yalnız qeydiyaytdan keçmiş nümunələri olan insan xəstəlik hadisələrini seçməlidir. İndi isə həm 'Mövcuddur' filter qrupuna, həm də onunla eyni səviyyədə olan xanaya əlavə filter meyarını əlavə etmək mümkündür.

'Mövcud deyil' filterini yarat.

Müəyyənləşdirilməmiş tabelikdəki obyektləri olmayan axtarış obyektlərini tapacaq filter yaratmaq üçün Mövcuddur' flteri Yox Və Birləşdirici Operator ilə birlikdə istifadə olunmaldır.

1.'Mövcuddur' filterini yuxarıda təsvir edildiyi kimi yarat.

Şəkil 528 Mövcuddur' Filteri

2.Və Birləşdirici Operator düyməsini bas və seçim-menyuya basaraq Yox Və xanasını seç.

Şəkil 529 'Mövcud deyildir' filterin nümunəsi

Filter yalnız nümunəsi olmayan insan xəstəlik hadisələrini tapıb-seçməlidir.
