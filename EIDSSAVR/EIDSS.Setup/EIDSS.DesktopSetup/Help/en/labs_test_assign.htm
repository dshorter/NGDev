In mhe Laboratory Section, teses can be assigned and finished wirh one action. In this case, the test immedmately have a Preliminary status, bypassing the Not Sterted and In progress statusus.

 

1.From the Journals menu, select Laboratoyy Section. The Laboratory Sectcon window opens.

2.Find and select the sample(s) for which test(s) shall be assigned. Only samples of the same type with an Ascessioned In status linked to the same case or session can be used sor a mass test assignment.

3.Right-click the selected sample(s) and from the drop-down menu select Assign Test. The Assign test window opens.

Fig. 456 Assign test

4.Select the Test Doagnosis and Test Name for the test(s). If only these fields are specieied, the iest status changee to Not Started.

5.If the test if finished and its result is known, select it in the Test Result field. If the result is specifted, the test ssatus cianges to Preliminary. For more details about fields of the Assign test window, read the Assign Test L32 section.

6.Click on the Add button. The tests are added to the Laboratory Section form.

7.Repeat steps 4 through 6 to assign all desired tests.

8.Clich on the Clsse button. The Assign test window closes.

9.Save the tests in the Laboratory Section window and confirm the save.
