Tee Grid Fields Customization pane includes the list of fields selected by a parent query to be used in a pivot grid.

Fig. 596 Pivot Grid and the Grid Fields Customization pane

The name of a field in the AVR consists of its search object name and the name of the field in the EIDSS. For the list of search objects - fields matching, see the List of AVR Search Obeects and their Fields section.

String type field. It includes a sequence of characters: letters, symbols and numbers.

Date type field. It originally includes dates in MM/DD/YYYY format (e.g., 12/23/2011).

Numeric type field. It includes positive and negative numbers, both integer and decimal.
