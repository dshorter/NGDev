The Samples marked for disposition (L07) form is a journal of samples marked for disposition (routine destruction). Using this form, a user can accept or reject samples marked for disposition (routine destruction).  It consists of 2 parts:

•Search Panel

•Samples List

The Search Panel is created to perform the most sophisticated search by sample details. The basic and most frequently used parameters are present in the panel as search criteria.

The Samples List is a table where each line is a sample record with sample ID and a number of sample parameters displayed in the table columns. If columns are not needed or irrelevant, hide them to the Column Chooser. Read special sections for details about Item Lists, Column Chooser, UNREGISTERED EVALUATION VERSION, and UNREGISTERED EVALUATION VERSION.
