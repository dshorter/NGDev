Указание результата теста, проводившегося во внешней лаборатории (ОФ)

В проводящей тест пробы лаборатории может быть (а может и не быть) установлена система ЭИСНЗ. Если она не установлена, результаты тестов должны регистрироваться лицом, отправившим туда пробы. В системе не будет информации о внешних тестах, проведённых в отношении переданных проб, пока их результаты не будут направлены в текущую организацию. Ввиду этого создаётся тест, а в него включается результат внешнего тестирования. Результат тестирования внешним учреждением может вводиться исключительно для проб с назначенными тестами и статусом Передана во внешнюю организацию.

Настоящая операция доступна только в случае отсутствия включённого флажка на кнопке с фиксацией Упрощённый режим лабораторного модуля, расположенной в окне Системные настройки.

 

1.Находясь в меню Журналы, выберите пункт Пробы. Откроется окно Список проб.

2.Найдите пробу, для которой предполагается ввести результат теста внешнего учреждения.

3.Проделайте одно из следующих действий:

•Дважды нажмите мышкой на пробе;

•Отметьте пробу и нажмите кнопку Редактировать.

Откроется окно Проба.

4.Нажмите на вкладке Тесты, чтобы открыть её. На вкладке Тесты содержится список тестов, проведённых в отношении пробы в текущей или внешней лаборатории.

Рис. 319 Вкладка «Тесты»

5.Нажмите кнопку Внешние результаты. В списке тестов создаётся новая строка для внешнего теста.

6.Добавьте информацию, имеющуюся в отношении теста, проведённого во внешнем учреждении. Следующие поля являются обязательными для заполнения:

•Название теста

•Проверяемый тестом диагноз

•Результат

•Дата результата

7.Нажмите кнопку Сохранить и подтвердите сохранение данных.

8.Для закрытия окна Проба, нажмите кнопку ОК.
