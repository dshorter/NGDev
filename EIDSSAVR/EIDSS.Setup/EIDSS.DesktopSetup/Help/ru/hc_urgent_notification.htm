Создание экстренного извещения

Экстренное извещение является упрощённой процедурой создания случая заболевания человека, когда информация заполняется на основании «Бумажной формы экстренного извещения».

1.Находясь в меню Создать, выберите Создать -> Случай заболевания человека. Откроется окно формы Сообщение о случае заболевания человека.

Рис. 170 Форма сообщения о случае заболевания человека

2.Заполните необходимые данные. Для получения полной информации о форме, просьба обратиться к форме Н02 Сообщение о случае заболевания человека.

3.При заполнении данных ЭИСНЗ проверяет базу данных на наличие в ней записи об аналогичном случае заболевания. Если такая запись уже существует, автоматически открывается окно Список случаев заболеваний людей, позволяя выбрать случай для проведения дедупликации.

4.Нажмите кнопку OK. После этого, создание экстренного извещения будет завершено.

Начнётся репликация сохранённого случая заболевания, и он будет передан на сервер ЭИСНЗ.
