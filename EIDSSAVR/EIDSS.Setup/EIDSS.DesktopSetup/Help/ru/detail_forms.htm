Подробные формы

В подробной форме содержатся поля объекта выбранного списка. Обязательные к заполнению поля отмечены красной рамкой; все остальные поля являются либо необязательными для заполнения, либо нередактируемыми.

Для примера, перейдите по пунктам Создать -> Случай заболевания человека. Откроется окно формы «Сообщение о случае заболевания человека».

Рис. 29 Пример подробной формы

Общее описание кнопок:

Кнопка

Описание

Сохранить

Её нажатие сохраняет данные, введённые в форму. Чтобы сохранить форму, необходимо заполнить все обязательные поля. После сохранения форма остаётся открытой.

ОК

Её нажатие сохраняет данные и закрывает форму. Чтобы сохранить форму, необходимо заполнить все обязательные поля.

Отменить

Её нажатие закрывает форму без сохранения данных. Появляется окно подтверждения совершения действия с кнопками Да/Нет. Нажмите Да для закрытия формы.

Новая

Нажмите кнопку для закрытия текущей формы без сохранения данных и открытия аналогичной пустой формы. Появляется окно подтверждения совершения действия с кнопками Да/Нет. Нажмите Да для продолжения.

Удалить

Нажмите кнопку для удаления соответствующего объекта (случай заболевания человека, случай заболевания животного, сессия надзора за переносчиками заболевания, и т.д.) из базы данных. Появляется окно подтверждения совершения действия с кнопками Да/Нет. Нажмите Да для продолжения.

Очистить

Её нажатие очищает данные, введённые в поля формы. Появляется окно подтверждения совершения действия с кнопками Да/Нет. Нажмите Да для продолжения.

Бумажные формы

Её нажатие открывает бумажную форму для печати. Открывается список всех возможных бумажных форм. Выберите необходимую бумажную форму. Для получения более полной информации о бумажных формах, просьба обратиться к разделу Бумажные формы.

Для выбора объекта из раскрывающегося списка необходимо:

1.Нажмите кнопку (), чтобы раскрыть раскрывающийся список.

Рис. 30 Раскрытый раскрывающийся список

2.Выберите из списка объект. Раскрывающийся список закроется и выбранный объект будет отображаться в поле.

Для выбора объекта из списка поиска необходимо:

1.Нажмите кнопку (), чтобы открыть список поиска.

Рис. 31 Открытый список поиска

2.Отметьте в списке объект и нажмите кнопку Выбрать. Список поиска закроется и выбранный объект будет отображаться в поле.

Для добавления объекта в раскрывающийся список необходимо:

1.Нажмите кнопку (), чтобы открыть окно подробной информации об объекте.

Рис. 32 Добавление нового лица в раскрывающийся список

2.Заполните обязательные и факультативные поля (если в их отношении есть какая-либо информация) формы.

3.Нажмите кнопку OK. Новый объект отобразится в поле, а также добавится в раскрывающийся список.

Для редактирования объекта в раскрывающемся списке необходимо:

1.Нажмите кнопку (), чтобы открыть окно подробной информации об объекте.

Рис. 33 Редактирование подробных сведений о человеке

2.При необходимости, отредактируйте информацию. Заполните обязательные и факультативные поля (если в их отношении есть какая-либо информация).

3.Нажмите кнопку OK. Отредактированный объект отобразится в поле, а также обновится в раскрывающемся списке.
