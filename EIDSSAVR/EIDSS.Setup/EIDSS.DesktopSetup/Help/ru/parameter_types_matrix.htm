Редактор типов параметров

Редакторе типов параметров предназначен для создания списка определённых параметров, которые могут использоваться в Конструкторе настраиваемых форм.

Для создания списка типов параметров:

1.Находясь в меню Система, нажмите пункт Настройка, а затем нажмите Редактор типов параметров. Откроется окно Редактор типов параметров.

Рис. 96 Редактор типов параметров

2.В левом панели содержится список параметров (заранее заданные стандартные параметры и уже добавленные специфические параметры). Для редактирования существующего параметра, его необходимо выбрать в левой панели, внести изменения в Текст по умолчанию, Текст на национальном языке и Тип, а в правой панели - исправить список его значений. Для удаления значения параметра, отметьте его в правой панели и нажмите кнопку Удалить , расположенную в правой панели; а чтобы добавить новое значение, нажмите кнопку Добавить , также расположенную в правой панели. Если тип параметра является справочным, корректировка списка значений невозможна. Для изменения порядка расположения значений используйте кнопки Вверх и Вниз.

3.Для добавления нового параметра, нажмите кнопку Добавить , расположенную в левой панели, и введите Текст по умолчанию и Текст на национальном языке; выберите Тип параметра. Если Тип представляет собой справочник, то для определения списка значений выберите Тип справочника из раскрывающегося меню в правой панели; на экране отобразится список выбранных справочных значений. Если Тип представляет собой фиксированные предустановленные значения (что значит, что список значений определяется пользователем), то для добавления нового значения нажмите кнопку Добавить , расположенную в правой панели, и укажите Текст по умолчанию и Текст на национальном языке. Для удаления значения параметра, отметьте его в правой панели и нажмите кнопку Удалить , расположенную в правой панели. Для изменения порядка расположения значений используйте кнопки Вверх и Вниз.

4.Для удаления параметра выберите его в левой панели и нажмите кнопку Удалить .

5.Сохраните данные и закройте окно.
