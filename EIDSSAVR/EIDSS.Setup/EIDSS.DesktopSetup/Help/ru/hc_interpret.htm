Интерпретация результатов тестов

Процедура интерпретации тестов необходима для постановки диагноза пациенту. Все результаты интерпретации тестов должны заноситься в соответствующий раздел формы Сообщение о случае заболевания человека. Пользователь может добавлять информацию об интерпретации результатов теста как во время регистрации случая заболевания, так и после неё, во время редактирования.

Рис. 179 Вкладка «Тесты»

1.Для запуска резюмирования и интерпретации результатов тестов, выберите тест с указанным значением Результат в разделе Тесты, и нажмите кнопку Новое.

Примечание: чтобы кнопка Новая стала активной, необходимо заполнить поле Результат / наблюдение в разделе Тесты.

Примечание: количество интерпретаций не зависит от количества тестов.

2.Заполните таблицу. Для получения полной информации о форме, просьба обратиться к форме Н02 Сообщение о случае заболевания человека.

Примечание: заполнение поля Не подтвердился / подтвердился является обязательным для сохранения данных.

Примечание: после того, как данные будут сохранены, запись с заполненным полем Не подтвердился / подтвердился перестанет быть доступной для редактирования в разделе «Интерпретация и резюмирование результатов».

3.Для сохранения добавленной информации резюмирования и интерпретирования, нажмите Сохранить.

 

Начнётся репликация сохранённого случая заболевания, и он будет передан на сервер ЭИСНЗ.
