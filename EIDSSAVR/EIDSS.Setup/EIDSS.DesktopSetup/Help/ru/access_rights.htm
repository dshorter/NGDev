Права доступа

Разные пользователи наделены в системе разными правами доступа.

Полномочия в системе могут определяться для:

•Отдельных объектов (диагноз или сайт) или класса объектов (все диагнозы или все сайты). Для получения более подробной информации о классах доступа, обратитесь к разделу Управление правами доступа.

•Списка системных функций. Например, доступ к данным о случаях заболеваний людей или животных. Для получения дополнительной информации о списке системных функций и их применении в системе, просьба обратиться к разделу Список системных функций.

 

Полномочия могут назначаться пользователю или группе пользователей. Полномочия группы и пользователя обладают равным приоритетом внутри одного уровня приоритета. В случае возникновения конфликта прав, полномочие Запретить имеет приоритет над полномочием Разрешить в рамках одинакового уровня приоритета.

Для изменения полномочий пользователя необходимо:

Перейдите по пунктам Система -> Администрирование -> Работники; откроется Список работников.

Рис. 42 Список работников

Выполните поиск пользователя, отметьте его/её в списке и нажмите кнопку Редактировать. Находясь в открытой форме Сведения о работнике, откройте вкладку Системные функции. При этом следует помнить, что полномочия пользователя настраиваются отдельно по каждому сайту; необходимый сайт выбирается в поле Сайт. Просмотрите список системных функций и полномочия пользователя в их отношении. Помимо этого, на вкладке Группы можно найти список групп пользователей, в которую включён нужный пользователь.

Для изменения полномочий группы пользователей необходимо:

Перейдите по пунктам Безопасность -> Группы пользователей; откроется Список групп пользователей.

Рис. 43 Список групп пользователей

Выполните поиск группы пользователей, отметьте её в списке и нажмите кнопку Редактировать. Находясь в открытой форме Сведения о группе пользователей, откройте вкладку Системные функции. При этом следует помнить, что полномочия группы пользователей настраиваются отдельно по каждому сайту. В поле Сайт выберите необходимый сайт. Просмотрите список системных функций и полномочия групп пользователей в их отношении.
