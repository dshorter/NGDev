Панель поиска формы Лабораторный раздел представляет собой специальный инструмент, созданный для нахождения лабораторных записей, проб или тестов, используя их параметры в качестве критериев поиска. Для открытия или закрытия панели поиска, нажмите кнопку Панель поиска, расположенную в левой части окна. Закрыть панель поиска также можно нажав на таблицу Лабораторный раздел.

Поля критериев поиска

Статус пробы

Используется для указания статуса необходимой пробы.

Дата статуса пробы: с/по

Используется для указания интервала времени, который начинается со времени последнего изменения статуса пробы.

Локальный / полевой идентификатор пробы

Используется для указания локального или полевого идентификатора необходимой пробы.

Идентификатор случая заболевания / сессии

Используется для указания случая заболевания или сессии, связанных с необходимой пробой.

Направлено в организацию

Используется для указания текущей организации, чтобы увидеть все пробы, предназначенные для обратного получения. После того, как пробы будут найдены, их можно получить / принять в систему. Для получения более подробной информации, прочитайте раздел Приём в систему / получение передававшихся(-ейся) существующих(-ой) проб(-ы), связанных(-ой) со случаем заболевания / сессией (ЛР).

Передана во внешнюю организацию

Используется для указания внешней организации, в которую передавалась соответствующая проба. После того, как пробы будут найдены, их результаты можно зарегистрировать. Для получения более подробной информации, прочитайте раздел Указание результата теста, проводившегося во внешней лаборатории (ЛР).

Направившая результаты организация

Используется для указания внешней организации, которая проводила тест соответствующей переданной ей пробы. Проба будет найдена только в том случае, если результат её внешнего теста был зарегистрирован в системе. Для получения более подробной информации, прочитайте раздел Указание результата теста, проводившегося во внешней лаборатории (ЛР).

Дата приёма в систему: с/по

Используется для указания интервала времени, который начинается со времени приёма пробы в систему.

Лабораторный идентификатор пробы

Используется для указания лабораторного идентификатора соответствующей пробы.

Тип пробы

Используется для указания типа соответствующей пробы из раскрывающегося списка.

Название теста

Используется для указания наименования соответствующего теста в раскрывающемся списке.

Проверяемый тестом диагноз

Используется для указания диагноза случая заболевания или сессии, связанной с соответствующей пробой, выбираемого из раскрывающегося списка. Используются следующие диагнозы:

•Для случая заболевания человека: текущий диагноз или изменённый диагноз;

•Для случая заболевания животного: окончательный или один из предположительных диагнозов;

•Для сессии активного надзора: любой диагноз, связанный с сессией.

Статус теста

Используется для указания статуса соответствующего теста в раскрывающемся списке.

Результат теста

Используется для указания результата соответствующего теста.

Дата статуса теста: с/по

Используется для указания интервала времени, который начинается со времени последнего изменения статуса теста.

Дата результата: с/по

Используется для указания интервала времени, который начинается со времени получения результата теста.

Кнопки

Поиск

Запускает поиск лабораторных записей в соответствии с указанными критериями выбора.

Очистить

Удаляет критерии поиска или возвращает Панель поиска к состоянию по умолчанию.
