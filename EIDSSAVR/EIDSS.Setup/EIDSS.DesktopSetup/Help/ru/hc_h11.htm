Форма Сведения о дедупликации случаев заболеваний людей используется для сравнения двух случаев заболеваний людей.

Рис. 230 Выбор остающегося в базе случая посредством сравнения случаев заболеваний

Остающийся представляет собой случай заболевания, оставляемый в системе; Замещаемый является удаляемым из системы случаем заболевания.

 

Вкладка «Извещение» открывается по умолчанию и используется для рассмотрения основных различий между двумя случаями заболеваний. Все различающиеся между собой поля выделяются красным цветом.

Поле / переменное значение

Описание

Идентификатор случая заболевания

Поле Идентификатор случая заболевания отображает идентификационный номер случая заболевания человека.

Локальный идентификатор

Поле Локальный идентификатор отображает локальный идентификатор случая заболевания.

Диагноз

Поле Диагноз отображает текущий диагноз случая заболевания.

Дата диагноза

Поле Дата диагноза отображает дату первоначального диагноза.

Фамилия

Поле Фамилия отображает фамилию пациента.

Имя

Поле Имя отображает имя пациента.

Отчество

Поле Отчество отображает отчество пациента.

Дата рождения

Поле Дата рождения отображает дату рождения пациента.

Возраст

Поле Возраст отображает возраст поле пациента в единицах измерения возраста.

Пол

Поле Пол отображает пол пациента.

Область

Поле Область отображает область текущего места проживания пациента.

Район

Поле Район отображает район текущего места проживания пациента.

Населённый пункт

Поле Населённый пункт отображает район текущего места проживания пациента.

Улица

Поле Улица отображает название улицы места текущего проживания пациента.

Почтовый индекс

Поле Почтовый индекс отображает почтовый индекс текущего адреса пациента.

Дом/корпус/квартира

Поле Дом/корпус/квартира отображает номер дома, корпуса и квартиры текущего места жительства пациента.

Номер телефона

Поле Номер телефона отображает телефонный номер места текущего проживания пациента.

Гражданство

В поле Гражданство указывается гражданское подданство пациента.

Место работы

В поле Место работы указывается место работы, школа, детский сад, учреждение или любое другое место, которое пациент посещает на регулярной основе, вживую контактируя с большим количеством людей.

Адрес места работы

Поле Адрес места работы отображает адрес организации, указанный в поле выше.

Дата появления симптомов

Поле Дата появления симптомов отображает дату появления первых симптомов заболевания у пациента.

Окончательное состояние пациента

В поле Окончательное состояние пациента указывается ситуация с пациентом: Скончался(-ась) или Жив(-а).

Значение для этого поля берётся из поля Состояние пациента на момент извещения.

Изменённый диагноз

В поле Изменённый диагноз отображается диагноз, который был выбран вместо первоначального диагноза во время процесса расследования случая заболевания.

Дата изменения диагноза

Поле Дата изменения диагноза отображает дату, в которую произошло изменение диагноза.

Тип текущего местонахождения

В поле Тип текущего местонахождения отображается значение поля Текущее местонахождение пациента во время извещения, которое обозначает местонахождение пациента.

Название текущего местонахождения

Поле Название текущего местонахождения отображает наименование или адрес текущего местонахождения пациента в момент извещения.

Дополнительные сведения

В поле Дополнительная информация указывается дополнительная важная информация.

Вкладка «Пробы» используется для просмотра списка проб, взятых в связи со случаями заболеваний. Все пробы имеют рядом с собой кнопки с фиксацией. Если кнопки с фиксацией активированы, это означает, что такие пробы были добавлены / оставлены вместе с оставляемым в базе случаем заболевания. Если проба была принята в систему в Лабораторном модуле, то с кнопки с фиксацией такой пробы невозможно снять флажок.
