Применение фильтров

Хотя в запросе имеются свои фильтры, иногда бывают необходимы и фильтры макета. Поскольку один запрос можно использовать для создания различных макетов, целесообразным является использование в запросе этих общих для всех макетов фильтров; а различающиеся для каждого макета фильтры - используются в своих соответствующих макетах.

Например, необходимо создать несколько отчётов, по отчёту на заболевание. Для этого в модуле АВО есть 2 способа:

Первый способ

1.Создайте запрос со всеми необходимыми данными.

2.Оставьте только один диагноз, остальные скройте при помощи фильтра.

3.Создайте для него макет.

4.Скопируйте запрос и измените фильтр.

5.Создайте для него макет.

6.Повторите шаги 4-5 нужное количество раз.

Второй способ

1.Создайте запрос со всеми необходимыми данными.

2.Создайте для него макет с фильтром, удаляющим неподходящие диагнозы.

3.Скопируйте макет и измените диагноз в его фильтре.

4.Повторите шаг 3 нужное количество раз.

2-й способ требует меньше усилий.

Создание фильтра в макете

Находясь в макете, нажмите пункт Редактировать фильтры. Откроется окно Фильтр.

Применение фильтров

После создания фильтров, активируйте их, что значит включите флажок на кнопке с фиксацией Применить фильтры сводной таблицы, расположенной в окне Информация вкладки Сводная таблица.
