Əgər bəzi pilterlər iərtib cədvəldə istifadə oluaacaq qruplaşoırıcı funksiyaların nəticələrinə tətbiq olunmalıdırsa onları Baxış rejimin alt-səhifədə yarat və tətbsq et. Filterin funksionallığı - əks istiqamətdə yerləşən seçim-xanalarını nişanlamaqla və daha çevir alət olan uygunlaşdırma filterlərini seçməklə cədvəldə görünməsini estədiyiniz məlumatları işarələmək üçün istifadə olunan standart filterlərdən ibarətdir.

Filterin açılan siyahılar xüsusi sütun üçün filter kriteriyalarını müəyyənləşdirmək üçün tərtib olunmuşdur.

Filterlə birlikdə Tərtib cədvəldə qurdşdırılmış spesifik parametrlər Baxış rejimin alt-pəncərəsində dəyişririlə bilməi.

Statdart filter

Standart filterdən istifadə etmək üçün sütun başlığında olan filter işarəyə bas və açılmış siyahısından istədiyin bəndi seç.

Şəkil 562 Sütun filteri

Fdlteri uyğunlaşdır

Daha tənmil filter tələb olunduğu hilda Istifadəçi tərəfindən köklənilən filteri istifadə et.

1.Sütun başlığında filter işarəyə bas və aşılmış siyabısından Istifadəçi tərəfindən köklənilən filter xanasını seç. Intifadəçi tərəfindən köklənilən filterin pəncərəsi açılacaq.

2.2 axtarış keyarıla qədər istifadəçi tərəfindən köklənilən filteri qur.

3.Bəli düyməsini bas.
