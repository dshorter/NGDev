Вкладка Краткая информация используется для упрощённой регистрации информации и материалов, собранных в ходе сессии. Данные вводятся в таблицу.

 

Рис. 266 Вкладка «Краткая информация»

Столбцы

Идентификатор записи

Идентификатор группы переносчиков / отдельного переносчика заболевания (новый) используется для данных о новом и ранее не сохранявшемся отдельном переносчике / группе переносчиков заболевания. После первого сохранения отдельного переносчика / группы переносчиков заболевания, вместо (нового) идентификатора, подставляется идентификатор, сгенерированный в соответствии со Схемой уникальной нумерации.

Местоположение

Выберите или введите точное местоположение отлова группы переносчиков / отдельного переносчика заболевания. Нажмите мышкой на поле и откроется специальное окно для ввода местоположения. После того, как местоположение будет указано, нажмите кнопку ОК для закрытия этого специального окна. Данные отобразятся в поле Местоположение.

Это поле является обязательным для заполнения.

Дата отлова

Отображается дата отлова отдельного переносчика или группы переносчиков заболевания.

Тип переносчика заболевания

Указывается тип отловленного отдельного переносчика / группы переносчиков заболевания. Это поле является обязательным для заполнения.

Биологический вид

Указывается биологический вид переносчик(-ов) заболевания. Это поле является обязательным для заполнения.

Пол

Указывается пол отловленного(-ых) переносчик(-ов) заболевания.

Количество отловленных групп переносчиков / отдельных переносчиков заболевания

Вводится количество отловленных отдельных переносчиков / групп переносчиков заболевания с выбранными параметрами.

Заболевание (количество групп переносчиков / отдельных переносчиков заболевания с положительной реакцией)

Добавляются подтверждённые тестами заболевания.

1.Нажмите кнопку «стрелка вниз» (). Откроется дополнительное окно.

2.Для добавления нового заболевания, нажмите на пустую строку с сообщением Нажать для добавления новой строки. Будет добавлена новая пустая строка.

3.Укажите подтверждённое заболевание и количество групп переносчиков / отдельных переносчиков заболевания, пробы которых дали положительную реакцию. Заболевание может быть введено всего лишь раз.

Если заболевание добавлено по ошибке, отметьте его и в дополнительном окне нажмите кнопку Удалить. Заболевания будет удалено из дополнительного окна, из столбца Заболевание (количество групп переносчиков / отдельных переносчиков заболевания с положительной реакцией) на вкладке Краткая информация, а также из поля Заболевания такой сессии.

Кнопки

Удалить

Используется для удаления выделенной(-ых) записи(-ей):

1.Выберите запись(-си) для удаления.

2.Нажмите кнопку Удалить. Откроется окно с запросом на подтверждение действия.

3.Нажмите Да для подтверждения удаления и закройте окно подтверждения.
