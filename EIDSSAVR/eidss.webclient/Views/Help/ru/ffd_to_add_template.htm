Конструктор настраиваемых форм - добавление нового шаблона

Данные основного шаблона:

1.В окне Конструктор настраиваемых форм нажмите мышью на вкладке Редактор шаблонов для его открытия.

2.Выберите необходимый Тип шаблона, а затем Добавить шаблон, используя кнопку ( ). В список добавится новый шаблон; при этом окно предварительного просмотра будет пустым.

Рис. 103 Создание нового шаблона

3.Укажите для шаблона Название по умолчанию (на английском) и Национальное название (на местном языке), а также Примечание.

4.Поставьте галочку на кнопке с фиксацией Является ли универсальным шаблоном?, если собираетесь использовать такой шаблон в качестве универсального; эта опция включена по умолчанию для шаблонов данного типа и страны (используется в том случае, если никаких конкретных шаблонов не определено).

Разделы и параметры:

1.В левой панели выберите шаблон, который необходимо отредактировать. Нажмите расположенную внизу на левой панели кнопку Параметры, чтобы открыть список параметров, которые можно использовать для текущего шаблона. Нажмите , чтобы раскрыть список существующих шаблонов этого типа, а затем вновь нажмите напротив шаблона для редактирования, чтобы раскрыть список доступных параметров. Если список не содержит нужный параметр, добавьте его..

Рис. 104 Нажмите кнопку «Параметры»

2.Для добавления раздела / параметра, отметьте его и нажмите в окне предварительного просмотра шаблона для его добавления. Если параметр относится к разделу, такой параметр тоже добавится в панель.

Рис. 105 Добавление параметра

3.Для передвижения и/или изменения размера параметра используйте курсор мыши.

4.Измените значение Схема параметра (укажите месторасположение его названия), и укажите, является ли параметр обычным или обязательным; это указывается в списке Режим редактирования. При необходимости, измените значения Ширина, Высота и Размер надписи.

5.Для добавления всех необходимых параметров, повторите шаги с 2-го по 4-й.

6.Для добавления надписей, нажмите кнопку Надпись, а затем нажмите на желаемом местоположении надписи.

7.Для добавления линейных диаграмм, нажмите кнопку Линейная диаграмма, а затем нажмите на желаемом местоположении линейной диаграммы в поле предварительного просмотра.

8.Для удаления параметра/раздела, выберите его и нажмите кнопку Удалить объект.

Правила для параметров:

1.Для определения правил параметров шаблона, выберите необходимый шаблон в окне Редактор шаблонов. Нажмите кнопку Правила, расположенную в нижней части левой панели. Откроется пустая панель Правила. В правой панели выберите параметр, для которого будет применяться правило.

Рис. 106 Правила

2.Нажмите кнопку Добавить правило. В список добавится новое правило.

3.Правой кнопкой мыши нажмите на поле, для которого создаётся такое правило. В раскрывающемся меню, выводимому при нажатии правой кнопки мыши, выберите необходимое правило. Если указан параметр в поле Значение, в нижней панели отображается дополнительное поле Константы.

Рис. 107 Правило для одного параметра

4.Укажите следующее:        

i.Нажмите Действия. В открывшемся окне из списка параметров выбранного шаблона выберите параметр, к которому будет применено действие после изменения параметра, для которого создаётся такое правило, а в столбце Действия, для выбранного параметра, активируйте флажок на кнопках Очистить и/или Отключено.

Пример: параметр B должен стать нередактируемым, если указано значение для параметра А. Откройте окно Действия, выберите в списке параметр В, а в столбце Действия активируйте флажок Отключено.

ii.В поле Точка проверки укажите процедуру применения правила: «При загрузке», «При изменении значения», «При сохранении» (с уведомлением), или «При сохранении» (с ошибкой).

iii.В поле Константы укажите то/те значения(-я) параметра, к которому будет применяться правило. Нажмите на раскрывающемся списке и в открывшемся окне нажмите Добавить константу, чтобы добавить в список значение; выберите его из раскрывающегося списка.

iv.В поле Название по умолчанию укажите название правила на английском языке.

v.В поле Национальное название укажите название правила на местном языке.

vi.В поле Текст сообщения укажите текст сообщения на английском языке (если это необходимо), который будет отображаться в случае отсутствия проверки правила.

vii.В поле Текст сообщения на национальном языке введите текст сообщения на местном языке (если это необходимо), который будет отображаться в случае отсутствия проверки правила.

viii.Поставьте флажок в поле Нет, если функция правила (Является значением или Является пустым) будет противоположной (Не является значением или Не является пустым).

Определители для параметров:

1.Для определения определителей параметров шаблона, выберите необходимый шаблон в окне Редактор шаблонов. Нажмите находящуюся внизу правой панели кнопку Добавить определитель.Откроется окно Редактор определителей. В левой панели содержится список типов определителей, доступных для выбранного типа шаблона.

2.В левой панели укажите тип определителя. Разверните список определителей значений нажав на , выберите значение добавляемого определителя, и нажмите ОК. Либо в находящееся в правой панели поле Критерии введите название добавляемого определителя, нажмите Поиск, и, после нахождения значения, нажмите ОК. Значение добавится в список определителей.

3.Для удаления определителя выберите его в списке определителей и нажмите Удалить определитель.
