Как правило, эта функция используется в двух различных ситуациях:

•Использование одного и того же поля / переменной с различным группированием даты в одной и той же таблице;

•Использование одного и того же поля / переменной с различными функциями агрегирования в одной и той же таблице.

Если в отчёте требуется отображать одно и то же поле с различным группированием, напр., по годам и месяцам в одной и той же таблице, поле даты необходимо использовать дважды: в первом случае оно должно группироваться по годам, а во втором - по месяцам. Для создания дубликата поля, необходимо использовать функцию Копировать. Для отображения месяцев, необходимо изменить индивидуальное группирование скопированного поля путём нажатия правой кнопки мыши на таком поле и выбрав Группировка по дате = месяц из раскрывающегося списка меню.

Рис. 417 Использование копий полей

 

Если в отчёте требуется отображать поле с другими функциями агрегирования в одной и той же таблице, напр., конкретное число заявленных случаев заболеваний (функция агрегирования Подсчёт) и коэффициенты заболеваемости (Функции агрегирования Количество (Количество на 10.000, Количество на 100.000, и т.д.). Копируются данные поля «Случай заболевания человека - идентификатор случая заболевания». Для создания дубликата поля, необходимо воспользоваться функцией «Копировать». Функцию агрегирования одного из столбцов необходимо поменять на Количество на 100.000 (или другое процентное соотношение) при помощи кнопки Функция агрегирования.

Копии и оригинал поля являются взаимозаменяемыми; их можно переименовывать и убирать (удалять) из сводной таблицы. При её удалении из сводной таблицы, такая копия перемещается в окно Настройка полей таблицы и хранится там для последующего использования.

Копирование поля

1.Правой кнопкой мыши нажмите на поле, которое необходимо скопировать в окно Настройка полей таблицы или в сводную таблицу.

2.Из раскрывающегося меню выберите Копировать.

Копия поля появится рядом с ним.

Удаление копии поля

1.Правой кнопкой мыши нажмите на удаляемом оригинале или удаляемой копии.

2.Из раскрывающегося меню выберите Удалить копию.

Копия будет удалена.
