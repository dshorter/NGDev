Настройки архивирования данных

Настройки архивирования данных представляет собой окно с нередактируемых данных, информирующее пользователя о графике архивирования данных. Как правило, пользователь работает только с неархивированными данными, но в случае необходимости в архивированных данных пользователь может активировать опцию «Подключиться к архиву», расположенную в меню «Файл». Активация этой опции переводит программу ЭИСНЗ в режим только для чтения.

Для просмотра настроек архивирования данных необходимо:

Находясь в меню Система, нажмите мышкой Настройки архивирования данных.Откроется окно Настройки архивирования данных.
