Форма «Редактор диаграмм»

Окно Редактор диаграмм включает в себя панель Структурная диаграмма и панель Настройка полей таблицы. В панели Структурная диаграмма содержится диаграмма, занимающая панель Вид представления данных, и поле Тип диаграммы, предназначенное для изменения типа диаграммы. Панель Настройка полей таблицы предназначено для изменения вида представления данных осей и диаграммы.

Рис. 462 Редактор диаграмм

Для настройки его вида, нажмите мышкой на оси. Панель Настройка полей таблицы поменяется. Трёхмерные диаграммы не имеют возможности настройки.

Рис. 463 Настройка оси диаграммы

 

Цвет линии

Цвет линии оси.

Ширина линии

Ширина линии оси.

Цвет сетки

Цвет дополнительных линий сетки. Это полезно в случаях, когда линии сетки являются видимыми (активирована кнопка с фиксацией Отображение линий сетки).

Стили линий сетки

Стиль дополнительных линий сетки (сплошной, штриховой, точечный, штрих-точечный или штриховой с двумя точками). Это полезно в случаях, когда линии сетки являются видимыми (активирована кнопка с фиксацией Отображение линий сетки).

Отображение линий сетки

Регулирует отображение линий сетки:

ВКЛ. - линии сетки диаграммы видны.

ВЫКЛ. - линии сетки диаграммы скрыты.

Значение угла наклона надписей

Угол наклона надписей на оси.

Значение шрифта надписей

Указывается шрифт надписей на оси.

Значение стиля размещения надписей

Служит для изменения порядка размещения надписей на оси.

ВКЛ. - значения располагаются в шахматном порядке.

ВЫКЛ. - значения располагаются в линейном порядке.

Текст

Указывается название оси. Это полезно в случаях, когда название является видимым (активирована кнопка с фиксацией Отображение).

Шрифт

Позволяет изменять шрифт названия оси. Это полезно в случаях, когда название является видимым (активирована кнопка с фиксацией Отображение).

Выравнивание

Выравнивает название оси Это полезно в случаях, когда название является видимым (активирована кнопка с фиксацией Отображение).

Отображение

Позволяет регулировать отображение оси.

ВКЛ. - название оси является видимым.

ВЫКЛ. - название оси является невидимым.

Подсчёт второстепенных отметок времени

Указывается количество отметок времени, не имеющих названий, расположенных между пронумерованными.

Выворотка

Зеркальное отображение диаграммы.

Автоматический диапазон

Указываются минимальный и максимальный концы диапазона. Следует помнить, что указанный вручную диапазон сохраняется и, может не соответствовать изменённой диаграмме.

ВКЛ. - диапазон устанавливается автоматически.

ВЫКЛ. - диапазон устанавливается вручную.

Максимальный диапазон

Указывается максимальное значение диапазона. Включение возможно только при отключённой кнопке с фиксацией Автоматический диапазон.

Минимальный диапазон

Указывается минимальное значение диапазона. Включение возможно только при отключённой кнопке с фиксацией Автоматический диапазон.

Позиционное расположение

Указывается расположение оси Y (слева или справа). К оси Х не применяется.
