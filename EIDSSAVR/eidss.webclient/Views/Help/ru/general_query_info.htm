Окно Общая информация запроса содержит общую информацию об открытом запросе (его наименование и описание) и кнопку с фиксацией Добавить все ключевые справочные значения, используемую для создания макетов тех административных единиц и диагнозов, которые не имеют записей в базе данных ЭИСНЗ.

Рис. 453 Название и описание запроса

Имя запроса

Имя запроса указывается на английском и местном языке (в зависимости от обстоятельств). Это поле является обязательным для заполнения. Имя запроса используется для идентификации запроса в окне «Дерево запросов / папок / макетов».

Описание

Описание запроса не является обязательным для заполнения полем, и может использоваться для указания подробных сведений о запросе: некоторые специальные фильтры, объекты поиска, и т.д.

Добавить все ключевые справочные значения

Представляет собой кнопку с фиксацией, созданную для добавления записей-заполнителей для недостающих значений диагнозов и административных единиц. Для получения дополнительной информации, прочитайте раздел Добавление ключевых значений.
