Добавление столбца агрегированных данных

Если в сводной таблице используется агрегирование данных, то первоначальные данные заменяются результирующими. Иногда, впрочем, в одной и той же таблице требуется присутствие как первоначальных, так и результирующих данных. В этом случае, на вкладке Вид представления данных требуется использование столбца агрегированных данных.

Создание агрегированного столбца

1.Используется для выбора столбца / группы столбцов, к которому(-ой) применяется функция агрегирования.

2.Правой кнопкой мыши нажмите на заголовок столбца / группы столбцов и из раскрывающегося меню выберите Добавить столбец агрегированных данных. Справа от столбца / группы столбцов добавится новый столбец с именем Столбец агрегированных данных.

3.Новый столбец можно переименовать; для этого нажмите правой кнопкой мыши на заголовке столбца агрегированных данных и выберите пункт Переименовать столбец.

4.Новый столбец ещё не содержит функций. Для добавления функции в столбец агрегированных данных выберите столбец и функцию в поле Функция, расположенное в верхней части вкладки.

5.При необходимости, укажите Параметры для функции агрегирования. Столбец агрегированных данных невозможно использовать в качестве параметра настройки другого столбца агрегированных данных.

6.Укажите параметр Точность для столбца агрегированных данных.

Точность

-1

Результат не округляется в большую сторону.

0

Результат округляется в большую сторону до целочисленного значения.

>0

1, 2, 3, и т.д. Результат округляется в большую сторону до указанного знака.

Функции агрегирования, доступные в таблице представления данных

Суммарный процент

Представляет собой сумму всех ячеек, расположенных над рассматриваемой ячейкой, делённую на итоговое значение столбца и умноженную на 100%. Для применения функции необходимо выбрать исходный столбец (в поле Параметры: по столбцу). Функцию можно применять исключительно к числовым / целочисленным столбцам.

Максимальное значение строки

Представляет собой максимальное значение всех ячеек одной и той же строки в группе столбцов. Функцию можно применять исключительно к числовым / целочисленным столбцам.

Минимальное значение строки

Представляет собой минимальное значение всех ячеек одной и той же строки в группе столбцов. Функцию можно применять исключительно к числовым / целочисленным столбцам.

Процент столбца

Представляет собой значение ячейки, делённое на итоговое значение столбца и умноженное на 100%. Функцию можно применять исключительно к числовым / целочисленным столбцам. Для применения функции необходимо выбрать ячейку исходного столбца (в поле Параметры: по столбцу).

Процент строки

Представляет собой значение ячейки, делённое на итоговое значение строки и умноженное на 100%. Функцию можно применять исключительно к числовым / целочисленным столбцам. Для применения функции необходимо выбрать ячейку исходного столбца (в поле Параметры: по столбцу).

Пропорция

Представляет собой значение ячейки, делённое на значение той же строки другого выбранного столбца (Пропорция = числитель/знаменатель). Функцию можно применять исключительно к числовым / целочисленным столбцам. Для применения функции необходимо выбрать два исходных столбца (в полях Параметры: числитель и знаменатель).

Соотношение

Представляет собой значение ячейки, делённое на значение той же строки другого выбранного столбца (Соотношение = числитель/знаменатель). Функцию можно применять исключительно к числовым / целочисленным столбцам. Для применения функции необходимо выбрать два исходных столбца (в полях Параметры: числитель и знаменатель).

Она отличается от функции Пропорция: если и числитель и знаменатель являются целыми числами, результаты отображаются в формате X:Y и уменьшаются до однозначного числа, напр.: 30:60 отображаются как 1:2.
