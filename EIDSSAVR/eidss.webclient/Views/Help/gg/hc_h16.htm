ადამიანის დაავადების აგრეგირებული შემთხვევების დედუპლიკაციის ფორმა გამოიყენება, როცა საჭიროა შემთხვევის დედუპლიკაცია.

ნახ. 589 "ადამიანის დაავადების აგრეგირებული შემთხვევების დედუპლიკაციის" ფორმა

ველი/ცვლადი

აღწერა

შემთხვევის ID

ეკრანზე ჩანს შემთხვევის ID

დასაწყისის თარიღი

დაწყების თარიღი დამოკიდებულია შემთხვევის დროის შუალედზე. ეკრანზე ჩანს დროის შუალედის დაწყება.

დროის შუალედი

ეკრანზე ჩანს შემთხვევის დროის შუალედი ("წელი", "კვარტალი", "თვე", "კვირა" ან "დღე").

რეგიონი

ეკრანზე ჩანს შემთხვევის "რეგიონი".

რაიონი

ეკრანზე ჩანს შემთხვევის "რაიონი".

დასახლება

ეკრანზე ჩანს შემთხვევის "დასახლება".

ორგანიზაცია

ეკრანზე ჩანს ის ორგანიზაცია, რომელმაც შექმნა შემთხვევა მონაცემთა ბაზაში (ორგანიზაციის აბრევიატურა).

თარიღი

ეკრანზე ჩანს შეყვანის თარიღი

მატრიცის ვერსია

ეკრანზე ჩანს "მატრიცის ვერსიის დასახელება".

შაბლონი

ეკრანზე ჩანს "შაბლონის დასახელება".

ღილაკები

დუბლირებული შემთხვევების ძებნა

ღილაკზე დაწკაპუნებით დაიწყება დუბლირებული შემთხვევების ალგორითმის ძებნა. ნაპოვნი დუბლირებული შემთხვევების ჩამონათვალი განახლებულია. თუ დუბლირებული შემთხვევები ვერ მოიძებნა, ეკრანზე გამოჩნდება შემდეგი შეტყობინება: "დუბლირებული შემთხვევები ვერ მოიძებნა".

რედაქტირება

დააწკაპუნეთ ღილაკზე, გახსენით ადამიანის დაავადების აგრეგირებული შემთხვევის დეტალების (H06) ფორმა და შეასწორეთ შემთხვევა.

წაშლა

შერჩეული შემთხვევის წასაშლელად დააწკაპუნეთ ღილაკზე.
