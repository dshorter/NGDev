ლაბორატორიული ტესტების ჩამონათვალის (L03) ფორმა შექმნილია ყველა იმ რეგისტრირებული ლაბორატორიული ტესტების საჩვენებლად, რომლებიც აკმაყოფილებს ძებნის კრიტერიუმებს. იგი შედგება 2 ნაწილისაგან.

•ძებნის პანელი

• ტესტების ჩამონათვალი

"ძებნის პანელი" შექმნილია იმისათვის, რომ ჩატარდეს ყველაზე სრულყოფილი ძებნა ტესტის დეტალების მიხედვით. ამ პანელში ძებნის ველების სახით მოცემულია ძირითადი და ყველაზე ხშირად გამოყენებული პარამეტრები.

ტესტების ჩამონათვალი არის ცხრილი, სადაც თითოეული სტრიქონი წარმოადგენს ტესტის ჩანაწერს, რომლის ლაბორატორიული ტესტის ID და ტესტის რამდენიმე პარამეტრი მოყვანილია ცხრილის სვეტებში. თუ სვეტები არ არის საჭირო ან არ არის შესაფერისი, დამალეთ ისინი "სვეტის შემრჩევის" ფუნქციის გამოყენებით. მეტი ინფორმაციის მისაღებად წაიკითხეთ ქვემოთ მოყვანილი; პუნქტების ჩამონათვალი, სვეტის შემრჩევი, ძებნის ჩამონათვალი და ჩამონათვალის და ძებნის ფორმების ინდივიდუალიზაცია.

ნახ. 281 "ლაბორატორიული ტესტების ჩამონათვალის" ფანჯარა

ძებნის კრიტერიუმების ველები

ტესტის სახელი

ჩამოსაშლელ სიაში შეარჩიეთ სასურველი ტესტის დასახელება.

პარტიის საიდენტიფიკაციო ნომერი

შეიყვანეთ იმ პარტიის უნიკალური ID, რომელსაც კონკრეტული ტესტი მიეკუთვნება.

სტატუსი

შეარჩიეთ სასურველი ტესტის სტატუსი ჩამოსაშლელი მენიუდან. ტესტის სტატუსების შესახებ მეტი ინფორმაციის მისაღებად, წაიკითხეთ სექცია ტესტის სტატუსები.

შედეგის თარიღი

შეიყვანეთ დროის შუალედი, როცა ტესტის შედეგი შეიყვანეს, დაადასტურეს ან შეასწორეს ბოლოს.

შემთხვევის საიდენტიფიკაციო კოდი / სესიის საიდენტიფიკაიცო კოდი

შეიყვანეთ დაავადების ის შემთხვევა ან სესია, რომელიც დაკავშირებულია ტესტების ნიმუშთან.

ტესტი დიაგნოზი

შეარჩიეთ დაავადების შემთხვევის ან სესიის ის დიაგნოზი, რომელიც დაკავშირებულია ტესტის ნიმუშთან.

პაციენტი

შეარჩიეთ პაციენტი, ვისთვისაც შეგყავთ დაავადების შემთხვევა.

ფერმის მფლობელი

ვეტდაავადების შემთხვევიდან შეარჩიეთ ნიმუშთან დაკავშირებული ფერმის მეპატრონე.

ცხოველის ID/სახეობის ID/ვექტორის ID

ფრინველის დაავადების შემთხვევაზე მიბმული ნიმუშისთვის დანიშნული ტესტი, შეიყვანეთ სახეობის ID.

შინაური საქონლის დაავადების შემთხვევაზე მიბმული ნიმუშისთვის დანიშნული ტესტი, შეიყვანეთ ცხოველის ID.

ვექტორული ზედამხედველობის სესიისთვის დანიშნული ტესტი, შეიყვანეთ ვექტორის ID.

ადამიანის დაავადების შემთხვევაზე მიბმული ნიმუშისთვის დანიშნული ტესტი, ეს ველი არ გამოიყენება.

აქტიური ზედამხედველობის სესიაზე მიბმული ნიმუშისთვის დანიშნული ტესტი, შეიყვანეთ სახეობის ან ცხოველის ID.

ლაბორატორიული ნიმუშის საიდენტიფიკაციო კოდი

შეიყვანეთ სასურველ ტესტთან დაკავშირებული ნიმუშის ლაბორატორიული ID.

ლოკალური/საველე ნიმუშის ID

შეიყვანეთ სასურველ ტესტთან დაკავშირებული ნიმუშის ადგილობრივი ან საველე ID.

მიღების თარიღი -დან/-მდე

მიუთითეთ დროის შუალედი, როცა მიღებული იყო სასურველ ტესტთან დაკავშირებული ნიმუში.

ნიმუშის ტიპი

მიუთითეთ სასურველ ტესტთან დაკავშირებული ნიმუშის ტიპი.

შედეგების ჩამონათვალის სვეტები

პარტიის საიდენტიფიკაციო ნომერი

იმ პარტიის ნომერი, რომელსაც კონკრეტული ტესტი მიეკუთვნება.

ტესტის სახელი

ტესტის დასახელება.

ტესტის კატეგორია.

სტატუსი

ტესტის მიმდინარე სტატუსი. ტესტის სტატუსების შესახებ მეტი ინფორმაციის მისაღებად, წაიკითხეთ სექცია ტესტის სტატუსები.

ტესტის შედეგი

ტესტის შედეგი.

შედეგის თარიღი

ტესტის შედეგის შეყვანის, დადასტურების ან შესწორების თარიღი.

ლაბორატორიული ნიმუშის საიდენტიფიკაციო კოდი

ამ ტესტთან დაკავშირებული ნიმუშის ლაბორატორიული ID.

ნიმუშის ტიპი

ამ ტესტთან დაკავშირებული ნიმუშის ტიპი.

ლოკალური/საველე ნიმუშის ID

ამ ტესტთან დაკავშირებული ნიმუშის ადგილობრივი ან საველე ID.

მიღების თარიღი

თარიღი, როცა მიღებული იყო ამ ტესტთან დაკავშირებული ნიმუში.

ფუნქცონალური არე

ფუნქციონალური ზონა (ორგანიზაციის განყოფილება), სადაც ინახება და გამოიყენება ამ ტესტთან დაკავშირებული ნიმუში.

შემთხვევის საიდენტიფიკაციო კოდი / სესიის საიდენტიფიკაიცო კოდი

ტესტის ნიმუშთან დაკავშირებული დაავადების შემთხვევის ან სესიის ნომერი.

ტესტი დიაგნოზი

ტესტის ნიმუშთან დაკავშირებული დაავადების შემთხვევის ან სესიის დიაგნოზი.

პაციენტი/ფერმის მეპატრონე

ტესტის ნიმუშთან დაკავშირებული ადამიანის დაავადების შემთხვევის ან ვეტდაავადების შემთხვევის ფერმის მეპატრონის სახელი.

ცხოველის ID/სახეობის ID

ფრინველის დაავადების შემთხვევაზე მიბმული ნიმუშის სახეობის ID.

შინაური საქონლის დაავადების შემთხვევაზე მიბმული ნიმუშის ცხოველის ID.

შეუძლებელია ადამიანის დაავადების შემთხვევის მიმართ გამოყენება, ეს ველი ცარიელია.

ღილაკები

შექმენით პარტია

ქმნის პარტიას.

მონიშნეთ ერთიდაიგივე ტესტის ტიპის მქონე ის ტესტები, რომლებიც არ არის პარტიაში და დააწკაპუნეთ ღილაკზე პარტიის შექმნა. გაიხსნება ჯგუფური ტესტის დეტალების L22 ფორმა შერჩეული ტესტებით.

ეს ღილაკი აქტიურია, თუ ნებადართული იქნება "ჩაწერის" ოპერაცია "ლაბორატორიულ ტესტებზე დაშვების" სისტემურ ფუნქციაზე.

რედაქტირება

ახდენს შერჩეული ტესტის რედაქტირებას. ეს ღილაკი გამოჩნდება, თუ ნებადართული იქნება "ჩაწერის" ოპერაცია "ლაბორატორიულ ტესტებზე დაშვების" სისტემურ ფუნქციაზე.

წაშლა

შლის მონიშნულ ტესტ(ებ)ს. ცხრილში შეარჩიეთ წასაშლელი ტესტ(ებ)ი და დააწკაპუნეთ წაშლის ღილაკზე.

დასრულებული ტესტების წაშლა შეუძლებელია. თუ შეუძლებელი იქნება ზოგიერთი შერჩეული ტესტის წაშლა, ეკრანზე გამოჩნდება შესაბამისი გაფრთხილება და ეს ტესტები არ წაიშლება; ყველა სხვა ტესტი წაიშლება.

ეს ღილაკი გამოჩნდება, თუ ნებადართული იქნება "წაშლის" ოპერაცია "ლაბორატორიულ ტესტებზე დაშვების" სისტემურ ფუნქციაზე.
