ვეტერინარული აგრეგირებული მოქმედების ჩამონათვალი V09

ნახ. 97 ვეტერინარული აგრეგირებული მოქმედებების ჩამონათვალის ფანჯარა

ძებნის პანელი განყოფილება; მოიცავს სხვადასხვა ველებს, რომლებიც გამოიყენება მოქმედების ძებნისთვის.

ნახ. 98 ძებნის მენიუ

ველი/ცვლადი

აღწერა

შემთხვევის ID

ამ ველში შეიძლება მიუთითოთ "შემთხვევის ID".

დროის შუალედის ერთეული

ველში დროის ინტერვალის ერთეული აისახება დროის ინტერვალის რეგულატორები, რომლისთვისაც მოხდება მონაცემთა შეჯამება.

დასაწყისის თარიღი

ველი საწყისი თარიღი მიუთითებს თარიღზე, როდესაც დაიწყო აგრეგირებული მონაცემების შეგროვება.

შერჩეული დროის ინტერვალის ერთეულის შესაბამისად, გამოითვლება შერჩეული საწყისი თარიღის პოზიცია და შესაბამისი დროის ინტერვალის დასაწყისი გამოყენებული იქნება ჩანაწერების გასაფილტრად.

დასრულების თარიღი

ველი დასრულების თარიღი მიუთითებს თარიღზე, როდესაც დასრულდა აგრეგირებული მონაცემების შეგროვება.

შერჩეული დროის ინტერვალის ერთეულის შესაბამისად, გამოითვლება შერჩეული დასრულების თარიღის პოზიცია და შესაბამისი დროის ინტერვალის დასასრული გამოყენებული იქნება ჩანაწერების გასაფილტრად.

ადმინისტრაციული დონე

ველი ადმინისტრაციული დონე ასახავს ადმინისტრაციული ფილტრის რეგულატორებს, რომლისთვისაც მოხდება მონაცემთა შეჯამება.

რეგიონი

ველში რეგიონი მითითებულია ზუსტად ის რეგიონი, სადაც მოხდა აგრეგირებული მონაცემების შეგროვება.

რაიონი

ველში რაიონი მითითებულია ზუსტად ის რაიონი, სადაც მოხდა აგრეგირებული მონაცემების შეგროვება.

დასახლება

ველში დასახლება მითითებულია ზუსტად ის დასახლება, სადაც მოხდა აგრეგირებული მონაცემების შეგროვება.
ძებნის შედეგების ჩამონათვალის განყოფილება; ეკრანზე გამოაქვს აგრეგირებული მოქმედებები, რომლებიც შეესაბამება ძებნის კრიტერიუმებს.

ნახ. 99 შედეგების ჩამონათვალის ძებნა

ველი/ცვლადი

აღწერა

მოქმედების ID

მოქმედების ID ეკრანზეა გამოტანილი.

დასაწყისის თარიღი

ეკრანზე ჩანს ის თარიღი, რომლიდან დაწყებულიც შეგროვდა კონკრეტული აგრეგირებული მონაცემები.

დროის შუალედი

ეკრანზე ჩანს დაავადების შემთხვევის დროის ის შუალედი, რომელიც გამოყენებული იყო ამ შემთხვევის შექმნის დროს.

რეგიონი

ეკრანზე ჩანს ის რეგიონი, სადაც შეგროვდა კონკრეტული აგრეგირებული მონაცემები.

რაიონი

ეკრანზე ჩანს ის რაიონი, სადაც შეგროვდა კონკრეტული აგრეგირებული მონაცემები.

დასახლება

ეკრანზე ჩანს ის დასახლება, სადაც შეგროვდა კონკრეტული აგრეგირებული მონაცემები.

ღილაკები

ახალი

დააწკაპუნეთ ღილაკს ახალი, რათა გაიხსნას ვეტ აგრეგირებული მოქმედების დეტალების ფორმა და შექმენით ახალი ვეტერინარული აგრეგირებული მოქმედება.

რედაქტირება

შეარჩიეთოქმედება და დააწკაპუნეთ ღილაკს რედაქტირება რათა გაიხსნას ვეტ აგრეგირებული მოქმედების დეტალების ფორმა და მოახდინეთ შერჩეული მოქმედების რედაქტირება.

წაშლა

შეარჩიეთ შემთხვევა და დააწკაპუნეთ წაშლის ღილაკზე, რათა წაშალოთ შერჩეული ვეტერინარული აგრეგირებული მოქმედება.
