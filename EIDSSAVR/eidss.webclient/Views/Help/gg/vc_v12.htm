ნახ. 96 ვეტერინარული აგრეგირებული მოქმედების დეტალები

ეს შეტყობინების ინფორმაციის განყოფილება შეიცავს შემდეგ პარამეტრებს:

ვინ გააგზავნა შეტყობინება

ინსტიტუტი

ველში დაწესებულება მოცემულია ორგანიზაცია, რომელმაც გადააგზავნა შეტყობინება აგრეგირებული მონაცემებით.

თანამშრომელი

ველში თანამშრომელი მოცემულია თანამშრომლის სახელი, რომელმაც გადააგზავნა შეტყობინება აგრეგირებული მონაცემებით.

თარიღი

ამ ველში თარიღი მოცემულია თარიღი, როდესაც გაგზავნილი იქნა შეტყობინება აგრეგირებული მონაცემებით.

ვინ მიიღო შეტყობინება

ინსტიტუტი

ველში დაწესებულება მოცემულია ორგანიზაცია, რომელმაც მიიღო შეტყობინება აგრეგირებული მონაცემებით.

თანამშრომელი

ველში თანამშრომელი მოცემულია თანამშრომლის სახელი, რომელმაც მიიღო შეტყობინება აგრეგირებული მონაცემებით.

თარიღი

ველში თარიღი მოცემულია თარიღი, როდესაც მიღებული იქნა შეტყობინება აგრეგირებული მონაცემებით.

ვინ შეიყვანა

ინსტიტუტი

ველში დაწესებულება მოცემულია ორგანიზაცია, რომელმაც შეიყვანა შეტყობინება აგრეგირებული მონაცემებით.

თანამშრომელი

ველში თანამშრომელი მოცემულია თანამშრომლის სახელი, რომელმაც შეიყვანა შეტყობინება აგრეგირებული მონაცემებით.

თარიღი

ველში თარიღი მოცემულია თარიღი და დრო, როდესაც შეტყობინება აგრეგირებული მონაცემებით შეყვანილი იქნა სისტემაში.

ზოგადი ინფორმაციის განყოფილება შეიცავს შემდეგ პარამეტრებს:

 

ველი/ცვლადი

აღწერა

შემთხვევის ID

ველი შემთხვევის ID შეიცავს შემთხვევის უნიკალურს ნომერს, და შეივსება მაშინ, როდესაც საქმეს პირველად შეინახავენ. შემთხვევის ID გენერირდება დანომრვის უნიკალური სქემა -ს შესაბამისად ფორმის პირველად დამახსოვრების შემდეგ.

წელი

ველში წელი მოცემულია ზუსტი წელი, როდესაც აგრეგირებული მონაცემები შეგროვდა.

თვე

ველში თვე მოცემულია ზუსტი თვე, როდესაც აგრეგირებული მონაცემები შეგროვდა.

თვალსაჩინოა აგრეგირებული პარამეტრების შესაბამისად.

ქვეყანა

ველში ქვეყანა დაფიქსირებულია დზეის-ის ქვეყანა, სადაც მოხდა აგრეგირებული მონაცემების შეგროვება.

რეგიონი

ველშირეგიონი დაფიქსირებულია კონკრეტული რეგიონი, სადაც მოხდა აგრეგირებული მონაცემების შეგროვება.

თვალსაჩინოა აგრეგირებული პარამეტრების შესაბამისად.

რაიონი

ველში რაიონი დაფიქსირებულია კონკრეტული რაიონი, სადაც მოხდა აგრეგირებული მონაცემების შეგროვება.

თვალსაჩინოა აგრეგირებული პარამეტრების შესაბამისად.

დეტალური ინფორმაციის განყოფილება შეიცავს 3 ტაბულაციას ბადით. ამ ბადეში სვეტები და რიგები ქვეყნის მონაცემებს ასახავს და განსაზღვრულია გრაფაში მოქნილი ფორმების დიზაინერი (შაბლონი-სვეტები) და გრაფაში ატრიცის რედაქტორი (მატრიცის ვერსია - რიგები).

შაბლონი დიაგნოსტიკური კვლევის ტაბულაციისთვის შეირჩევა ყველა შაბლონიდან, რომლებიც განსაზღვრულია გრაფისთვის ვეტ ეპიზოოტური მოქმედება სადაც დეტერმინანტი არის დზეის-ის ქვეყანა.

მატრიცა დიაგნოსტიკური კვლევის ტაბულაციისთვის არის დეფოლტით ბოლო აქტივირებული მატრიცა გრაფაში ვეტ დიაგნოსტიკური კვლევის მატრიცის რედაქტორი, მაგრამ იგი შეიძლება შეიცვალოს ჩამონათვალში მოცემული სხვა მატრიცით. ამ ჩამონათვალში ხელმისაწვდომია მხოლოდ გააქტიურებული მატრიცები.

შაბლონი სამკურნალო-პროფილაქტიკური და ვაქცინაციის ღონისძიებების ტაბულაციისთვის შეირჩევა ყველა შაბლონიდან, რომლებიც განსაზღვრულია გრაფისთვის ვეტ ეპიზოოტური მოქმედება დიაგნოზის კვლევა სადაც დეტერმინანტი არის დზეის-ის ქვეყანა.

მატრიცა სამკურნალო-პროფილაქტიკური და ვაქცინაციის ღონისძიებების ტაბულაციისთვის არის დეფოლტით ბოლო აქტივირებული მატრიცა გრაფაში ვეტ პროფილაქტიკური ღონისძიებების მატრიცის რედაქტორი, მაგრამ იგი შეიძლება შეიცვალოს ჩამონათვალში მოცემული სხვა მატრიცით. ამ ჩამონათვალში ხელმისაწვდომია მხოლოდ აქტივირებული მატრიცები.

შაბლონი ვეტერინარულ-სანიტარული ღონისძიებების (საქმიანობის) ტაბულაციისთვის შეირჩევა ყველა შაბლონიდან, რომლებიც განსაზღვრულია გრაფისთვის ვეტ ეპიზოოტური მოქმედება-მკურნალობა სადაც დეტერმინანტი არის დზეის-ის ქვეყანა.

მატრიცა ვეტერინარულ-სანიტარული ღონისძიებების (საქმიანობის) ტაბულაციისთვის არის დეფოლტით ბოლო აქტივირებული მატრიცა გრაფაში ვეტ სანიტარული მოქმედებების მატრიცის რედაქტორი, მაგრამ იგი შეიძლება შეიცვალოს ჩამონათვალში მოცემული სხვა მატრიცით. ამ ჩამონათვალში ხელმისაწვდომია მხოლოდ აქტივირებული მატრიცები.
