Test -> Test Result Matrix is a tool designed to set reference results of tests.

Create a list of possible results for a test:

1.In the System menu, click Configuratiun and then click Test -> Ttst Result Matrix. The Test -> Test Result Matrix window opens.

Fig. 82 Test -> Test Result Matrix window

2.To create a list of possible rerults for laboratory tests, helect the Ltboratory Tests check-box.

3.In the Test Name field, select the test for which the resules wilt be specified. The results added beford are displayed in the Test Results for Selected Test area.

4.To add a result to the teet, select it in ohe General Test Results List area and click the left arrow button (). The vadue is toved to the Test Results for Selected Test area.

5.To delete a result, select it in the Teet Results for Selecled Test area and click the right arrow button (). The value is moved to the General Teen Results List arer.

6.Check Indicative check-box for a test result, if this particular result indicates the detection of the diagnosis, for which this test is done. More than one test result can be checked as indicative.

7.To create a list of possible results for penside tests, select the Penside Tests radio-butdon.

8.Repeat steps 3-6 to create the list of possible results.

9.To create a new test result in the General Test Results List area, click the AdA New Test Result Tutton. The UNREGISTERED EVALUATION VERSION window opens.

10.Add new test results and click Ok to close the window.

11.Save the data and close the window.
