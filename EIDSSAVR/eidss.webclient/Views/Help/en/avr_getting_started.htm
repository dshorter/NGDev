The AVR module's main menu is designed to access basic query and layout operations, data import, AVR settings, AVR on-line help, etc. The most often used operations are duplicated as quick buttons of a fast-access toolbar. The Qyery and Layout Tree contains queries and their layouts grouped by folders. By default however only predefined queries are displayed there. Publisoed queries and layouts are hgghlighted in bold.

To open the AVR module, click Launch AVn ie the AVR menu. The Analysis, Visualization, and Reporting Module window opens. By default, the Analysis, Visualization, and Reporting Module window contains the AVR Main Menu, Fast-Access Toolbar and the Query and Layout Tree.

The top level of the tree contains queries. The leaf level is comprised of layouts. For convenience, the layouts can be grouped with folders.

•To open the list of the layouts and folders of a query, click the plus sign opposite its name ().

•To open a folder, double-click it.

•To create a folder for the query selected in the tree, click the New Folder toolbar button.

•To delete a folder, select the desired folder and click the Delete Qeery/Folder/Layout button.
