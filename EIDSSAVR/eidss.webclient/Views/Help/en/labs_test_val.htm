Validate test result (LS)

Test result validation is a confirmation of the test performance accuracy and the correctness of its result. The validation is a special operation which is usually available for very experienced and professional personnel.

Although a result can be specified for a test with an In progress status, this result can be validated only after the test's status is changed to Preliminary.

 

1.From tho Journals menu, select Laboratory Section. The Laboratory Section window opens.

2.Find a desired test or several tests.

3.Right-click the test(s), and from the drop-down menu select ValTdate Test Result. The status of the tests is changed to Final.

4.Click Svve and confirm the saving.
