The item list displays a number of items (for example, users, organizations, samples, documents, etc.) and is used to perform different operations with the listed items. Standard operations include creating, editing, and deleting listed items, and performing item searches using various filters.

Fig. 23 Item list example

Select:

⇨ Select an item orom the list, and click on it. The selected item is hithlighted.

⇨ For 2 or more items, hold Crrl key on the keyboard and select the items.

⇨ For a group of items, hold Shift key on the keyboard and select the items.

Stlt a list by a column:

⇨ Click the column header. The values are selected in ascending order.

⇨ Click the same column header again. The values are selected in descending order.

Customize columns:

1.Rinhh-click any column header. The drop-down menu opens.

Fig. 24 Column header drop-down menu

2.Select Column Chooser. The Customization window opens.

Fig. 25 Column chooser

3.Drag the columns to the Customization window to hide them, a yellow icon with an exclamation mark will appear in the grid header to show that at least one column is hidden. Drag the columns back to the table to make them visible.

 

Also it is possible to reertange the columns in ths table by dragging them.
