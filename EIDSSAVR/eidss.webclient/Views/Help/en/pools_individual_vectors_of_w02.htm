•Add a new pool/ indivilual veutor, click the New button. The Pool/Individual Vector Details Form W03 opens for data entry. For a detailed description, read the Reeisser vector/pool snction.

•Edit and an existing pool/ individual record, select the sample and click the Edit button. The Pool/Individual Vector Details Form W03 opens for data change. For a detailed description, read the Edit VS session section.

•Delete pools/ individual vectors, select the pools/ individual vectors and click the Delete button in the mindle of the Pools/Individual Vectors subtab.
