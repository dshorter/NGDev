In the pivot grid it is possible to select how to display the data - as a number of items, the maximum item, etc. A function that performs a computation on a set of values rather than on a single value is called an aggregate function. For example, finding the average or mean of a list of numbers is an aggregate function.

By default, each EIDSS field has an aggregate function applied to it. If the default aggregate function does not meet the requirements, either change the default aggregate function (this should be done if the new aggregate function is going to be applied in most cases), or change the aggregate function is to be applied here and now.

The arithmetic average value, often simply called the mean, of two numbers, such as 2 and 8, is obtained by finding a value A such that 2 + 8 = A + A. One may find that A = (2 + 8)/2 = 5.

If we increase the number of terms in the list for which we want an average, we get, for example, that the arithmetic mean of 2, 8, and 11 is found by solving for the value of A in the equation 2 + 8 + 11 = A + A + A. One finds that A = (2 + 8 + 11)/3 = 7.
