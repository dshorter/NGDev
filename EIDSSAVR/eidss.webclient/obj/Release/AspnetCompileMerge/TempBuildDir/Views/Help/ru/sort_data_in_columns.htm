Сортировка данных в столбце

Для целей сортировки любого столбца в таблице данные на вкладке Вид представления данных можно мгновенно выстраивать в обратном порядке. Просьба иметь в виду, что скрытие столбца удаляет сортировку.

Сортировка столбца

1.Проделайте одно из следующих действий:

•Находясь в таблице, нажмите на заголовок столбца для сортировки его содержимого. Рядом с заголовком появится стрелка, указывающая вверх или вниз, означающая, что значения поля сортированы по возрастанию или по убыванию.

•Нажмите правой кнопкой мыши на заголовке столбца и из раскрывающегося меню выберите пункт Сортировать по возрастанию / сортировать по убыванию.

Рис. 428 Сортировка столбца по возрастанию

2.Для отображения сортировки в обратном порядке, нажмите на стрелку ещё раз. Сортировать можно текст, цифры и даты.

Рис. 429 Таблица с данными, сортированными по октябрю

Удаление сортировки

Нажмите правой кнопкой мыши на заголовке столбца и из раскрывающегося меню выберите пункт Очистить сортировку.
