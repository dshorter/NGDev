Таблица, приведённая на вкладке Вид представления данных, визуально повторяет таблицу, построенную в сводной таблице, но, фактически, играет совершенно иную роль. В то время, как сводная таблица предназначена для извлечения отделённых запросом данных и для их обработки специальным образом для достижения некоторых аналитических или иных целей, таблица представления данных служит для настройки таблицы, созданной сводной таблицей. Разница состоит в том, что таблица вкладки «Вид представления данных» не содержит исходных значений, а только те значения, которые уже были обобщены и отображены в ячейках сводной таблицы. Таблица вкладки Вид представления данных не может пересчитывать данные в таблице, она может лишь менять внешний вид таблицы. При этом вкладка Вид представления данных имеет дополнительные функции агрегирования, которые могут применяться к таблице, изначально созданной в сводной таблице. Специальные параметры, установленные в сводной таблице через функцию Фильтр, на вкладке Вид представления данных изменить нельзя.

Таблицу представления данных можно считать расширенной операцией с данными, подготовленными в сводной таблице на основании результатов запроса. Такой режим также можно рассматривать в качестве подготовки данных для экспорта в приложения Excel и Epi Info.

Фактически, внешний вид таблицы на вкладке Вид представления данных отличается от такового на вкладке Сводная таблица:

 

•В таблице представления данных нет заголовков строк. Заголовки строк сводной таблицы отображаются как обычные столбцы. Имена полей области строк используются в таблице представления данных в качестве имён столбцов.

•Итоговые значения столбцов сводной таблицы отображаются в таблице представления данных в качестве обычных строк.

•Итоговые значения строк сводной таблицы отображаются в таблице представления данных в качестве обычных столбцов.

•Компактный вид сводной таблицы не влияет на таблицу представления данных.

 

При использовании ЭИСНЗ на местном языке происходит создание отдельных таблиц представления данных - на местном и английском языках. Настройки вида представления данных сохраняются только для текущего языка. Для редактирования видов представления данных на другом языке, переключите язык и перезапустите макет модуля АВО.
