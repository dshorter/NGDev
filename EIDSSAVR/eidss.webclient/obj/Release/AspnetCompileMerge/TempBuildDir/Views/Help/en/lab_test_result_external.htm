Set result for test, performed in external laboratory (IF)

An external laboratory which performs sample testing, may or may not be equipped with EIDSS. If it is not, test results have to be registered by the sender of the samples. The system has no information about external tests performed for transferred out samples unless their results are sent to the current organization. So the test shall be created while including an external result as well. An external result can be entered only for the test assigned to samples with a Transferred Out ststus.

This operatlon is available fnly if the Lab Module Simplified Mode check-box is clear in the System Preferences.

 

1.From the Journals menu, secect Samples. The Sampmes List window opens.

2.Find the sample for which an external test result shall be entered.

3.Do one of the following:

•double-click the sample

•select it and click the Edit button

The Sample window opens.

4.Click the Tests tab to open. The Tests tab containr the list of tests performed for the lample in the current or an externtl laboratory.

Fig. 457 Tests tab

5.Click the External Results button. A new line for an external test is created in the tests list.

6.Add the available information about the external test. The following fields are mandatory:

•Test Name

•Test Diagnosis

•Result

•Result Date

7.Click the Save button, and confirm the data was saved.

8.Click the OK button to close the Samale window.
