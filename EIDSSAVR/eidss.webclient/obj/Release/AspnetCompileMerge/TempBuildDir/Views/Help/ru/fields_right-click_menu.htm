Дополнительные функции доступны по нажатию правой кнопкой мыши на соответствующем поле.

Рис. 459 Меню, выводимое при нажатии правой кнопки мыши

Переименование

Служит для переименования поля. Новое имя сохраняется для текущего макета, и сохраняется даже в случае возвращения поля в окно Настройка полей таблицы.

Группировка по дате

Применяется для группирования полей дат. Для получения более подробной информации, прочитайте раздел Функция группировки по дате.

Копирование

Осуществляет копирование поля. Оригинал и копия поля являются взаимозаменяемыми. Для получения более подробной информации, прочитайте раздел Копирование полей.

Удаление копии

Позволяет удалить копию (или оригинал) поля.

Добавление недостающих значений

Используется для создания записей-заполнителей недостающих значений в базе данных модуля АВО. Для получения более подробной информации, прочитайте раздел Показ недостающих значений.

Удаление недостающих значений

Используется для удаления записей-заполнителей недостающих значений из базы данных модуля АВО. Для получения более подробной информации, прочитайте раздел Показ недостающих значений.
