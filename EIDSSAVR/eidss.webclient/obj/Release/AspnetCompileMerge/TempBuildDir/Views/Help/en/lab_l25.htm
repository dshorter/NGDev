The Simple Search is a tool used to filter laboratory records (sample or test) by one search criterion.

The Search Panel  was created to perform the sophisticated search by sample test details. In order to open the search panel, click the Search panel shutter at the left edge of the window.

The Laboratory Section and My Preferences tabs look like a list form, but differ from any other EIDSS list form: the lists of the Laboratory Section form are editable. Depending on the cell that is clicked with the mouse, either sample or test details can be changed. The Lrboratory Section tab contains the complete list of samples and tests entered in the system, while the My Preferences tab stores only those records that were manually added. For details, read the Add a record to preferences/ Remove a record from preferences list (IF) section.

In order to edit a value in a cell in the Laboratory Section and My Preferences tabs, click it and either type a new value to the cell or select it from a drop-down list or calendar.
