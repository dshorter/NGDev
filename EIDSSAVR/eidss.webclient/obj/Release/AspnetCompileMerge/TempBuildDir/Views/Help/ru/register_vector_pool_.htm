Регистрация отдельных переносчиков / групп переносчиков заболевания

Если в отношении какой-либо сессии предполагается ввести подробные сведения, то переносчика или группу переносчиков заболевания необходимо вначале зарегистрировать. Отдельный переносчик заболевания представляет собой переносчика, отловленного в индивидуальном порядке (напр., крыса, мышь, и т.д.). Группа переносчиков заболевания представляет собой группу из нескольких переносчиков, пойманных скопом (напр., комары, мухи, и т.д.). Список переносчиков заболеваний, предназначенных к поимке отдельно или совместно, является заранее заданным, но его можно просматривать и/или изменять в Редакторе справочника типов переносчиков заболеваний.

 

1.Нажмите кнопку Новая/новый, расположенную в форме Сессия надзора за переносчиками заболевания. Откроется окно Сведения о группе переносчиков / отдельном переносчике заболевания.

2.Введите основную информацию о группе переносчиков / отдельном переносчике заболевания в раздел «Краткая информация» формы «Сведения о группе переносчиков / отдельном переносчике заболевания».

3.Введите общую информацию об отдельном переносчике / группе переносчиков заболевания на вкладку «Общая информация» формы «Сведения о группе переносчиков / отдельном переносчике заболевания».

4.Введите специфическую информацию по отдельному переносчику / группе переносчиков заболевания на вкладку «Специфические данные о переносчике заболевания» формы «Сведения о группе переносчиков / отдельном переносчике заболевания».

5.Введите сведения о пробах взятых от отдельного переносчика / группы переносчиков заболевания на вкладке «Пробы» формы «Сведения о группе переносчиков / отдельном переносчике заболевания».

6.Заполните поля результатов тестов по отдельному переносчику / группе переносчиков заболевания на вкладке «Полевые тесты» формы «Сведения о группе переносчиков / отдельном переносчике заболевания».

7.Проведите приём проб сессии в лабораторном модуле.

8.Назначьте этим пробам лабораторные тесты.

9.Проведите лабораторные тесты и зарегистрируйте результаты тестов в лабораторном модуле. Тесты с результатами можно просматривать на вложенной вкладке «Лабораторные тесты» формы «Сессия надзора за переносчиками заболевания».

10.Нажмите кнопку ОК для сохранения изменений, и закройте окно Сведения о группе переносчиков / отдельном переносчике заболевания.

 

Созданная запись о новом отдельном переносчике / группе переносчиков заболевания отображается в форме Сессия надзора за переносчиками заболевания. Заболевания, получившие подтверждение посредством проведённых над отдельным переносчиком / группой переносчиков заболевания тестов, добавляются в поле Заболевания сессии (форма Сессия надзора за переносчиками заболевания).
