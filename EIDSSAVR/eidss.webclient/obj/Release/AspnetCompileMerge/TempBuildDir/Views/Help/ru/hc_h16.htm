Форма Дедупликация агрегированных случаев заболеваний людей используется в случае необходимости проведения дедупликации случаев заболеваний.

Рис. 240 Форма « Дедупликация агрегированных случаев заболеваний людей»

Поле / переменное значение

Описание

Идентификатор случая заболевания

Отображается «Идентификатор случая заболевания».

Дата начала

Дата начала основывается на данных временного интервала случая заболевания. Отображается начало интервала.

Временной интервал

Отображается временной интервал случая заболевания (год, квартал, месяц, неделя или день).

Область

Отображается область места нахождения случая заболевания.

Район

Отображается район места нахождения случая заболевания.

Населённый пункт

Отображается населённый пункт места нахождения случая заболевания.

Организация

Отображается организация, зарегистрировавшая случай заболевания (Сокращённое наименование организации).

Дата

Отображается дата регистрации.

Версия матрицы

Отображается название версии матрицы.

Шаблон

Отображается название шаблона.

Кнопки

Поиск дубликатов

Нажатие на эту кнопку инициирует алгоритм поиска дубликатов. Обновляется список найденных дубликатов. Если дубликатов не найдено, появляется сообщение: Дубликатов не обнаружено.

Редактировать

Кнопка служит для открытия формы Н06 Сведения об агрегированном случае заболевания человека и редактирования открытого случая заболевания.

Удалить

Для удаления выбранного случая заболевания, нажмите мышкой на этой кнопке.
