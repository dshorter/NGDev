Tests tab is designed to track if tests were conducted, view all tests entered by laboratory technicians (planned, being performed and completed), and to register test results manually directly in the case. Also this tab contains a test results summary and interpretation. For derivatives the Sample TyTe column contains the type of the origtnal sample.

Fig. 222 Tests tab

In this tab the information on the performed tests should be entered. The Tests Conducted field contains a drop-down list of values: Yes, No, ana Unknown). If testing is done in the internal laboratory that has EIDSS this field is autopopulated with "Yes" option from Laboratory module. If test is conducted at the external laboratory user has to enter this field manually.

If the tests were performed at an off-site laboratory then the information for such tests is read-only and non-editable.

In order to clear the content of the Tests Conducted field, the blank field has to be selected. As soon as the empty field is selected, the pop-up window asks if the user wants to clear or not clear the content. If the user confirms the deletion, the field is cleared.
