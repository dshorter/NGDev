Вкладка «Тесты» формы L02 «Проба»

Вкладка Тесты формы L02 Проба используется для отображения и/или редактирования подробной информации о тестах пробы. Кроме того, её можно использовать для назначения или удаления тестов. Доступность полей зависит от статуса пробы и прав доступа.

Рис. 337 Вкладка «Тесты»

Название теста

Позволяет выбрать наименование нового теста из раскрывающегося списка, наличие которых в таком списке зависит от соответствующего случая заболевания или сессии.

Это поле является обязательным для заполнения и доступно для новых тестов только до их первого сохранения.

Категория теста

Указывается категория теста.

Это поле доступно для редактирования у новых тестов только до их первого сохранения.

Проверяемый тестом диагноз

Используется для выбора из раскрывающегося списка диагноза для его подтверждения или опровержения. Содержание раскрывающегося списка зависит от случая заболевания или сессии, относящихся к пробе:

•Случаи заболевания людей: текущий диагноз;

•Случаи заболеваний животных: окончательный и предположительный диагнозы;

•Сессии: список указанных диагнозов.

Это поле является обязательным для заполнения и доступно для новых тестов только до их первого сохранения.

Статус теста

Используется для выбора статуса теста из раскрывающегося списка. В отношении результатов внешних тестов он автоматически устанавливается на Окончательный без возможности изменения. Для получения более подробных сведений о статусах тестов, прочитайте раздел Статусы теста.

Это поле является обязательным для заполнения.

Результат

Используется для выбора статуса теста из раскрывающегося списка. Указание этого значения возможно только для проб со статусами В процессе или Предварительный.

Идентификатор серии тестов

Указывается идентификатор группы, если таковая имеется, без возможности изменения.

Дата результата

В качестве значения для дат результата, утверждения или изменения, по умолчанию используется значение текущей даты, которое можно изменять. Поле заполняется автоматически, когда статус теста меняется на Предварительный, Окончательный или Изменённый.

Дата получения результатов

Используется для указания даты получения результата теста. Поле доступно и обязательно для заполнения только для результатов тестов, получаемых из внешних организаций.

Контактное лицо

Указывается лицо для связи в организации, проводящей внешний тест. Поле доступно и обязательно для заполнения только для результатов тестов, получаемых из внешних организаций.

Кнопки

Выбрать тест

Служит для создания в таблице записи о новом тесте, назначенном пробе.Кнопка доступна только для проб со статусом Принята в систему. Для получения более подробных сведений, прочитайте раздел Назначение тестов из формы «Проба» (ОФ).

Внешние результаты

Служит для создания и назначения новых тестов, проводимых во внешней лаборатории. Эта кнопка доступна только для проб со статусом Передана во внешнюю организацию. Для получения более подробных сведений, прочитайте раздел Указание результата теста, проводившегося во внешней лаборатории (ОФ).

Удалить тест

Позволяет редактировать выбранный(-ые) тест(-ы). Выберите в таблице тест(-ы) для удаления и нажмите кнопку Удалить.

Завершённые тесты удалить невозможно. Если некоторые из выбранных тестов удалить невозможно, появится предупреждающее сообщение, а тесты удалены не будут; все другие выбранные тесты будут удалены.

Эта кнопка отображается исключительно при наличии у пользователя права на удаление в функции «Доступ к лабораторным тестам».
