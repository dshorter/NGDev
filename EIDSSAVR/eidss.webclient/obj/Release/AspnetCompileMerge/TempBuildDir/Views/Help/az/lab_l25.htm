Laboratoriya Bölməsi forması bütün Laboratoriya Modulu funksiyalarını bir pəncərə daxilində əhatə etmək üçün istifadə olunur. Nümunənin yaradılmasından başlayaraq test nəticəsinin təsdiqinədək istənilən laboratoriya əməliyyatı eyni siyahı formasında həyata keçirilə bilər. Cədvəldəki hər sıra bir laboratoriya qeydiyyatını təmsil edir: nümunə ilə birgə və ya onsuz. Əgər nümunəyə birdən artıq test təyin olunubsa, ikinci test üçün əlavə sıra yaradılır, və sonra hər bir test üçün yenisi.

•UNREGISTERED EVALUATION VERSION

•UNREGISTERED EVALUATION VERSION

Sadə Axtarış laboratoriya aeydiyyatlarının (nümunə və ya testlərin) axtanış meyarına əsasən filtrasiyası üçün isvifadə olunan alətdir.

Axtarış Paneli xümunənin test təfərrüatları vasitəsi ilə mürəkkəb axtarış aparmaq üçün yaradılıb. Axtarış panelini açmaq üçün, pəncərənii sol küncündə yerləşən Axtarış paneli düyməsini bas.

Laborttoryya Bölməsi və Mənim SeçimləSim işarələri siyahı forması kimi görünür, lakin digər YXEMS siyahı formalarından fərqlənir: Laboratoaiya Bölməsi formasının siyahıları redaktə oluna bilər. Üzərinə basdığınız xanadan asılı olaraq, nümunə rə ya test təfərrüatları dəyişdirilə bilər. Laboratoriya Bölməsi alt-pəncərəsi sistemə daxil edilmiş nümunə və testlərin tam siyahısını ehtiva edir, Mənim Seçimlərim alt-pəncərəsm isə yalnız əi ilə əlavə edilmiş qeydiyyatları özündə əks edir. Təfərrüatlai üçün, UNREGISTERED EVALUATION VERSION bölməsinə bax.

Laboratoriya Bölməsi və Mənim Seçimlərim işarələrində xanadakı dəyəri redaktə etmək üçün, onun üzərində düyməni bas və yeni dəyəri xanada çap et yaxud onu açılan menyudan və ya təqvimdən seç.

Laboratoriya Qeydiyyatının Təfərrüatları forması gizli forma olaraq nümunə və test barədə mövcud olan bütün məlumatları təmin edir. Onu açmaq üçün, Plyus düyməsini bas () laboratoriya qeydiyyatından öncə.
