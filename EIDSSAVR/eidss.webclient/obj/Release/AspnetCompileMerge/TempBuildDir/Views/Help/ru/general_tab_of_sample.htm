Вкладка Общая информация формы L02 Проба, используется для отображения и/или редактирования информации о пробе. Доступность полей зависит от статуса пробы и прав доступа.

 

Идентификатор исходной пробы

Отображается исходная проба аликвот и дериватов. Для просмотра сведений об идентификаторе исходной пробы, нажмите кнопку Сведения (). Поле заполняется только в отношении аликвот или дериватов.

Обрабатывающий отдел

Используется для указания отдела организации, который работает с пробой.

Место хранения

Указывается положение места хранения пробы:

1.Нажмите кнопку Сведения (). Откроется окно Холодильники.

2.Выберите необходимое место.

3.Нажмите кнопку ОК. Окно Холодильники закроется. В поле будет отображаться назначенное местоположение.

Тип случая заболевания

Отображается тип случая заболевания, относящийся к пробе:

•Заболевание человека;

•Заболевание домашней птицы;

•Заболевание домашнего скота;

•Надзор за переносчиками заболеваний.

В отношении сессии АН (активного надзора) не отображается ничего.

Информация о случае заболевания

Указывается идентификатор случая заболевания и диагноз.

Для открытия формы случая заболевания, нажмите кнопку Сведения (). Доступ к форме случая заболевания регулируется правами доступа пользователя к форме случая заболевания.

В отношении сессии АН (активного надзора) не отображается ничего.

Идентификатор сессии

Отображается идентификатор сессии, связанной с пробой.

Для открытия формы сессии, нажмите кнопку Сведения ( ). Доступ к форме регулируется правами доступа пользователя к такой форме.

Для случаев заболеваний не отображается ничего.

Информация о пациенте / биологическом виде / переносчике заболевания

Отображается следующая информация:

•По случаям заболеваний домашней птицы: биологический вид;

•По случаям заболеваний домашнего скота: идентификатор животного;

•По случаям заболеваний людей: фамилия, имя и отчество пациента.

•По сессиям АН: вид и идентификатор животного.

•По сессиям НЗПЗ: идентификатор переносчика заболевания.

Примечания

Вносятся примечания.

Учреждение

Отображается взявшая пробу организация.

Работник

Отображается Ф.И.О. взявшего пробу работника.
