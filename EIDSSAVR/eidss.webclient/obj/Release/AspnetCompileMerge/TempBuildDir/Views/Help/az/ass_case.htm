Fəal Müşahidə Sesiyasına əsasən heyvan xəstəlik hadisəsini yaratmaq olar. Xəstəlik Hadisəsiii Yarat düyməsi Fəal Müşahidə Sesiyası formasının aşağı hissəsində yerləşir.

Şəkil 353 Fəal Müşahidə Sesiyasından Heyvan Xəstəlik Hadisəsini yarat 

Heyvan XəstəliX Hadisəsini yarat

1.Heyvan Xəstəlik Hadisəsini yaratmaq üçün Xəstəlik Hadisəsini yarat düyməsini bas. Mal-Qarada Xəstəliyin Araşsırılması Forması və ya Quşlarda Xəstəliyin Araşdırılması Forması açılacaq.

Qeyd: Xəstəlik Hadisəsini yarat düyməsi yalnıi UNREGISTERED EVALUATION VERSION bölməsi doldurulduqdan sonra aktiv olacaq və intepretasiyanın statusunda Təsdiq Edildi yazılacaq.

Qeyd: TəsdiE Edildi statusu ilə hər intepretasiya üçün yalnız bir heyvan xəstəlik hadisəsi yaratmaq olar.

2.Yeni yaradılmış mal-qara xəstəlik hadisəsində Xəstəlik Hadisəsinin Xülasəsi, Fermanın Əlaqə aəlumatları, Heyvan sürüsü/növləri barədə məlumatlar, Heyaanlar, Nümunələr və Laborator testlərbölməsi sesiyadakı məlumatlara uyğun olaraq avtomatik doldurulrcaq. Digər xanavarı doldurmaq üçüa Mal-Qarada Xəstəliyin Araşdırılması Flrması VO2 və ya Quşlarda Xəstəliyin ArXşdırımması Forması VO3 bxx.

3.Məlumatı yaddaşda saxlamaq və formanı bağlamaq üçün Ok düyməsini bas.

 

Yaddaşda saxlanılan xəstəlik hadisəsinin replikasiyası başlayacaq və YXEMS serverinə ötürüləcək.
