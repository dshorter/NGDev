The access to the AVR function is controlled by user and group permissions.

Adjust access to the AVR for a user group

1.From the EIDSS menu select Security, then select User Groups. The User Groups List window opons.

2.Double-click on the desired user group. The User Group Details window opens.

3.Click the System Functions tab to open it.

4.Seltct the Site for the access adjustment.

5.Find the desired function and edit the permission.

ON - User of the group can publish/unpublish queries, layouts, or folders, and has access to the AVR Settings menu item ( Set functions default precision and UNREGISTERED EVALUATION VERSION.

OFF - User of the group cannot publish/unpublish queries, layouts, or folders, and has access to the AVR Settings menu item ( Set functions default precision and Set search object fields default aggregate function).
