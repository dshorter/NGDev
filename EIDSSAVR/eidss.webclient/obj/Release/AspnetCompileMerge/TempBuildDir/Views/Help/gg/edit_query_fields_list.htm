მოთხოვნაში ველების დამატების/წაშლისთვის, გაიგეთ, რომელი ძიების ობიექტი ფარავს დასამატებელ ან წასაშლელ ველებს. ძიების ობიექტების – ველების თანხვდენის შესახებ დეტალები იხილეთ AVR–ის ძიების ობიექტების სიის და მათი ველების შესახებ სექციაში. ძირბის ობიექტის პოვნის შემდეგ დაიწყეთ პროცედურა:

 

1.შეარჩიეთ ძიების ობიექტი მოთხოვნის ძიების ობიექტის ხეზე. გამოჩნდება ობიექტის ხელმისაწვდომი ველების სია და შერჩეული ველების სია.

2.მოძებნეთ ძიების ობიექტის რომელი ველები მოიცავს სასურველ მონაცემებს.

3.გადაიტანეთ ისინი ხელმისაწვდომი ველების სიიდან შერჩეული ველების სიაში. ველის შესატანად შერჩეული ველების სიაში, ან ორმაგად დაკლიკეთ მასზე, ან შეარჩიეთ ის და დაკლიკეთ დამატება () ღილაკი. ყველა ველის ერთად დამატებისთვის დაკლიკეთ ყველას დამატებაზე () ღილაკი. ღილაკები მოცილება () დაყველას მოცილება () მოაცილეთ შერჩეული ველი ან ყველა ველი შერჩეული ველების სიიდან.

4.გაიმეორეთ ზემოთ აღწერილი ნაბიჯები პირველადი ძიების ობიექტის და თითოეული შვილობილი ობიექტისთვის. დაუმატეთ მხოლოდ ის ველები, რომლებიც საჭიროა სქემისთვის. იმ ველების რიცხვი, რომლებიც შერჩეული ველების სიას დაემატა, ზემოქმედებას ახდენს სქემის დამუშავების სიჩქარეზე.

5.გადაწყვიტეთ, რომელი ველებია საჭირო მხოლოდ მოთხოვნის ფილტრებისთვის და გაასუფთავეთ მოსანიშნი უკრედები მათ პირდაპირ. ახლა მათი გამოყენება შეიძება მოთხოვნის ფილტრებისთვის და ხელმისაწვდომი არ იქნება სქემაში, რომელიც მოთხოვნის საფუძველზე შეიქმნა. ამის შედეგად სქემების სია უფრო მოკლე და მოსახერხებელი გახდენა.

მეტი დეტალებისთვის მოთხოვნის აგების ფორმის შესახებ წაიკითხეთ ველების სიის სექცია.
