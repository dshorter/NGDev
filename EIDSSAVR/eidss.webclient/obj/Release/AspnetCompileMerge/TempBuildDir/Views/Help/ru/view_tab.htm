Вкладка Вид представления данных создана для настройки отчётов, создания географических карт и диаграмм.

Рис. 461Вкладка «Вид представления данных».

Функция

Позволяет указать функцию для дополнительного агрегированного столбца. Для получения более подробной информации, прочитайте раздел Добавление столбца агрегированных данных.

Для столбца агрегированных данных

Нередактируемое поле, отображающее столбец агрегированных данных, для которого выбрана функция.

Точность

Позволяет указать точность отображения данных в столбце агрегированных данных после применения функции. Для получения более подробной информации о точности, прочитайте раздел Настройка точности функций по умолчанию; сведения о столбце агрегированных данных, доступны в разделе Добавление столбца агрегированных данных.

Серия диаграммы по умолчанию

Служит для выбора данных, которые будут представлены в качестве диаграммы. Для использования доступны только данные из области данных сводной таблицы. В диаграмме возможно использование более одной серии.  Для получения более подробной информации о построении диаграммы, прочитайте раздел Диаграммы в модуле АВО.

Диаграмма оси Х по умолчанию

Служит для выбора данных, представляемых на оси Х диаграммы. Для использования доступны поля из области данных сводной таблицы.

Открыть диаграмму

Необходима для открытия диаграммы, создаваемой на основе вида представления данных таблицы. Диаграмму можно настраивать в окне Редактор диаграмм. Для получения более подробной информации о настройке диаграмм, прочитайте раздел Редактор диаграмм.

Данные, по умолчанию используемые для построения слоя карты с диаграммами

Служит для выбора данных, которые будут представлены в качестве диаграммы на карте. Для использования доступны только данные из области данных сводной таблицы. В диаграмме возможно использование более одной серии.  Для получения более подробной информации о построении диаграммы, прочитайте раздел Географические карты в модуле АВО.

В виде градиентов

Служит для выбора данных, которые будут использованы при окраске карты. Из области данных сводной таблицы доступна только одна последовательность данных. Для получения более подробной информации о построении карты, прочитайте раздел Географические карты в модуле АВО.

Административная единица карты по умолчанию

Служит для выбора административной единицы по умолчанию, используемой для построения карты. Позже её можно будет поменять в окне Редактор карт.

Открыть карту

Необходима для открытия карты, создаваемой на основе вида представления данных таблицы. Карту можно настраивать в окне Редактор карт. Для получения более подробной информации о настройке карты, прочитайте раздел Географические карты в модуле АВО.
