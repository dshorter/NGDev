Форма L11 «Список холодильников»

Форма L11 Список холодильников является перечнем лабораторных холодильников, в которых хранятся пробы. Она состоит из панели поиска и списка холодильников, которые соответствуют указанным в такой панели критериям поиска.

Находясь в меню Журналы, выберите пункт Холодильники. Откроется окно Список холодильников.

Рис. 356Список холодильников

Параметры поиска

Обрабатывающий отдел

Указывается полное или частичное наименование холодильника для нахождения.

Примечание

Указывается полное или частичное примечание по холодильнику для нахождения.

Тип хранения

Из раскрывающегося списка выбирается Холодильник.

Список результатов

Обрабатывающий отдел

Наименование холодильника.

Примечание

Указывается примечание в отношении холодильника.

Тип хранения

Указывается тип холодильника.

Кнопки

Копировать

Используется для копирования записи о выбранном холодильнике со всеми его элементами хранилища.

1.Выберите исходный холодильник.

2.Нажмите кнопку Копировать. Откроется окно Копировать холодильник.

3.Введите наименование необходимого холодильника и нажмите кнопку ОК. В окно Список холодильников добавится запись о новом холодильнике.

Новый

Нажатие создаёт запись о новом холодильнике.

Нажмите кнопку Новый. Откроется форма L12 Схема хранения.

Редактировать

Служит для открытия записи выбранного холодильника для редактирования.

1.Выберите холодильник, информацию о котором необходимо изменить.

2.Нажмите кнопку Редактировать. В режиме редактирования откроется форма L12 Схема хранения, в которой содержатся сведения о холодильнике.

Удаление

Служит для удаления записи выбранного холодильника.

После удаления записи о холодильнике, ни холодильник, ни элементы его хранилища нельзя будет больше ни использовать, ни автоматически восстановить.
Удалять можно записи только о тех холодильниках, в которых нет хранящихся проб.

1.Выберите холодильник, запись о котором необходимо удалить.

2.Нажмите кнопку Удалить. Для подтверждения совершения действия откроется окно Удалить запись.

3.Нажмите кнопку Да. Окно Удалить запись закроется. Запись об удалённом холодильнике будет удалена из ЭИСНЗ и из окна Список холодильников.
