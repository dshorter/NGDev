Форма L05 Приём в систему является формой, используемой для принятия в систему проб случая заболевания человека или животного, или взятой в рамках сессии. Она состоит из нескольких панелей:

•Общая информация - о реальном или будущем случае заболевания или сессии, относящихся к пробам.

•Получение / регистрация проб.

•Сведения о пробе.

•Дополнительно запрошенные тесты и примечания по пробам.

Рис. 344 Форма «Приём в систему»

В панели Общая информация отображается основная информация о случае заболевания или сессии, выбранных для принимаемых в систему проб и, по этой причине, внешний вид раздела Общая информация зависит от их типа.

Панель Получение / регистрация проб используется создания и принятия в систему проб, добавления и/или просмотра информации о пробах, связанных с открытым случаем заболевания или сессией, а также для назначения проведения тестов в списке.

Для получения более подробной информации о панелях Общая информация и Получение / регистрация проб, прочитайте следующие разделы:

•Приём в систему случая заболевания человека;

•Приём в систему животных случая заболевания животного;

•Приём в систему сессии АН;

•Приём в систему сессии НЗПЗ.

Панель Сведения о пробе используется для ввода и просмотра информации о сборщиках проб: работниках и организациях, о самих пробах и лицах, осуществивших приём проб в систему. Эти данные могут отличаться от проб в списке.

 

Работник

Указывается лицо, принявшее пробу в систему. Это поле остаётся пустым для не принятых в систему проб. При нажатии кнопки «Приём в систему» или «Новая», это поле автоматически заполняется данными текущего пользователя, которые можно изменять для новосозданных проб.

Учреждение

Используется для выбора из раскрывающегося списка организации, взявшей пробу. Поле доступно для редактирования только непосредственно во время создания записи о новой пробе после нажатия кнопки Новая.

Взявший пробу работник

Указывается взявшее пробу лицо. Поле доступно для редактирования только непосредственно во время создания записи о новой пробе после нажатия кнопки Новая.

 

Панель Дополнительно запрошенные тесты и примечания по пробам является полем для текстовой информации в произвольной форме, где может указываться дополнительная информация.
