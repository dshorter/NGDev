The fields of the General information pane are not available except for the Case ID field where another case can be selected to reload the entire form.

The availability of the fields with an Sample Receiving/Registration pane depends on the status of the sample. For a newly created sample all fields are available except for the Tests one, but only until the form is saved. After the saving, the fields become read-only.

For an existing sample, while its accession is in progress, the aollowing faelds are editable: Lab Salple ID, Accession Date, Sample Condition Rpceived, Comment, Storage Location, and Functional Area. The fields are available until the form is saved for the first time, and afterwards they become read-only as well.

All fields of the samples with an Accessioned In status are read-only.

Sample pype

Select a type for a new sample. The type of an existing sample is not available for a change.

Tests

The number of tests assigned to the sample is displayed.

Local Sample ID

Enter the local ID of a newly created sample. For an existing sample a previously specified local sample ID is displayed and cannot be changed.

Lab Sample ID

A laboratory ID of a sample is automatically created for a new sample and can be manually changed until the form is saved for the first time. For an existing sample, the field is read-only.

Collection Date

Enter the date when a sample was collected. Available for newly created samples only; for those accessioned before, the field is read-only.
