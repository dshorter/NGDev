ეს ვეტერინარული აგრეგირებული მოქმედებების დუბლირების მოხსნის ფორმა გამოიყენება მაშინ, როდესაც მოითხოვება მოქმედებების დებლირების მოხსნა.

ნახ. 101 აგრეგირებული ვეტერინარული მოქმედების დუბლირების მოხსნის ფორმა

ველი/ცვლადი

აღწერა

დუბლირების მოხსნის ჯგუფი

დუბლირების მოხსნას დაქვემდებარებული მოქმედებები გამოსახულია ჯგუფებში. ჯგუფი შეიძლება გაფართოვდეს ან შემცირდეს "+/-" ღილაკზე დაწკაპუნებით.

შემთხვევის ID

ეკრანზე ჩანს შემთხვევის ID

დასაწყისის თარიღი

დაწყების თარიღი გამოითვლება შემთხვევის დროის ინტერვალის შესაბამისად. ეკრანზე ჩანს დროის შუალედის დაწყება.

დროის შუალედი

შემთხვევის დროის ინტერვალი გამოდის ეკრანზე (წელი. კვარტალი, თვე, კვირა ან დღე)

რეგიონი

ფერმის რეგიონი ეკრანზეა გამოტანილი.

რაიონი

ფერმის რაიონი ეკრანზეა გამოტანილი.

დასახლება

შემთხვევის დასახლება ეკრანზეა გამოტანილი.

ორგანიზაცია

შეყვანილია დაწესებულების მახასიათებლით (ეკრანზე გამოდის ორგანიზაციის აკრონიმი).

თარიღი

ეკრანზე ჩანს შეყვანის თარიღი

მატრიცის ვერსია

ეკრანზე ჩანს "მატრიცის ვერსიის დასახელება".

შაბლონი

ეკრანზე ჩანს "შაბლონის დასახელება".

ღილაკები

დუბლირებული შემთხვევების ძებნა

დააწკაპუნეთ ღილაკზე რათა მოძებნოთ დუბლიკატები. მოძიებული დუბლიკატების ჩამონათვალი განახლდება. თუ დუბლირებული შემთხვევები ვერ მოიძებნა, ეკრანზე გამოჩნდება შემდეგი შეტყობინება: "დუბლიკატები ვერ მოიძებნა".

რედაქტირება/გადახედვა

დააწკაპუნეთ ღილაკზე რათა გაიხსნას შერჩეული აგრეგირებული შემთხვევა რედაქტირების/კითხვის (edit/read-only) რეჟიმში.

წაშლა

დააჭირეთ ღილაკს, რათა წაშალოთ შერჩეული მოქმედება.

ჯამური ინფორმაცია სექცია შეიცავს ბადისებურ ცხრილს, რომელშიც აგრეგირებულია ყველა შერჩეული შემთხვევის მონაცემები. ამ განყოფილებაში სვეტები ქვეყნის მონაცემებს ასახავს და განსაზღვრულია გრაფებში მოქნილი ფორმების დიზაინერი და ვეტ აგრეგირებული შემთხვევის მატრიცის რედაქტორი.
