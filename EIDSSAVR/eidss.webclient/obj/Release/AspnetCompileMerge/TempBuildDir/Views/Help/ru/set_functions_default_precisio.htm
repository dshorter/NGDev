Настройка точности функций по умолчанию

В зависимости от требований, возможно указание точности по умолчанию для каждой функции агрегирования, используемой в сводной таблице или таблице представления данных.

 

1.Находясь в модуле АВО, нажмите пункт Запустить модуль АВО. Откроется окно Модуль анализа, визуализации и отчётности.

2.В меню модуля АВО, нажмите пункт Опции, а затем пункт Настройки. Откроется окно Настройки модуля АВО.

3.Для её открытия, нажмите вкладку Точность функций по умолчанию. Откроется вкладка Точность функций по умолчанию.

4.В столбце Функция агрегирования найдите нужную функцию и укажите для неё точность в столбце Точность.

5.Повторите предыдущий шаг для каждой функции, чья функция агрегирования по умолчанию требует изменения.

6.Нажмите ОК. Откроется окно с запросом на подтверждение совершения действия.

7.Для сохранения изменений и закрытия настроек, нажмите кнопку Да.

Точность

-1

Результат не округляется в большую сторону.

0

Результат округляется в большую сторону до целочисленного значения.

>0

1, 2, 3 и т.д. Результат округляется в большую сторону до указанного знака.
