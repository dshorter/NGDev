აქტიური ზედამხედველობის კამპანია V16

ნახ. 73 აქტიური ზედამხედველობის კამპანია

აქტიური ზედამხედველობის კამპანიის ფანჯარა:

კამპანიის ტაბულაცია ასახავს ზოგად ინფორმაციას კამპანიის შესახებ, ასევე დიზგნოზებს, სახეობათა ჩამონათვალს, და დამატებით კომენტარებს.

ნახ. 74 აქტიური აზედამხედველობის კამპანია - კამპანიის ტაბულაცია

ველი/ცვლადი

აღწერა

კამპანიის საიდენტიფიკაციო კოდი

ველი შეიცავს კამპანიის უნიკალურს ნომერს, და შეივსება მაშინ, როდესაც კამპანიას პირველად შეინახავენ. კამპანიის ID-ის გენერირება ხდება დანომვრის უნიკალური სქემის შესაბამისად.

კამპანიის სახელწოდება

კამპანიის სახელწოდება მოცემულია ამ ველში. ეს არის სავალდებულო ველი.

კამპანიის ტიპი

კამპანიის ტიპი (აქტიური ზედამხედველობა, გამოკითხვა ან კვლევა) ასახულია ამ ველში. ეს არის სავალდებულო ველი.

კამპანიის სტატუსი

კამპანიის სტატუსი (ახალი /New/, ღია ან ' დახურული) ასახულია ამ ველში. ეს არის სავალდებულო ველი.

კამპანიის დაწყების თარიღი

კამპანიის დაწყების თარიღი მოცემულია ამ ველში.

კამპანიის დასრულების თარიღი

კამპანიის დასრულების თარიღი მოცემულია ამ ველში.

კამპანიის ადმინისტრატორი

კამპანიის ტიპი მოცემულია ამ ველში.

 

დაავადებათა და სახეობათა ჩამონათვალის განყოფილება შეიცავს შემდეგ პარამეტრებს:

ველი/ცვლადი

აღწერა

დავადება

ჩამოსაშლელი სარჩევიდან შეარჩიეთ დაავადება რომელთა კვლევაც დაგეგმილია.

სახეობა

სახეობები, რომელთა კვლევა დაგეგმილია შერჩეული დაავადებისთვის,შეარჩიეთ ჩამოსაშლელი სარჩევიდან ან შექმენით ახალი სახეობათა ტიპის მიმართვის რედაქტორში.

დაგეგმილი #

შეიყვანეთ კვლევისთვის დაგეგმილი ცხოველების რაოდენობა.

ღილაკები

ქვევით

დააჭირეთ ღილაკს, რათაამოძრაოთ შერჩეული სტრიქონი ჩამონათვალში. ფუნქცია გამოიყენება ბრძანებისთვის.

ზემოთ

დააჭირეთ ღილაკს, რათა ამოძრაოთ შერჩეული სტრიქონი ჩამონათვალში ზემო მიმართულებით. ფუნქცია გამოიყენება ბრძანებისთვის.

წაშლა

დააჭირეთ ღილაკს, რათა წაშალოთ შერჩეული სტრიქონი ჩამონათვალში.

კომენტარების/აღწერილობების განყოფილება გამოიყენება ნებისმიერი დამატებითი ინფორმაციის შესაყვანად.

 

სესიის ტაბულაცია ეკრანზე გამოაქვს აქტიური ზედამხედველობის სესიები, რომლებიც დაკავშირებულია კამპანიასთან.

ნახ. 75 აქტიური ზედამხედველობის კამპანია - სესიების ტაბულაცია

ველი/ცვლადი

აღწერა

სესიის ID

სესიის ID, რომელიც დაკავშირებულია მოცემულ კამპანიასთან, ასახულია ამ ველში.

რეგიონი

სესიის რეგიონი ასახულია ამ ველში.

რაიონი

სესიის რაიონი ასახულია ამ ველში.

დასახლება

სესიის დასახლება ასახულია ამ ველში.

სესიის დაწყების თარიღი

კამპანიის დაწყების თარიღი ასახულია ამ ველში.

სესიის დასრულების თარიღი

სესიის დასრულების თარიღი ასახულია ამ ველში.

სტატუსი

სესიის სტატუსი (ღია ან ' დახურული) ასახულია ამ ველში.

ღილაკები

ახალი სესია.

დააჭირეთ ამ ღილაკს, რათა შექმნათ აქტიური ზედამხედველობის სესია.

შეარჩიეთ სესია

დააჭირეთ ამ ღილაკს, რათა შეარჩიოთ აქტიური ზედამხედველობის სესია ამ კამპანიისთვის.

სესიის ამოღება

დააჭირეთ ამ ღილაკს, რათა ჩამონათვალიდან აირჩიოთ აქტიური ზედამხედველობის სესია.

დეტალების დათვალიერება

დააჭირეთ ამ ღილაკს, რათა ნახოთ შერჩეული აქტიური ზედამხედველობის სესიის დეტალები.

 

სესიის ტაბულაცია გამოიყენება აქტიური ზედამხედველობის კამპანიის დასრულებისთვის; ნებისმიერი კომენტარი და შენიშვნა პროცესისა და კამპანიის შედეგის შესახებ.
