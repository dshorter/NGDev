Создание агрегированного случая заболевания человека

1.Для создания агрегированного случая заболевания человека, выберите Создать -> Агрегированный случай заболевания человека. В новом окне откроется форма Сведения об агрегированном случае заболевания человека.

Рис. 196 Сведения об агрегированном случае заболевания человека

2.Заполните поля формы. Для получения полной информации о форме, просьба обратиться к разделу Сведения об агрегированном случае заболевания человека H06.

3.Выберите Версию матрицы в раскрывающемся списке или создайте новую.

4.В раскрывающемся списке выберите Шаблон или создайте новый.

5.Нажмите кнопку ОК для сохранения случая заболевания, и закройте форму.

Начнётся репликация сохранённого случая заболевания, и он будет передан на сервер ЭИСНЗ.
