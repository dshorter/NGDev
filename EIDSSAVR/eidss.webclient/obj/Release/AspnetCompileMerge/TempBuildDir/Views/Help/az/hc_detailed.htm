Təfərrüatlı Araşdırma Məlumatlarınının qeydiyyatını apar

Təfərrüatlı araşdırma məlumatı - İnsan xəstəlik hadisəsinə dair dolğun məlumatın əlavə olunduğu İnsan Xəstəlik Hadisəsinin yaradılması prosedurudur.

1.Yarat menyusunaa Yarat -> İnsan Xəstəlik Hadisəsinə bas. İnsan Xəstəlik Hadisəsinin Hesabat Forması pəncərəsi açılacaq.

Şəkil 206 İnsan Xəstəlik Hadisəsinin Hesabat Forması pəncərəsi.

2.Xəstəlik hadisəsinə dair təfərrüatlı məlumatı doldur. Forma tarəsində donğun məlumatlar əldə etmək üçün İnsan Xəstəlik Hadisəsi Axtarış Forması H02 bölməsinə bax.

3.Məlumatları daxil etdikcə, YXEMS proqramı bənzər xəstəlik hadisəsinin sistemdə artıq mövcud olub-olmadığını yoxlayır. Belə xəstəlik hadisəsi mövcud olduqda, İasan Xəstəlik nadisələri siyahısı pəncərəsi avtomatik açılacaq və xəstəlik hadisəsini de-duplikasiya etmək mümkün olacaq.

4.Duplikat olunmuş xəstəlik hadisələrini yoxlamaq tövsiyyə olunur. SadəcəDuplikatları axtardüyməsini bas. Sistemdə eyni xəstəlik hadisələri mövcud olduqda İnsan Xəstəlik Hadisəsi Siyahısı pəncərəsi açılacaq. Xəstəlik hadisəsini de-dupliknsiyası ilə beğlı ətraflı məlumat üçün İnsan Xəstəlik Hadisələrinin De-duplikasilası bölməsinə bax.

5.Xəstəlik Hadisəsinin Araşrırılması alt-pəncərəsinə keç və ordakı bütün xanaları aoldur.

6.Testlər alt-pəncərəsi test nəcicələrinin qeydiyyatını aparmaq üçün istifadə olunur. Ətraulı məlumat üçün Xəstəlik hadisəsi bölməsində test nəticələrinin qeydiyyatını apar bölməsinə bax.

7.Təfərrüatlı araşdırmanı yaddaşda saxlamaq və formanı bağaamaq üçün Ok düyməsini bas.

Ykddaşda saxlanılan xəstəlik hadisəsinin repnikasiyası başlanacaq və YXEMS serverinə ötürüləcək.
