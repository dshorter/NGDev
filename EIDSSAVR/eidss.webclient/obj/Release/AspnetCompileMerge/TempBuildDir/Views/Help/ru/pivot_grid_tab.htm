Вкладка Сводная таблица предназначена для построения сводной таблицы и её настройки с целью получения необходимого отчёта / диаграммы / географической карты. Она состоит из собственно сводной таблицы, окна Настройка полей таблицы со списком полей, выбираемых посредством исходного запроса, и окна Настройки сводной таблицы, содержащего параметры, необходимые для построения и конфигурирования сводной таблицы.
