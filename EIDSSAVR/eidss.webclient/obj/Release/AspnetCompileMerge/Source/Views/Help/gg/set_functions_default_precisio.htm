მოთხოვნიდან გამომდინარე, შესაძლებელინაგულისხმევი/დეფოლტით დაზუსტების მითითება გადაწყობად ბადეში ან ხედის ცხრილში გამოყენებული ყოველი აგრეგატული ფუნქციისთვის.

 

1.AVR მენიუში, დააწკაპუნეთ AVR-ის ჩართვაზე. გაიხსნება ანალიზის, ვიზუალიზაციის და ანგარიშების მოდულის ფანჯარა.

2.AVR მენიუში, დააწკაპუნეთ ოპციებზე და შემდეგ დააწკაპუნეთ პარამეტრებზე. გაიხსნება AVR პარამეტრების ფანჯარა.

3.გასახსნელად დააწკაპუნეთ ფუნქციებისნაგულისხმევი/დეფოლტით დაზუსტების ჩანართზე გაიხსნება ფუნქციებისნაგულისხმევი/დეფოლტით დაზუსტების ჩანართი.

4.იპოვეთ სასურველი ფუნქცია აგრეგატული ფუნქციების სვეტში და მიუთითეთ დაზუსტება მისთვის დაზუსტების სვეტში.

5.გაიმეორეთ წინა ნაბიჯი ყოველი ფუნქციისთვის, რომლისნაგულისხმევი/დეფოლტით აგრეგატული ფუნქციაც უნდა შეიცვალოს.

6.დააწკაპუნეთ "დიახ"-ზე. იხსნება დადასტურების ფანჯარა.
