Создание макета

При создании макета, количество доступных для использования текущим пользователем полей ограничено правами доступа, которыми он обладает в системе.

1.Вначале решите, как должны выглядеть получаемые при помощи запроса данные.

2.Находясь в меню ЭИСНЗ, нажмите на пункт Запустить модуль АВО. Откроется окно Модуль анализа, визуализации и отчётности.

3.Выберите исходный запрос для макета.

4.Проделайте одно из следующих действий:

•В меню Файл модуля АВО, нажмите на пункт Новый макет.

•Нажмите кнопку панели инструментов Новый макет. Вкладка Дерево запросов и макетов закроется, и откроется вкладка для нового макета.

5.Введите имя и описание для макета на вкладке Информация.

6.Постройте сводную таблицу на вкладке Сводная таблица.

7.Настройте вид макета, используя вкладку Вид представления данных. Вид представления данных является окончательным отчётом. Данные в нём могут отображаться или в виде диаграммы, или в виде географической карты.

8. Настройте свойства диаграммы или карты.

9.Сохраните макет.

10.Закройте макет.

 

Созданный макет будет доступен только для его автора, пока он/она не откроет к нему общий доступ (в этом случае макет станет доступен для редактирования каждым пользователем того же сайта, обладающего правами доступа к модулю АВО), или не опубликует его (в этом случае макет становится видимым в режиме только для чтения каждому пользователю страны, имеющему права доступа к модулю АВО).

Для получения более подробных сведений о форме Редактор макетов, прочитайте раздел Редактор макетов.
