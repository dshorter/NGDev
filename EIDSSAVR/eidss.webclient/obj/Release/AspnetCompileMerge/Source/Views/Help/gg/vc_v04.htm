ნახ. 102 ფერმების ფანჯარა

ძებნის პანელი განყოფილება; მოიცავს სხვადასხვა ველებს, რომლებიც გამოიყენება ფერმის ძებნისთვის.

ნახ. 103 ძებნის მენიუ

ველი/ცვლადი

აღწერა

ფერმის საიდენტიფიკაციო ნომერი

ფერმის ID შეყვანილია

ფერმის დასახელება

ფერმის სახელწოდება შეყვანილია.

საკონტაქტო ტელეფონი

ფერმის ტელეფონი შეყვანილია.

რეგიონი

რეგიონი შერჩეულია.

რაიონი

შერჩეულია რაიონი; ჩამონათვალი შეიცავს მონაცემებს რაიონის მითითებიდან, რომელიც გაფილტრულია შერჩეული რეგიონის მიხედვით.

ქალაქი ან სოფელი

შერჩეულია დასახლება; ჩამონათვალი შეიცავს მონაცემებს დასახლების მითითებიდან, რომელიც გაფილტრულია შერჩეული რაიონის მიხედვით.

გვარი (ფერმის მფლობელის)

ფერმის მფლობელის გვარი შეყვანილია.

სახელი

(ფერმის მფლობელი)

ფერმის მფლობელის სახელი შეყვანილია.

მამის სახელი

(ფერმის მფლობელი)

ფერმის მფლობელის მამის სახელი შეყვანილია.

ფერმის ტიპი

ფერმის ტიპი შერჩეულია ჩამოსაშლელი სარჩევიდან: ფრინველი ან შინაური ცხოველი.

       ძებნის შედეგების ჩამონათვალის განყოფილება ეკრანზე გამოაქვს ფერმათა ჩამონათვალი, რომელიც შეესაბამება ძებნის კრიტერიუმებს.

ნახ. 104 შედეგების ჩამონათვალის ძებნა

ველი/ცვლადი

აღწერა

ფერმის საიდენტიფიკაციო ნომერი

ფერმის ID ეკრანზეა გამოტანილი.

ფერმის დასახელება

ფერმის სახელწოდება ეკრანზეა გამოტანილი.

ფერმის ტიპი

ფერმის ტიპი ეკრანზეა გამოტანილი.

ღილაკები

ახალი

დააწკაპუნე ღილაკზე ახალი, რათა შექმნა ახალი ფერმა. ფერმის ინფორმაცია გაიხსნება ახალ ფანჯარაში,

რედაქტირება

რედაქტირებისთვის შეარჩიეთ ფერმა ძებნის ჩამონათვალში, და დააწკაპუნეთ რედაქტირების ღილაკზე. შერჩეული ფერმა გაიხსნება ახალ ფანჯარაში, რედაქტირების რეჟიმში.

წაშლა

წაშლისთვის შეარჩიეთ ფერმა ძებნის ჩამონათვალში, და დააწკაპუნეთ წაშლის ღილაკზე. გამაფრთხილებელი მესიჯი წაშლა/გაუქმება გამოჩნდება ღილაკების სახით. "ჩანაწერი წაიშლება. წაიშალოს ჩანაწერი?", დააწკაპუნეთ წაშლა, რათა წაშალოთ ჩანაწერი.
