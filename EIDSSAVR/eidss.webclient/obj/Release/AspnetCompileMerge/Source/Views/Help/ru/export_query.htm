Если извлечённые при помощи запроса из базы данных ЭИСНЗ сведения требуются для проведения какой-либо дополнительной внешней обработки, или извлекаются по иным причинам, такие сведения можно сохранять в файлах форматов Excel или Access. Это можно делать при помощи имеющейся в модуле АВО функции «Экспортирование запроса». Данные сохраняются как есть, т.е., без каких-либо параметров настроек, конфигурации или функций агрегирования.

При превышении определённого количества записей (а именно, более 65536) доступен только формат XLSX (Excel 2007).

Экспорт запроса

1.Проделайте одно из следующих действий:

•В окне Дерево запросов и макетов модуля АВО выберите запрос для экспортирования.

•В окне Дерево запросов и макетов модуля АВО откройте запрос для экспортирования.

2.В меню АВО выберите пункт Экспортировать, а затем Запрос.

3.Из раскрывающегося меню выберите версию 2003 или 2007 формата Excel или Access. Откроется окно Сохранить как.

4.Выберите место сохранения файла, введите его наименование в поле Имя файла, и нажмите кнопку Сохранить.
