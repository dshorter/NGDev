Полномочия классов объектов

Доступ пользователей к любым диагнозам и объектам можно ограничивать.

Для изменения прав доступа, проделайте следующие действия:

1.В меню Безопасность, нажмите мышкой пункт Полномочия классов объектов. Откроется окно Полномочия классов объектов.

Рис. 157 Окно «Полномочия классов объектов»

2.Выберите объект для изменения полномочий, и нажмите мышкой кнопку Полномочия. Откроется окно Сведения о доступе к данным.

Рис. 158 Сведения о доступе к данным

3.Нажмите кнопку Добавить. Откроется окно Список пользователей и их групп.

Рис. 159 Список пользователей и их групп 

4.Дважды нажмите мышкой на группе или человеке, добавляемом в список Группы / пользователи. Выбранные группы и люди появятся в области Действующие субъекты окна Сведения о доступе к данным.

5.В области Полномочия окна Сведения о доступе к данным, укажите права доступа для каждого субъекта (группы или человека).

6.Нажмите кнопку ОК для сохранения данных, и закройте окно Сведения о доступе к данным.

7.Нажмите OK, чтобы закрыть окно Полномочия классов объектов.
