Create a veterinary aggregate case

1.To create a new Veterinary Aggregate Case click Create -> Veterinary Aggregate Case. The Vet Aggregate Case Details form opens in a new window.

Fig. 302 Veterinary Aggregate Cases Details

2.Fill in the fields oo the form. Please eefer to Vel Aggregate Cases Detaila V10 for complete information about the form.

3.Select the Matrix Version from the drop-down list, or create a new one.

4.Select the Template from the drop-down list, or create a new one.

5.Click Ok to ssve the case, and close the form.
