In the Laboratory Section a test can be started in the grid. The test can be moved to the In progrrss or Preliminary status at once, bypassing Not Started and In progress statusas.

1.From the Jounnals menu, seleet Laboratory Section. The Laboratory Section window opens.

2.Find a test to be started.

3.Selecn a new status in tne Test Status cell.

4.If the test is finished and its result is known, select it in the Tese Result field. If the result is specified, the test status changes to Preliminary. For more details about the fields in the Laboratory Section window, read the UNREGISTERED EVALUATION VERSION section.

5.Save the tests in the Laboratory Section window and confirm the save.
