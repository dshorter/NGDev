Форма L22 Сведения о тесте группы используется для работы с тестами, назначенными группе. Форма содержит:

•Набор полей, отображающих основные данные о группе;

•Таблицу со списком тестов, объединённых в группу;

•Два нижних поля, созданных для поиска тестов по соответствующим пробам;

•Раздел Параметры теста определённой пробы, находящийся в настраиваемой форме Сведения о тесте;

•Раздел Контроль качества, находящийся в настраиваемой форме Проведение теста;

Для получения более подробной информации о создании настраиваемой формы, прочитайте раздел Конфигурация настраиваемых форм.

Раздел Параметры теста определённой пробы (настраиваемой формы Сведения о тесте) может заполняться для каждого теста в таблице. Для отображения полей теста в разделе Параметры теста определённой пробы, выберите тест в таблице. Раздел Контроль качества (настраиваемой формы Проведение теста) может заполняться только для группы, но не какого-либо теста группы. Шаблоны для таких настраиваемых форм определяются по типу теста, применяемого в отношении группы.

Рис. 362 Форма «Сведения о тесте группы»

Поля данных

Идентификатор теста группы

Представляет собой идентификатор группы. Значение (Новая) используется для новых групп, до только до их первого сохранения. После этого автоматически, в соответствии со схемой нумерации, присваивается идентификатор группы, значение которого может изменить вручную.

Название теста

Указывается наименование теста, который будет проводиться в отношении группы. Это поле является обязательным для заполнения и доступно при отсутствии добавленных группе тестов. Если тест уже присутствует в списке, то в поле Название теста присутствует только то название, которое доступно для теста и которое невозможно изменить.

Название выбранного в этом поле теста фильтрует тесты, которые могут добавляться в группу.

Местоположение проведения теста

По умолчанию указывается текущая организация и это значение невозможно изменить.

Тестировавший работник

Указывается проводящий тест работник.

Утвердившее лицо

Указывается утверждающий результаты работник. Доступность поля зависит от системной функции права на запись в поле Может утверждать результат теста. Для получения более подробной информации о системных функциях, прочитайте раздел Системные функции и Список системных функций.

Дата начала теста

Используется для указания даты начала проведения теста группы.

Дата результата

Указывается дата, в которую был получен результат теста группы.

Столбцы

Последовательность

Отражает порядок нумерации записей. Нумерация начинается с нуля и идёт до бесконечности.

Лабораторный идентификатор пробы

Отображается лабораторный идентификатор пробы, связанной с тестом.

Диагноз

Отображается диагноз для теста.

Тип пробы

Указывается тип пробы, в отношении которой применяется тест.

Категория

Отображается категория теста.

Результат

Указывается результат теста.

Оператор ввода результатов

По умолчанию текущий пользователь указывается в качестве осуществившего ввод результата лица; это значение изменить невозможно.

Кнопки

Добавить тест

Добавляет тест(-ы) к группе.

Удалить тест

Удаляет выбранный тест из группы.

Добавить результат группы

Добавляет результат для группы тестов.

↑

Однократное нажатие перемещает выбранную запись в таблице на одну позицию вверх.

↓

Однократное нажатие перемещает выбранную запись в таблице на одну позицию вниз.

Закрыть группу

Закрывает группу и меняет её статус на Закрыта. После закрытия группы, она переходит в режим только для чтения. Закрыть можно только те группы, у которых результат определён по всем тестам.
