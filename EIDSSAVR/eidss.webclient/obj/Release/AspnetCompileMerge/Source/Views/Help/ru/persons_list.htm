Любое лицо, указываемое во время создания случая заболевания человека либо иной процедуры, добавляется в Список лиц. Используйте этот список для поиска пациента, владельца фермы, имени связанного со случаем заболевания лица, и т.д.

Для просмотра «Списка лиц» необходимо:

1.Находясь в меню Журналы, нажмите мышкой на пункте Лица.Откроется Список лиц.

Рис. 125 Список лиц 

2.Используя панель поиска, введите критерии поиска и нажмите кнопку Поиск.

3.Список отвечающих критериям поиска лиц отобразится в правой части окна Список лиц.

 
Для добавления человека в «Список лиц» необходимо:

1.Находясь в поле Список лиц, нажмите кнопку Новое. Откроется окно Сведения о лице.

Рис. 126 Сведения о лице

2.Заполните поля и нажмите кнопку ОК. В окно Список лиц добавится запись о человеке.

Для редактирования сведений о лице необходимо::

1.Находясь в поле Список лиц, выберите необходимого человека и нажмите кнопку Редактировать, чтобы изменить сведения о нём.

Рис. 127 Сведения о лице

2.Внесите необходимые изменения и нажмите кнопку ОК. Сведения о человеке будут сохранены.

 
Для удаления человека из «Списка лиц» необходимо::

1.Находясь в поле Список лиц, выберите человека для удаления из списка, и нажмите кнопку Удалить.

2.Откроется окно с запросом на подтверждение совершения действия. Нажмите Да.

Рис. 128 Удаление человека из списка

3.Человек будет удалён из Списка лиц.

Примечание: запись из Списка лиц удалить невозможно, если указанный в этой записи человек связан со случаем заболевания или сессией.
