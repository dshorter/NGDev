Принятие уничтожения пробы (ОФ)

После отметки пробы на уничтожение (ликвидацию) и смены её статуса на Отмечена для стандартного уничтожения, может быть утверждена операция её уничтожения.

 

1.Находясь в меню Журналы, выберите пункт Уничтожение проб. Откроется окно Отмеченные для уничтожения пробы. Оно содержит все пробы, отмеченные для уничтожения.

Рис. 324 Отмеченные для уничтожения пробы

2.Отметьте уничтожаемые пробы и нажмите кнопку Принять. В режиме подтверждения откроется окно Уничтожение пробы. Оно содержит все пробы, отмеченные для стандартного уничтожения.

Рис. 325 Уничтожение пробы

3.Нажмите кнопку ОК. Откроется окно подтверждения на совершение действия.

4.Для подтверждения уничтожения пробы, нажмите кнопку Да. Статус пробы поменяется на значение Уничтожена.
