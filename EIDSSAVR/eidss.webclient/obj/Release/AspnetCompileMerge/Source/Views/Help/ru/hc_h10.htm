Форма Список дедупликации случаев заболеваний людей используется в случае необходимости проведения дедупликации случаев заболеваний.

Рис. 227 Окно «Список случаев заболеваний людей»

Раздел панели поиска содержит различные поля, используемые для поиска случаев заболеваний.

Рис. 228 Раздел «Параметры поиска»

В разделе «Параметры поиска» пользователь может указать критерии, которые будут использоваться для фильтрации записей. Раздел «Параметры поиска» обладает той же функциональностью, что и форма Список случаев заболеваний людей.
В разделе «Список результатов поиска» отображается перечень случаев заболевания, чьё описание подходит введённым критериям поиска.

Рис. 229 Раздел «Результаты поиска»

В разделе Список результатов поиска отображается список записей, соответствующих значениям введённых параметров поиска. Раздел Список результатов поиска обладает той же функциональностью, что и форма Список случаев заболеваний людей.
Функциональные кнопки, присутствующие в форме.

Добавить для дедупликации

Выбранный случай заболевания из списка результатов поиска добавляется в верхний раздел формы. В верхний раздел можно добавлять только по два случая заболеваний.

Удалить

Выберите случай заболевания в верхнем разделе и нажмите на эту кнопку для его удаления.

Дедупликация

После выбора двух случаев заболевания для проведения дедупликации и их помещения в верхнюю часть, нажмите на такую кнопку для начала процедуры дедупликации. Откроется форма Н11 « Сведения о дедупликации случаев заболеваний людей».
