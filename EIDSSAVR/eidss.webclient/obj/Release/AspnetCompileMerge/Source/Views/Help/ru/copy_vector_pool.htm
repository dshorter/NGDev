Копирование записи об отдельном переносчике / группе переносчиков заболевания

При необходимости создания расширенного списка похожих записей об отдельных переносчиках / группах переносчиков заболевания, целесообразным будет создание одной записи об отдельном переносчике / группе переносчиков заболевания, с её последующим копированием необходимого количества раз. После этого можно приступить к редактированию созданных копий.

В процессе копирования записи о группе переносчиков / отдельном переносчике заболевания, можно выбирать набор данных, которые копируются с такой записью.

 

1.На вложенной вкладке «Группы переносчиков / отдельные переносчики заболевания» формы «Сессия надзора за переносчиками заболевания» выберите запись той группы переносчиков / отдельного переносчика заболевания, которую вы хотите копировать.

2.Нажмите кнопку Копировать. Откроется окно Копировать группу переносчиков / отдельного переносчика заболевания.

Рис. 258 Окно «Копировать группу переносчиков / отдельного переносчика заболевания»

3.Выберите наборы данных, предназначенные для копирования с группой переносчиков / отдельным переносчиком заболевания. Для получения более подробной информации, прочитайте раздел формы W05 Копирование группы переносчиков / отдельного переносчика заболевания.

4.Нажмите кнопку Выбрать. Окно Копировать группу переносчиков / отдельного переносчика заболевания закроется. Откроется форма Сведения о группе переносчиков / отдельном переносчике заболевания, в которой будет новая запись о группе переносчиков / отдельном переносчике заболевания, готовая для редактирования.

Поля копируются в соответствии со следующими правилами:

•Новый Идентификатор группы переносчиков / отдельного переносчика заболевания создаётся в соответствии со Схемой уникальной нумерации.

•Значения полей Идентификатор сессии и Тип переносчика заболевания копируются из первоначальной группы переносчиков / отдельного переносчика заболевания.

•Поле Полевой идентификатор группы переносчиков / отдельного переносчика заболевания остаётся пустым и его можно редактировать вручную.

•Пробы сессии тоже копируются. Если пробы копируются, укажите для них идентификаторы.

Другие данные копируются в соответствии с выбранными параметрами.

5.Отредактируйте новую запись о группе переносчиков / отдельном переносчике заболевания.

6.Нажмите кнопку ОК, расположенную на форме Сведения о группе переносчиков / отдельном переносчике заболевания, чтобы сохранить изменения и подтвердить создание копии. Если на этом этапе нажать кнопку Отменить, никакой новой записи о группе переносчиков / отдельном переносчике заболевания создано не будет, а копирование будет отменено.
