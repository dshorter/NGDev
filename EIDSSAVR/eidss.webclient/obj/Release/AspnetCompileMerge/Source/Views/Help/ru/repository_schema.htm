При помещении пробы на хранение, в ЭИСНЗ можно зарегистрировать её точное местоположение внутри хранилища.

Но ещё до указания местоположения пробы, вначале необходимо указать схему хранения. Наилучшим способом сделать это будет создание виртуальной копии хранилища объекта в базе данных ЭИСНЗ.

Для создания схемы хранилища необходимо::

1.Находясь в меню Журналы, нажмите мышкой на пункте Холодильники. Откроется окно Список холодильников.

Рис. 48 Схема хранилища

При открытии окна в первый раз, в нём не будет перечислено никаких объектов, поскольку схему хранения ещё только предстоит определить.

2.Для создания записи о новом холодильнике нажмите кнопку Новый. Откроется окно Схема хранения.

3.Заполните поля, находящиеся в правой части окна Схема хранения, и нажмите кнопку Сохранить. Введённый элемент будет добавлен и будет отображаться в левой части окна.

Рис. 49 Элемент схемы хранения

4.Выберите холодильник на схеме слева, и нажмите кнопку Новый, чтобы создать элемент хранилища холодильника.

Рис. 50 Субэлемент схемы хранения

5.Заполните поле Наименование элемента хранилища и укажите Тип элемента хранилища. При необходимости, укажите Количество мест хранения и Штрихкод секции. Добавьте любые дополнительные сведения в поле Примечания и нажмите кнопку Сохранить. Введённый элемент будет добавлен и будет отображаться в левой части окна.

Примечание: Количество мест хранения определяет возможное количество проб для текущего элемента хранилища.

6.Для создания большего количества субэлементов, повторите шаги 4-5. Все введённые элементы будут добавлены и будут отображаться в левой части окна.

Для копирования элемента схемы необходимо::

1.Выделите целевой объект из списка и нажмите кнопку Копировать.

2.Структура скопированного элемента схемы будет аналогична оригиналу. В активированном диалоговом окне, измените атрибуты нового элемента схемы.

3.Для сохранения изменений, нажмите Сохранить.

Для удаления элемента схемы необходимо::

1.Выберите элемент для удаления.

2.Для удаления элемента или суб-элемента, нажмите кнопку Удалить.

Для печати отчёта о содержимом контейнера необходимо::

1.Нажмите кнопку Бумажные формы и укажите Содержимое контейнера. Отчёт отобразится в окне Программа просмотра отчётов.

2.Нажмите пиктограмму Принтер, чтобы распечатать отчёт.
