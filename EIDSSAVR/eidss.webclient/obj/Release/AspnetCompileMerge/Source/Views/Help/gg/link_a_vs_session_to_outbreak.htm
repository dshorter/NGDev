ზოგჯერ, ვექტორული ზედამხედველობის სესიები დაკავშირებულია ეპიდაფეთქებებთან. თუ სესიის მიზეზია ეპიდაფეთება, მიზანშეწონილი იქნება მისი განსაზღვრა დზეის-ში.

 

1.ჟურნალების მენიუში შეარჩიეთ ვექტორული ზედამხედველობის სესიები. გაიხსნება ვექტორული ზედამხედველობის სესიების ჩამონათვალის ფანჯარა.

2.მოძებნეთ სასურველი ვექტორული ზედამხედველობის სესია. დეტალური აღწერილობისთვის, თუ როგორ ვიპოვოთ სესია, წაიკითხეთ სექცია ვექტორული ზედამხედველობის სესიის ძებნა.

3.გააკეთეთ შემდეგიდან ერთ–ერთი:

•სესიის რედაქტირებისთვის ორჯერ დააწკაპუნეთ მასზე.

•მონიშნეთ დასარედაქტირებელი სესია და დააწკაპუნეთ რედაქტირების ღილაკზე.

სესია გაიხსნება ვექტორული ზედამხედველობის სესიის ფორმაში და იგი მზადაა რედაქტირებისთვის.

4.ეპიდაფეთქების ID ველში ან ხელით შეიყვანეთ ეპიდაფეთქების ნომერი, ან დააწკაპუნეთ ჩამონათვალის ღილაკზე () და მონიშნეთ ეპიდაფეთქება ეპიდაფეთქებების ჩამონათვალში. დეტალური აღწერილობისთვის, თუ როგორ ვიპოვოთ სასურველი ეპიდაფეთქება, წაიკითხეთ სექცია ეპიდაფეთქების ძებნა.

5.სესიისთვის შეყვანილი ეპიდაფეთქების ID-ის დასამახსოვრებლად დააწკაპუნეთ დამახსოვრების ღილაკზე.

 

სესიასა და ეპიდაფეთქებას შორის კავშირის მოსახსნელად, წაშალეთ ეს ეპიდაფეთქება ამ სესიის ეპიდაფეთქების ID ველიდან.
