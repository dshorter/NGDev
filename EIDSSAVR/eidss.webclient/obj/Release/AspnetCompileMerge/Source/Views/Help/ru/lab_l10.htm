Форма L10 Передача пробы используется для передачи проб из лаборатории сайта и их получения обратно.

Рис. 355 Форма «Передача пробы» (Принятие на сайт)

Поля данных

Идентификатор передачи пробы

Идентификатор операции передачи или обратного получения.

•Для операций передачи, поле Идентификатор передачи пробы заполняются автоматически, в соответствии со схемой нумерации, и такой номер можно изменять вручную до фактического момента передачи.

•При операции обратного получения, отображается присвоенный направителем пакета идентификатор, которой изменить невозможно.

Цель передачи

Указывается цель передачи во внешнюю организацию. Это поле является доступным до завершения операции.

Передача откуда

Организацию-источник передачи изменить невозможно:

•При передаче во внешнюю организацию, отображается название текущей организации;

•При обратном получении из внешней организации, отображается название такой внешней организации, направившей пакет.

Отправитель

Отправителя пакета изменить невозможно:

•При передаче во внешнюю организацию, по окончании выполнения операции передачи будет отображаться имя текущего пользователя;

•При обратном получении из внешней организации, будет отображаться имя работника внешней организации, направившего пакет.

Передаётся в

Указывается получающая организация. Используется исключительно для операций передачи во внешнюю организацию.

Дата отправки

Дату отправки поменять невозможно.

•При передаче во внешнюю организацию, по окончании выполнения операции передачи будет отображаться значение текущей даты.

•При обратном получении из внешней организации, будет отображаться дата, когда пакет покинул исходную организацию.

Столбцы

Идентификатор лаборатории-источника

Идентификатор пробы первоначальной лаборатории:

•При передаче во внешнюю организацию, используется идентификатор пробы, присвоенный текущей лабораторией;

•При обратном получении из внешней организации, используется лабораторный идентификатор пробы, присвоенный внешней организацией.

Тип пробы

Указывается тип пробы.

Дата принятия

Указывается дата, в которую была получена проба, и которую можно менять вручную до сохранения формы. Поле предназначено исключительно для операций обратного получения.

Получатель

Отображается имя текущего пользователя в качестве работника, получившего пробу. Поле предназначено исключительно для операций обратного получения.

Лабораторный идентификатор пробы

Указывается лабораторный идентификатор пробы новосозданной записи, генерируемый в соответствии со схемой нумерации для получаемых обратно проб, который можно менять вручную. Поле не применяется для записей о передачах во внешнюю организацию.

Состояние пробы

Указывается состояние получаемых обратно проб. Если состояние получаемой обратно пробы плохое или она отклонена, в поле Комментарий необходимо внести пояснение. Поле состояния становятся доступным после нажатия кнопки Обратное получение, и должно быть заполнено ещё до первого сохранения формы. После первого сохранения, поле становится нередактируемым.

Комментарий

Указывается комментарий для полученных обратно проб. Если состояние получаемой обратно пробы плохое или она отклонена, в поле Комментарий, в обязательном порядке, необходимо внести пояснение. Поле состояния становятся доступным после нажатия кнопки Обратное получение, и должно быть заполнено ещё до первого сохранения формы. После первого сохранения, поле становится нередактируемым.

Место хранения

Указывается положение места хранения принятой обратно пробы. Поле состояния становятся доступным после нажатия кнопки Обратное получение, и должно быть заполнено ещё до первого сохранения формы. После первого сохранения, поле становится нередактируемым.

1.Нажмите кнопку Сведения (). Откроется окно Холодильники.

2.Выберите необходимое место.

3.Нажмите кнопку ОК. Окно «Холодильники» закроется. В поле будет отображаться назначенное местоположение.

Обрабатывающий отдел

В раскрывающемся списке указывается отдел организации, получающий пробу обратно.

Поле состояния становятся доступным после нажатия кнопки Обратное получение, и должно быть заполнено ещё до первого сохранения формы. После первого сохранения, поле становится нередактируемым.

Кнопки

Обратное получение

Отмечает все пробы в списке на обратное получение и даёт возможность внесения в них изменений (см. описание в столбцах выше).

1.Нажмите кнопку Обратное получение.

2.Измените данные в столбце.

3.Сохраните форму. Для получения более подробных сведений об операции обратного получения, прочитайте раздел Обратное получение пробы.

Передача во внешнюю организацию

Отмечает все пробы в списке на передачу во внешнюю организацию, и даёт возможность внесения в них изменений. См. описание Столбцы выше.

1.Укажите для них необходимую организацию.

2.Нажмите кнопку Передача во внешнюю организацию.

3.Измените данные в столбце.

4.Сохраните форму. Для получения более подробных сведений об операции передачи во внешнюю организацию, прочитайте раздел Передача пробы во внешнюю организацию (ОФ).

Кнопка становится доступной после указания организации назначения.

Добавление

Открывает форму L01 «Список проб» для выбора проб, предназначенных для добавления к пакету.

Удаление

Удаляет выбранные пробы из пакета передач.

ОК

Служит для сохранения данных в форме и подтверждения операции передачи.

Отмена

Служит для отмены всех сделанных в форме изменений и операции передачи.
