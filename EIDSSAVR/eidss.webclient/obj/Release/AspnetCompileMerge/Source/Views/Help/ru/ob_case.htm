Связывание вспышки заболевания со случаем/сессией заболевания

1.Приступите к созданию или редактированию вспышки заболевания.

Рис. 246 Регистрация новой вспышки заболевания

2.Нажмите кнопку Добавить случай / сессию заболевания и выберите случай заболевания человека, случай заболевания животного или сессию НЗПЗ (надзора за переносчиками заболевания), чтобы связать такой случай или сессию со вспышкой заболевания. Откроется окно Случай заболевания животного, Случай заболевания человека, или Список сессий надзора за переносчиками заболеваний.

3.Выберите тот случай / ту сессию заболевания, связь с которой необходимо установить; для поиска используйте фильтрацию списка.

Рис. 247  Выбор случая заболевания животного, предназначенного для связывания со вспышкой заболевания

4.Нажмите кнопку Выбрать. Окно Случай заболевания животного, Случай заболевания человека, или Список сессий надзора за переносчиками заболеваний закроется. Выбранный случай заболевания отобразится в форме Связанное со случаем заболевания сообщение окна Сведения о вспышке заболевания.

Примечание: если диагноз или группа диагнозов вспышки заболевания определены, добавить случай / сессию с отличающимся диагнозом не получится.

5.Для создания списка случаев / сессий заболевания, связанных со вспышкой заболевания, повторите шаги со 2-го по 4-й выше.

6.Нажмите кнопку ОК для сохранения данных, и закройте окно Сведения о вспышке заболевания.

Начнётся процедура репликации сохранённой записи о вспышке заболевания и сведения о ней будут переданы на сервер ЭИСНЗ.
