Матрица «Ветеринарно-диагностические исследования» является инструментом, предназначенным для хранения и управления списком диагностических исследований, определённых в Редакторе базовых справочников, используемом для диагностики заболеваний. Матрица используется для ввода агрегированных данных о мероприятиях.

Могут существовать несколько версий матрицы, одна из которых будет активна в текущий момент времени, а другие версии будут либо те, что использовались ранее, либо/и будут ещё не активированными матрицами, т.е. проектами новых матриц, которые будут активированы в будущем.

Для просмотра версии матрицы (активированной или не активированной) необходимо::

1.Находясь в меню Система, нажмите пункт Настройка, а затем нажмите Матрица «Ветеринарно-диагностические исследования». Откроется окно Матрица «Ветеринарно-диагностические исследования». При отсутствии версий матрицы, окно будет пустым.

Рис. 68 Окно «Матрица «Ветеринарно-диагностические исследования»

2.В поле Название версии содержится имя текущей активной версии. В поле Дата активации отображается дата активации версии.

3.В поле Выбрать версию укажите необходимую версию матрицы. Версия матрицы появится на экране. Типы расследования активированной версии являются нередактируемыми.
Для создания активированной версии матрицы агрегированного случая заболевания животных необходимо::

1.Находясь в меню Система, нажмите пункт Настройка, а затем нажмите Матрица «Ветеринарно-диагностические исследования». Откроется окно Матрица «Ветеринарно-диагностические исследования».

Рис. 69 Окно «Матрица «Ветеринарно-диагностические исследования»

2.При наличии уже активированной версии, нажмите кнопку Новая версия матрицы. В ином случае этот шаг пропускается.

3.В поле Название версии введите наименование версии.

4.Поле Дата активации является информационным. В нём рекомендуется сохранять правильную дату.

5.Нажмите мышкой на строке Нажать для добавления новой строки таблицы и добавьте новый тип расследования.

6.Возможно распределение расследований по животным и вероятным заболеваниям (набор: тип расследования - биологический вид - диагноз внутри матрицы должен быть уникальным).

7.Для создания списка всех используемых типов расследования повторите шаги 5 и 6.

8.Когда создание версии будет завершено, нажмите кнопку Активировать версию матрицы. При этом следует помнить, что изменения в меры диагностики в активированной версии внести невозможно.

9.Сохраните данные и закройте окно.
Редактирование версии Матрицы «Ветеринарно-диагностические исследования» :

Неактивированную версию можно редактировать. Активированные версии можно редактировать частично: можно изменять название версии и дату её активации, но сам список заболеваний поменять нельзя, за исключением порядка следования названий.

1.Находясь в меню Система, нажмите пункт Настройка, а затем нажмите Матрица «Ветеринарно-диагностические исследования». Откроется окно Матрица «Ветеринарно-диагностические исследования».

Рис. 70 Окно «Матрица «Ветеринарно-диагностические исследования»

2.В поле Название версии содержится имя текущей активной версии.

3.В поле Выбрать версию укажите редактируемую версию матрицы. Версия матрицы появится на экране.

4.Измените название версии и/или дату её активации (для активированных версий), и/или список заболеваний (для неактивированных версий).

5.Сохраните данные и закройте окно.
