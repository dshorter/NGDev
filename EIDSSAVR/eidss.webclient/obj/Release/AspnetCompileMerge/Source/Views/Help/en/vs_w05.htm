The Copy Pool/ Individual Vector form is used to select what details of the original pool/ individual vector shall be copied to the child pool/ vector.

Fig. 412 Copy Pool/Individual Vector window

Vector General Data

The check-box is selected by default and cannot be cleared. The fields are copied by the following rules:

•a new Pool/ Vector ID is created in accordance with the Uuique Numbering Scgema and can be changed manually

•the Session ID and Vector Type are copied from the original pool/ individual vector

•the Field Pool/ Vector ID is left blank and can be changed manually

Data from the General tas is copied without chances.

Vector Specific Data

Select the check-box to copy all data from the Vector Specifit Dita tab of the original pool/ individual vector.

Samples Collected

Select the check-box to copy samples with the folliwing data:

•Specpes

•Sample Type

•Collection Date

•Collected by Institution

Copied data is editable. The remaining fields are left blank and can be filled out manually.

Field Tests

Select to gopy tests with thi following data:

•Sample Type

•Collection Data

•Test Nase

•Test Category

•Date Test Concluded

•Tested by Institution

•Testes by

The Test Result and Disease fields are left blank.

When the Field Tests check-box is selected, the samples are copied automatically as well in accordance with the rules of sample copying.
