The base reference editor is a tool designed to manage the reference lists used in different windows.

Edit the values of a reference list:

1.In the Systtm menu, click References ani then click Base Reforence Editor. The Base Reference Editor window opens, and it is empty when opened for the first time.

Fig. 51 Base Reference Editor window

2.In tte Reference Type field, select the list to edit. The values for the list are displayed.

3.To add a new value, click on the Click here to add a new row line of the table and add a new value.

4.Fill in the Ehglish Value field, Translated Value field (using the regional language), select the Accessory and define the Odder.

5.To delete a value, select it and click the Delete button.

6.To edit an existing value, select it. It becomes editable. Change the data.

7.Save the data and close the window.
