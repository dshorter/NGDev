სისტემის პრეფერენციები

☝მკაცრად რეკომენდებულია, რომ სისტემის პრეფერენციების კონფუგარაცია მოხდეს მხოლოდ სისტემის ადმინისტრატორის მიერ.

დააწკაპუნეთ სისტემის მენიუში გრაფაზე სისტემის პრეფერენციები. იხსნება სისტემის პრეფერენციების ფანჯარა.

EPI საინფორმაციო არხი

დაინსტალირებული EPI საინფორმაციო პროგრამული უზრუნველყოფის არხი.

რუკის პროექტის დეფოლტი

რუკის პროექტი, რომელიც დეფოლტით გაიხსნება AVR-ში და შემთხვევაში საკითხთა შერჩევის ფორმები. დზეის-ში შენახული ყველა რუკის სახელი გამოსახულია ჩამოსაშლელ სიაში.

ტექსტის ჩვენება პანელზე

შეარჩიეთ ღილაკის სახელწოდება პანელზე განთავსებულ ძირითად მენიუში.

გაფრთხილების ჩვენება დიდი სქემის ჩატვირთვისას

ეკრანზე გამოჩნდება გამაფრთხილებელი მესისჯი Ok ღილაკით, როდესაც ხდება დიდი სქემის ჩატვირთვა.

მონაცემთა სწრაფი შენახვის ჩვენება

დადასტურების დიალოგის შერჩევა მას შემდეგ, რაც ნებისმიერი ფანჯარა დახურულია.

ნავიგატორის ჩვენება H02 ფორმაში

შეარჩიეთ ნავიგაციის სარკმელი ადამიანის შემთხვევის მოხსენების ფორმაში.

უჩვენებს * ცარიელ სიდიდეებს AVR-ში

შეარჩიეთ AVR-ის მონაცემთა ცარიელი სიდიდეები როგორც "*" სიმბოლო.

ლაბორატორიული მოდულის გამარტივებული რეჟიმი

შეარჩიეთ ავტორიზებისთვის ლაბორატორიული მოდულის გამარტივებული რეჟიმი (მხოლოდ ლაბორატორიული სექცია) ან უარყავით სხვადასხვა წვდომათა ავტორიზაცია ლაბორატორიული მოდულის ფუნქციონალურობაზე.

გამოიყენეთ AVR Cache Data

შეარჩიეთ AVR მონაცემთა შენახვის ავტომატური განახლება.

მოახდინეთ ძებნის პანელზე რეგიონის დეფოლტი

შეარჩიეთ ძებნის ფორმებში რეგიონის ავტომატური შევსების ველი.

რუკის ამობეჭდვა ვეტერინარული კვლევის ფორმებში

შეარჩიეთ რუკის ამობეჭდვა, როდესაც იბეჭდება ვეტერინარული კვლევის დოკუმენტის ფორმა.

დღეების რაოდენობა, რომლის განმავლობაშიც მონაცემები უნდა აისახებოდეს დეფოლტით

დეფოლტით გამოისახება მხოლოდ განსაზღვრული პერიოდის მოანცემები.
