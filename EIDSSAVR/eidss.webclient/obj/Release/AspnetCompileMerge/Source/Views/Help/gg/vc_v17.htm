აქტიური ზედამხედველობის სესიების ჩამონათვალი V17

ნახ. 77 აქტიური ზედამხედველობის სესიების ჩამონათვალი

ძებნის პანელი განყოფილება მოიცავს სხვადასხვა ველებს, რომლებიც გამოიყენება სესიების ძებნისთვის.

ნახ. 78 ძებნის მენიუ

ველი/ცვლადი

აღწერა

სესიის ID

კამპანიის ID შეიძლება კონკრეტულად აისახოს ამ ველში.

ველზე ხელმისაწვდომობის ID

საველე ნიმუშის ID შეიძლება კონკრეტულად აისახოს ამ ველში

სესიის სტატუსი

სესიის სტატუსი (ღია ან ' დახურული) შერჩეულია.

სესიის დაავადება

სესიის დაავადება შერჩეულია

რეგიონი

სესიის რეგიონი ასახულია ამ ველში.

რაიონი

შერჩეულია რაიონი (ჩამონათვალი შეიცავს მონაცემებს რაიონის მითითებიდან, რომელიც გაფილტრულია შერჩეული რეგიონის მიხედვით).

ქალაქი ან სოფელი

შერჩეულია დასახლება (ჩამონათვალი შეიცავს მონაცემებს დასახლების მითითებიდან, რომელიც გაფილტრულია შერჩეული რაიონის მიხედვით).

დასაწყისის თარიღი

კამპანიის დაწყების თარიღის ფარგლები შეყვანილია.

დასრულების თარიღი

კამპანიის დასრულების თარიღის ფარგლები შეყვანილია.

შეყვანის თარიღი

მონაცემების შეყვანის თარიღის ფარგლები შეყვანილია.

შეყვანილია

სესიის შემქმნელი შერჩეულია.

კამპანიის საიდენტიფიკაციო კოდი

სესიასთან დაკავშირებული კამპანიის ID შეყვანილია.

კამპანიის სახელწოდება

სესიასთან დაკავშირებული კამპანიის სახელწოდება შეყვანილია.

კამპანიის ტიპი

კამპანიის ტიპი (აქტიური ზედამხედველობა, გამოკითხვა ან კვლევა), რომელიც სესიასთანაა დაკავშირებული, ამ ველშია შერჩეული.

მონაცემთა შეყვანის სამუშაო ობიექტი

მონაცემთა შეყვანის ადგილი შერჩეულია.
ძებნის შედეგების ჩამონათვალის განყოფილებას ეკრანზე გამოაქვს სესიების ჩამონათვალი, რომელიც შეესაბამება ძებნის კრიტერიუმებს.

ნახ. 79 შედეგების ჩამონათვალის ძებნა

ველი/ცვლადი

სესიის ID

სესიის ID ასახულია ამ ველში.

სესიასთან დაკავშირებული კამპანიის ID გამოტანილია ეკრანზე.

კამპანიის სახელწოდება

სესიასთან დაკავშირებული კამპანიის სახელწოდება გამოტანილია ეკრანზე.

ღილაკები

ახალი

დააწკაპუნე ღილაკზე ახლი რათა შექმნათ ახალი სესია. აქტიური ზედამხედველობის სესიის ფორმაიხსნება ახალ ფანჯარაში.

რედაქტირება

რედაქტირებისთვის შეარჩიეთ შემთხვევა ძებნის ჩამონათვალში, და დააწკაპუნეთ რედაქტირების ღილაკზე. შერჩეული კამპანია გაიხსნება ახალ ფანჯარაში რედაქტირების რეჟიმში.

წაშლა

წაშლისთვის შეარჩიეთ სესია ძებნის ჩამონათვალში, და დააწკაპუნეთ წაშლის ღილაკზე. გამაფრთხილებელი მესიჯი წაშლა/გაუქმება გამოჩნდება ღილაკების სახით.
