Поиск отчёта «Основной синдромный надзор»

1.Перейдите Журналы -> Формы «Основной синдромный надзор. Откроется Список форм «Основной синдромный надзор».

Рис. 276 Список форм «Основной синдромный надзор»

2.Введите критерии поиска в поля панели поиска. Для получения полной информации о форме, просьба обратиться к разделу Список форм «Основной синдромный надзор».

3.Нажмите Найти. Результаты отобразятся в правой части формы.

4.Нажмите на отчёте в списке результатов. После этого отчёт можно будет редактировать или удалить.
